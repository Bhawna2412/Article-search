# Setting Up Reply Templates

**Reply Templates** are predefined messages that an agent can create and utilize as a form of communication while replying to a ticket. Such automated responses help reduce messaging time and help in interacting with customers more effectively.

Creating reply templates is quite easy and hassle-free. You and other agents can create such responses and use them while replying to a client’s ticket.

## Steps to Create a Reply Template

1. Log in to your Service Cloud account, then click the user profile menu at the top of the application bar and choose **Settings**.

![userprofile7.png](https://us.v-cdn.net/6038474/uploads/E1S3S5DDKNI5/userprofile7.png)
2. On the **Settings** page, select **Reply Templates**.

![userprofile8.png](https://us.v-cdn.net/6038474/uploads/N9B3SD2E5GK0/userprofile8.png)
3. To create responses, click the **New Reply Template**.

![userprofile9.png](https://us.v-cdn.net/6038474/uploads/EKI0CUBS2ULB/userprofile9.png)
4. Under the **New Reply Template **slider, enter the details that the replies will contain. Such templates will contain the following fields:

- **Template Name:** Enter a title for the new reply template.
- **Response Message: **Enter a pre-saved response that you and your agents can use while replying to a ticket.

![userprofile10.png](https://us.v-cdn.net/6038474/uploads/EU8XSC1MVBDL/userprofile10.png)

> **Note:** **Note**: **Template title **and **Response Message** are **mandatory **fields to be entered.

5. Optionally, you can use the **Insert Tag **drop-down field to include tags in the reply template. This is helpful since it adds more information and context to your tickets.

![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/temp18.05T5g00000rL1tQEAS.png)

> **Note:** **Note**: Taglines include basic ticket and agent-related contact details that can be inserted into the pre-saved replies.

6. Additionally, agents can upload documents, images, or brochures with the help of the **Attachment **tool while creating a template.

![userprofile11.png](https://us.v-cdn.net/6038474/uploads/Z3BVVJAYI3D6/userprofile11.png)

> **Note:** **Important Note**: Attachments need to be uploaded in **JPG, PNG, PDF, **and **DOC **formats only.

7. Save your template.

![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/temp20.05T5g00000rL0BtEAK.png)
8. Agents can view their pre-saved responses on the **Reply Templates **Index page. The Service Cloud provides agents the option to either edit their saved templates or discard them.

![userprofile12.png](https://us.v-cdn.net/6038474/uploads/SR7RV3PP4494/userprofile12.png)

## Using Reply Templates

1. These templates are normally used by agents while they are replying to a ticket. Agents can simply click on the **Reply Templates** icon >> search and select a template from the list that is displayed >> and insert the response template into the **reply to** textbox.

![userprofile13.png](https://us.v-cdn.net/6038474/uploads/UQRIMA1NHOZJ/userprofile13.png)
2. The pre-saved information and attachments will be displayed in your replies.

![userprofile14.png](https://us.v-cdn.net/6038474/uploads/WUU5DN9M1UKV/userprofile14.png)
