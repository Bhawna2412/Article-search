# Setting Up Ticket Types

The **Ticket Types** functionality in Service Cloud allows you to create and manage various ticket categories based on service requests or queries you receive from users.

With the help of such classifications, you can organize and easily identify the ticket type, enabling you to handle user requests much more effectively.

Ticket classification becomes especially important if you are handling multiple incoming tickets and you need to resolve them as quickly as possible. To do that, you need to sort out these tickets depending on the nature of the request.

Some of these queries could be customer-related problems, such as login issues, payment transactions, checkout errors, etc., or could be vendor queries, which normally include inventory management, website configuration, lead inquiry, and so on.

## Creating a Ticket Type

1. To access Ticket Types, click on your account's user profile menu and select **Settings **from the dropdown list.

![userprofile7.png](https://us.v-cdn.net/6038474/uploads/UPRICKOTIPKG/userprofile7.png)
2. Then, choose **Ticket Types**.

![userprofile25.png](https://us.v-cdn.net/6038474/uploads/85IASBUIYO9V/userprofile25.png)
3. On the Ticket Type Index page, click on **New Type**.

![userprofile26.png](https://us.v-cdn.net/6038474/uploads/T4O84GQX1FK6/userprofile26.png)
4. Mention the title and enter some descriptions about the ticket type. Click on **Save**.

![userprofile27.png](https://us.v-cdn.net/6038474/uploads/FDE4X8MO5GNH/userprofile27.png)
5. A new ticket type has been created. You can manage such labels from the Index page itself.

![userprofile28.png](https://us.v-cdn.net/6038474/uploads/WQTWKZU3C3P4/userprofile28.png)

## Using Ticket Types

1. Depending on the user query, ticket labels (types) can be assigned while creating a new ticket.

![userprofile29.png](https://us.v-cdn.net/6038474/uploads/P9BDVVNUNRXI/userprofile29.png)
2. You can also apply ticket labels to an existing ticket via the **Type drop-down** and thereby reply to the users accordingly.

![userprofile30.png](https://us.v-cdn.net/6038474/uploads/PW1NYXRNZSBC/userprofile30.png)
3. You can view such labels for every ticket allocated to them from the **Ticket Index** page.

![userprofile31.png](https://us.v-cdn.net/6038474/uploads/325TY2D9SVU1/userprofile31.png)
