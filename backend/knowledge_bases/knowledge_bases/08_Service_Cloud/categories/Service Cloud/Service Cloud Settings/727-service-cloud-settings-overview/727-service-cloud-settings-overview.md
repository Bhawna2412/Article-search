# Service Cloud Settings Overview

The application-related settings for Service Cloud are available on the **Settings** page, which is accessible through the account's user profile menu located on the top application bar.

![userprofile6.png](https://us.v-cdn.net/6038474/uploads/PYXEUY02UEAU/userprofile6.png)
The settings on the page are categorized into two sections:

- **Workflow**: In this section, you'll find all the settings that are relevant to your ticket management workflow, such as settings that help you create reply templates, custom ticket types, and tags. To learn more, see:  - [**Setting Up Reply Templates**](https://galaxy.maropost.com/s/article/setting-up-reply-templates)
  - [**Setting Up Ticket Types**](https://galaxy.maropost.com/s/article/Setting-Up-Ticket-Types)
  - [**Setting Up Tags**](https://galaxy.maropost.com/s/article/Setting-Up-Tags)
- **Channel**: In this section, you'll find the Support Email setting that helps you set up your support email addresses. To learn more, see:  - [**Setting Up Support Email Addresses**](https://galaxy.maropost.com/s/article/Setting-Up-Support-Email-Addresses)
