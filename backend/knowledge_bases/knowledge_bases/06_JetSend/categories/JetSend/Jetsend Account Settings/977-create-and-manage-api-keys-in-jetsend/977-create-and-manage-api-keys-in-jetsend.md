# Create and Manage API Keys in JetSend

The API KEYS tab on the Account Settings page is where you can create new and manage existing API Keys. Click your user name on the top-right corner of the application and from the drop-down menu, select Account Settings. Then, on the Account Settings page, select the API KEYS tab.

![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/1.05T5g00000Ao69AEAR.png)

### Creating an API Key

To create an API Key, do the following:

1. Click your user name on the top-right corner of the application and from the drop-down menu, select Account Settings.

   ![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/2.05T5g00000Ao69KEAR.png)
2. Select the API KEYS tab, and then click CREATE API KEY.

   ![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/3.05T5g00000Ao69PEAR.png)
3. Enter an API key name and click CREATE.

   ![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/4.05T5g00000Ao69jEAB.png)
