# Adding a Shopify Integration

Maropost Marketing Cloud lets you easily create and manage multiple integrations with your Shopify accounts.

To learn about the integration, see [**Integration with Shopify**](https://galaxy.maropost.com/s/article/Integration-with-Shopify).

In Navigation, click INTEGRATIONS and then select Shopify to go to the Shopify Integration index page. If you don't have any existing integrations with Shopify, you'll see the Integration Settings dialog box to add a new integration instead of the index page.

![Shopify_Integration_Index.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/integration2.05T5g000018nEYgEAM.png)
