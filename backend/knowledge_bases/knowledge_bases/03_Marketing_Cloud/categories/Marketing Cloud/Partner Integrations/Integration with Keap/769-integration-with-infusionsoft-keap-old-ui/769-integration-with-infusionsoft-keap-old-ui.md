# Integration with InfusionSoft/Keap (Old UI)

Marketing Cloud provides comprehensive integration capabilities with InfusionSoft/Keap through multiple connection types, enabling sophisticated data synchronization for email marketing campaigns. These integrations help streamline your workflow by automating contact management, purchase tracking, and preference synchronization across platforms, improving operational efficiency while ensuring consistent customer experiences.

When properly configured, InfusionSoft/Keap integrations help optimize your email marketing effectiveness by connecting e-commerce data with targeted messaging capabilities, enabling personalized campaigns based on purchase behavior, contact preferences, and customer lifecycle stages.

Maropost Marketing Cloud offers a number of different integration points with **Infusionsoft/Keap**. The integration is done using both Maropost's APIs and Infusionsoft's APIs.

> **Note:** Use the latest UI for a simplified process of creating and managing Keap Integrations. See Integration with Keap (New UI) to learn more.

The InfusionSoft index page displays the previously created integrations. Here, you can find the POST URL for the integrations and edit or delete them. On this page, you can create new integrations by clicking the Add Settings button.

![InfusionSoft_Index_Page.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/05T5g00000BkWuI.05T5g00000BkWuIEAV.png)
We support the following types of integration with Infusionsoft:

- **Contact Import** - Imports new contacts from Infusionsoft into Maropost. See [**Importing New Contacts from Infusionsoft/Keap**](https://galaxy.maropost.com/s/article/Importing-New-Contacts-from-Infusionsoft-Keap) for detailed steps.
- **Product Contact Import** - Imports contacts, those in the process of purchasing a product, from Infusionsoft to Maropost. The import fetches contacts from both failed and successful transactions. See [**Importing Contacts from Infusionsoft/Keap Transactions**](https://galaxy.maropost.com/s/article/Importing-Contacts-from-Infusionsoft-Keap-Transactions) for detailed steps.
- **Product Revenue** - Imports e-commerce data from Infusionsoft into Maropost. See [**Importing E-Commerce Data from Infusionsoft/Keap**](https://galaxy.maropost.com/s/article/Importing-E-Commerce-Data-from-Infusionsoft-Keap) for detailed steps.
- **Form Contact Import** - Imports contact information from the contact forms. See [**Importing Contacts from Infusionsoft/Keap Forms**](https://galaxy.maropost.com/s/article/Importing-Contacts-from-Infusionsoft-Keap-Forms) for detailed steps.
- **Tag Import** - Imports contact tags from Infusionsoft to Maropost. See [**Importing Contact Tags from Infusionsoft/Keap**](https://galaxy.maropost.com/s/article/Importing-Contact-Tags-from-Infusionsoft-Keap) for detailed steps.
- **Add to DNM List** - Imports contacts from Infusionsoft into Maropost's DNM list. See [**Importing Unsubscribers Into Maropost Do Not Mail List**](https://galaxy.maropost.com/s/article/Importing-Unsubscribers-Into-Maropost-Do-Not-Mail-List) for detailed steps.
- **Contact Custom Fields** - Imports custom field data from Infusionsoft into Maropost. See [**Importing Custom Field Data from Infusionsoft/Keap**](https://galaxy.maropost.com/s/article/Importing-Custom-Field-Data-from-Infusionsoft-Keap) for detailed steps.
- **Contact Tag Import** - Imports the tagged contacts from Infusionsoft into Maropost. See [**Importing Tagged Contacts from Infusionsoft/Keap**](https://galaxy.maropost.com/s/article/Importing-Tagged-Contacts-from-Infusionsoft-Keap) for detailed steps.
- **DNM to Infusionsoft** - Imports new additions in Maropost's DNM list into Infusionsoft. See [**Importing Do Not Mail List into Infusionsoft/Keap**](https://galaxy.maropost.com/s/article/Importing-Do-Not-Mail-List-into-Infusionsoft-Keap) for detailed steps.
- **Import History** - Imports contact and order historical data from InfusionSoft to Maropost. See [**Importing InfusionSoft History**](https://galaxy.maropost.com/s/article/Importing-InfusionSoft-Keap-History) for detailed steps.
