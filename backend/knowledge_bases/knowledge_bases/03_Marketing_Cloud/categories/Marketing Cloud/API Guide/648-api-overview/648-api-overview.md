# API Overview

API Overview enables you to integrate Marketing Cloud with your existing systems and automate email marketing operations, helping maximize marketing effectiveness through programmatic campaign management and data synchronization. This helps optimize marketing ROI by allowing you to connect your customer data sources directly to email campaigns and automate routine marketing tasks.

When properly implemented, API integrations can substantially improve marketing efficiency by enabling real-time data updates, automated campaign triggers, and seamless workflow connections between your marketing tools and customer databases.

Marketing Cloud offers both **REST APIs** and **GraphQL APIs**, allowing you to choose the best solution that suits your needs. To retrieve data from your Marketing Cloud account, you can use the GraphQL APIs instead of REST. However, to submit and alter data through our API network ecosystem, you must use the REST APIs.

To have further insights on such query languages, see:

- [**REST APIs**](https://galaxy.maropost.com/s/article/REST-APIs)
- [**GraphQL APIs**](https://galaxy.maropost.com/s/article/GraphQL-APIs)

### API Reference Guides

The API Reference Guides for REST and GraphQL APIs are accessible via the account's application interface and contain information about every API supported by Marketing Cloud.

To access the API reference guides, click on your username and select **Connections**.

![api12.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05TJ100000Iq0GyMAJ.jpg)
On the Connections page, select the **REST API** tab to see all the supported REST APIs, and the **GraphQL API** tab to view all the supporting GraphQL APIs.

![api13.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05TJ100000Iq0P2MAJ.jpg)

### Request Authentication

Marketing Cloud's API uses API Keys to validate requests from several IP addresses. The Settings tab allows you to generate, view, and administer your API keys. For more details, read our [**API Keys**](https://galaxy.maropost.com/s/article/API-Keys) article.
