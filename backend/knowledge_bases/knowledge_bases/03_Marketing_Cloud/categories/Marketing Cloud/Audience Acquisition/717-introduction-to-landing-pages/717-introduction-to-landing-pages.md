# Introduction to Landing Pages

You can create dedicated landing pages to significantly improve your email campaign effectiveness by providing focused, conversion-optimized destinations for your marketing messages. Landing pages enable you to deliver targeted experiences that match your email content while maintaining brand consistency across the customer journey. This helps enhance subscriber interaction by reducing distractions and guiding recipients toward specific actions like purchases, registrations, or downloads.

When properly configured with relevant templates and clear calls-to-action, landing pages typically perform better than directing traffic to general website pages, as they streamline the user experience and improve marketing effectiveness for your campaigns.

Marketing Cloud allows you to create standalone pages that customers can visit upon clicking a link. Such links can be sent to customers via email or SMS marketing. The page theme may vary from your main website since these landing pages may serve a different purpose each time you run a marketing campaign.

Maropost’s Marketing Cloud enables you to create a landing page from scratch, i.e., it allows you to add your own headers, page URLs, page tracking functionality, HTML codes, etc. Additionally, it offers a wide range of tools to help you create aesthetically appealing and dynamic web content for your customers.

If you have difficulty creating a new landing page from the start, you can choose a large selection of our web page templates and customize them as per your requirements.

## Managing Landing Pages

The Landing Pages index page consists of all the web pages that you have previously created. Here you can manage your existing web pages as well as create new ones.

To access the Landing Pages module, open your Maropost Marketing Cloud account and click on **Acquisition** >> **Landing Pages**.

![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/land0.05T5g00000w679XEAQ.png)

The web pages are listed with the following information:

![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/land30.05T5g00000w69TvEAI.png)

- **Name: **It is the title of your web page.
- **Editor Type:** It specifies the format/editing tool used to create a web page. Marketing Cloud allows you to create web pages using either a "WYSIWYG" or a "Drag & Drop" editor.
- **Domain Status:** Displays a verification status indicating whether a page URL's domain is present on a landing page or not.
- **Publish At:** Depicts the time at which the web page will be activated.
- **Expire At:** Depicts the time at which the web page will be inactivated.
- **Updated At:** Displays the date and time at which the web page details were modified.
- **Created At:** Displays the date and time at which the web page was initially created.
- **Actions:** Allows users to verify a domain, preview, duplicate, and discard a landing page. To know more about previewing landing pages, kindly read this [**article**](https://galaxy.maropost.com/s/article/Preview-a-Landing-Page).

To view a list of web pages belonging to a particular format, use the **View** drop-down and select the format you want to view.

![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/land31.05T5g00000w69UZEAY.png)

Besides viewing and managing your existing web pages, you can create a new landing page via the **New Page** button. To learn more about how to create a landing page, kindly read our [**Creating Landing Pages**](https://galaxy.maropost.com/s/article/Creating-Landing-Pages) content.
