# Surveys

Survey functionality enables you to gather detailed customer insights that can substantially improve engagement by providing data-driven understanding of customer attitudes, preferences, and satisfaction levels. This feedback mechanism helps optimize your marketing effectiveness by allowing you to segment audiences based on survey responses and create more targeted, relevant campaigns that typically perform better than generic messaging.

You can leverage survey data to automates routine audience segmentation tasks and improves operational efficiency by automatically triggering campaigns based on customer responses and interests.

A **Survey** is a detailed study of a market or geographical area to gather data on attitudes, impressions, opinions, satisfaction level, etc., through a poll conducted amongst a select audience.

Adding a survey to your website helps promote a better understanding of the attitude or the requirements of the individual customer. Based on the interest of the user, you can send broadcasts/campaigns and set up automation.

### Managing Surveys

You can create new surveys and manage previously created ones on the Surveys index page. In Navigation, go to ACQUISITION, and then select Surveys.

![Survey_Index_Page.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/05T5g00000BkWJW.05T5g00000BkWJWEA3.png)
The following actions are available to you:

- **Preview Survey**: Use the option to see how your survey will appear to your customers.
- **Survey Questions:** Use the option to add questions to the survey. To learn about the types of questions that you can add to the survey, see Step 5 in the Creating Survey Forms section.
- **Reports**: Use the option to see the reporting data based on the responses received for the survey.
  ![Results.jpg](https://us.v-cdn.net/6038474/uploads/attachments/migrated/05T5g00000BkWJX.05T5g00000BkWJXEA3.png)

  ![Results2.jpg](https://us.v-cdn.net/6038474/uploads/attachments/migrated/05T5g00000BkWJY.05T5g00000BkWJYEA3.png)
- **Edit Survey: **Use the option to update the survey details.
- **Delete All Responses**: Use the option to clear the survey responses.
- **Delete Survey Permanently**: Use the option to delete the entire survey along with the data associated with it.
- **Create a Duplicate Survey**: Use the option to create a replica of the survey.

### Creating Survey Forms

To create a survey form, do the following steps:

1. In Navigation, go to ACQUISITION, and select Surveys. Then, on the Surveys index page, click New Survey.
2. 
  In the New Survey dialog box, enter the survey details.

  ![New_Survey.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/05T5g00000BkWJZ.05T5g00000BkWJZEA3.png)
3. Click Create. The new survey form now appears on the index page.
4. Click the Actions drop-down button for the new survey that you created. And then from the drop-down menu, select survey questions.
5. On this questionnaire page, you can add and edit questions to be shown in the survey. You have the option of choosing from a variety of question types. To add a question, click a question type on the left.
  ![Survey_Questions.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/05T5g00000BkWvZ.05T5g00000BkWvZEAV.png)

The following types of questions are available for survey forms:

- **Free Text:** Add a question in text format.
- **Multiple Choice:** Add a question in multiple-choice format.
- **Multiple Selection:** Add a question with the option to choose multiple answers.
- **Drop Down:** Add a question with options that appear in a drop-down list.
- **Name:** Add a field to fetch the first and last name of a user.
- **Email address:** Add a field to fetch the email address of a user.
- **Phone number:** Add a field to fetch the phone number of a user.
- **Address:** Add a field to fetch the address of a user.
- **Date/Time:** Add a field to fetch any date and time input from the customer.

The changes are saved as you add and edit the questions. Once done, close the window and return to the Surveys index page.
