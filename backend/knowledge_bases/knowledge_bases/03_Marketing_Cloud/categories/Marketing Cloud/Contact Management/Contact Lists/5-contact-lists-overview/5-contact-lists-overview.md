# Contact Lists: Overview

You can organize your contacts into targeted lists to enable more relevant messaging and significantly improve subscriber engagement rates. Contact lists help optimize your email marketing effectiveness by allowing you to segment audiences based on specific criteria, reducing unsubscribe rates while delivering personalized content that resonates with different customer groups.

When properly organized, contact lists streamline your workflow by providing the foundation for targeted campaigns, automated workflows, and personalized product recommendations. This approach helps ensure your marketing messages reach the most relevant audiences, typically leading to better campaign performance and stronger customer relationships.

A **contact list** enables you to store details of several contacts and use them for various marketing purposes, such as workflows, product recommendation emails, subscription renewals, feedback forms, surveys, and more. There are no restrictions on the number of lists an account can have or the number of contacts that can be added to a list.

You can view all previously created contact lists from the contact list's index page. From the Index page, you can edit an existing list or create a new one and add new contacts to the list. We have also provided you with options to export contacts from a list and import contacts from other platforms through various import methods.

You can access this section from the **Navigation** panel under the **Audience** menu.

## Viewing the Contact List Overview

Click the list name to see a **360-degree** view of the contacts. Here, you can see an overview of all the list's details, along with quick insights. From the list itself, you can additionally manage your list and take any steps if necessary. For more information, see the [**Contact Lists Overview**](https://galaxy.maropost.com/s/article/Contact-Lists-Overview-Page-New) article.

## Creating a List

To create a list, click the **New List** tab on the Contact Lists index page. Then, on the list creation page, add relevant details such as name, description, brand list, subscription criteria, third-party integration links, etc. After generating the list, you can add contacts to it either manually or by importing them. For more information, see the [**Creating a Contact List**](https://galaxy.maropost.com/s/article/Creating-a-Contact-List-New) article.

## Importing a List

To import contacts to a list, select the **Import List** option from the Actions drop-down menu. In the Import List dialog box, you have three options for importing a list: **File Import, FTP Import,** and **Automated Imports**. For more information, see the [**Importing Contact Lists**](https://galaxy.maropost.com/s/article/Creating-and-Managing-Contacts-New#CreatingContactsThroughImport) article.

## Exporting a List

To export all of the contacts in the list, select **Export List** from the Actions drop-down menu. When you specify the details in the Export List dialog box and click **Save**, the contact exporting process begins. When the contacts are exported, you will be informed via the notifications panel. For more information, see the [**Exporting a Contact List**](https://galaxy.maropost.com/s/article/Exporting-Contacts) article.

## Viewing Contacts in a List

To see all the contacts in a list, select **View Contacts** from the Actions drop-down menu. The contacts page displays all contacts in the list, along with their associated information. On the [**Contact 360 Overview**](https://galaxy.maropost.com/s/article/Contact-360-Overview-New) page, click a contact's name to reveal their 360-degree viewpoint.

To add a new contact manually, utilize the **New Contact** tab. You can also use the icons to export and import contacts from the list.

## Editing a List

To make changes to a list, select the **Edit List** option from the Actions drop-down menu and make necessary modifications to the list thereafter.

## Deleting a List

If you wish to delete a list, select the **Delete List** option from the Actions drop-down menu. However, do reconsider since a deleted list cannot be recovered.

> **Note:** **Note:** Deleting a list is not recommended, as it can cause reporting issues.

## Refreshing a List

The Refresh icon is located next to every contact list, ensuring the list's contents are up-to-date. Upon refreshing, you can see the latest count of contacts.
