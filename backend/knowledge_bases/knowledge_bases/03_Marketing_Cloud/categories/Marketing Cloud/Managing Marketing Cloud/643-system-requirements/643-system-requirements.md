# System Requirements

**System Requirements** provide optimal performance for your email marketing campaigns by ensuring your browser and operating system can fully support all Marketing Cloud features. This helps optimize your email marketing effectiveness by preventing technical issues that could impact campaign creation, analytics viewing, or automation setup.

When your system meets these recommended standards, you can expect improved performance and reliability in managing your email marketing operations, resulting in more efficient campaign development and monitoring.

Ensure that your system configurations are as per the following recommended standards for best performance:

| **Operating Systems:** |  |
| --- | --- |
| **Operating System** | **Supported Version** |
| Windows | Windows 10 |
| Mac | macOS |
| **Browsers:** |  |
| **Browser** | **Supported Version** |
| Chrome | Version 79.0.3945.88(official build) (64 bit) |
| Firefox | Version 75.0 or above |
| Safari | Version 13.0.4 (13608.4.9.1.4) |
| Edge | v79.0.309 |
| **Note:** If you are using any of the following browsers, we recommend that you upgrade to one of the browser versions we support:   - Internet Explorer (IE) 6 / XP - IE 7 / Vista - IE 8 / XP - IE 8 - 10 / Win 7 - IE 10 / Win Phone 8.0 - Safari 5.1.9 / OS X 10.6.8 - Safari 6.0.4 / OS X 10.8.4 |  |
| **Other Configurations:** |  |
| Screen Resolution | 1280 x 800 (Recommended) |
