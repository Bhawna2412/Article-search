# Folder Management

**Folder Management** enables you to organize your email marketing campaigns, content, and segments efficiently using the updated folder interface, helping streamline your workflow and improve team collaboration. This helps optimize marketing effectiveness by providing an enhanced way to categorize and access your email marketing assets, reducing time spent searching for resources.

When properly organized through the new folder system, marketing teams can substantially improve operational efficiency with improved drag-and-drop functionality and better privilege management for collaborative email marketing projects.

Folders are a great way to organize your data. The folder panel is available in various index pages of the application. You can create multiple folders and drag-and-drop the data into them.

> **Note:** **Note:** Currently, we support both legacy and new folder panels. The new folder panel is currently available in the index pages of the following: **Email Campaigns, Segments,** and **Content**. For information on the legacy folder panel, see Folder Management - Legacy.

## Accessing Folders

Along the right side of the **Navigation** panel, you’ll notice a small folder icon. Click the icon, and the folder panel for that page expands.

![Folder_Closed.jpg](https://us.v-cdn.net/6038474/uploads/attachments/migrated/05T5g00000BkWVj.05T5g00000BkWVjEAN.png)

## Managing Folders

You can use the + New Folder button to create new folders. Use the hover-over action menu (three dots) in the folder panel to rename or delete folders and assign privileges to other users. When you leave the page, the folder panel retracts back to the left side. If you want to keep the Folder panel open when loading the Index page, turn on the Always Open toggle button.

![Folder_Open.jpg](https://us.v-cdn.net/6038474/uploads/attachments/migrated/05T5g00000BkWVk.05T5g00000BkWVkEAN.png)

## Setting Folder Privacy or Assigning Privileges

To edit the privacy settings of a folder, i.e., assigning privileges to other users, select the folder and click the Privacy icon. To assign a user as one of the owners of the folder, select the user in the OWNERS tab. To assign a user as one of the editors of the folder, select the user in the EDITORS tab. You can remove the privileges by deselecting the user in the relevant tab. After you are done making the changes, remember to click Save.
