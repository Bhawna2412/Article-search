# Creating Growth & Attrition Reports

You can create Growth & Attrition reports to enable data-driven decisions about your email marketing performance by tracking subscriber acquisition and loss patterns over time. These reports help identify improvement opportunities in your list management strategy and measure campaign effectiveness by showing trends in list growth, subscriber engagement, and contact attrition rates. When properly configured with relevant date ranges and performance metrics, growth and attrition analysis typically leads to better subscriber retention strategies.

Growth & Attrition reporting also helps optimize your email marketing ROI by revealing which campaigns and time periods generate the highest quality subscribers while identifying potential issues that might be causing subscriber loss before they significantly impact your list health.

To learn more about Growth & Attrition reports and what the report metrics mean, see [**Growth & Attrition Report: Overview**](https://galaxy.maropost.com/s/article/Growth-Attrition-Report-Overview).

Perform the following steps to create a Growth & Attrition report:

1. In the Navigation, go to ANALYTICS, and then select Custom Reports.
2. On the Custom Reports Index page, click the New Report button.
3. On the New Custom Report Type page, select the report type as Growth & Attrition.
4. Select a schedule and enter the report name and delivery details for the report. See Step 1 - Schedule & Delivery for more details.
5. Choose the report details, such as the report file type, date format, lists from which to retrieve the data, and performance metrics to be included in the report. See Step 2 - Report Details for details.
6. Finally, when you are done setting up the report, click Submit.

Once the report is generated, you can download the report from the Notification panel. You can also view and download the report from the Custom Reports Index page. In the Navigation, go to Analytics and then select Custom Reports to open the Custom Reports Index page.

### Step 1 - Schedule & Delivery

In this step, you select the schedule and enter the report name and delivery details for the report.

![G_A_Step_1.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/05T5g00000BkXFF.05T5g00000BkXFFEA3.png)
Select from or enter values for the following fields:

- **Report Name:** Enter the name of the report.
-   -     - **From Date/To Date:** Enter the date range for the report. The date range you enter will be used to select all campaigns that were delivered and responses received within that range. Both 'from' and 'to' dates are inclusive, that is, the report will be generated from 12 AM of the 'from' day to 11:59 PM of the 'to' day.

> **Note:** The dates and times are in Eastern Time Zone.

    **Once** - Select this option if you just want the report to run once.
  -     - **Recur Every:** Select whether to have the report sent to you on a daily, weekly, or monthly basis.
    - **Time:** Specify the time of day (in the Eastern Time Zone) at which the report is to be generated.
    - **Delivery Date:** Specify the date on which the report starts getting generated.

    **Recurring** - Select this option if you want the report to be sent to you on a recurring basis.

  **Schedule Report:** Select the report schedule.
-   - **Recipient Email:** Enter the email address of the recipient.
  - **Subject:** Enter a subject line for the email.
  - **Message:** Enter a message to be sent in the email.

  **Delivery Details: **Specify the delivery details to send the report through email.

**Note**:

- You must provide an email address to receive the recurring report.
- The totals in the report reflect the delivery and response data collected from the preceding time period based on the recurring interval that you have selected. For example, if on Wednesday, you created the custom report and selected the option to have the recurring report generated every day at 6 AM, then Thursday's report will include total delivery and response data collected from 6 AM Wednesday to 6 AM Thursday.

### Step 2 - Report Details

In this step, you choose the report details such as report file type, date format, date range, and lists from which to retrieve the data. Here, you also choose the metrics to be included in the report.

![G&A Step 2.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000C9VRdEAN.jpg)
Select from or enter values for the following fields:

- **Report Type:** Select the file type for the output of the report.
- **Date Format:** Select the date format in which the dates in the report are to be shown. Currently, the supported date formats are MM/DD/YYYY and DD/MM/YYYY.
-   - **Lists **- Select the contact lists that you want to include in the report. You can start typing the name of the list in the dynamic search bar to quickly find the campaigns you want.

  **Report Metrics:**
- **Performance Metrics:** Select the metrics that you want to include in the output file. When making a copy of an existing report, the fields that you selected in the original report will be pre-selected by default in the copy. For more information on the performance metrics, see [**Growth & Attrition Report: Overview**](https://galaxy.maropost.com/s/article/Growth-Attrition-Report-Overview).
