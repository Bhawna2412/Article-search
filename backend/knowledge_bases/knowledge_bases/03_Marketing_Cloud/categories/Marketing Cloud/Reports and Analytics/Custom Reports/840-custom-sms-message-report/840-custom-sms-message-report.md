# Custom SMS Message Report

A **Custom SMS Message Report** is a summary of your outbound and inbound message segments for specific SMS campaigns within the chosen time period.

When using the SMS campaign feature, you are invoiced based on the SMS message segments (campaigns, test messages) that you send and the inbound messages (replies) that you receive. You can create one-time reports or set up recurring reports where you can find these details to help you understand what you are being billed for.

Here's a screenshot of a sample report:

![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/SMSMessageReport.05T5g00000pm4pIEAQ.png)

An SMS message report consists of the following information:

- **Sending Number**: The number from which the message originated. For the outbound messages, this will be your number, and for the inbound messages, this will be your customers' numbers.
- **Receiving Number**: The number on which the message is received. For the outbound messages, this will be your customers' number, and for the inbound messages, this will be your number.

- 
- 
- **Segment Count**: The message segment count required to send an SMS. Segments are the character batches used to measure a text message. The mobile carriers have a specific character limitation for each segment, and therefore, a long SMS message is sent in multiple segment batches.
- **Message Type**: Indicates the type of the message, such as campaign, test, valid keyword, and invalid keyword.
