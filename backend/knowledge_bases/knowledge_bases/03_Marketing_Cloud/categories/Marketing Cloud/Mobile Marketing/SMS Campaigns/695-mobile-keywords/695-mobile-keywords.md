# Mobile Keywords

**Mobile Keywords** are texts or phrases that you can define to be associated with certain actions. You can use these keywords in SMS campaigns to interact with your audience. Your contacts can text the keyword in response to your SMS campaign to indicate their preferred response action.

The primary use case for Mobile Keywords is building contact lists; however, you can have many creative uses of the feature. See [**Mobile Keywords FAQs**](https://maropost.vanillacommunities.com/home/leaving?allowTrusted=1&target=https%3A%2F%2Fgalaxy.maropost.com%2Fs%2Farticle%2FMobile-Keywords-FAQs) for details of a few such use cases and also answers to some other frequently asked questions.

**Points to Remember**: 

- 
- 
- Each Mobile Keyword must be unique. You can duplicate a Keyword's settings, but the Keyword text has to be different.
- You cannot delete any keywords. However, you can disable the Keywords that you don't want to use.

### Creating a Mobile Keyword

Perform the following steps to create a Mobile Keyword:

1. 
  In **Navigation**, go to **CONTENT**, and then select **Mobile Keywords**.

  ![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/mobkey0.05T5g00000xqNkOEAU.png)
2. 
  On the **Mobile Keywords** index page, click **NEW KEYWORD**.

  ![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/NewKeyword1.05T5g00000xprPYEAY.png)
3. 
  On the Add New Mobile Keyword dialog box, enter a unique keyword text and select the choice of action for the keyword. The available actions are **Subscribe, Unsubscribe,** and **Add To DNM** Lists.

  ![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/NewKeyword2.05T5g00000xprSIEAY.png)
4. 
  Select the lists to which the action is to be associated. For example, if the action is ‘**Subscribe**’, then the customer texting the keyword will be added to the list selected in this field.

  ![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/mobkey4.05T5g00000xqN07EAE.png)
5. 
  Now, enter a message in the response field that the customer will receive after texting the keyword. While drafting the response message, you can use additional keywords within it to enhance your interaction with your customers.

  ![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/NewKeyword4.05T5g00000xprhcEAA.png)
6. 
  Enable click tracking for the keyword. By default, the Click Tracking toggle is enabled.

  ![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/NewKeyword5.05T5g00000xprjJEAQ.png)

### Setting Up Invalid Response Message

The invalid response message is sent in response to the texts that are received with the wrong or missing keyword. This message is set up on the account level and is applicable to all the Keywords created in that account.

On the Mobile Keywords index page, click the Invalid Response Message button. In the dialog box, draft your response message for the invalid texts. This message can contain contact tags and Keywords. You can also use the Send Test feature to test this message. Currently, you can send the test message to one mobile number at a time.

1. 
  On the **Mobile Keywords** index page, click **INVALID RESPONSE MESSAGE**.

  ![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/ResponseMsg1.05T5g00000xpnuMEAQ.png)
2. 
  In the dialog box, draft your response message for the invalid texts. You can add keywords in your response message and enable click tracking too.

  ![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/ResponseMsg2.05T5g00000xpsXOEAY.png)
3. 
  Enter the **Sender** and **Receiver** mobile numbers in the given fields. You can use the Send Test option to send a test message. Finally, click **SAVE**.

  ![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/ResponseMsg3.05T5g00000xpsXnEAI.png)

### Using a Keyword in an SMS Campaign

When composing a new SMS Campaign, you can add Keywords to the message. All the active Keywords in your account are available from the Keywords drop-down list.
