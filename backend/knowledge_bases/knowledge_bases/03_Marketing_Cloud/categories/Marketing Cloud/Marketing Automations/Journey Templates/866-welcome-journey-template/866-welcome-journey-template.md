# Welcome Journey Template

You can configure welcome journey templates to nurture leads automatically and deliver timely, relevant content to new subscribers. This automated workflow significantly improves subscriber engagement during the critical first impression period, helping establish stronger relationships and enhancing long-term campaign effectiveness.

Welcome journey automation enables you to scale personalized communication without manual effort. When properly configured, welcome journeys typically perform better than generic email campaigns, driving improved open rates and setting clear expectations that reduce unsubscribe rates during the crucial onboarding phase.

**Welcome Templates** can be used to create a journey for sending introductory emails to your new subscribers.

## Usage Scenario

You want a simplified way to automate the process of sending out a series of welcome emails to your new subscribers. The welcome emails are a great opportunity to introduce your business, build a personal connection, and help users utilize your products or services.

> **Note:** To learn about Journey Templates, see Journey Templates: Overview.

## Before You Begin

In order to use the Welcome template to set up a journey, you must already have:

- **Contact lists** that are to be used to trigger the journey. To learn about contact lists, see [**Introduction to Contact Lists**](https://galaxy.maropost.com/kb/articles/5-contact-lists-overview-new).
- **Email Content** to be used in the emails. To learn about email content, see [**Content Builders: Overview**](https://galaxy.maropost.com/s/article/Content-Builders-Overview).

## Step-by-Step Process

Setting up a Welcome Journey is simple with the use of the template and requires you to perform two steps. In step 1, you configure your Journey settings, and then in step 2, you set up the trigger for the Journey and content for emails to be sent.

The following steps show how to create a journey using the Welcome Template:

1. In step 1 (Settings for Welcome Journey), enter a name for your journey in the **Journey Name** text box.

![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/WTStep1NameEndDate.05T5g00000fa89OEAQ.png)
2. Optionally, enter an end date and time for the journey in the **End Date** and **End Time** fields, respectively. If you specify a date and time here, the journey becomes inactive on the specified date and time.

3. Optionally, select the **Enable Journey** checkbox to activate the Journey at the time of creation. If the checkbox is selected, the journey becomes active right after it is created.

![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/WTStep1checkboxes.05T5g00000fa8juEAA.png)
4. Optionally, select the **Retrigger Journey** checkbox to enable retriggering of the journey. If the checkbox is selected, the journey is retriggered for the contacts who have already finished the journey if they match the trigger criteria again.

5. Click **Next** to go to step 2.

6. In step 2 (Setup for Welcome Journey), enter the sender's name in the **From Name** text box and the sender's email address in the **From Email Address** text box. The name and email address you specify here are used in the welcome emails.

![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/WTStep2SenderInfo.05T5g00000fa7PmEAI.png)
7. Next, set up the trigger for the journey by selecting one or more contact lists from the **Select List** drop-down list. A trigger here refers to the event that begins the journey for a contact. For a welcome journey, the trigger event is when a contact is added to any of the specified lists. So, as part of setting up the trigger, you must select the lists.

![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/WTStep2Trigger.05T5g00000fa8RhEAI.png)
8. In the **Set Up Content** section, select the content for the emails to be sent. Following the industry standard, Marketing Cloud enables you to send out three welcome emails.

> **Note:** The delay between each of these emails is 1 day.

![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/WTStep2Content.05T5g00000fa8MwEAI.png)

- **Email 1: Introduction: **Select the content for the first email. We recommend using email content to introduce your business and services.
- **Email 2: Expectations:** Select the content for the second email. We recommend using email content to set clear expectations.
- **Email 3: Expectations:** Select the content for the third email. We recommend using email content to talk about how your business can help them.

9. Click **Finish** to create the journey.
