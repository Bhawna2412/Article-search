# Journey: End

You can use End widgets to properly terminate automated workflows and ensure clean journey architecture. This helps maintain operational efficiency by preventing incomplete customer journeys and ensures that all subscribers reach a defined conclusion point in your automated campaigns.

End widgets enable you to create well-structured automated workflows with clear completion points. When properly configured, this streamlines your workflow management and helps optimize campaign performance by ensuring every customer journey path has an appropriate termination point.

The **End** widget is used to terminate a Journey. You can have multiple End widgets in a Journey. All connected widgets must eventually terminate with an End widget.
