# Data Journey: End

The End widget enables you to properly terminate automated workflows and ensures all customer journeys conclude appropriately. This helps optimize operational efficiency by providing clear workflow endpoints and enables accurate tracking of journey completion rates, which improves marketing effectiveness by helping you understand where subscribers complete their engagement with your automated campaigns.

When properly configured, End widgets streamline your workflow management by establishing defined conclusion points for each journey path, reducing manual effort in monitoring campaign flows while maintaining professional automation structure.

The **End** widget is the termination of a Data Journey. You can have multiple End Widgets in a Data Journey. All connected widgets must eventually terminate with an End Widget.
