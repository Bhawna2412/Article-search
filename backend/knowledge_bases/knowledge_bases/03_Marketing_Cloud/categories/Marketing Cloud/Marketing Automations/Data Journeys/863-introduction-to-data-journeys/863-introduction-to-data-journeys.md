# Introduction to Data Journeys

**Data Journeys** also present a schematic for automation similar to [**Journeys**](https://galaxy.maropost.com/s/article/Introduction-to-Journeys); however, while the Journeys are for individual contacts, the Data Journeys are for the management of bulk data. You can use Data Journeys to import or export a list, execute an SQL Query, or send a campaign to a group of contacts.

![Data_Journey.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/05T5g00000BkX7R.05T5g00000BkX7REAV.png)

You can use the following widgets to create a journey:

- **Data Journey Triggers:** Use a Data Journey Trigger widget to define the rules by which a contact enters the Data Journey. Every trigger has a different function and can be used to initiate the management of bulk data. [**Learn More**](https://galaxy.maropost.com/s/article/Data-Journey-Triggers)
- **Data Journey Actions:** Use a Data Journey Action widget to define the course of action to be initiated; for example, start the import, trigger a Journey, toggle data Journey, FTP upload, HTTP post, secure list import, and so on. [**Learn More**](https://galaxy.maropost.com/s/article/Data-Journey-Actions)
- **Data Journey Filters:** Use a Data Journey Filter widget to add branching logic to the Journey. There are two filter widgets available to you: % Split and A/B Split. [**Learn More**](https://galaxy.maropost.com/s/article/Data-Journey-Filters)
- **Data Journey Delays:** Use a Data Journey Delay widget to delay the next step in a Data Journey for a certain time period. [**Learn More**](https://galaxy.maropost.com/s/article/Data-Journey-Delays)
- **Data Journey End:** Use the Data Journey End widget to terminate a Data Journey. [**Learn More**](https://galaxy.maropost.com/s/article/Data-Journey-End)

### Managing Data Journeys

You can create a new Data Journey and manage your existing Data Journeys on the Data Journeys index page. To access the page, in Navigation, go to AUTOMATION and then select Data Journey.
