# DNS Setup

**DNS Setup** is essential for ensuring emails reach intended recipients and protects your domain's reputation. Properly configured sending and link tracking domains optimize deliverability performance and help maintain professional email communications that recipients trust.

This foundational configuration significantly improves inbox placement by establishing an authenticated email infrastructure, which is required before you can begin sending email campaigns.

DNS configuration is a mandatory step before you can begin sending email campaigns from your account.

Marketing Cloud enables you to easily set up the sending and link tracking domains as part of the application settings. To navigate to the Settings page, click your user name in the top-right corner of the application, and then from the drop-down menu, select Settings.

As part of the DNS configuration, you must set up the following:

- **Sending Domains** – The domain that will be used for sending your email campaigns. You must add and verify at least one domain as a sending domain. For more details, see [**Set Up and Manage Sending Domains**](https://galaxy.maropost.com/s/article/Set-Up-and-Manage-Sending-Domains).
- **Link Tracking Domains** – The domain that will be used for tracking the sent email campaigns. You must add and verify at least one domain as a link tracking domain. For more details, see [**Set Up and Manage Link Tracking Domains**](https://galaxy.maropost.com/s/article/Set-Up-and-Manage-Link-Tracking-Domains).

> **Note:** **Note:** You must have the **DNS Setup** privilege enabled to be able to set up and manage the DNS. To learn about user privileges, see [**Managing Users and Their Privileges**](https://galaxy.maropost.com/s/article/Managing-Users-and-Permissions-in-Marketing-Cloud).
