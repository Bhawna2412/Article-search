# Mobile App Profiles

You can create mobile app profiles to expand your marketing reach beyond email and SMS campaigns through push notifications and in-app messaging. This helps optimize marketing effectiveness by enabling multichannel communication strategies that engage customers through mobile applications, improving operational efficiency through centralized mobile marketing management.

When properly configured, mobile app profiles reduce manual effort by streamlining mobile SDK development and help ensure consistent communication across all your marketing channels.

**Mobile App Profiles** is a software kit consisting of tools that help users build platform-oriented mobile applications and implement new features in existing mobile apps. The resources contained in an SDK include libraries, documents, code samples, guides, and APIs.

**SDK** stands for “**Software Development Kit**.**” **It helps developers build app functionalities without the need for in-depth knowledge, thus preventing them from writing code from scratch whenever a new implementation is needed.

Maropost Marketing Cloud provides a dedicated section wherein you can create and manage such mobile profiles.

## Managing Mobile App Profiles

On the **Mobile App Profiles** page, you can create mobile SDKs for different testing environments and manage them from a single place. Click your user name in the top-right corner of the application, and from the profile drop-down menu, select **Settings**. Then, on the Settings page, select **Mobile App Profiles**.

![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/MobileAppNav.05T5g00000xVrvUEAS.png)The index page displays all your existing mobile app profiles and provides you with an option to create a new one.

![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/MobileAppIndexPage.05T5g00000xVrxQEAS.png)For each profile, you can find the following details and action items:

- 
- **Status:** Shows the current status of the software that is being tested and utilized.
- 
- 
- **Update At:** Shows the date and time at which the Mobile SDK software was last modified.
- **Actions:** Enables you to edit or delete a mobile app profile.

## Downloading and Configuring the Mobile App

To view and download configuration files for your mobile apps, select the Mobile App Profile on the index page (**Settings** >> **Mobile App Profiles**). This opens an overview page showing the SDK details and provides you with the option to download the SDK files.

![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/MobileAppOverview.05T5g00000xVtEcEAK.jpg)On this page, you can find the following information:

- 
- **Push Ready Platforms**: Indicates the platforms for which the app is built, i.e., iOS and/or Android.
- **iOS Setup** and **Android Setup**: Contains the SDK **.Zip**, and **configuration files** for downloading.

Once you have downloaded the configuration and .zip files, you can proceed with the rest of the installation and configuration required for your setup. 

To learn more about how to implement the Maropost Mobile SDK and the Mobile SDK configuration process for iOS and Android, see [**Mobile SDK: In-App Messaging page**](https://galaxy.maropost.com/s/article/Mobile-SDK-In-App-Messaging), [**Mobile SDK for iOS – Installation and Configuration**](https://galaxy.maropost.com/s/article/Mobile-SDK-for-iOS-Installation-and-Configuration), and [**Mobile SDK for Android – Installation and Configuration**](https://galaxy.maropost.com/s/article/Mobile-SDK-for-Android-Installation-and-Configuration), respectively.
