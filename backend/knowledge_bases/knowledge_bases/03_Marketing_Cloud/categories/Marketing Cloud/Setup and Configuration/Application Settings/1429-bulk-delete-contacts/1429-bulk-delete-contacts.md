# Bulk Delete Contacts

The **Bulk Delete Contacts** feature in Marketing Cloud allows you to remove a large number of contacts with ease.

If you have a large number of contacts that you want to delete, create a file with the email addresses of all the contacts. Then, import the file on the **Bulk Delete Contacts** page to initiate the delete process.

The following are the steps to delete the contacts in bulk:

1. Click your username in the top-right corner of the application, then from the drop-down menu, select Settings.

2. On the **Settings** page, in the Contacts section, click Bulk Delete Contacts.

![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/SettingsBulkDelete.05T5g00000SRjegEAD.png)
3. On the **Bulk Delete Contacts** page, click the Import icon.

![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/BulkDeleteContacts.05T5g00000SRjevEAD.png)
4. In the **Import Emails to Delete** dialog box, select the file (TXT or CSV format) containing the email addresses of the contacts that you want to delete. Then, select the delimiter used in your uploaded file. Finally, select the column name or column number in which your contacts' email addresses are stored.

![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/ImportEmailstoDelete.05T5g00000SRjfeEAD.png)
5. Click IMPORT to start the process of importing and deleting the contacts.
