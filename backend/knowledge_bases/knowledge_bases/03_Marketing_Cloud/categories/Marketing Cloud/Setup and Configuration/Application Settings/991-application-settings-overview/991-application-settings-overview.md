# Application Settings: Overview

You can access centralized application settings to configure essential features that enable more relevant messaging and improve subscriber engagement. This streamlines your workflow by providing organized access to all configuration options needed for effective email marketing operations, from DNS setup to contact management.

When properly configured, these settings help optimize your email marketing infrastructure and reduce manual effort by organizing all administrative functions in one accessible location.

The **Settings** page is where you can find the application-related setups and configurations. These setup tasks are categorized into the following categories: **Connections, DNS Setup, Contacts,** **Campaigns,** and **Mobile App Configuration**.

Click your user name in the top-right corner of the application, and then from the drop-down menu, select Settings.
