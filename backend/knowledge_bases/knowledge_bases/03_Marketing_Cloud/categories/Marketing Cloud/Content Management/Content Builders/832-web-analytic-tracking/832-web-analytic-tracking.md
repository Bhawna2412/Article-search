# Web Analytic Tracking

You can use web analytics tracking parameters to measure campaign effectiveness and enable data-driven decisions about your email marketing performance. This significantly improves your ability to track subscriber engagement and identify improvement opportunities by automatically appending tracking parameters to all external links in your email content.

**Web Analytics Tracking** helps optimize your email marketing ROI by providing detailed insights into how contacts interact with your content. When properly configured, this approach reduces manual effort while ensuring comprehensive tracking across all your email campaigns.

Marketing Cloud provides you with the ability to include web analytics tracking parameters in your content. The purpose of this feature is to track how your contacts interact with your content and which aspects of the content they engage with the most.

Normally, when you need to append web analytics tracking parameters to your email links, you have to create the links and then append them manually to each and every link you want to track. However, with the Web Analytics feature in content builders, you can create the tracking parameters once and then auto-append them to every non-Marketing Cloud link in your content.

## Adding Web Analytics Tracking Parameters

To add analytics tracking parameters to your content, follow the steps below:

1. In Navigation, go to **Content**, and then select **Email Content.**
2. On the **Email Content** index page, create new content (clicking the **New Content** button) or edit existing content (**Actions** > **Edit Content**). To create new content, you'll be asked to select the content builder you want to use.
  ![conteditor54.png](https://us.v-cdn.net/6038474/uploads/DREJ13F3LBPR/conteditor54.png)
3. On the content builder, click the **Advanced** tab.
  ![conteditor55.png](https://us.v-cdn.net/6038474/uploads/G1PKK9RLW9B4/conteditor55.png)
4. ***Note:**** Instead of populating the value yourself, you can use our *[***tags available for content***](https://galaxy.maropost.com/s/article/Tags-in-Content-Overview)* (for example, the ****{{campaign.name}}**** tag) to dynamically set the values of your tracking parameters.*
  ![conteditor56.png](https://us.v-cdn.net/6038474/uploads/AOCYA344XK7M/conteditor56.png)
5. Optionally, you can add more parameters by clicking **Add Parameter** and entering the name and value for the parameter.
6. Click **Save**.
