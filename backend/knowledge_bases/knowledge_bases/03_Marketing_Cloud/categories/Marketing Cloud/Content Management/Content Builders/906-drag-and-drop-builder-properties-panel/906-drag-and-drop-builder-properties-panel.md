# Drag and Drop Builder: Properties Panel

The **Properties** panel enables you to create professional, branded email designs with precise control over visual elements and layout structure. Understanding how to use the Content, Rows, and Settings subtabs helps you build engaging email experiences that maintain brand consistency while optimizing for different devices and email clients.

When properly configured, these design controls help ensure your emails display correctly across various email platforms while maintaining the visual appeal that drives subscriber engagement and campaign effectiveness.

The Properties panel, along with the Canvas, helps you design your content in the drag-and-drop builder. It is divided into three subtabs: **Content, Rows,** and **Settings**.

## Content Subtab

The **Content** subtab contains the content elements that you can choose from and add to your message. Just drag and drop a content element (for example, a button, an image, a text block, a transparent or solid divider, a social media sharing widget, and so on) onto the canvas area.

### **Content Properties**

When you click a content element in the message, the Content Panel switches to Properties mode, allowing you to change the settings of that particular element. The Content Properties area changes depending on the content element that has been selected. If you click on a button, for example, you will see several tools to help you style the button and link it to your desired action (such as opening a web page, sending an email, making a phone call, and so on).

> **Note:** **Note:** If you click outside a content element, the row in which the content element is located is selected, and its properties are available (for more information, see [**Drag and Drop Builder: Custom Row Categories**](https://galaxy.maropost.com/kb/articles/905-drag-and-drop-editor-custom-row-categories)).

In the example screenshot shown below, the "Check Out the Deals!" button has been selected:

![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/DnDContentProperties.05T5g00000o5eBOEAY.png)The properties panel switches to button mode and allows you to set the button’s width, background color, text color, alignment, the roundness of the corners, padding around the text, and much more.

With buttons and numerous other content items, the builder provides you with complete control over the design of your email messages. Some of the controls, including content padding and borders, have both a simple editing mode (where only one setting is shown) and an advanced editing mode (where more control is provided, for example, specifying padding for the four sides of the object independently of each other).

## Rows Subtab

The **Rows** subtab contains different types of structural elements for your message. This lets you create various sections of the message.

Why different sections? Can’t you just have different content blocks within the same section? You can, but that won’t always work. Rows with different structures allow you to organize content in a way that is different from the preceding and following rows. For instance, you may need:

- 
- A two-column row below it to display some of the store’s best-selling products

Whenever you need to switch to a different number of columns, that’s when you can introduce a new section by using the Rows subtab.

![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/DnDContentRows.05T5g00000o5owtEAA.jpg)The very first time you click the Rows subtab, you'll see in the drop-down selection list that you have two options for row categories:

- **Empty** (Default): This category displays a selection of empty rows with the number of columns and their widths already preset.
- **Default:** This category displays a selection of rows with preformatted content, including images, text blocks, and buttons, to enhance your user experience.

Once you have created and saved your own custom rows, the names of the custom categories also appear in the Rows subtab. To learn more about custom row categories, see [**Drag and Drop Builder: Custom Row Categories**](https://galaxy.maropost.com/s/article/Drag-and-Drop-Editor-Custom-Row-Categories).

Selecting a row will switch the properties panel to the row and column settings. The builder provides great flexibility and customization at both the row and column levels.

## Settings Subtab

The **Settings** subtab allows you to configure the general settings applicable to the entire message with ease.

![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/ContentSettings.05T5g00000o5e2lEAA.jpg)On this tab, you'll find the following settings:

- **Content Area Width**: Indicates the width of the content area. According to email marketing best practices, it should typically be set to 600 px or less.

> **Note:** **Note:** The width specified here is the maximum width that is to be used for devices with larger screens. When the device screen is smaller, the message is automatically adjusted to it.

- **Background Color**: Indicates the background color of the content message. This setting is effective for rows and columns in which color has not been specified. Otherwise, the background color specified for that content structure is used.
- **Content Area Background Color**: Indicates the background color for the content area. This setting is effective for rows and columns in which color has not been specified.
- **Default Font**: Indicates the default font that is used when another font family has not been specified. This setting affects all text that is set to use the global font.
- **Link Color**: Indicates the color of the link's text. This setting affects all text links set to use the default link color.
