# Content: Dynamic Tags

You can use dynamic tags to create more relevant and personalized email experiences by displaying different content based on recipient attributes or behaviors. This feature enables you to deliver targeted messaging within a single campaign, helping increase click-through performance and subscriber engagement.

Dynamic content helps optimize email marketing effectiveness by ensuring each recipient sees the most relevant information for their specific situation. When properly configured, this feature streamlines your workflow by eliminating the need for multiple campaign versions while maintaining the personalization that drives better campaign results.

The dynamic contents you create in Maropost Marketing Cloud are created as dynamic tags. These dynamic tags are then available in the Dynamic Areas drop-down list for you to use in the content.

To learn more about dynamic content, see [**Dynamic Content**](https://galaxy.maropost.com/s/article/Dynamic-Content).
