# Dynamic Content

Dynamic content enables you to deliver personalized email experiences that speak directly to each subscriber's interests, location, or behavior, which typically leads to better engagement rates compared to generic messaging. This capability helps create more relevant email campaigns by automatically displaying different content blocks to different audience segments within the same email template.

When properly configured, dynamic content streamlines your campaign production workflow by allowing you to create a single email template that serves multiple audience segments, rather than building separate campaigns for each group. This approach maintains brand consistency while delivering targeted messaging that enhances subscriber interaction and reduces unsubscribe rates.

**Dynamic Content** is the content within an email that changes based on the viewer. The relevant content is displayed to the people who match the criteria you've set for your dynamic content. Your criteria could be based on industry, job title, or even a product purchased.

Consider a scenario in which you wish to send a campaign to two different locations. The content needs to be tailored slightly differently for each location. The text in the email for the first location should read "Celebration in London," and the text in the email for the second location should read "Celebration in Toronto." In this case, you can use dynamic content to show the text differently depending on whether the viewer is located in London or Toronto.

## Managing Dynamic Content

The Dynamic Content page* *(in **Navigation** >> **Content** >> **Dynamic Content**)* *serves as the starting point for creating, editing, archiving, and deleting dynamic content. The Dynamic Content Index page shows the list of dynamic content. You can use the drop-down list at the top-left corner to choose whether you want to view the active or archived dynamic content.

> **Note:** **Note:** Archived dynamic content isn't available in any content editor and cannot be edited. However, you can still use the archived dynamic content to create content and templates.

## Creating Dynamic Content

Perform the following steps to create dynamic content:

1. In Navigation, go to **Content**, and select **Dynamic Content**.
2. On the **Dynamic Content** index page, click **New Dynamic Content**.
3. In the editor page, enter the name for the new dynamic content.
4. Next, use the editor to create the original content. The original content is shown to those audiences who do not meet the criteria specified in the rules.
  ![dyncontent2.png](https://us.v-cdn.net/6038474/uploads/AXF1ITERI54N/dyncontent2.png)
5. Click the **Add Rule** button to add criteria. The rule enables you to select the audience and then create alternative content to be shown to the specific audience. You may add multiple rules as per your requirement.
  ![dyncontent3.png](https://us.v-cdn.net/6038474/uploads/XGAOY5E1VLCW/dyncontent3.png)
6. To create a rule, do the following:  1. In the Rule section, select a segment for the rule. The segment selection determines the audience.
  2. Create alternate content that is to be shown to the audience. Instead of original content, the alternate content is shown to the audience from the specified segment.
***Note:**** Repeat these two steps to add more rules as needed.*
7. To learn more about tags and product recommendations in email marketing, read our [**Tags in Content: Overview**](https://galaxy.maropost.com/s/article/Tags-in-Content-Overview) and [**Product Feed Template**](https://galaxy.maropost.com/s/article/Product-Feed-Templates) articles.
  ![dyncontent4.png](https://us.v-cdn.net/6038474/uploads/34DENQXDISRI/dyncontent4.png)
8. Click **Save**.

## Using Dynamic Content

Our content editors enable you to use dynamic content within the content. You can simply insert the dynamic content tag when creating your content. In a content editor, select the place in the content where you want dynamic content to appear. Then, select the dynamic content tag from the Dynamic Areas drop-down list.
