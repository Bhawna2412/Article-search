# Image Library

The **Image Library** helps streamline your email design workflow by providing a centralized location for all your visual assets, ensuring consistent branding across campaigns while reducing the time spent searching for images. This organized approach to asset management enables your team to quickly access approved brand imagery and maintain visual consistency that enhances subscriber recognition and trust.

When properly organized with folders and naming conventions, your image library significantly improves operational efficiency by eliminating duplicate uploads and ensuring team members can easily locate the right visual elements for their email campaigns.

The **Image Library** acts as a repository where you can store all the images used in your content. Simply add images or create multiple folders and then add images into those folders.

To learn about how to create and manage folders, see [**Folder Management**](https://galaxy.maropost.com/s/article/Folder-Management).

> **Note:** **Note:** You can add an image of **PNG, JPG,** or **GIF** type up to a maximum of **2 MB** in size.

## Managing Image Library

On the Image Library index page, you can view and manage your previously added images and add new images. To access the page, in Navigation, go to CONTENT, and then select Image Library.

![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/ImageLibraryIndexPage.05T5g00000SQg2SEAT.png)By default, the images on the index page are displayed in the list view. You can use the Change View icons at the top to toggle between the list view and grid view. You can also see the details of an image (such as the name, link URL, and the date on which it was last updated) by clicking the image name.

## Adding New Images

To add a new image, click the Add Image button in the top-right corner. In the New Images dialog box, you can either drag and drop an image file or click the Upload button to select an image from your computer. You can add multiple images by using the Add More link before clicking the Save button.

![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/img0.05T5g00000zfTbyEAE.png)

> **Note:** **Note:** You can upload images that support **PNG, JPG, GIF,** and **JPEG** extensions.
