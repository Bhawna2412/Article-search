# Release Notes 3.15.9 - Shopify

# Release Notes 3.15.9 - Shopify

This release will be available to all customers by the 6th September 2022.

* Inventory Buffers will no longer reduce the available outlet inventory when checking the in-store available quantity.
* Resolved an issue where previously users were unable to access additional tabs within the Shopify Integration Settings in Retail Express unless they refreshed or reloaded the page.
* Resolved an issue so that Payment Methods can now be mapped to different Payment Types without encountering an error.
