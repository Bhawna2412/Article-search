# Release Notes v3.15.1 - API Updates

In this release, we have made one improvement to the EDS API, and one resolution to the Unified/Rest API.

This update will be made available to all customers between **7th February** - **15th February**.

## Improvements

We have added two new fields to the **EDS API** in preparation for a future [Shopify Pre-Order](https://portal.productboard.com/xjc5eqgr3wpp4lregjvharq3/c/38-shopify-preorders-feature) feature (you can read more about it on our [Product Portal](https://portal.productboard.com/xjc5eqgr3wpp4lregjvharq3/c/38-shopify-preorders-feature)).

These fields will not impact existing integrations. Further information on this feature will be available in the future.

```
<PreOrdersEnabled>false</PreOrdersEnabled><StockForPreOrder xsi:nil="true"/>
```
