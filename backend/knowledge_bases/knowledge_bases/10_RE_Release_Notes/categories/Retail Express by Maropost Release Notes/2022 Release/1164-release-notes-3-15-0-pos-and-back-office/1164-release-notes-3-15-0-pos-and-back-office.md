# Release Notes 3.15.0 - POS and Back Office

# Release Notes 3.15.0 - POS and Back Office

This release will be available to all users between 17th - 25th of January 2022.

## New Features

### Gift Voucher EDS API Integration

![Gift Voucher Integration (1).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppf7sEAA.jpg)

Omni Channel Gift Vouchers has been a huge success for our Shopify merchants, so we're extremely excited to announce we've updated the Event-Driven Synchronisation (EDS) API to allow integrators and third-party developers to build their own Gift Voucher integrations!

Using the EDS API, developers and integrators can build a direct integration with your web store so your customers can:

* Purchase Gift Cards online
* Redeem Gift Cards online (as well as in-store - redemptions can be synchronised for up-to-date balances)
* View Gift Card balances

Full details can be found in our API Release Notes. We strongly recommend forwarding this link to your developers/integrators.

[Click here to read more.](https://galaxy.maropost.com/s/article/Release-Notes-3-15-0-AP)

### Access Support from Back Office

We've added a new Help Widget to the Back Office so you can get the help you need without ever leaving the page. You can search for articles, or contact the Support Team, right from Back Office!

![kbwidget.gif](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppf87EAA.jpg)

The new widget includes the ability to:

* Search the Knowledge Base
* View articles
* Open the article in a new window
* Contact Support

Take a look at the [announcement](https://galaxy.maropost.com/s/article/Announcing-the-Knowledge-Base-in-Back-Office) for all the details.

## Improvements

* We've made improvements to the **Shippit integration** when fulfilling in bulk from Retail Express.
* We've introduced a **notification in POS** when a user tries to apply a global discount beyond their User or Product Maximum Discount Rule e.g. entering a discount of 25% when the user is only authorised to offer 10%  
  ![usernotification.gif](https://ress.zendesk.com/hc/article_attachments/4413576386063/usernotification.gif)

## Resolutions

* An error has been resolved with the **Stock Adjustment Summary Report** (following a Stocktake) where under certain circumstances values were displayed incorrectly as a positive or negative.
