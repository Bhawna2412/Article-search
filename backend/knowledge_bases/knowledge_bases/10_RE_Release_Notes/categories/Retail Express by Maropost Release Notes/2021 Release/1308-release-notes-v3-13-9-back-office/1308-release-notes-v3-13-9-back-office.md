# Release Notes v3.13.9 - Back Office

**Release Date:** 2nd August 2021

In this release we've made an update to the Stocktake functionality.

* **Stocktake Excel file** - Where an additional barcode has been listed, the Excel file will retain the zeroes at the start of the code e.g. 03892743811, where previously the 0 was incorrectly removed.
