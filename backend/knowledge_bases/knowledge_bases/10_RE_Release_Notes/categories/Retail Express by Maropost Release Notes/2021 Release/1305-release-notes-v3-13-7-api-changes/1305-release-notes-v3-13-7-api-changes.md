# Release Notes v3.13.7 - API Changes

**Release date:** 22nd June 2021

For this release we've updated the **Warehouse Management System** API to add a new method, "AllocateITOs". This method allows you to update the Allocated quantity for a list of Transfer items.

To use the new method, you will just need to enter the following parameters:

* the Items
* the Outlet/Warehouse the Transfer is being sent from
* Quantity to be allocated

For detailed information, including a sample request and response, please view the detailed API documentation below (refer to page 18 for the AllocateITOs method specifically).
