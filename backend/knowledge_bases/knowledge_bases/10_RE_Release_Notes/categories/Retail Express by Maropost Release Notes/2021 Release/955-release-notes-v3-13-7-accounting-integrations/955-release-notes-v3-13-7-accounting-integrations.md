# Release Notes v3.13.7 - Accounting Integrations

# Release Notes v3.13.7 - Accounting Integrations

**Release Date:** 22nd June 2021

In this update we've made several improvements and resolutions as part of our commitment to continuous evolution and improvement to Retail Express.

---

### Exclude Transaction Types from Xero

We have added the ability to configure which transactions should be included/excluded from the Upload to Xero.

Within the Xero Integration configuration in Retail Express (Settings > Integrations > Xero) we've added a new "Transaction Types" tab. To exclude a Transaction Type from Xero simply untick the option and click **Save all**.

**Note:**These will all be ticked by default to match the existing integration behaviour
