# Release Notes - 3.13.5 - POS and Back Office

**Release Date:** 27th April 2021

In this release we're excited to announce the support for multiple barcodes for a single product within Retail Express, making it easier than ever to manage products.

**New:**

* Manage multiple barcodes in Retail Express ([read the announcement here](https://galaxy.maropost.com/s/article/Announcing-Multiple-Barcodes))

**Updated:**

The following areas have been updated for compatibility with Multiple Barcodes:

* Global Search in Back Office
* Fulfilment Report
* Purchase Orders
* Product Search & Management
* Product Search
* Stocktake
* Stock Adjustment
* Transfers
* POS (New Sales, Refunds, Transfers)
* Unified API and Inventory Planning API - as new products are created the barcode will now also be validated against additional barcodes to ensure the new product code is unique
