# Release Notes - 3.13.1 - POS and Back Office

**Release Date:** 2nd February 2021

We're excited to announce the "**Quick Create Products**" feature, streamlining the process of creating products and saving hours of your time! On top of that, we've also made some improvements and resolutions to our Shippit integration. Read on for all the details.

In this release:

- [Purchase Orders - Quick Create Products](#quick-create)
- [Improvements and Resolutions](#improvements)

## []()**Purchase Orders - Quick Create Products**

The Quick Create product feature is available within Purchase Orders, allowing you to create both individual products and styles. In fact, creating styles has never been easier - you can do it all from within the same screen, instead of creating each style product individually!

Combine Quick Create Products with the Quick Receive feature (announced in [version 3.12.11](https://galaxy.maropost.com/s/article/Release-Notes-3-12-11-POS-and-Back-Offic)), and you can create and receive Purchase Orders faster than ever, no matter whether the products exist in Retail Express or not.

![mceclip11 (2).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppeEEEAY.jpg)

> **Note:** To access this feature users must have the "Create/Edit a Product/Package" feature enabled for their Security Profile.

Using this feature you can:

- Create an**individual product ****style/group** with multiple sizes and colours in only a few short steps (instead of creating each variant individually)
- **Auto generate Supplier SKU** codes for styles/groups
- Enter the same**Short Description**for all products, or automatically create unique descriptions for products in a style e.g. Short Description + Size + Colour - "Basic tee 10 White"
- **Create new attributes **- Product Type, Size, Colour etc
- **Add quantities **for the new products
