# 2021 Release Overview

The 2021 release cycle brought a variety of updates to Retail Express across core modules like **POS**, **Back Office**, **API integrations**, and **Accounting connectors**. 

These releases focused on improving user experience, enhancing reporting accuracy, and expanding the flexibility of integrations for businesses managing growing inventories and sales volumes.
