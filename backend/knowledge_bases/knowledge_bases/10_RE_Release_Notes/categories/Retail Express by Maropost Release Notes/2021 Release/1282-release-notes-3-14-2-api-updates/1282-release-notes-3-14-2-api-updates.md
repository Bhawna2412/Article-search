# Release Notes 3.14.2 - API Updates

In this update, we've made several changes to the API Integrations with Retail Express.

This release will be rolled out gradually to all users between **1st - 11th November 2021**.

  

---

## Unified API

You can now use the Supplier Returns feature with the Unified API. Using the API you can create and update Supplier Returns, retrieve information about Supplier Returns and associated Return Items and more!

For more information please refer to the [Developer Portal](http://galaxy.maropost.com/home/leaving?allowTrusted=1&target=https%3A%2F%2Fdeveloper.retailexpress.com.au%2Fgetting-started) (information will be available once the update has been rolled out).

## Financial Summaries API

Supplier Credits are now also available to the Financial Summaries API, the API used to return daily summarised financial data.

[Click here](http://galaxy.maropost.com/home/leaving?allowTrusted=1&target=https%3A%2F%2Fv1apidocumentation.retailexpress.com.au%2Fapis%2Fvd0ld51rce) to view more detail.

## Resolutions

* The **eCommerce/Web Services API** has been updated so an error message will now be correctly returned when an order is sent with an invalid Product ID
* Performance and stability improvements to **EDS**
* The **Unified API**was updated to resolve an issue where the Customer Loyalty Information was missing for results after the first page
