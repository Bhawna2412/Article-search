# Release Notes 3.14.3 - API

This release will be available to all users between 29th November and 9th December 2021.

In this release we've made the following updates to the Unified API:

* Queries to the **Customer** method will now correctly display results from the second page onwards including the following data:
  + Customer ID
  + Customer Name
  + Loyalty Program Enabled
  + Loyalty Program Points Available
  + Loyalty Program Points Redeemed
  + Loyalty Program Points to Date
