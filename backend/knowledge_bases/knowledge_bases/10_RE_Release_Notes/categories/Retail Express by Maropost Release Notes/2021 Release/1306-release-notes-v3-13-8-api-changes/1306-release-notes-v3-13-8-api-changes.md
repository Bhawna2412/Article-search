# Release Notes v3.13.8 - API Changes

**Release Date:** 12th July 2021

In this release we've made several changes to the our APIs as part of our commitment to continuous evolution and improvement to Retail Express.

## Inventory Planning API

|  |  |
| --- | --- |
| **Method** | **Details** |
| GetProducts | As [previously advised](https://galaxy.maropost.com/s/article/Release-Notes-3-12-11-Inventory-Planning-AP), the deprecated "DisableProduct" field has now been removed |
| GetStockByOutlet | The correct Global Price is now returned for a product when there is no Outlet-specific price configured |

## Webstore API

* The Allocated quantity is now updated correctly when cancelling a sale through the **Webstore API**
