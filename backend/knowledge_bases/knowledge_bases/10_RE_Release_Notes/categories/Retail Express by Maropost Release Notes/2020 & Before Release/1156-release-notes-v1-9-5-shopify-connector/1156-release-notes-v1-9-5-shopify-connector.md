# Release Notes v1.9.5 - Shopify Connector

**Release Date:** 16th December 2019

In this release we've made an improvement to the Order synchronisation process.

|  |  |
| --- | --- |
| External Order Reference | The External Order Reference in Retail Express has been updated to synchronise from the Shopify "Order Name". It will now include any prefixes and suffixes configured in Shopify instead of order number only e.g. The Order Name of #ORD1001 will be synchronised as ORD1001 (where previously it would have been 1001).  The External Reference field appears in POS, both in the Search feature and within the order itself.  mceclip4 (29).png  The External Reference number can also be viewed from within the Fulfilment Report.  mceclip10 (5).png |
