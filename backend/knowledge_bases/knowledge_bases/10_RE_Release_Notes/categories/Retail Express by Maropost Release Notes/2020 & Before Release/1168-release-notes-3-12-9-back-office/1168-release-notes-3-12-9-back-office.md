# Release Notes 3.12.9 - Back Office

**Scheduled Release Date:** 5-11-2020

In this release we've made some essential changes to several key features for increased functionality and security - Zebra Label Printing and the Xero integration.

In this release:

- [Zebra Label Printing Update](#label-printing)
- [Xero Integration](#xero)

## []()**Zebra Label Printing Update**

As part of the [Adobe Flash end-of-life](https://galaxy.maropost.com/s/article/Adobe-Flash-End-of-Life) updates, Retail Express will be introducing an updated Zebra Label Presets settings page. This page will retain the same great functionality, but with an updated look and feel that's no longer dependent on Adobe Flash.

Previously we advised the designs would be reset - this is no longer the case, and **no impact is expected. **Existing label designs will be updated to be compatible with the new functionality. No changes are required to continue printing existing labels using the new functionality.

In addition to that, we've also added some extra fields available to display on the labels:

- Custom Attributes
- Promotional Pricing
- Promotional Expiry Date
- Season

We have performed an update to the integration to support our integration with Xero OAuth2.0 Security Authentication.

There's no expected downtime or impact for our customers using the Xero integration.
