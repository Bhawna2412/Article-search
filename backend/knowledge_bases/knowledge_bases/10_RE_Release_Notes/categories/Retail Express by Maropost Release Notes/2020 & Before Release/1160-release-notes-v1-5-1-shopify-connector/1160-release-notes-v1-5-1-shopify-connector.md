# Release Notes v1.5.1 - Shopify Connector

**Release Date:** 25/02/2019 **| Version Note:** v1.5.1

With this feature release we have made it even easier to manage your Shopify products directly from Retail Express. With one click of a button you will be able to control Shopify's ability to sell products even when they're out-of-stock.

### **New - Sell out-of-stock Products**

Using the "Prevent Disabling" feature in Retail Express, you can now control the Shopify option "Allow customers to purchase this product when it's out of stock".

![products_preventdisabling.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppfL1EAI.jpg)Prevent Disabling in Retail Express

![shopify_inventory_purchaseproductoutofstock.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppfLBEAY.jpg)Out of Stock sales feature in Shopify

**How it works:**

When the "Prevent Disabling" option is ticked in Retail Express, the system will trigger a synchronisation to Shopify to update the product. The feature "Allow customers to purchase this product when it's out of stock" will become enabled. Shopify will then ignore the Inventory Quantity for that product and allow sales regardless of the stock level.

This feature can be enabled:

- Per product
- For multiple products using the Retail Express "Product Search & Management" feature

Click on the heading below for detailed step-by-step instructions.

#### [Individual products](#individual-product)

To control this feature for individual products:

1. Log into **Retail Express**
2. Open the **Edit Product** window e.g. enter the Product PLU code into the search bar in the top right-hand corner of Back Office
3. Scroll to the **Advanced Options **section
4. Tick the **Prevent Disabling **check box
  ![product_preventdisabling.gif](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppfLQEAY.jpg)
5. Click **Save Changes**

#### [Multiple products](#multiple-products)

To enable the option for multiple products:

1. Log into Retail Express
2. Navigate to **Inventory > Product Search & Management**
3. Use the **Filters **to search for products as required
4. Click **Search**
5. From the **Apply Product Edit Setting **select the Prevent Disabling option
6. From the **Apply To **option select All Search Results

> **Note:** You can also select to apply this setting to Core Products only

7. Click **Apply**
  ![productsearchandmanagement_preventdisabling.gif](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppfKHEAY.jpg)

Once the changes have been applied the page will refresh. Note the "PD" column for the products is now ticked - this is the Prevent Disabling column.

> **Note:** Hover over the column heading for an explanation of the column
