# Release Notes v3.8.0 - API Changes

**Release Date:** 18th September 2019

In this release we have made **2** changes to the below APIs. For detailed information about all API changes, please refer to the [API Updates Release Notes](http://galaxy.maropost.com/home/leaving?allowTrusted=1&target=https%3A%2F%2Fress.zendesk.com%2Fhc%2Fen-us%2Fsections%2F360000149596).

|  |  |
| --- | --- |
| **API** | **Details** |
| Warehouse Management System | In the GetCustomerOrders call, the response returned the fields **Unit Price** and **Tax Rate Applied**. In this release these fields are now populated. |
| Inventory Planning | A small number of users had experienced an issue with duplicate customers in the **GetCustomers **call. This has now been resolved. |
