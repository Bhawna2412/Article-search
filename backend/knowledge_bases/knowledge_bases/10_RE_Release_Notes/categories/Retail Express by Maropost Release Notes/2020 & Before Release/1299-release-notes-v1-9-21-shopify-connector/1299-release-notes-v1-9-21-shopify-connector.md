# Release Notes v1.9.21 - Shopify Connector

# Release Notes v1.9.21 - Shopify Connector

**Release Date:** 7th July 2021

In this release we've made some improvements to resolve issues with synchronising product prices between Retail Express and Shopify.

* Resolved an issue where the "Compare At" price (discounted price) was not removed when the RRP price was less than or equal to the sell price in Retail Express
