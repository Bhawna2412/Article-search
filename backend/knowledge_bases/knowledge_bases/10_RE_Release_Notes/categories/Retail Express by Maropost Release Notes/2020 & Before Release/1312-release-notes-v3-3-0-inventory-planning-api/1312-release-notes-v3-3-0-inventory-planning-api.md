# Release Notes v3.3.0 - Inventory Planning API

**Release Date:** 27-03-19 **| Version Note:** v3.3.0

In this release we have extended the functionality of the Inventory Planning API.

### **Inventory Planning updates**

Using the **SaveProducts **method, on a success response the following fields will be returned for each product created and updated:

- Supplier SKU
- Product ID
