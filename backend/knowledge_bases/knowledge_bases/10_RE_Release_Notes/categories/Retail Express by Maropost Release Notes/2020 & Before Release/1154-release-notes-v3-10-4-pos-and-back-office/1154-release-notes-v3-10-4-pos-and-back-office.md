# Release Notes v3.10.4 - POS and Back Office

**Release Date:** 12th December 2019

In the last release for 2019, we've introduced the ability to create your own Customer Types, and released** 4** resolutions to existing features!

**In this release:**

- [Create new Customer Types](#customer-type)
- [Improvements and Resolutions](#improvements)

### []()**Create new Customer Types**

You can now create your own Customer Types for targeting and reporting on your customers!

![settings_customertype.gif](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppfRTEAY.jpg)
Using the new Customer Types feature you can:

- **Create unique Customer Types **to fit your business, for example Trade customers, Affiliates and more
- **Target your Customer Types** in the **Customer Activity Report** e.g. selectively add Customer Types to Newsletters, award bonus loyalty points and more!
- **Filter reports **by your Customer Types e.g. the Sales Report
  ![mceclip0 - 2023-07-09T184020.645.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppfRYEAY.jpg)

Customer Types can be assigned to existing customers at POS, so your staff can update the information quickly and easily!

![pos_customertype.gif](https://ress.zendesk.com/hc/article_attachments/360002982535/pos_customertype.gif)
The existing system Customer Types will remain (Employee, Retail, Partner etc) but you can add and delete your own unique Customer Types as required!

To access the new Customer Types configuration settings navigate to **Settings > Customer Types**.

### **Resolutions**

Below is a list of resolutions included in the latest Maintenance Release:

|  |  |
| --- | --- |
| Voucher Expiry | An issue has been resolved where in specific circumstances the "Forever" voucher expiry period was unable to be selected. |
| Newsletters | The [[unsubscribe]] placeholder now correctly replaces the [[YourStoreName]] placeholder with Store Name in the Unsubscribe message. |
| Shopify | Retail Express vouchers created through the Shopify Connector integration are now created with the correct voucher expiry date. |
| Sales and Fulfilment Report | An issue affecting some clients when editing a sale from back office reports has been resolved. |
