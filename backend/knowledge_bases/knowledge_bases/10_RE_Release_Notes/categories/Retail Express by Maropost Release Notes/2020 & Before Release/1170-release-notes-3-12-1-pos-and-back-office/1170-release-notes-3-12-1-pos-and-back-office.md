# Release Notes 3.12.1 - POS and Back Office

**Release Date:** 17th June 2020

In this release we've made some improvements to functionality as part of our ongoing commitment to evolve and enhance Retail Express.

**Note:**This release also includes an update to the Warehouse Management System API - refer to the [API Release Notes](https://galaxy.maropost.com/s/article/Release-Notes-3-12-1-Warehouse-Management-System-WMS) for more information.

## Improvements

|  |  |
| --- | --- |
| Outlets | We have added the Outlet/Warehouse ID number to the Outlet maintenance window, making it easier to know which Outlet is being referred to within API integrations.  The Outlet ID can be seen in the Outlets table via **Settings > Locations/Outlets > Outlets**.  mceclip0 - 2023-07-09T162845.755.png |

## Resolutions

|  |  |

| End of Day Report | Restored the ability to use the Sort feature by clicking on the column headings |
