# Account Profile

In your InboxAware control panel navigate to the **Account Settings** page.

As an Account Admin you’ll use this screen to update the name and postal address for your account. You can also set Deliverability Metric Threshold settings, making it easy to spot warning signs with your inbox placement of campaigns and mailbox providers.

* **When Inbox % is Below**: The inbox percentage for a campaign or mailbox provider will appear in red shading when the value is below the threshold you set.
* **When Spam % is Above**: The spam percentage for a campaign or mailbox provider will appear in red shading when the value is above the threshold you set.
* **When Missing % is Above**: The missing percentage for a campaign or mailbox provider will appear in red shading when the value is above the threshold you set.
