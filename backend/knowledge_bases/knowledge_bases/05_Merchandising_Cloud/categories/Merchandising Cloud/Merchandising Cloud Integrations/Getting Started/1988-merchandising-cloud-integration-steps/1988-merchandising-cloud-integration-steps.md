# Merchandising Cloud Integration Steps

You can integrate Merchandising Cloud with your ecommerce platform using platform-specific guides for Shopify, Maropost, Norce, or custom implementations, which helps streamline the setup process and accelerate time to value. Following the appropriate integration path for your platform ensures proper data synchronization and feature compatibility, enabling you to start leveraging AI-powered search and merchandising capabilities faster.

Completing the integration provides access to the Merchant Dashboard where you can configure and customize search, recommendations, and smart collections to match your merchandising strategy. The integration process establishes the foundation for improving product discovery and conversion optimization, with platform-specific tools designed to minimize technical complexity while maintaining flexibility for custom requirements.

## Step 1: Integrate Merchandising Cloud

We provide different tools to help you integrate your shop with Merchandising Cloud. Follow the relevant link to integrate:

- [Shopify](https://developers.findify.io/docs/shopify)
- [Maropost](https://developers.findify.io/docs/integrate-maropost)
- [Norce](https://developers.findify.io/docs/norce-flight)
- [Custom](https://developers.findify.io/docs/integrations-custom-steps)

If your platform is not listed here, you can still integrate using the [Custom](https://developers.findify.io/docs/integrations-custom-steps) integration guide.

## Step 2: Login to Merchant Dashboard

- Once you have installed and integrated Merchandising Cloud you will have access to our Merchant Dashboard
- See further instructions on [Merchant Dashboard](https://developers.findify.io/docs/dashboard-access-and-management).
