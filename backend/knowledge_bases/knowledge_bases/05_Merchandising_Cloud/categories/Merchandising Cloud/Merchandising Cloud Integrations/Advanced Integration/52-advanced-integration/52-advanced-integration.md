# Advanced Integration

Advanced integration techniques enable you to create highly customized search and merchandising experiences tailored to your specific business needs. When you leverage advanced implementation capabilities, you can build differentiated ecommerce experiences that align precisely with your brand and customer expectations, which tends to enhance conversion rates through more sophisticated, business-specific product discovery implementations.

Advanced integration knowledge empowers you to maximize the value of your merchandising platform. This helps you create competitive advantages through custom implementations that solve unique business challenges, supporting better conversion optimization and product discovery outcomes through technical capabilities that extend beyond standard implementations.

There are several way to integrate Maropost Merchandising Cloud (formerly Findify) with your ecommerce store. Each merchant can choose the best way for them, depending on their needs (in terms of flexibility and control); time constraints; and the development resources available to them.

## Direct API Integration[https://developers.findify.io/docs/advanced-integration-overview#direct-api-integration](https://developers.findify.io/docs/advanced-integration-overview#direct-api-integration)

API integration will give you the maximum flexibility and control over how Merchandising Cloud is integrated with your store.

This is usually the most time consuming way to integrate Merchandising Cloud, however it is the method that will provide the most flexibility and speed. You can integrate only the resources that you use and customize them (or use your existing UI) as you wish.

The downside of this approach is the increased development time and development resources needed for this type of integration.

Approximate development time: **weeks to months**.

## JavaScript SDK[https://developers.findify.io/docs/advanced-integration-overview#javascript-sdk](https://developers.findify.io/docs/advanced-integration-overview#javascript-sdk)

If you have decided to go with the API integration, we have a pre-built SDK for JavaScript language that can make it easier to integrate the API if you are using Node.js on the server or are building your own frontend implementation.

Approximate development time: **weeks to months**.

## React Connectors Library[https://developers.findify.io/docs/advanced-integration-overview#react-connectors-library](https://developers.findify.io/docs/advanced-integration-overview#react-connectors-library)

If you are looking to build your own custom frontend experience using React framework, you can utilize our React Connectors library that creates an abstraction over the JavaScript SDK and makes it even easier to communicate with the API.

Approximate development time: **weeks to months**.

## Merchant.JS[https://developers.findify.io/docs/advanced-integration-overview#merchantjs](https://developers.findify.io/docs/advanced-integration-overview#merchantjs)

Merchant.JS (or MJS) is a pre-built frontend library that is built using React framework. This library includes a pre-built frontend experience for all the components needed to show Merchandising Cloud's Personalized Search (including autocomplete), Smart Collections, and Recommendations solutions

Although MJS is pre-built with a certain UI, we have included a way to customize the look and feel by utilizing our [DevTools Chrome Extension](https://developers.findify.io/docs/findify-react-devtools).

This is the fastest way to integrate with Merchandising Cloud, as you don't need to build the integration from scratch. You can also utilize our professional services to customize MJS to your needs and free up your developers.

Approximate development time: **days to weeks**.
