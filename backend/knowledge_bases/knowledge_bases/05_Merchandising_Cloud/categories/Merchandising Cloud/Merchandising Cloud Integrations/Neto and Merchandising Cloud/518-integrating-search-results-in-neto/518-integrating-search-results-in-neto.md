# Integrating Search Results in Neto

Integrating advanced search results in Neto helps you deliver powerful product discovery capabilities within your ecommerce platform. When Neto stores leverage enhanced search functionality, you help improve product findability and search relevance, which tends to enhance conversion rates through better search experiences that help customers find products more effectively.

Neto search integration enables sophisticated search capabilities tailored to your platform. This helps you provide better product discovery experiences for Neto customers, supporting improved conversion performance through more relevant search results and enhanced filtering options that align with Neto's platform architecture.

## To integrate Merchandising Cloud search results follow these steps:[https://developers.findify.io/docs/maropost-integrating-findify-search-results#to-integrate-findify-search-results-follow-these-steps](https://developers.findify.io/docs/maropost-integrating-findify-search-results#to-integrate-findify-search-results-follow-these-steps)

1. Add Merchandising Cloud script
2. Create Search Results Page
3. Remove the default Neto search

### Step 1: Add script[https://developers.findify.io/docs/maropost-integrating-findify-search-results#step-1-add-findify-script](https://developers.findify.io/docs/maropost-integrating-findify-search-results#step-1-add-findify-script)

2. In the Neto admin panel, go to "Settings & tools" → "All settings & tools".

![46ad96e-search_results_1.png](https://us.v-cdn.net/6038474/uploads/C8HUS5MOV49P/46ad96e-search-results-1.png)
2. Scroll down to "Analytics & 3rd Party Scripts" and click on "Custom Scripts".

![7c55499-search_results_2.png](https://us.v-cdn.net/6038474/uploads/83YWCZYZ2L2H/7c55499-search-results-2.png)
3. Click on "Add New" located on the left side of your dashboard.

![6187c7a-search_results_3.png](https://us.v-cdn.net/6038474/uploads/1NFSL80VKHKO/6187c7a-search-results-3.png)
4. Provide a Name for the custom Script such as "Merchandising Cloud Script" and leave all "key" fields empty.

![24b57ff-1.png](https://us.v-cdn.net/6038474/uploads/B7N26I360UJK/24b57ff-1.png)
5. Scroll down and select 'Page Header' as the place where the script will be inserted.

![8446af9-search_results_5.png](https://us.v-cdn.net/6038474/uploads/5CPRZR709MDC/8446af9-search-results-5.png)
6. Insert the following script in the code editor, ex.

```plaintext
<script src="//assets.findify.io/your_store_name.min.js" async defer></script>
```

### Step 3: Remove the default Neto search[https://developers.findify.io/docs/maropost-integrating-findify-search-results#step-3-remove-the-default-maropost-commerce-cloud-search](https://developers.findify.io/docs/maropost-integrating-findify-search-results#step-3-remove-the-default-maropost-commerce-cloud-search)

In order to remove the default search from your Neto store, please remove the class `<span class="cm-s-neo" style="box-sizing: border-box;">ajax_search</span>` from your search input field. The template should be found within 'headers' → ' template.html'.

So, for example, your search input field could look like this:

```plaintext
<span class="cm-s-neo" style="box-sizing: border-box;"><input class="form-control ajax_search" value="" id="name_search" autocomplete="off" type="search" aria-label="Input search"></span>
```

And after removing `<span class="cm-s-neo" style="box-sizing: border-box;">ajax_search</span>` it would look like this:

```plaintext
<span class="cm-s-neo" style="box-sizing: border-box;"><input class="form-control" value="" id="name_search" autocomplete="off" type="search" aria-label="Input search"></span>
```

After this setup, please, proceed on [integrating the analytics tracking](https://developers.findify.io/docs/maropost-integrating-findify-analytics).
