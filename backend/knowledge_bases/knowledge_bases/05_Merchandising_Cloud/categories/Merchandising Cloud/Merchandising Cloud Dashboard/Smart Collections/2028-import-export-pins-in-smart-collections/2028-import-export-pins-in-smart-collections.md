# Import / Export Pins in Smart Collections

You can import and export product pins across Smart Collections using CSV files, enabling efficient management of strategically positioned products at scale. This bulk operation capability helps save time when applying consistent merchandising strategies across multiple collections or when migrating pin configurations, supporting efficient collection management while maintaining control over featured product placement that drives conversion.

Managing pins through import/export helps streamline merchandising operations for stores with many collections. You can quickly replicate successful pin strategies across similar collections, backup pin configurations before making changes, or apply seasonal merchandising updates en masse-enabling data-driven collection optimization while reducing manual effort required to maintain strategic product positioning across your catalog.

## What is Pinning in Smart Collections?

Pinning allows merchants to customize the display order of specific products in their Smart Collections by "pinning" them to a desired position, ensuring they appear prominently at the top of the collection. This feature is particularly useful for promoting certain products without altering other dynamic sorting or personalization settings.

The Import/Export Pins feature enables merchants to quickly and easily make bulk changes to these pinned products, providing a more efficient workflow for managing large collections and/or multiple collections.

## How to Use Bulk Import/Export for Pins

> **Note:** This feature is only available only for select merchants. Contact us if you are interested in enabling this feature.

### Export Pins

1. Go to **Smart Collections > Pinning** within the Dashboard.
2. Click **Export Pins** to download a CSV file containing the current pin configurations.

![d8b184f701d69be25673f1d7d96e2497373ca270cfcf3b67cca52444ac9a0065-Export_Pins.png](https://us.v-cdn.net/6038474/uploads/N5PAI2FYJ5EJ/d8b184f701d69be25673f1d7d96e2497373ca270cfcf3b67cca52444ac9a0065-export-pins.png)

### Edit the Exported CSV

1. Open the exported CSV file in a spreadsheet application (e.g. Google Sheets).
2. Adjust the pin configurations as needed.
3. Save your changes and export the file in CSV format.

### Import Pins

1. Within the Dashboard, return to **Smart Collections > Pinning**.
2. Click **Import Pins** and select your updated CSV file.
3. Once uploaded, the changes will then be applied automatically within the dashboard.

## Troubleshooting and FAQ

### Q: My CSV generates a General Error. What is the issue?

### Q: I have trouble exporting in the *CSV (Comma Delimited)* format. What method do you recommend?

*A: We recommend using Google Sheets. It is free, easy to use, and very easy to import/export CSV's in the correct format.*

### Q: Can I import and export pins in both automatic and manual collections?

*A: Yes*, this functionality works the same for both automatic and manual collections.

### Q: What is the maximum amount of pins that can be imported?

*A: There is no limit imposed, but for optimal performance we recommend batches of maximum 1,000 pins at a time.*

### Q: Can I delete pins in bulk?

*A: Yes, you can delete pins in bulk on the pinning page*
