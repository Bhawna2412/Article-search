# Seamless Collections (Shopify)

You can manage all your Merchandising Cloud collections directly from the Shopify dashboard through Seamless Collections, which automatically syncs changes instantly without manual intervention. This streamlined workflow eliminates the need to manage collections in two separate systems, helping you save time and reduce errors while maintaining dynamic, AI-powered collection pages that adapt to customer preferences for improved conversion.

Seamless Collections integration helps simplify collection management for Shopify merchants. You can create or modify automated collections in Shopify using standard rules and tags, and Merchandising Cloud automatically syncs them in real-time-enabling you to leverage familiar Shopify workflows while benefiting from AI-powered personalized ranking and unlimited filtering that typically helps improve collection page performance and customer engagement.

## What Are Smart Collections?

Smart Collections by Maropost Merchandising Cloud (formerly Findify) covers a selection of products displayed with a static URL, mainly for browsing purposes.

Most common are divisions based on categories or brands, but any field in the database can be used to set up collections.

The dynamic abilities ensures they are constantly up to date without any manual work required.
*e.g. If a collection is set up to include all products where* **Brand = Findify**, *new products fulfilling the criteria will automatically be added as soon as they enter the catalogue.*

Using our Smart Collections feature enables the full suite of features such as **Merchandising Suite**, **Personalization**, **Dynamic Filters & Sorting**, **Promo Cards** etc.[Read More](https://developers.findify.io/docs/smart-collections)

## How does Seamless Shopify Collections Work?

Our Seamless Shopify Collections feature improves the way Shopify merchants work with their Merchandising Cloud collections. Merchants can manage all their Merchandising Cloud collections directly from the Shopify dashboard - making the process quicker and easier than ever before. ## Requirements

In order to import collection settings from Shopify, there needs to be settings to be imported. I.e. the collection needs to be setup as Automated (we are only able to pull Automated collections that are based on the regular product data, but not metafields) and not Manual (where individual ID's are selected). 95% of all merchants collections are in general Automatic given that they stay up to date and are a lot quicker to configure.
