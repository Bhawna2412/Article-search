# Seamless Collections (BigCommerce)

You can enable automatic collection synchronization for BigCommerce, allowing Merchandising Cloud to sync your category configurations automatically without manual setup. This seamless integration helps streamline collection management by automatically importing and updating collections from your BigCommerce platform, enabling you to maintain AI-powered personalized collections without duplicating configuration work across systems.

Implementing Seamless Collections for BigCommerce helps reduce operational overhead while enhancing collection performance. The automatic synchronization ensures your collections stay up-to-date with platform changes, while AI-powered ranking and filtering capabilities improve customer experience and conversion rates-enabling you to focus on merchandising strategy rather than technical synchronization while benefiting from dynamic, behavior-driven product positioning.

## What Are Smart Collections?

Smart Collections by Maropost Merchandising Cloud (formerly Findify) covers a selection of products displayed with a static URL, mainly for browsing purposes.

Most common are divisions based on categories or brands, but any field in the database can be used to set up collections.

The dynamic abilities ensures they are constantly up to date without any manual work required.
*e.g. If a collection is set up to include all products where* **Brand = Findify**, *new products fulfilling the criteria will automatically be added as soon as they enter the catalogue.*

Using our Smart Collections feature enables the full suite of features such as **Merchandising Suite**, **Personalization**, **Dynamic Filters & Sorting**, **Promo Cards** etc.[Read More](https://developers.findify.io/docs/smart-collections)

## How does Seamless BigCommerce Collections Work?

This feature works in a similar way to the [Seamless Shopify Collections capability](https://developers.findify.io/docs/bigcommerce-seamless-collections).

We will automatically sync any changes made to BigCommerce collections (that are based on categories and brands) within a few minutes of the change being made. This takes place automatically - without the need to do any extra configuration in the Merchant Dashboard.
