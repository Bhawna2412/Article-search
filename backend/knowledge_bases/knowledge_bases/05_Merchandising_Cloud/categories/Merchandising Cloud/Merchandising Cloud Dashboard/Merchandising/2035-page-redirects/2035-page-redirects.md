# Page Redirects

You can configure search query redirects to send customers directly to specific pages when they search for certain terms, helping guide them to landing pages, category pages, or informational content more effectively than search results.

This capability enables you to optimize the customer journey for specific queries like brand names, campaign keywords, or informational searches-potentially improving conversion by directing customers to curated experiences designed for specific intents rather than generic search results.

Implementing query redirects helps optimize search outcomes for strategic terms. You can redirect promotional searches to campaign landing pages, brand searches to brand pages, or informational queries to help content-enabling you to control the experience for high-value search terms while allowing AI-powered search to handle product discovery queries, supporting both strategic marketing objectives and the personalized search experiences that drive general product discovery and conversion.

## When to use Page Redirects

![0.jpg](https://us.v-cdn.net/6038474/uploads/I1UHZRBRBA1T/60JDGAD999J6.jpg)
There can be other applications for page redirects, you can technically redirect any search query to any page.

There can be multiple additional applications for this feature:

- Redirect search queries to dedicated category\collection pages. If you have a dedicated *men's* collection, then when someone searches for *men* or *men's*, you can redirect the user to that page as it has more relevant content (e.g. more images and descriptions) that can be useful for the user
- Redirect zero result searches. In some cases, it might be useful to redirect zero result searches to another page of your store suggesting to look at something else available.

## How to Set Up a Page Redirect

1. Go to the [Merchant Dashboard](https://dashboard.findify.io/solutions/search/redirects)
2. Define the search terms - you can define multiple search terms.
3. Define the URL they will be re-directed to.
4. Click 'Save'
