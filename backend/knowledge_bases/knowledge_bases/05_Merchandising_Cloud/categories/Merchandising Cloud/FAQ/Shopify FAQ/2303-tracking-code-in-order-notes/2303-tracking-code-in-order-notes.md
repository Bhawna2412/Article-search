# Tracking code in order notes

**Merchandising Cloud adds tracking code to Shopify orders notes, that may look something like this:**
_{"uniq_id":"gKpWKl6VQhULXIZF","visit_id":"L7HzjOyO2BHStW2g"}_

**We do this for a number of reasons:**

- It helps our search engine to correctly calculate and dynamically update products' popularity and as a result, produce more relevant search results for customers.
- We use this data for our analytics platform that allows you to check different metrics in [Merchandising Cloud dashboard](https://dashboard.findify.io/analytics/snapshot).
- We use this data for our personalization platform that allows us to produce relevant search results and suggestions based on current customer actions.

> 📘 Remove Merchandising Cloud tracking code from Shopify order notes
> This tracking code can be removed but will affect the granularity of your Analytics and is hence advised to be kept. 
> If you don't want Merchandising Cloud to insert tracking code into your Shopify orders you can contact us at [support@maropost .com](mailto:support@maropost.com) and our team will disable this feature.
