# SKU Capacity

# How many SKUs can Merchandising Cloud handle?

To data, Merchandising Cloud has load tested with more than 3 million individual SKUs. These tests were successful and did not result in any issues.

Currently, Merchandising Cloud works without issue with clients running up to almost 1 million SKUs.

> 👍 In the future, Merchandising Cloud may decide to load test with larger SKU numbers, should the need arise. If and when that happens, this page will be updated accordingly.
