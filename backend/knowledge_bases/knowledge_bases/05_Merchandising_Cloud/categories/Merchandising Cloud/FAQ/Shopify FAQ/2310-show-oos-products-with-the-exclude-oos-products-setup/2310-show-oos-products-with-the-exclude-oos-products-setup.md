# Show OOS products with the exclude OOS products setup

If you have a setup in our dashboard to exclude OOS products from Merchandising Cloud widgets:

![fa8efc6-Shopify-findify-include-oos-products-if-oos-setup.png](https://us.v-cdn.net/6038474/uploads/ZD5C5U1Z72DE/fa8efc6-shopify-findify-include-oos-products-if-oos-setup.png)
And you still want to show OOS products as if they are in-stock, you have 2 options:

## 1. Use **Continue Selling when OOS** option on the variant level in Shopify.

If the variant has this setup, Merchandising Cloud will read the variant as available and will still show it in our results.

![Image](https://files.readme.io/fd16ae2-Shopify-findify-include-oos-products-if-oos-setup-continue-selling.png)

## 2. Use findify-live:true tag.

If the setup above is non-applicaple, you can setup a product tag: **findify-live:true** on the products that you still want to show even if OOS. Once the tag is in place, please contact our support [support@maropost .com](mailto:support@maropost.com) to adjust the setup internally.

![Image](https://files.readme.io/23456d6-Shopify-findify-include-oos-products-if-oos-setup-tag.png)
