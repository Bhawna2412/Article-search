# Framework

You can integrate Merchandising Cloud with your Shopify store using the native Liquid-based app, which streamlines implementation without requiring React developers or custom development work. This Shopify-native approach enables faster deployment and makes it easier for any Shopify-familiar merchant or developer to customize the search and discovery experience.

The Liquid framework integration represents a simpler, more accessible approach to implementing Merchandising Cloud on Shopify stores. By using Shopify's own language rather than requiring React expertise, you can get search, collections, and recommendations running quickly with pre-designed layouts, then customize them using standard Shopify development practices that your team likely already knows.

## Historical Context

In the past, our integration with Shopify merchants primarily relied on MJS React for the front end. While this approach delivered excellent results in terms of appearance and functionality, the integration process itself wasn't as streamlined as it could have been. To connect with Maropost Merchandising Cloud (formerly Findify) through MJS React, Shopify merchants had three options: (a) employ in-house developers skilled in the React framework, (b) hire a web agency with React-savvy personnel, or (c) engage our developers to execute the integration on their behalf.

As for the latter - while we initially provided a customization service, this has become increasingly impractical as Merchandising Cloud has expanded. I'm sure you would agree that our developers need to continue their focus on pioneering new innovations for you, the client, rather than tailoring solutions for individual stores!

## Introduction of a Game-Changer

This situation prompted us to innovate, resulting in the development of a native Shopify app created using Shopify's own language - Liquid, optimized for Shopify 2.0. Now, ecommerce merchants on the Shopify platform need only to find Maropost Merchandising Cloud (formerly Findify) in the Shopify app store and follow straightforward on-screen integration instructions, which encompass:

- Creating a Merchandising Cloud account and linking your store.
- Synchronizing your product data.
- Confirming your email address.
- Installing Merchandising Cloud into your chosen Shopify theme.

In sum, you can have Merchandising Cloud up and running on your store with pre-designed layouts in just a matter of minutes. After integration, it's a breeze to use Shopify's native app blocks to position Merchandising Cloud components precisely where you want them on your store. Any merchant familiar with Shopify can do this effortlessly.

For those who desire customizations, our components can be easily modified by any developer well-versed in Shopify Liquid.

Thanks to this new native Shopify app integration, integrating Merchandising Cloud has never been more straightforward for Shopify merchants.
