# Configure Color Swatches in Shopify Editor

You can configure color swatches directly within the Shopify editor without needing to edit code, making it easy to customize how color options display on your product cards. 

This simplified configuration approach enables merchants to adjust swatch visibility, style, and behavior settings through a visual interface, streamlining the setup process.

Setting up color swatches through the Shopify editor provides flexible control over the customer experience, allowing you to choose between different display styles like sliders or "+x more" formats for both mobile and desktop. 

By enabling the appropriate fields in your Merchant Dashboard and then configuring display preferences in the Shopify editor, you can create an intuitive color selection experience that helps improve product discovery and customer engagement.

Color swatches improve your Shopify store by giving customers a visually intuitive way to browse and select products based on color. This guide walks you through configuring Merchandising Cloud-powered color swatches directly within the **Shopify editor**.
