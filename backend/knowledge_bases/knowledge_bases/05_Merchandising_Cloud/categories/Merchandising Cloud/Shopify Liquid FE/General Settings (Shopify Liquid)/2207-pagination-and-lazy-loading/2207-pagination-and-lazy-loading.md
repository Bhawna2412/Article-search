# Pagination and Lazy Loading

You can configure pagination and lazy loading options to control how customers navigate through search results and collections. These settings affect page load performance and browsing experience, with lazy loading providing seamless infinite scroll while traditional pagination offers discrete page navigation.

Choosing between pagination styles impacts both technical performance and user experience. Lazy loading creates a continuous browsing experience that can help improve engagement by eliminating pagination clicks, while traditional pagination provides clearer navigation points and may perform better for certain use cases. Selecting the appropriate approach helps optimize the browsing experience for your specific catalog size and customer preferences.

This guide provides detailed instructions on how to implement and configure pagination and lazy loading for our search and collection pages in your Shopify store using the Liquid language.

## Default Pagination

By default, our search and collection pages display results using pagination. You can customize this behavior through the Shopify Theme Editor.
