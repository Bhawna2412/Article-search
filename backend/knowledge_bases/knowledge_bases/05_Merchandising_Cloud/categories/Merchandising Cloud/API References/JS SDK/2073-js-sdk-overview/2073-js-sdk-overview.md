# JS SDK Overview

## Overview

The JavaScript SDK provides a high-level wrapper around the low-level JSON API, simplifying integration by handling request formatting, authentication, and response parsing. This enables developers to implement search, recommendations, and other features more efficiently using familiar JavaScript patterns rather than managing raw API calls.

Using the SDK helps streamline development by providing typed interfaces and helper methods that reduce implementation complexity while ensuring correct API usage. This tends to accelerate integration timelines and reduce errors compared to working directly with the JSON API.

Maropost Merchandising Cloud (formerly Findify) JS SDK is a JavaScript wrapper of the low-level JSON API.
