# Github: JS SDK

The JavaScript SDK is available on GitHub with complete source code, documentation, and examples for implementing search and personalization features. This open-source availability enables developers to review implementation details, contribute improvements, and stay current with SDK updates while integrating platform capabilities.

Accessing the GitHub repository helps ensure you can understand SDK internals, review best practices, and implement features with confidence by referencing official examples and documentation maintained alongside the codebase.
