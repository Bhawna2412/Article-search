# Analytics JS Overview

The Analytics JS library enables you to integrate the Feedback API and gather user behavior data that powers machine learning and personalization features. This helps ensure the AI recommendation engine and personalization capabilities have the behavioral data needed to deliver relevant product suggestions and search results tailored to individual customer patterns.

Implementing Analytics JS involves initializing the library with your API key and configuring event tracking for customer interactions like clicks, purchases, and page views. This data collection helps optimize search relevance, recommendation quality, and overall personalization effectiveness across your ecommerce experience.

Maropost Merchadising Cloud (formerly Findify) Analytics JS helps you integrate our Feedback API on the website and gather all the user behavior analytics that our ML and personalization require.

This is documentation for Analytics JS >= 3.x. If you are looking for the old version of Analytics JS (< 3.x) it can be found here:

- [v2.x on github](https://github.com/findify/analytics-js)
- [v2.x on npm](https://www.npmjs.com/package/findify-analytics)
- [v1.x on github](https://github.com/findify/analytics-js/tree/v1.1.43)
