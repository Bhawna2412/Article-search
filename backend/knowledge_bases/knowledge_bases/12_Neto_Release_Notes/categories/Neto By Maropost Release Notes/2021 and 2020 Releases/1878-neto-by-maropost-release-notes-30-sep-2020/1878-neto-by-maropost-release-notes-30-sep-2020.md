# Neto by Maropost Release Notes – 30 Sep 2020

**Release Version**: 6.122

## **Minor Improvements**

- Added a fallback email address field for Fastway shipping labels. If a shipped order doesn't include an email address, the fallback is used instead.

## **Bug Fixes**

- Fixed an issue where a lower-case country code on a sales order caused the tax rate to be 0.
- Fixed an issue in the eBay integration where weight in pounds was converted to kilograms when listing on the eBay Motors US site.
