# Neto by Maropost Release Notes – 27 Jan 2021

**Release Version**: 6.136

The latest version, 6.136, is rolling out soon to all users. The big news in this release is multi-factor authentication for staff user accounts. 

MFA is optional per user and requires a smartphone app (such as Google Authenticator) to verify a user's credentials.

More information about MFA can be found in our support hub article here: [https://galaxy.maropost.com/kb/articles/1016-multi-factor-authentication-mfa](https://galaxy.maropost.com/kb/articles/1016-multi-factor-authentication-mfa)
