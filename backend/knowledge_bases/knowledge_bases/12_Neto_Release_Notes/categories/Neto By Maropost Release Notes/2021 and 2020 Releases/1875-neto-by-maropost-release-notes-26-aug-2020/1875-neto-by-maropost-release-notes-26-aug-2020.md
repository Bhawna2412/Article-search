# Neto by Maropost Release Notes – 26 Aug 2020

**Release Version**: 6.117

## **Minor Improvements**

- Amazon and Catch category mapping will now display notifications if categories have been updated and need to be remapped in Neto.
- 
- Updated the text on shipping rate tables to explain that transit cover doesn't change the rate charged to customers. The transit cover will still appear during consignment creation if set up.
- 

## **Bug Fixes**

- Fixed an issue where cubic wasn't calculated correctly for multiple items when creating consignments.
