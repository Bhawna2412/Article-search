# Neto by Maropost Release Notes – 19 Jan 2021

**Release Version**: 6.135

## **Minor Changes**

- Hunter Express shipping labels have been updated to the latest version.
- The Shipping Date Groups link has been added to the All settings & tools page. Date groups let you set specific days when orders aren't picked up or shipped, such as public or local holidays.
