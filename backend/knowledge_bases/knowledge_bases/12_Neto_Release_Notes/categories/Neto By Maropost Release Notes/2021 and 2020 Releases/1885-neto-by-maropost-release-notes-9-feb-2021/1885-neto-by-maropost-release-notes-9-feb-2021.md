# Neto by Maropost Release Notes – 9 Feb 2021

**Release Version**: 6.138

## **Minor Changes**

- Klayvio for Neto (by Keetrax) add-on has launched, and is available to install from the **Addons** menu in your control panel.

## **Bug Fixes**

- Fixed an issue with the eBay integration where orders were marked as **On Hold** with payment mapping messages, despite being mapped correctly.
