# Neto by Maropost Release Notes – 29 Jul 2020

**Release Version:** 6.113

## **Minor Improvements**

- Neto's Xero integration has been updated to allow more control over Cost of Goods Sold. COGS journals can now be batched by date invoiced or date completed.

## **Bug Fixes**

- Fixed an issue in Neto POS where image thumbnails weren't updated after being changed in the control panel.
- Fixed an issue with the eBay integration where automated revisions caused a retail price invalid error when the RRP field is blank.
