# Neto by Maropost Release Notes – 22 Sep 2020

**Release Version**: 6.121

## **Bug Fixes**

- Fixed an issue in sales orders where the suburb selected wouldn't load if there were too many discounts active.
- 
- Fixed an issue that prevented vouchers from being used to pay for orders in the control panel if a Stripe payment method was set as the default.
- Fixed an issue that caused an error when importing images from the imagedrop folder.
