# Neto by Maropost Release Notes – 23 Jun 2020

**Release Version**: v6.108

### **Minor Improvements**

- When making changes to a sales order and then changing the status, you will now be notified that the changes will be lost. You'll then have the opportunity to go back and save the changes before changing the status.

### **Bug Fixes**
