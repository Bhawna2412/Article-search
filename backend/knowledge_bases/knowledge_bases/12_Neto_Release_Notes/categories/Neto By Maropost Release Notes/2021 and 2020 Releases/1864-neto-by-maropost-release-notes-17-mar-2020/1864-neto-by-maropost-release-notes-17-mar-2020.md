# Neto by Maropost Release Notes – 17 Mar 2020

**Release Version**: v6.94

### **Bug Fixes**

- Fixed an issue with the eBay integration that didn't update the weight of products correctly on eBay, when the product weight changed.
- Fixed an issue when editing a sales order price in Safari where the price groups wouldn't display.
- Fixed an issue with editable kits in sales orders, where components couldn't be added to an orderline.
