# Neto by Maropost Release Notes - 7 Jul 2021

## OVERVIEW

Point of Sale version 1.25 is rolling out soon. For iOS users, there will be an app update in the coming week. Also, rolling out this week are updates from the eCommerce version 6.159.

## **Minor Updates**

- 
- When a barcode is scanned that does not match a product a beep noise will now be made. Browser users need to allow POS to autoplay audio, consult your browser's settings or help site for instructions.
- The Crossbeam Xero integration has launched and is available to install. More information [here](https://www.netohq.com/support/s/article/crossbeam).
- MYOB by Channelup add-on has launched and is available to install. More information [here](https://www.netohq.com/support/s/article/myob-accountright-channelup).

## **Bug Fixes**

- Fixed an issue where a notification would always appear prompting merchants to update their iOS version on iOS 13+.
- Fixed an issue in Xero where credits and RMAs would not export with an error saying "The document total does not equal the sum of the lines.".
