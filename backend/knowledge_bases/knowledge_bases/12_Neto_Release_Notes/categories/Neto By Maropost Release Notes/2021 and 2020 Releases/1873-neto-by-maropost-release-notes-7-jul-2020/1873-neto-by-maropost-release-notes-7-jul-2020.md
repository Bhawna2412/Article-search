# Neto by Maropost Release Notes – 7 Jul 2020

**Release Version**: 6.110

## **Bug Fixes**

- Fixed an issue in the Amazon integration where eParcel tracking numbers weren't sent to Amazon if they were updated through the Neto API.
- Fixed an issue in webstores where creating a customer account didn't show any password validation errors.
- Fixed an issue where you couldn't log into the control panel in a second browser, and end the original session.
