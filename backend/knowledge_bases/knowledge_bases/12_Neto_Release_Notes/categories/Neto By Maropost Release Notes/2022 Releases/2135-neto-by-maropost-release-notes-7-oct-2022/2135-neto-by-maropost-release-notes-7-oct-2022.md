# Neto by Maropost Release Notes – 7 Oct 2022

## **Overview**

 In this release, we are implementing some new options for the [PayPal Payment Method](https://galaxy.maropost.com/s/article/paypal-payment-method).

Read on for more information!

## **New Feature**

### **PayPal Checkout**

![PayPal_Logo](https://us.v-cdn.net/6038474/uploads/attachments/migrated/PayPalLogo.0685g000009vX6YAAU.png)This new method can be used by customers to pay for an order from their My Account page on the webstore. Eligible customers will find this option available under the **Select your payment method** dropdown on the **Pay Now** page.

This new method can be used by merchants to pay for an order from the control panel including on the Receipt Payments page.

- Ability to set up multiple instances of the payment method, allowing for differing Configuration options based on user group, min/max spend, etc.
- Ability to Set the colour of the PayPal Checkout button that appears on the Maropost Checkout. Choose from Gold, Blue, Black, White or Silver.
- Ability to choose whether to display a separate PayPal “Pay in 4” button on the checkout (Note: this dependant on Pay in 4 being enabled on your PayPal account).
- Ability to set your own display name, which will be used on the checkout.

Successful setup of this method will automatically disable any existing PayPal Express or PayPal Standard payment methods you have active, meaning you don’t have to do this manually.

### **PayPal Advanced Credit and Debit Cards**

![PayPal_ACaDC](https://us.v-cdn.net/6038474/uploads/attachments/migrated/PayPalACaDC.0685g000009vX6iAAE.png)This new method can be used by customers to pay for an order from their **My Account** page on the webstore. Eligible customers will find this option available under the **Select your payment method** dropdown on the **Pay Now** page.

This new method can be used by merchants to pay for an order from the control panel including on the Receipt Payments page. 

- Ability to set up multiple instances of the payment method, allowing for differing Configuration options based on user group, min/max spend, etc.
- Ability to set your own display name, which will be used on the checkout.

### **Venmo via PayPal (US Stores Only)**

Venmo through PayPal will be available for merchants with US-based stores.
