# Neto by Maropost Release Notes – 1 Oct 2019

**Release Version: **v6.72

### **Minor Features**

 **Amazon Australia**

The Amazon Australia integration has been updated to allow listing products that are fulfilled by Amazon. 

You will need to map the fulfilment centre ID field in the field mapping, to a product field that contains the ID. Leave the ID field blank for products that are not fulfilled by Amazon.

 Stock levels will not be sent when the fulfilment centre ID is populated on a listing, these still need to be loaded through seller central.
