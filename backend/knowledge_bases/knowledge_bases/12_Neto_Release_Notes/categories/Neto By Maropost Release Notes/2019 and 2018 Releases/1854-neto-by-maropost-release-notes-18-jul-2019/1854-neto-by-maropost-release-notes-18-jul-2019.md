# Neto by Maropost Release Notes – 18 Jul 2019

**Release Version**: 6.61

## **Minor Features**

- New Australia Post charge codes have been added for the eParcel API.

## **Bug Fixes**

- 
- 
- Fixed a bug where custom credit was incorrectly calculated when quotes, cancelled orders, or unapproved orders existed.
- Fixed an issue in Catch descriptions that wasn't removing <style> tags in HTML.
- Corrected the Aged Receivables report to show the correct data when **Show Zero Balances** is set to No.
- 
- Fixed an issue in the GetWarehouse API call so it no longer returns a blank for the OutputSelector WarehouseState.
