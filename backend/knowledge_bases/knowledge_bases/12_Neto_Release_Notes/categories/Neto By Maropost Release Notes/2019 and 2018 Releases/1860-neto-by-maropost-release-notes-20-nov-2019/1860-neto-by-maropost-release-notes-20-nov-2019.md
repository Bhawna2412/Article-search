# Neto by Maropost Release Notes – 20 Nov 2019

**Release Version**: v6.79

### **Minor Features**

The OpenFreight carrier configuration has new fields that can be used during consignment creation:

- Book Pickup Now: Yes/No

- Pick Timezone: Country/City

- Pickup Ready: Time

- Pickup Closing: Time

 These fields are optional, and if used will be transmitted to OpenFreight when the manifest is submitted. They can be set up in the **Shipping** > **Other shipping setup** > **Carriers & Labels**.
