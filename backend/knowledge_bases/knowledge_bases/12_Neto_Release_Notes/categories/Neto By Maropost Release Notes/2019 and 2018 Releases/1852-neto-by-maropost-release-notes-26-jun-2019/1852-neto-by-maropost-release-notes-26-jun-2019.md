# Neto by Maropost Release Notes – 26 Jun 2019

**Release Version**: 6.58

## **Minor Features**

- The **Refresh Caches** button in your control panel will now load new CSS and JS assets, effectively clearing their cache as well.

## **Bug Fixes**

- Fixed an issue that prevented users from logging into the POS.
- Resolved sales order payments that showed an **Uninitialised value** error so that this no longer occurs.
- Fixed a problem where Neto would try to update the payment status of orders that had been marked paid in eBay.
