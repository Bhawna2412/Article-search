# Neto by Maropost Release Notes – 19 Jun 2019

**Release Version**: 6.57

## **Minor Features**

- Catch sellers can now list products without a barcode through Neto. Instead, the product SKU will be used as the listing's MPN. Note that Catch still reviews the data for each product registered to ensure integrity.
- A new setting has been introduced to turn the newsletter subscription checkbox on the checkout page to "opt out", rather than opt in. You can find this in the **Webstore **> **Checkout Settings **page.
