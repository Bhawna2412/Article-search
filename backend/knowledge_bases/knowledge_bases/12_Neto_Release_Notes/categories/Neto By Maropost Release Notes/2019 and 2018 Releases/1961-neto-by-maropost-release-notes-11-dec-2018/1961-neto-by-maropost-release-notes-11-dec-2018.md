# Neto by Maropost Release Notes – 11 Dec 2018

**Release Version: **6.33

### **Amazon AU**

**Carrier mapping**

You can now specify which shipping carrier you used to send an Amazon order. This is set on the shipping rate table, much like eBay carrier mapping. A step by step guide is available in our†support hub.

**Origin country**

This field has been added to the product field mapping, for product registration. Instructions can be found in our†support hub.
