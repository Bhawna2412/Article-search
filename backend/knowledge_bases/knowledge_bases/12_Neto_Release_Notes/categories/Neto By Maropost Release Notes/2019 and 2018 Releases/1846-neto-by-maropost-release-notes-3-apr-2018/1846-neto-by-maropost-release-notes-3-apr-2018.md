# Neto by Maropost Release Notes –  3 Apr 2018

**Release Version**: 6.14

We're excited to announce that Neto version 6.14 has begun rollout to all of our customer's sites. Some will already be on 6.14, others will be updated this week.

This release brings some new features to our eWay and eBay integrations, as well as the usual minor changes and bug fixes.

We're planning a webinar to showcase the new eBay features, so stay tuned to for more details and to register your spot.
