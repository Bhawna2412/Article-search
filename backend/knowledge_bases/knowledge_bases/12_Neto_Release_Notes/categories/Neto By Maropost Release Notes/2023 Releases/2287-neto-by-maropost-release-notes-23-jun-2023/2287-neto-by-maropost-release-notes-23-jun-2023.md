# Neto By Maropost Release Notes – 23 Jun 2023

## **Overview**

This release notes covers the eCommerce releases **6.252.0**, **6.253**, and **6.254**.

## **Enhancements**

### **Variation (Child) Products Visible on Sitemap**

If you have set up your web store to show variation/child products as separate URLs, these variation products will now be included in the sitemap. Earlier, only standalone and parent products were included in the sitemap.

To learn more about sitemaps, see the ‘Set up XML sitemaps For Google’ section in [**Google Search Console Setup**](https://galaxy.maropost.com/s/article/Google-Search-Console-Setup). And to learn about variation (child) products, see [**Add Variation Products**](https://galaxy.maropost.com/s/article/add-variation-products).

### **Continued Improvements for PayPal Cards**

Continued improvements for PayPal Cards around failed PayPal authorisation not releasing pre-authorisation amounts when transactions could not be completed.

### **Optimized Addon Scripts for Google Analytics 4 and PayPal - Pay In 4**

The addon scripts for Google Analytics 4 and PayPal - Pay In 4 have been optimized. The Google Analytics 4 script changes will help improve reporting the odd data. The PayPal - Pay In 4 script changes will help improve your web store performance.

To learn about Google Analytics 4 addon, see [**Google Analytics 4**](https://galaxy.maropost.com/s/article/google-analytics-4) and to learn about PayPal - Pay In 4 addon, see [**Add PayPal Pay in 4 to your Webstore**](https://galaxy.maropost.com/s/article/add-paypal-pay-in-4-to-your-webstore).

### **Updated Hunter Express Postcodes and Zones**

The Hunter Express postcodes and zones were updated to reflect the newest changes for June.

The [**Stamped.io**](https://galaxy.maropost.com/home/leaving?allowTrusted=1&target=http%3A%2F%2Fstamped.io%2F) addon is no longer supported.
