# Neto by Maropost Release Notes – 1 Dec 2023

## **Overview**

This release includes eCommerce releases **6.275.0** and **6.276.0**.

## **Enhancements and Updates**

### **Ability to Fetch Multilevel Pricing Data Through API**

Following up on one of the top ideas on the [**Idea Portal**](https://galaxy.maropost.com/s/ideas?category=Commerce+Cloud), we are happy to introduce the ability to fetch multilevel pricing information for the products through the API.

A new OutputSelector PriceGroups.MultilevelBands has been added to the [**getItem**](https://galaxy.maropost.com/home/leaving?allowTrusted=1&target=https%3A%2F%2Fdevelopers.maropost.com%2Fdocumentation%2Fengineers%2Fapi-documentation%2Fproducts%2Fgetitem) API, enabling you to retrieve the multilevel pricing information.

### **Optimized the Order Listing page in the Control Panel and the Checkout page**

For the Order Listing page in the Control Panel and the Checkout page, the backend code was refactored to aid in faster loading and overall performance improvement.
