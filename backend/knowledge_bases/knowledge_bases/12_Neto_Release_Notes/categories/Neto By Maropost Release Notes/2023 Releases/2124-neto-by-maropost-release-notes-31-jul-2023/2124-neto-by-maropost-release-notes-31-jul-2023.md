# Neto by Maropost Release Notes – 31 Jul 2023

## **Overview**

This release note covers the eCommerce release **6.258.0** and **6.259.0**. 

## **New Features**

### **Paypal 3D Secure 2 Authentication (3DS2) Available**

Paypal 3D Secure 2 authentication (3DS2) is now available for PayPal Advanced Credit and Debit Cards. 

3D-Secure is a protocol designed to ensure enhanced security and strong authentication for online credit and debit card transactions. To learn more about 3D Secure, see [**PayPal’s 3D-Secure FAQs**](https://galaxy.maropost.com/home/leaving?allowTrusted=1&target=https%3A%2F%2Fwww.paypal.com%2Fie%2Fwebapps%2Fmpp%2F3dsecure-faqs).

It is by default disabled and you can enable it on the Paypal Cards configuration page. To learn how to configure 3DS2 for PayPal Cards, see [**Configure 3DS2 for PayPal Cards**](https://galaxy.maropost.com/s/article/paypal-payment-method#3) section in the [**PayPal Payment Method**](https://galaxy.maropost.com/s/article/paypal-payment-method) article.

![PayPal 3DS2](https://us.v-cdn.net/6038474/uploads/attachments/migrated/PayPal3DS2.0685g00000G70NyAAJ.png)

### **Introduced a New Add-on: Address Guard**

We introduced a new add-on **Address Guard**. It is built by Address Guard and is available under the** **Shipping and Fulfillment section of the Addons page. 

Address Guard helps growing online brands with global address verification powered by the most reliable location data. With the help of Address Guard, you can verify, standardize, cleanse, and format address data for 240+ countries, all via an easy-to-integrate solution with your Neto web store.

For more information, including the setup instructions, see [**Address Guard**](https://galaxy.maropost.com/s/article/address-guard).

## **Enhancements**

### **Updated PayPal Cards Logo and Default Name**

The logo and default display name for PayPal Cards has been updated. In the Control Panel and the Checkout process, the PayPal Cards payment option is now displayed as **Credit or Debit Card** accompanied by a generic unbranded icon. The changes are done in accordance with industry-wide practices and to make the payment process more intuitive.

![PayPal Cards](https://us.v-cdn.net/6038474/uploads/attachments/migrated/PayPalCards.0685g00000G70ODAAZ.png)

### **Ability to View the Shipping Address on PayPal Dashboard and Invoices**

For orders paid through PayPal Cards, your customers can now see shipping address information on their PayPal dashboard and invoices.

![Invoice-Address](https://us.v-cdn.net/6038474/uploads/attachments/migrated/InvoiceAddress.0685g00000G70PGAAZ.png)

### **Updated Hunter Express Postcodes and Zones**

The Hunter Express postcodes and zones have been updated to reflect the newest changes for July.

### **Updated Courier Please Postcodes and Zones**

The Courier Please zone and sort codes have been updated to reflect the latest changes.

## **Fixes**

Fixed an issue in which users were unable to select a suitable category compatibility tree to download when updating eBay Compatibility Category data. As part of the fix, a disclaimer message has been introduced along with a checkbox to confirm your agreement to the terms of usage for the eBay compatibility category list.
