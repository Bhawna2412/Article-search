# Neto by Maropost Release Notes – 4 Sep 2023

## **Overview**

In this release, we are pleased to launch the redesigned UI for the webstore theme feature.

## **Enhancements**

### **Revamped Webstore Theme and Design Pages**

Working towards improving your experience with how you add and customize your webstore theme, we have made several enhancements to the Theme Store, Theme Editor, and other related pages.

#### **Adding a Theme**

The **Theme Store** page has been redesigned to simplify how you search for and add a theme and how you manage the themes you’ve used in your webstore.

﻿To learn about webstore themes, see [**Maropost Commerce Cloud Webstore Themes**](https://galaxy.maropost.com/s/article/maropost-commerce-cloud-webstore-themes).

The notable improvements include:

**Introduced Filtering Options on the Theme Page**: You can now search for themes using the following newly introduced filters:

- 
- 
- **By Cost**: Enables you to choose if you want to see free or premium themes.

**Segregated All the Themes That You’ve Already Used**: All the themes that you’ve installed in your webstore (including the one currently in use) are grouped under a section - **My Themes** - for quick access. This makes it easy to find all the themes that you have already installed including the one that’s currently live.

**Additional UI Improvements to the Theme Store Page**: We adapted a modernized design offering a more intuitive and user-friendly UI. There are several improvements around UI presentation and label texts. For example, UI changes to the Themes thumbnails, Your Current Theme section, and Extra Themes section.

![Webstore Theme](https://us.v-cdn.net/6038474/uploads/attachments/migrated/WebstoreTheme.0685g00000GAiF1AAL.jpg)

#### **Editing a Theme**

The **Theme Editor** page and the **Add Banner/Ad Unit Campaign** page have been redesigned to give you convenience and control over the editing process.

To learn about customising your webstore themes, see [**Webstore Design Options**.](https://galaxy.maropost.com/s/article/webstore-design-options)

The notable improvements include:

**Highlighted Style Elements on the Editor**: One of the most useful functionalities that has been introduced in this release is the ability to see the style elements on the editor and the ability to quickly toggle between them. Now, the style elements are highlighted when you hover over the page components on the editor. You can click the hover texts on the components to navigate between the style customisation options on the sidebar.

![Highlighted_Elements](https://us.v-cdn.net/6038474/uploads/attachments/migrated/HighlightedElements.0685g00000GAiErAAL.gif)**Collapsible Sections on the Sidebar**: The customisation options on the sidebar are now grouped under collapsible sections, making it easier for you to scroll through the sidebar to find the part that you wish to interact with.

**Intuitive UI for Logo Upload**: You can now see a preview and resize the image before saving it. Also, you’ll find the **Webshop Logos** section at the top of the sidebar for greater visibility.
