# Neto by Maropost Release Notes – 13 Feb 2023

**Fixes**

- Fixed an issue with eBay Business Processes where an error stating “The requested scope is invalid, unknown, malformed, or exceeds the scope granted to the client” occurred when attempting to (re)generate a new eBay authorisation token.
- Fixed an issue with checkout where some hosted icons, including navigational icons, were not loading. For example, in the date chooser, the arrows allow scrolling of the month.
