# Neto by Maropost Release Notes – 31 Aug 2023

## **Overview**

This release includes the e-Commerce releases **6.260.0 to 6.263.0**.

## **Updates and Enhancements**

### **Toll IPEC Renamed to IPEC**

Since the rebranding of Toll IPEC to IPEC, we have been changing all the references in our application UI to IPEC.

** **

### **Updated Default Name for PayPal Checkout**

The default name for** PayPal Checkout** has been changed to** PayPal – Pay now** (if Pay in 4 is disabled) and **PayPal – Pay now or Pay later** (if Pay in 4 is enabled).

![PayPal - Pay now](https://us.v-cdn.net/6038474/uploads/attachments/migrated/PayPalPaynow.0685g00000GAX2xAAH.png)** **

### **Updated Hunter Express Postcodes and Zones**

The Hunter Express Postcodes and Zones have been updated to reflect the newest changes for August.

### **Ability to Send** **DiscountPriceInfo for Variation Products**

You can now send the DiscountPriceInfo data to eBay for implementing the eBay Strike Through Pricing (STP) option for variation products. Earlier, it was supported only for standalone products.

> Note that STP is a deprecated feature, and therefore, this update is applicable for the existing STP-enabled accounts only.
