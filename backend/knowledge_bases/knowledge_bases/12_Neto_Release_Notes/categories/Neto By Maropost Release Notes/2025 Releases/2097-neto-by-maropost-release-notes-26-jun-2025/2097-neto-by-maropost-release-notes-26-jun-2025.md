# Neto by Maropost Release Notes – 26 Jun 2025

## Overview

This release note consists of the launch announcement of the Maropost Identity login.

## New Feature

### Launching Maropost Identity Login

Maropost Identity login is finally here!! Replacing the legacy login, the unified login experience will enable you to access various Maropost products and services through a Single Sign-On (SSO). It is faster, secure, and simpler to set up.

You can sign up to create a new account on Maropost Identity or link your Neto account to an existing Identity account. You can also update your email address when signing up or linking to Maropost Identity.

> **Note**: While we will support both Maropost Identity and legacy login for the near future, giving you plenty of time to switch to Maropost Identity, we recommend making the move right away to start leveraging the SSO.

To learn more about the Maropost Identity and how to set up your account, see [Upcoming feature launch: Maropost Identity for Neto - Maropost Community](https://galaxy.maropost.com/discussion/3268/upcoming-feature-launch-maropost-identity-for-neto).
