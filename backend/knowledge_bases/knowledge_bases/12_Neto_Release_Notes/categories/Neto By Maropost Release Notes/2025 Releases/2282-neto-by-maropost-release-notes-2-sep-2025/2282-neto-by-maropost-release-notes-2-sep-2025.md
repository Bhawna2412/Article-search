# Neto by Maropost Release Notes – 2 Sep 2025

This release consists of some exciting enhancements to our Amazon integration.

## Enhancements to Neto’s Amazon Integration

We are thrilled to announce the following improvements to our Amazon integration:

### Improved Error Visibility on Amazon Dashboard

It is now easier for you to troubleshoot your Amazon listing errors. Previously, only some errors were visible within Amazon Seller Central. Now, leveraging new capabilities of the Amazon APIs, the Amazon dashboard on Neto Control Panel displays all the listing errors that are preventing a successful listing creation.

This will enable you to understand why listings aren’t created so that you can take appropriate actions.

### Updated Category and Product Attribute Mapping Data

You now have more flexibility when it comes to mapping categories and product attributes, enabling you to list more products on Amazon.

The updates include the following:

- The Amazon Category and Product Type list in the Category Mapping page has been updated. This enables you to list products in categories that were previously not supported by the Amazon Selling Partner API (SP-API).
- The Attribute lists in the Product Field Mapping page has been updated to the latest version of Amazon’s taxonomy requirements. This enables you to map all required or optional attributes.

## Action Required – Next Steps for You

Please review your category mapping page and ensure you re-save each mapped category.

If you experience any issue mapping your categories and attributes, contact [support@maropost .com](mailto:support@maropost.com). And, if you have any trouble understanding the listing errors, please contact Amazon support for assistance.
