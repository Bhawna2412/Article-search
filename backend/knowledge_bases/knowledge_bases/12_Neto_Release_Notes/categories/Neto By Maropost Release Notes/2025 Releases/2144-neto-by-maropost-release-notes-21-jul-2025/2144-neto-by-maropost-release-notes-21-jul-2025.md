# Neto by Maropost Release Notes – 21 Jul 2025

## Overview

This release contains updates for the eCommerce versions 6.337.0, 6.338.0, and 6.339.0.

## Enhancements and Changes

### New Services Added for My Post Business

Neto has updated its My Post Business integration to include their newly introduced lightweight parcels - Extra Small/250g options.

Available for Australia Post packaging and customers' packaging, in both Parcel Post and Express Post services, the new options are:

- Parcel Post Flat Rate Satchel 5 kg (Extra small)
- Express Post Flat Rate Satchel 5 kg (Extra small)
- Parcel Post Flat Rate Own Packaging Up to 250 g (Extra small)
- Express Post Flat Rate Own Packaging Up to 250 g (Extra small)

### Updated AfterPay Checkout Message Per Compliance

As per Afterpay’s new compliance guidelines, the Afterpay message on the Checkout page has been updated to include the total product price alongside the repayment breakdown.

## Fixes

- Fixed an issue where Apple/Google Pay Express Checkout was overriding the customer-selected shipping method. This has been fixed, and the shipping method selected for the cart or checkout form is retained.
- Fixed an issue where some users' exported Stock Evaluation reports displayed an internal server error at the bottom of the report.
- Fixed an issue where the Auspost International Print Labels were missing the itemized weight information.
- Fixed an issue where the guest users are unable to print a receipt from the Order Success page at Checkout.
