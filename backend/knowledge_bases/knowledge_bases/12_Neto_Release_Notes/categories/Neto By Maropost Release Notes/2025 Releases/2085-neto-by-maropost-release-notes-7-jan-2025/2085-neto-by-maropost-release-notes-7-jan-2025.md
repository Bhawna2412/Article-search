# Neto by Maropost Release Notes – 7 Jan 2025

## **OVERVIEW**

This release includes updates from eCommerce releases **6.316.0** and **6.317.0**.

## **ENHANCEMENTS**

### **Added UserGroupId Parameter to GetShippingQuote API**

A new parameter - **UserGroupId** - has been added to the **GetShippingQuote API**. The parameter enables you to return shipping methods that are only available to a specific User Group. 

To learn more about the API, see the [**GetShippingQuote API**](https://galaxy.maropost.com/home/leaving?allowTrusted=1&target=https%3A%2F%2Fdevelopers.maropost.com%2Fdocumentation%2Fengineers%2Fapi-documentation%2Fshipping%2Fgetshippingquote).

> **Note**: This enhancement was requested by several customers on Idea Portal.
