# Neto by Maropost Release Notes – 4 Nov 2024

## **Overview**

This release contains updates for POS version **v.1.31.0.**

## **Enhancements and Updates**

### **Add Notes Against Orders in Neto POS**

POS Operators can now add notes against the orders in Neto POS. The **Add Note +** button, available on the Order Completion page, enables you to add a note for the order, which is then visible in the control panel as [sticky notes](https://galaxy.maropost.com/s/article/edit-a-sales-order#9).

You can edit or delete the note in Neto POS before completing the sale.

![NotesPOS](https://us.v-cdn.net/6038474/uploads/attachments/migrated/NotesPOS.068J1000006W7jXIAS.png)

### **View Order Notes in Neto POS Sales History **

A new column - Notes - has been added to the POS Sales History page. Here, you can click the Notes icon to see the notes added for the specific sale.
