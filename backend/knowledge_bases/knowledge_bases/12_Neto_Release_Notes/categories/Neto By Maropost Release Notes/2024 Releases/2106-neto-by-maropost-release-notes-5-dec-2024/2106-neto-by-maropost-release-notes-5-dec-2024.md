# Neto by Maropost Release Notes – 5 Dec 2024

## Overview

This release contains updates of the eCommerce hotfix version **6.314.3**.

## **Enhancement**

**WebP Image Format Supported in Data Imports**

You can now upload the WebP image format in the Product Data Import Wizard. The WebP image format is supported for both Simple Import (manual and From URL options) and Complex Import.
