# Neto by Maropost Release Notes – 7 Feb 2024

## **Overview**

This release includes eCommerce releases **6.278.0** and **6.279.0**.

## New Features and Enhancements

### **Ability to Add Images to Disputes**

You can now collect image evidence of the problem with an order (broken/damaged item, wrong item, overcharge, and so on) as part of the dispute process itself.

**What’s New?**

Your customers can upload an image when raising a new dispute and also when adding a new message to an existing dispute. So, while they can upload one image when creating the dispute, they can add more images through messages.

The following is a screenshot from the raising dispute process showing the newly added **Upload Image** component:

![RaiseDispute](https://us.v-cdn.net/6038474/uploads/attachments/migrated/RaiseDispute.0685g00000Kj5pOAAR.png)

The following is a screenshot of the **Dispute Details** page with the changes highlighted in green:

![DisputeDetails](https://us.v-cdn.net/6038474/uploads/attachments/migrated/DisputeDetails.0685g00000Kj5mCAAR.png)

You can see the uploaded images in the **Message History** section of the **View Item Dispute** page. In Navigation, go to **SALES ORDERS**, and select **Disputes**. Then, on the **Resolution Centre** page, click the Dispute ID to go to the **View Item Dispute** page.

![DisputeView](https://us.v-cdn.net/6038474/uploads/attachments/migrated/DisputeView.0685g00000Kj5ouAAB.gif)

**What’s Updated and Is Your Action Required? **

To support the dispute image feature, the Neto webstore templates have been updated. If you’re installing a webstore template now or in the future, you’ll find the feature included in the template, and there’s no action required from your end.

However, to enable the feature in your existing active templates (the ones already installed), you must manually customize the templates. For detailed instructions, see [Add Image Upload to Disputes](https://galaxy.maropost.com/home/leaving?allowTrusted=1&target=https%3A%2F%2Fdevelopers.maropost.com%2Fdocumentation%2Fcommerce-cloud-developer-documentation%2Ftweak-documents%2Fgeneric%2Fadd-image-upload-to-disputes%2F).

> **Note:** As an exception, the following Themes do not support the dispute image feature by default and require manual customization: **Brave**, **Frame**, **Geometric**, **Personal Best**, **Twenty3**, **Wholesale deluxe**, **Rapid**, **Amplify**, **Pet Perfect**, **Mi Casa**, and **Knick Knack**.

### **New Addon: ShipStation**

We added a new addon - **ShipStation** - to our Addon library. [ShipStation](https://galaxy.maropost.com/home/leaving?allowTrusted=1&target=http%3A%2F%2Fwww.shipstation.com%2F) is a scalable shipping solution that’s ideal for businesses of all sizes. Connect to your Neto store in seconds, seamlessly sync your shipping updates back and forth, and get products to your customers fast.

To learn more, see the [ShipStation article](https://galaxy.maropost.com/s/article/shipstation).
