# Neto by Maropost Release Notes – 20 Nov 2024

## **Overview**

This release contains updates of the eCommerce version **6.314.0**.

## **Enhancements**

### **EWay Available on Neto Control Panel**

You can now install the eWay payment method from the Neto control panel. If you already use eWay, you can add more eWay payment configurations in the control panel. For example, you can add Amex if you already have Mastercard and Visa. To learn how to set up eWay, see [eWAY Credit Card Payment Method](https://galaxy.maropost.com/s/article/eway-credit-card-payment-method).

> **Note**: New accounts will have the eWay available by default.

![eWay](https://us.v-cdn.net/6038474/uploads/attachments/migrated/eWay.068J1000007OAVWIA4.png)

### **WebP Image Format Supported When Downloading From URL**

You can now download images in WebP format from a URL when using the "Download Image from URL" addon.

![DownloadImageFromURL](https://us.v-cdn.net/6038474/uploads/attachments/migrated/DownloadImageFromURL.068J1000007OAqVIAW.png)

## Bug Fixes

- 
- Fixed the Startrack tracking link which was showing a DNS error. It has been fixed and now loads the Startracks tracking page.
- Fixed an issue in which the Arrival date was not displaying in the orderline details in the control panel.
