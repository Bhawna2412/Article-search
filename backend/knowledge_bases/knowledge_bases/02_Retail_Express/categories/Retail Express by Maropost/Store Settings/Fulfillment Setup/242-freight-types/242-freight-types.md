# Freight Types

You can categorize and prioritize customer shipping methods by creating freight types that label different delivery services like standard, express, or priority shipping, making order fulfilment more efficient. The Fulfilment Report allows filtering by freight type, helping retail operators process time-sensitive express shipments first while managing standard deliveries efficiently, which improves customer satisfaction through faster order processing.

Properly configured freight types automatically import from Shopify integration and help retail operators organize fulfilment workflows based on shipping urgency, ensuring premium shipping services receive appropriate priority in your order processing queue.

Freight Types are labels applied to different shipping methods selected by customers when placing an order to be delivered, making it easier to categorise and priority the dispatch of customer orders.

The Fulfilment Report allows you to filter by Freight Types, which makes it easier to target the fulfilment of specific freight methods before others e.g. if you had a Freight Type "Priority Shipping" or "Express Shipping", you would be able to see outstanding orders with this Freight Type, dispatch those, and then continue on with the remaining standard sales.

Freight Types are automatically imported by the Retail Express Shopify Connector tool. If using a third-party integration with Retail Express, contact your developer to request this feature.

## []()Create a Freight Type

To create a Freight Type:

> **Note:** If using the Retail Express Shopify integration, Freight Types are will be automatically created for you. If using a third-party integration with Retail Express, contact your developer to request this feature.

1. Navigate to **Settings > Fulfilment Setup > Freight Types**
2. Enter a **Freight Type **name
  ![mceclip0 (58).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppNmOEAU.jpg)
3. Click **Create**

If the Freight Type name matches a Freight Type previously deleted, a prompt will be displayed to confirm the Freight Type should be reactivated.

## []()Rename Freight Type

To rename an existing Freight Type:

1. Navigate to **Settings > Fulfilment Setup > Freight Types**
2. Locate the Freight Type to be renamed
3. Click the **Pencil **icon ![mceclip2 (26).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppNmTEAU.jpg)
4. Update the name
5. Click the **Tick **button ![mceclip3 (20).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppNmdEAE.jpg) (press the X to cancel the changes)

## []()Delete a Freight Type

Deleting a Freight Type will only prevent the Freight Type from being able to be used at POS. The Freight Type will still remain associated to all previous transactions, and able to be selected within the Fulfilment Report.

To delete a Freight Type:

1. Navigate to **Settings > Fulfilment Setup > Freight Types**
2. Locate the Fulfilment Type to be deleted
3. Click the **Trash Can** icon ![mceclip1 (41).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppNmiEAE.jpg)

The Freight Type will be removed from the list.

> **Note:** If using the Shopify Connector or an API integration, any incoming sales using a Freight Type name that was previously deleted will cause the Freight Type to be reactivated.
