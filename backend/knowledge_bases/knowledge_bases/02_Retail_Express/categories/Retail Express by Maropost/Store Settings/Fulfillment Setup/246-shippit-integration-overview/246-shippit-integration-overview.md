# Shippit Integration Overview

You can access comprehensive shipping automation by integrating **Retail Express** with Shippit, streamlining order fulfilment from POS and online sales through automated courier booking and label printing. This integration helps reduce fulfilment processing time and manual shipping errors, enabling faster order dispatch and improved customer delivery experiences across your retail operations.

Understanding the Shippit integration benefits and requirements helps retail operators decide if this shipping automation solution aligns with their fulfilment workflow needs, with options for single or multi-outlet configurations depending on your retail operations structure.

![shippit_integration (1).jpg](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000pphfnEAA.jpg)
The Shippit integration combines the strength of Retail Express' inventory management with Shippit's advanced shipping features, allowing businesses to streamline their shipping processes and reduce manual double handling.

## []()Using Shippit

When a sale is Fulfilled in Retail Express, the sale is automatically sent to Shippit. Sales can be fulfilled either individually within POS, or in bulk using the Fulfilment Report.

The Delivery Service in Shippit is automatically selected using the Freight Type selected on the sale (either by the customer shopping online or manually in POS - as per below).

Freight Types are mapped to corresponding Shippit Delivery Services within Retail Express.

![pos_freight_type (1).gif](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000pphvpEAA.jpg)*Manually selecting a Freight Type in POS*

Once the sale has been synchronised, the Tracking Details are displayed in POS.

![mceclip5 (26).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000pphc6EAA.jpg)*Tracking details in POS*

Switching to the Shippit portal, the staff member will then be able to print address labels and book the order for delivery.

![mceclip3 (57).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000pphvuEAA.jpg)

## []()Features of Shippit Integration:

The Retail Express Shippit integration is compatible with a range of features, listed below. We're always considering opportunities to expand the feature, so if there's something you'd like to be added to the Shippit integration why not [let us know](https://forms.zohopublic.com/virtualoffice112/form/ShippitFeatureRequest/formperma/PtX4ngXp_sImKlKQ2g2g8CZDGB0ytGTRdUQAyOu61Ls)?

### Features

Synchronise both **instore** and **online** orders from Retail Express to Shippit

**Multi-location** shipping

**Split order fulfilment** (ship one order from multiple stores)

**Partial Fulfilment** (multiple fulfilments for the same order)

**Domestic** and **International** orders

**Product Details synchronised** (SKU, Product Title/Short Description, Price, Quantity, Quantity Fulfilled/Packed - used to indicate when the number of products sent is less than the quantity ordered)

Synchronise **Total Product Weight **for shipment

View the **Shippit Tracking Number** within POS sale

Link to view **Tracking Status** available when viewing the sale in POS

**Automatically create new Freight Types** from incoming online sales (Shopify Connector and eCommerce API integrations using the FreightType field)

**View selected Freight Type** for a sale (POS and Fulfilment Report)

**Filter Freight Types** in the Fulfilment Report (e.g. fulfil Express Shipping orders first)

**Configurable Customer Invoice Number** (select between Retail Express Order Number or External Order Reference - recommended if using Shopify Connector)

### Limitations

**Product Dimensions **not synchronised (only total shipment product weight)

**Tracking Number **not visible from the Fulfilment Report (only POS)

**Tracking Numbers** not synchronised with Shopify/API integrations

**Printing Labels** and **booking shipment** managed via Shippit only

**Multiple parcels per SKU** (you can manually add additional parcels via Shippit)

**Live Quotes** not available

**Resynchronising orders** not available

**Automatic Freight Type creation** available in the eCommerce API for custom integrations

**One way synchronisation** from Retail Express to Shippit; changes made in Shippit will not be synchronised to Retail Express
