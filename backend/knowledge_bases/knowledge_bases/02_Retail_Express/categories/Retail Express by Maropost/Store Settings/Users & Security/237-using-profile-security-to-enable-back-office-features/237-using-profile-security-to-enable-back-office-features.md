# Using Profile Security to enable Back Office features

You can efficiently manage back office feature access across your retail team by creating security profiles that enable or disable specific functionality for groups of users based on their roles and responsibilities. This helps ensure staff can access the retail operations features they need for their position while protecting sensitive areas like purchase orders, financial reports, and inventory management from unauthorized access.

Setting up role-based security profiles helps retail operators maintain appropriate access controls across owner, manager, sales assistant, warehouse, and accounting roles, with the Security Settings Matrix providing tested configuration recommendations for common retail positions.

The **Profile Security** area controls which features users can access within Retail Express Back Office, also commonly referred to as User Permissions. Within each Profile all features are listed, and access to the feature is enabled by ticking the corresponding box.

Security Profiles make it efficient to manage user access by enabling/disabling multiple features for a group of users at once, instead of individual configurations.

> **Note:** Each User must have an associated Profile to be able to log into Retail Express.

Common Profiles include Administrators, Warehouse Managers and POS User Basic (typically assigned to casual staff members).

> **Note:** To access the Profile Security area you must be logged in as a user with the Feature **Profile Security **assigned to your profile. If you cannot access this feature please contact your company's System Administrator.

## []()Create a new Profile

To create a new Security Profile:

1. Navigate to **Staff > Profile Security**
2. Enter a **Profile Name**
3. Click **Insert**
4. The profile will be added to the list of profiles at the bottom of the page

![mceclip0 - 2023-07-10T045935.123.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppjEvEAI.jpg)

> **Note:** To change the name of the Security Profile click the Pencil icon on the row for that Profile.

## []()Enable/disable Features

To add/remove features from a Security Profile:

1. Click the **Edit Features **link on the row for the profile
2. All the individual functions/features available to be enabled/disabled will be displayed
3. **Tick** the box in the **Selected **column to **enable** a feature

> **Note:** The box in the **Selected** column to **disable **a feature.

4. **Tick** the **Essentials **feature
5. Click **Save Changes**

![mceclip1 - 2023-07-10T045939.807.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppjZlEAI.jpg)

> **Important:** All Profiles **must have Essentials enabled** as a minimum to allow the Users to perform basic tasks within Retail Express.

## []()Profile Security suggestions (Security Settings Matrix file)

The Retail Express Security Settings Matrix is an Excel spreadsheet, containing suggestions on which features should be enabled in Retail Express depending on the following user roles:

- Owner/CEO/Business Decision Maker
- Store Manager
- Assistant Manager
- Sales Assistant
- Inventory Manager
- Warehouse Staff
- Bookkeeper/Accountant
- eCommerce Manager

The Matrix provides suggestions for both Profile Security Features and POS Security Features i.e. Back Office and POS.

[**Security Settings Matrix Download**](https://www.retailexpressmedia.com/support/Security Setting Matrix.xlsx)

> **Note:** The Matrix is supplied as a guide only. Ensure you review all settings prior to configuring your security settings within your business.
