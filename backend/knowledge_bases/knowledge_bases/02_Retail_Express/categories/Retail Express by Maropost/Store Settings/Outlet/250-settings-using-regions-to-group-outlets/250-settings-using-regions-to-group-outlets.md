# Settings - Using Regions to group Outlets

Regions can be used to group your Stores/Outlets, usually by physical location or area e.g. if you have several stores spread throughout Queensland, you may choose to configure Regions as: South-East QLD, Central QLD and Northern QLD.

![blobid0 (39).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000pphcFEAQ.jpg)

## Create a Region

To create a new Region:

1. Navigate to **Settings > Locations/Outlets > Regions**
2. Enter a unique name into the **Name** field
3. Highlight the **Country**
4. Click the **Outlets** to associate with the Region

> **Note:** Hold Ctrl to select multiple Outlets.

5. Click **Create**

> **Note:** Outlets already assigned to a region will not be displayed in the list; an Outlet can only belong to a single Region.

## Edit a Region

To edit an existing Region:

1. Navigate to **Settings > Locations/Outlets > Regions**
2. Click the **Pencil **icon of an existing Region in the Regions table
3. Update the fields as required
4. Click **Update**
