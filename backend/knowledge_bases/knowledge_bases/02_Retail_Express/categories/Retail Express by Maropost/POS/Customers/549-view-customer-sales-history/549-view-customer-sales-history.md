# View Customer Sales History

You can deliver personalized service and informed recommendations through viewing customer sales history in **Retail Express** POS, which provides access to customer purchase records supporting product suggestions and service quality. This purchase visibility helps retail operators understand customer preferences and provide relevant recommendations, with effective sales history use typically improving customer satisfaction and supporting increased sales through personalized service across Australian retail operations.

Customer sales history in **Retail Express** enables retail staff to access past purchases, return patterns, and buying preferences, helping support informed product recommendations and personalized customer interactions. This history visibility helps ensure your retail operations can deliver knowledgeable service while identifying opportunities for complementary product suggestions and building stronger customer relationships.

Once a customer has been added to a sale in POS you can view their entire sales order history by clicking the Orders section on the right-hand side. This is particularly useful when the customer has outstanding laybys or makes an enquiry related to a previous order.

## Viewing the Customer Order History

To view the customer order history when in POS:

1. Ensure the customer has been added to the sale
2. On the Customer tab click the blue **Orders **tile
  ![mceclip0 - 2023-07-10T054206.099.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000pphV2EAI.jpg)

  The Value displayed will be a total of all sales for the customer for any Outlet, regardless of whether the user has access to the Outlet or not
3. The **Orders **window will be displayed
4. Click the **Order History Search **tab to display the order history
  ![mceclip1 - 2023-07-10T054212.386.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppkJKEAY.jpg)
5. Use the **Search Filters **to find a particular order
  If the sale isn't displayed, the user may be restricted from viewing sales for that Outlet.
6. Click **Preview **to view the invoice for the order (you can also **Print **the invoice if required)
7. Click **Cancel **to return to the Order History Search window
8. Click **Done **to return back to the Customers tab in the POS sale

## Order History Search tab filters:

|  |  |
| --- | --- |
| **Field** | **Description** |
| Invoice # | The invoice number of a historical order |
| Date | The date of the previous transaction |
| Outlet | The Outlet where the sale was made |
| PLU/Description Search | The "Search...." field will search both the PLU code (Supplier or Manufacturer SKU) and the Description based on the characters entered. Note that you can enter partial letters to match results e.g. Typing "Oak Ba" will still display "Oakley Mens Head Band" in the results |
| Size | Used to search for orders with a product matching the size selected. Clicking the filter will display the Sizes in the system, multiple sizes can be selected if required |
| Colour | Used to search for orders with a product matching the colour selected. Clicking the filter will display a list of Colours, multiple colours can be selected if required. |
| Qty | The number of individual products added to the order |
| Filled | The number of products that have already been dispatched to the customer |
| Price | Used to search for the price of the product |
| Total | The total sum of the order |
