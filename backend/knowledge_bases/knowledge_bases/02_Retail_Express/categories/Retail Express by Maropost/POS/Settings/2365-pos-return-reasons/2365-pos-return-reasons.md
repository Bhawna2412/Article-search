# Return Reasons

This feature allows you to create and configure custom reasons for product returns. Staff will be able to select the reason when process a return or exchange. View more details in the [Return Reason](https://galaxy.maropost.com/kb/articles/316-return-reason-report) report. 

To configure your Return Reasons:

1. Navigate to **Settings > Return Reasons**
2. Enter the **Return Reason** name
3. 
  ![image.05T5g00000pphaEEAQ.png](https://us.v-cdn.net/6038474/uploads/XN60LCCNS71U/image-05t5g00000pphaeeaq.png)
4. Tick **Enabled**
5. Click **Insert**
6. Use the **pencil** icon to edit the name for an existing Reason
7. Click the **trash can **icon to delete Return Reasons (you will be unable to delete the Return Reason if it's already been used for a sale)
