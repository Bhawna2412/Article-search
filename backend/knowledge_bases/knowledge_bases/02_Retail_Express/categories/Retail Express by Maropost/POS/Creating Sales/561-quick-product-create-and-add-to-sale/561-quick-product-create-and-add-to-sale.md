# Quick Product Create and Add to Sale

You can streamline your retail operations by creating and adding new products directly during the sales process in Retail Express, helping your retail operators handle unexpected inventory without interrupting customer transactions. This capability allows on-the-fly product creation when items haven't been pre-entered into your system, which helps reduce lost sales opportunities and improves checkout efficiency during busy retail periods.

Setting up quick product creation workflows enables your retail team to maintain sales momentum even when dealing with new or specialty items, helping optimize point-of-sale operations while ensuring all transactions are accurately recorded. This helps retail operations maintain flexibility and responsiveness to customer needs while keeping proper inventory and sales tracking.

The "Quick Create and Add Product" is a significant time-saving feature in the POS system, allowing users to swiftly create new products directly during a sale transaction. With this feature, users have the flexibility to start from scratch or duplicate an existing product, similar to the "Duplicate a Product" functionality available in the back office.

This feature allows users to immediately add the product to the sale upon creation, streamlining the workflow and eliminating the need to switch to the back office or resort to using a generic or miscellaneous product code.

> **Note:** A new profile security feature has been added to determine access to the feature.
> After the feature has launched, to allow 'Create Product/s in POS' access to a user, Go to Staff > POS Security features > Select Profile > Enable "Create Product/s in POS."

### **Create New Product/s at POS**

![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/Screenshot20240611at32926AM.05TJ10000047H1GMAU.png)

- Initially create a product in the system with basic information and then optionally customise any specifics or details of the product by adding more details on the 'Additional details Page'; such as a unique description or selling price tailored for a particular customer invoice.
- Users can generate SKUs when creating products, particularly when the specific SKU value is unknown. Users can utilize the "Generate SKU" option within the POS system during product creation. Later, if necessary, they can update the SKU value via the Edit Product page in the Back Office.
- Click the "Save and Add New" button to create additional products without navigating away from the page.
- Quickly specify only the minimum required fields for new products, or they optionally provide further details by accessing the "Additional Details" page.
- Create an entirely new product, or pre-fill from an existing product to save time when creating similar items or additional variants.

## []()**Pre-Fill From Existing Product/s**

To Pre-fill details from a product, On the Quick Create & Add Product page >> select **Pre-Fill from the Product** >> Enter **Product Code** >> **Search and Pre-Fill.**

Once the Pre-fill product is found, it's details are auto-populated. Edit the SKU for the new item, (it must be unique) along with any other changes required, then proceed further by clicking Create Product.

![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/Screenshot20240611at33229AM.05TJ10000047H32MAE.png)

### **Additional Details Page**

Users can access the '**Additional Details Page**' when using the 'Create New Product' option on the 'Quick Create & Add Product page'. To access this page, use **Create New Product** >> Enter Basic **Product Details** >> **Additional Details**

![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/Screenshot20240611at33407AM.05TJ10000047GUYMA2.png)

### **Order Details**

Proceed to the next page to update anything about the product that applies just to this specific invoice. This includes the description and fulfilment information, POS price and Qty.
