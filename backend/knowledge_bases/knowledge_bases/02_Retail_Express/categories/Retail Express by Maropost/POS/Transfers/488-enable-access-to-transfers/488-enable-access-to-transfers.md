# Enable Access to Transfers

You can streamline inter-store inventory management through enabling transfer access in **Retail Express** POS, which allows front-line staff to create and receive stock transfers without accessing Back Office systems. This access configuration helps retail operators improve operational efficiency, with proper transfer access enabling typically supporting faster stock movement and reduced administrative overhead across Australian retail operations.

Transfer access configuration in **Retail Express** enables retail operators to grant appropriate staff permissions for POS-based transfer operations, helping ensure efficient inventory distribution while maintaining security controls. This access management helps ensure your retail operations can empower staff with transfer capabilities while maintaining appropriate authorization controls supporting operational efficiency.

The Transfers feature in POS allows you to easily send and receive stock to/from another outlet. To use Transfers in POS you will need to enable two security permissions within the POS Security Features section.

These features will be enabled for all users with a matching Security Profile.

![rtaImage (72).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000powrREAQ.jpg)
To configure the permissions:

1. Navigate to **Staff > POS Security Features**
2. Select a **Profile** from the drop-down
3. Tick the feature **Allow Transfers**
4. Tick the feature **Allow Transfers - Void**
5. Click **Save**

|  |  |
| --- | --- |
| **Permission** | **Description** |
| Allow Transfers | Users with this Security Profile will be able to access the Transfers feature |
| Allow Transfers - Voice | Allows users with this Security Profile to void transfers i.e. cancel a Transfer. |
