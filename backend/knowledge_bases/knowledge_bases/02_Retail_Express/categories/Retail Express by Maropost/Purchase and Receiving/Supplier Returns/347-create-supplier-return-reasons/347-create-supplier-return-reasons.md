# Create Supplier Return Reasons

You can create customized Supplier Return Reasons to categorize why you're claiming credit from suppliers, helping standardize return documentation and enable future reporting on return patterns. This improves your retail operations by providing clear classification for damaged goods, short shipments, overcharges, and other supplier issues, with properly configured return reasons typically enabling better supplier relationship management and identification of recurring quality or fulfillment problems.

Setting up supplier return reasons involves understanding reason name requirements, editing and deletion capabilities, and visibility on supplier documentation to help retail operators focus on the right return categorization workflows, ensuring your retail strategy consistently drives better supplier accountability and operational tracking outcomes.

Supplier Return Reasons are labels used to categorise the reasons why you're claiming credit from your supplier e.g. damaged, short shipped, overcharged etc

The Return Reason will be visible on the Supplier Return document provided to your supplier.

## Create Reasons

To create a Supplier Return Reason:

1. Navigate to **Settings > Supplier Returns Setup > Supplier Return Reasons**
2. Click **Create New**
3. Enter the **Name **for the supplier reason (max of 50 characters)
  ![mceclip0 (29).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000pow0iEAA.jpg)
4. Click **Save**
5. The new Reason will be added to the table below

## Edit Existing Reasons

To edit the name of a Supplier Return Reason:

1. Click the line to be amended
2. Update the **Name** (click Cancel to discard the changes)
3. Click **Save** (or press Enter on your keyboard)

## Delete Return Reasons

Return Reasons can be deleted, meaning they will no longer be available when creating and editing a Supplier Return. Any returns that have already been processed using that Reason will continue to display the Return Reason correctly.

To delete a Supplier Return Reason:

1. Tick the box for the Reason (or click into the line)
  Tick multiple boxes to bulk-delete Reasons
2. Click **Delete**
