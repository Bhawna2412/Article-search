# Use the Product Audit tab to find PO numbers

You can use the Product Audit Tab to track purchase order history for specific products, helping quickly identify which POs contain particular items and understand ordering patterns. This improves your retail operations efficiency by providing product-level purchase order visibility for troubleshooting and research, with properly configured audit tab usage typically enabling faster issue resolution and better understanding of product procurement history.

Understanding product audit tab functionality involves recognizing PO number lookup capabilities, transaction history viewing, and product-specific filtering to help retail operators focus on the right product research workflows, ensuring your retail strategy consistently drives better operational problem-solving and procurement transparency outcomes.

The Audit tab displays a list of Purchase Orders the product belongs to. This is extremely handy when you're trying to find information on a PO for the product but you're not sure of the Purchase Order Number.

You can use the PO Status filter to further refine the view by On-Order, Available etc.

![mceclip0 - 2023-07-10T040633.222.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppioLEAQ.jpg)
To use the Product Audit tab:

1. Select a **PO Status** from the drop-down menu
2. Click **Search**
3. A list of orders will be displayed

> **Note:** Click **Export to Excel **at the bottom of the page to download the list of Purchase Orders into an Excel spreadsheet.

|  |  |
| --- | --- |
| **Column** | **Details** |
| POID | The Purchase Order Number |
| Created | Date the PO was created |
| Supplier | The Supplier Code |
| Warehouse | The Outlet against the Purchase Order |
| PO Status | The current status of the Purchase Order |
| Ship Status | The status if entered on the PO |
| ETA | The estimated date of arrival (if entered on the PO) |
| Supplier Ex 			Buy Ex 			COGS | The cost information from the order |
| Qty Ordered | The quantity ordered for that specific product on the order |
| Qty Received | If the PO has been received or made Available the quantity of products received |
| View | A shortcut icon to open the Purchase Order for Editing |
