# Back Office Receipt Printing Configuration

You can configure receipt printing settings in **Retail Express** Back Office, controlling receipt content, formatting, and printer selection supporting professional customer receipts and audit requirements. This helps optimize your retail operations by ensuring receipts include required information, maintaining brand consistency, and supporting customer service with clear transaction documentation.

The Receipt Printing Configuration window allows you to customise your receipts, including default comments.

To update your Receipt Configuration:

1. Navigate to **Settings > Label Printing > Receipt Printing**
2. Update the details as required
3. Click **Save Changes**

|  |  |
| --- | --- |
| Field | Details |
| Comments | Default Comments to be printed on receipts. For unique comments per store refer to the Outlet Configuration settings instead.  > **Note:** Outlet-specific comments take priority over Receipt Comments |
| Page Width | Changes the width of text on the receipt |
| Display Address | Enables store address details to be displayed on receipts |
