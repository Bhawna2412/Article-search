# Getting started with Offline POS

You can implement Offline POS capability for **Retail Express**, enabling continuous retail operations even during internet outages by maintaining local transaction processing and automatic synchronization when connectivity restores. This helps optimize your retail operations by preventing revenue loss during internet disruptions, maintaining customer service continuity, and ensuring business continuity regardless of connectivity challenges.

Offline POS has been designed as an emergency POS system. This feature allows your business to continue serving customers in the event that your internet connection is interrupted or there is an issue with access to Retail Express.

## []()Available features while trading offline

Using Offline POS you are able to continue trading and perform the following:

|  |  |
| --- | --- |
| **Feature** | **Available in Offline Mode?** |
| Scan and search for products | Yes |
| Use Disabled Products | No |
| View stock levels | No |
| Discount products | Yes |
| Max discount rules | Yes |
| Max discount rule override | Yes |
| Delete products from a sale | Yes |
| Search and select customers | Yes |
| Add new customers | Yes |
| Add a new delivery address to a customer | Yes |
| Process payments | Yes |
| Use Integrated EFTPOS | Yes - provided the issue isn't your internet connection |
| Process laybys | No |
| Redeem vouchers | No |
| Use loyalty points | No |
| Create a refund using an original sales invoice | No - however, you can process a Blank Refund |
| Create a blank refund | Yes |
| Process transfers | No |
| Print receipts and invoices | Yes - provided the issue isn't with your Local Network |
| Process a cash Up | No - however, you are able to perform a Cash Up after connection is restored and synchronisation is complete |
| Use Money In/Out | Yes - you are able to enter Money In/Out, but cannot use the Money In/Out Report until the connection is restored and synchronisation is complete |
| Custom attributes | No |
| Use Promotional Campaigns | Yes, except those with Usage Limits (as Offline POS is unable to validate the history of the Coupon Code for the customer) |
| Outlet Access restrictions | Users will be restricted to Outlets as per the Outlet Access settings in the User Account. |

## Limitations

|  |  |
| --- | --- |
| **Feature** | **Available in Offline Mode?** |
| Integrated EFTPOS | Without the internet, some Integrated EFTPOS methods may not work.  Check if your Payment terminal has an inbuilt modem - if the Computer is offline but can still communicate with the terminal you may still be able to process payments. |
| Historical Customer Transactions | No |
| View Stock Levels | No |
| Process Laybys | No |
| Redeem Vouchers | No |
| Use Loyalty Points | No |
| Create a Refund using an original sales invoice | No - however, you can process a Blank Refund |
| Process Transfers | No |
| Process a Cash Up | No - however, you are able to perform a Cash Up after connection is restored and synchronisation is complete |
| Custom Attributes | No |
| Disabled Products | No |
