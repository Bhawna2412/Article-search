# Using Mac OS X with Retail Express

You can operate **Retail Express** on Mac computers by installing virtual machine software running Windows 10, enabling full POS functionality including receipt printing and integrated EFTPOS for retail locations that prefer Mac hardware. This flexibility helps retail operators choose the hardware platform that best suits their business needs while maintaining complete access to retail operations features.

Understanding Mac compatibility requirements helps retail teams configure appropriate virtual machine setups using VMware Fusion or Parallels, ensuring your Mac-based retail locations can access both POS transaction processing and back office management capabilities effectively.

Retail Express is designed for use on Windows computers, but can still be used on Mac with some additional requirements.

|  |  |
| --- | --- |
| **POS** | Google Chrome (Virtual machine required for receipt printing/integrated EFTPOS) |
| **Back Office** | Google Chrome or Safari |

For detailed information please refer to the Hardware and Software Requirements.

To enable hardware you will need to install a virtual machine running Windows 10. Virtual Machines can be installed by using software like Vmware Fusion ([hardware requirements](https://www.vmware.com/products/desktop-hypervisor/workstation-and-fusion)) or Parallels ([hardware requirements](https://www.parallels.com/au/products/desktop/resources/#sysreq)).
