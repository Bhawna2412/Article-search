# Installing Barcode Font and Drivers

You can enable proper barcode label printing and document formatting throughout **Retail Express** by installing the required drivers and barcode fonts, ensuring product labels, purchase orders, and inventory documents display and print correctly. This helps streamline your retail operations by enabling accurate barcode generation for inventory management and product labeling workflows.

Properly installing drivers with administrator privileges helps avoid printing issues and ensures barcode functionality works consistently across your retail locations, supporting efficient inventory tracking and product identification throughout your retail operations.

The Retail Express Drivers are responsible for installing the barcode font used for previewing/printing labels, and on various other documents e.g. Purchase Orders.

> **Note:** The steps below require a computer account with **Administrator privileges**; if you're unsure please refer to your Systems Administrator.

[**Click here to download Drivers**](https://www.retailexpressmedia.com/support/drivers/firstinstall.exe)

## Installing the Drivers

To install the Retail Express Drivers:

1. Click **Download** to prompt the download of the Installation file
2. Click **Save **to save the file to your computer (depending on your browser it may automatically download the file to a preconfigured location)
3. If using Chrome the download may be blocked; you will need to temporarily override your security settings to download the file (and then reset them) - refer to the FAQs for more information
4. **Run **the downloaded Installation File
5. If a **User Access Control **window appears to request permission, click **Yes **to allow the changes
6. When the application runs click **Install**
  ![drivers_installation.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppPrpEAE.jpg)
7. The installation process will run
8. Select **Yes, restart the computer now **(you may need to close other applications and save your work)
9. The computer will restart, completing the installation process

## []()Frequently Asked Questions

[Installation process "not responding" error (frozen/hanging installation)](https://galaxy.maropost.com/kb/articles/229-installing-barcode-font-and-drivers#not-responding)

If the installation process displays a "Not Responding" error it will be typically due to the Antivirus software on the computer. This software will prevent the installation from completing the steps required to install the software.

To resolve the issue:

- Add **C:\RetailExpress\** to the list of exclusions for the Real Time Scanner, Web Browser Protection and Scheduled Scans
- Allow the file **firstinstall.exe**

> **Note:** The steps required to complete the above will be different depending on the antivirus program in use. Refer to the Antivirus Vendor's website for more information, or your Systems Administrator.

["Error occurred while trying to copy a file" message](https://galaxy.maropost.com/kb/articles/229-installing-barcode-font-and-drivers#reinstall)

If you encounter an error similar to the following, it indicates the drivers were installed previously on the computer. This message can be disregarded, click **OK **to continue the installation.

"An error occurred while trying to copy a file: C:\RetailExpress\Updates\FirstInstall\TextPrinter.dll Click OK to proceed anyway or Cancel to cancel installation."

![install drivers 3.jpg](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppPruEAE.jpg)
[Chrome is blocking the download](https://galaxy.maropost.com/kb/articles/229-installing-barcode-font-and-drivers#blocked)

If trying to download the installation file and Chrome is blocking it, you can temporarily disable the browser's security to download the file.

1. Click the **three-dot menu button** at the top right of the window
2. Click **Settings**
3. Type **Privacy and Security**
4. Select **Security**
  ![mceclip2 (36).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppNQvEAM.jpg)
5. Locate the **Safe Browsing **section
6. Note the option your computer is currently set to (Standard or Enhanced)
7. Select **No protection (not recommended)**
  ![mceclip3 (25).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppPs4EAE.jpg)
8. Click **Turn Off**
9. **Download** the file
10. **Repeat the step**s to re-enable the Browser security (in step 7 select the option your computer was originally set to)
  It's strongly recommended to re-enable the Browser Security immediately after downloading the file to protect you from websites, extensions and downloads known to be dangerous
