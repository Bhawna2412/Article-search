# API & Plugin Documentation

You can integrate **Retail Express** with various third-party systems through comprehensive API options, enabling seamless data exchange for eCommerce, warehouse management, inventory planning, and accounting operations. These API integrations help improve operational efficiency by automating data synchronization, reducing manual entry, and connecting your retail operations with specialized business systems.

Understanding the available API documentation involves reviewing the specific integration options that match your business requirements, from modern REST APIs to specialized interfaces for accounting and inventory planning. When you implement appropriate API integrations, you can expect reduced administrative overhead, improved data accuracy, and better operational coordination across your retail technology ecosystem.

The following documents contain information on the various API integrations available with Retail Express. These documents are available to review to determine if the integrations are available for your business requirements.

## API Documentation

| **API** | **Details** | **Link** |
| --- | --- | --- |
| Modern REST API | A unified API to replace all the legacy Soap XML APIs, with functionality for:  - Webstore/eCommerce - Warehouse Management - Inventory Planning - Accounting | [Click here](https://developer.retailexpress.com.au/getting-started) |
| Financial Summary | Provides daily summarised financial data and is the recommended API for all integrations with Accounting systems. | [Click here](https://v1apidocumentation.retailexpress.com.au/apis/vd0ld51rce) |
| Web Store | The eCommerce API is designed to allow a retailer to integrate a website / eCommerce system with Retail Express. This is a bi-directional Web Service. In summary, it can export products and prices, stock levels and customer information, while importing sales, payments and customers. | [Click here](https://www.retailexpressmedia.com/documentation/api/v2/Retail Express - V2 Web Store API.pdf) |
| EDS API | Event Driven Synchronisation, used for eCommerce integrations using a trigger-based system for prompting synchronisations. | [Click here](http://www.retailexpressmedia.com/documentation/api/v2/Retail Express - EDS API.pdf) |
| Inventory Planning | The Retail Express IPS API is a web service that allows Inventory Planning Systems to communicate with Retail Express point of sale and stock control software. This enables the synchronisation of products, pricing, purchase orders, internal transfers and other business-specific information to help reduce the amount of administration required to maintain an IPS. | [Click here](https://www.retailexpressmedia.com/documentation/api/v2/Retail Express - V2 Inventory Planning System API.pdf) |
| Warehouse Management | The Retail Express WMS API is a bi-directional web service that allows Warehouse Management  Systems to communicate with Retail Express point of sale and stock control software. This enables the synchronisation of products,  pricing, orders, customers and other business-specific information to help reduce the amount of administration required to maintain a WMS. | [Click here](https://www.retailexpressmedia.com/documentation/api/v2/Retail Express - V2 Warehouse Management System API.pdf) |
| Accounting Integration | Designed to send data to your Accounting System, including sales, purchases, stock adjustments and more. | [Click here](https://www.retailexpressmedia.com/documentation/api/v2/Retail Express - V2 Accounting API.pdf) |
