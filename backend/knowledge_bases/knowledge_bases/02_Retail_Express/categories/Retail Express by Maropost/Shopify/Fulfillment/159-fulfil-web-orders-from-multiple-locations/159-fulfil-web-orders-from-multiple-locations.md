# Fulfil Web Orders from multiple locations

You can enable split web order fulfillment in **Retail Express** to dispatch products from multiple outlets when a single location cannot fulfill an entire order, improving delivery speed and reducing unnecessary stock transfers. This feature helps optimize multi-store operations by allowing direct shipment from the locations with available inventory, potentially saving shipping costs and getting products to customers faster.

Setting up split web order fulfillment involves configuring fulfilment groups within your Shopify integration settings to allow orders to be fulfilled from multiple outlets. When properly configured, this capability helps ensure customers receive their orders more quickly through direct shipment from each location, reducing processing time and eliminating the delay of inter-store transfers.

Using the "Fulfil from Multiple Outlets" feature, if a web order cannot be fulfilled from a single Outlet you can separate the order into multiple fulfilments, reducing the number of Transfers, save shipping costs and dispatch the stock straight to the customer!

Split Web Order Fulfilment can be enabled/disabled per Fulfilment Group as required e.g you may wish to enable it for Outlets in one particular state or country only.

## []()To enable fulfilment from Multiple Outlets:

1. Navigate to **Settings > Integrations > Shopify > Edit > Fulfilment Locations **tab
2. Edit your **Fulfilment Group** (or use the fields at the top to create a new one)
3. Update the **Fulfil from Multiple Outlets **field
4. Click **Save**

## []()Example Scenario

With Split Web Order Fulfilment enabled, Retail Express receives the order and determines the Source Outlet to be Maroochydore and Gold Coast.

The Fulfilment Outlet is different for each line - the black shoe is fulfilled straight from the Gold Coast, and the blue shoe fulfilled from Maroochydore. This saves the staff from needing to manually intervene with the order to update the Fulfilment Location (since it's been calculated automatically), or transferring the stock between Outlets!

![rtaImage - 2023-07-08T101645.085.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppNnREAU.jpg)

1. Staff at the Gold Coast send the stock to the customer
2. Staff at Maroochydore send the stock to the customer
3. The customer receives their order in two shipments

In this example, by enabling Split Web Order Fulfilment the Maroochydore staff only have to handle stock once, dispatching it straight to the customer. The company saves down on processing time and gets the stock to the customer days faster than it would have been if they had to wait for stock to arrive first from the Gold Coast.
