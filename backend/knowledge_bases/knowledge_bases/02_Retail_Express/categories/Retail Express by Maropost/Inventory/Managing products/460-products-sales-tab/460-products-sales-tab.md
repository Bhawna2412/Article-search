# Products - Sales tab

The Sales tab provides a basic table displaying monthly sales for the previous 12 months. This is great for when you need to see figures at a glance; for more detailed reporting Retail Express offers an extensive Sales Reports via Reports > Sales Reports.

![products_editproduct_sales.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppWmEEAU.jpg)

|  |  |
| --- | --- |
| **Column** | **Details** |
| Units | The number of units (products) that sold during the month |
| Sales (INC) | The Inc Tax value of the units sold |
| GP $ | The Gross Profit $ value of the units sold |
| GP % | The Gross Profit amount as a percentage of the Sales value |
| Avg Sales (INC) | The average value that the product was sold for in the month |
