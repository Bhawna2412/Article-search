# Accounting Integration Overview

You can comprehend the complete accounting integration landscape for **Retail Express**, understanding available options including Xero, MYOB Cloud, MYOB AccountRight, and data export methods supporting various accounting workflows. This helps optimize your accounting technology decisions by understanding which integration approach best fits your business size, accounting platform choice, and operational preferences, enabling informed selection supporting efficient financial management.

Understanding accounting integration options enables you to select the most appropriate approach for your retail operations and accounting requirements. When you evaluate accounting integrations comprehensively, you can expect selection of the best-fit solution for your needs, appropriate implementation planning, realistic expectations for capabilities and requirements, optimized accounting workflows leveraging chosen integration method, and successful accounting automation supporting business growth and compliance.

Retail Express offers a variety of integration options, depending on your preferred accounting package.

Retail Express can export information to your accounting package including:

- Sales
- Purchases
- Stock adjustments
- and more

The Manual Integration option is available to use where a direct integration or partner integration is not available. To use the Manual Integration you can select from either of the following options:

- **Manual Journals** - A Financial Summary Report is available which provides details of Sales, Receipts and Inventory movements.  to assist in the creation of manual journals.
- **File Upload** - Users can download CSV files in the MYOB AccountRight 19+ file format to import summary Sales, Receipts, Supplier Invoice and Inventory Movement via the **Accounting Data Export **feature. For an automatic solution take a look at the MYOB Cloud integration.

**Supported Versions:** MYOB AccountRight 19+

> **Note:** While possible, please note Retail Express does not support using the File Upload method with alternate accounting packages. You will need to be able to perform the manual mapping process correctly for this solution to work.

View detailed information here:Accounting Data Export Overview

The Cloud Integration uses automatic, overnight synchronisation with your chosen accounting package - MYOB or Xero. The synchronisation will automatically create journals, daily invoices and more.

For MYOB, the Cloud integration supports AccountRight 2021+ where the Company File is stored online.

For more control over the accounting data, take a look at the Manual Integration method using exported files from Retail Express.
