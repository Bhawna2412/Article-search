# Advanced Excel Stocktake support for multiple Barcodes

You can streamline stocktake processes in Retail Express using Excel-based counting with multiple barcode support, helping your retail operations complete inventory audits more efficiently. This capability enables flexible stock counting workflows that accommodate products with multiple barcode identifiers, which helps reduce stocktake time and improve counting accuracy for complex inventory scenarios.

Setting up advanced Excel stocktake functionality enables your retail operations to handle inventory counts systematically, helping optimize audit efficiency and accuracy. This helps retail businesses maintain reliable inventory records through efficient counting processes that accommodate real-world product complexities and reduce the disruption caused by physical stock audits.

# Advanced Excel Stocktake support for multiple Barcodes

**Release date:** 28th June 2021

We're happy to announce updates to the Advance Excel Scan Stocktake file, including include support for Multiple Barcodes!

> **Note:** The Scan Tool is the file available to use for the Excel Advanced Stocktake method, and not the Excel file downloaded directly out of Retail Express.

Using this file you can scan your products using a barcode scanner, and it will automatically update the count of your products on the file for the matching product!

## Using additional barcodes

To use multiple barcodes, as you scan the products it will find the matching product in the list. Continue scanning to update the "Your Count" field, or click in "Your Count" and manually type in the field

> **Note:** You can use the radio buttons to filter by Barcodes specifically (excluding Product ID) if required

## Also included:

As part of this update we've also made multiple fixes to the Stocktake file where products were not scanning correctly.

In addition, error message explanations are now more clear e.g. whether the code scanned was not found in the list or if the code was matched multiple products (scanning the Manufacturer SKU for example).

Ensure you check the "Errors" tab to ensure you have resolved all identified issues before finalising your stocktake.
