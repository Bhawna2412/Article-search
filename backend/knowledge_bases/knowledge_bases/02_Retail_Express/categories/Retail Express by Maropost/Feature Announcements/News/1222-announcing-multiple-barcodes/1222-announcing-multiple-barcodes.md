# Announcing Multiple Barcodes

You can assign multiple barcodes to products in **Retail Express** to accommodate various supplier codes, promotional tags, and regional variations. This feature helps improve operational flexibility by allowing products to be scanned using different barcode identifiers, supporting supplier relationships, promotional activities, and multi-region operations across your retail network.

Setting up multiple barcodes involves configuring additional barcode fields and establishing barcode management procedures. When properly utilized, multiple barcode support typically leads to improved checkout efficiency through flexible product scanning, better supplier integration through native barcode support, and enhanced promotional capabilities across retail operations.

# Announcing Multiple Barcodes

**Release Date:** 27th April 2021

If your suppliers often change barcodes for the same product, or you have different SKUs for the same product, you can now manage multiple barcodes for a single product within Retail Express!

Announcing Multiple Barcodes - making it easier than ever search and manage your products!

## About Multiple Barcodes

Multiple Barcodes for the same product are used by suppliers for a variety of reasons, for example:

- The same product sourced for different markets e.g. Australian vs United States
- Different barcodes for different batches of products
- Updated packaging

With this new update to Retail Express you'll be able to enter up to a maximum of 20 barcodes for each product, scan any of these codes, and find the same product! Multiple Barcodes are available in POS, Stock Ordering and Receiving, Stocktakes and more!

## []()Creating Multiple Barcodes

When creating a product you'll still be required to enter a primary barcode - the Supplier SKU, as well as a Manufacturer SKU (used for grouping styles).

Additional barcodes can be added under the new "Barcodes" tab at the top:

![mceclip0.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000plspuEAA.jpg)
On the Multiple Barcodes tab you'll be able to type/scan a barcode to enter the barcode against the product. Any Users that have the rights to edit a product will be able to manage additional barcodes automatically - no additional security configuration is required.

> **Note:** Your barcode will still need to be unique i.e. the same barcode cannot be used for more than one product

![multiplebc.gif](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000plsq4EAA.jpg)

## []()Using Multiple Barcodes

Once you've created the barcodes, you'll be able to scan the code into a number of areas of Retail Express:

|  |  |
| --- | --- |
| Back Office | - Global Search - Mass Upload - Fulfilment Report - Purchase Orders - Stock Receipts - Product Search & Management - Stock Adjustments - Stock Take - Transfers |
| POS | - Create a sale - Refund - Transfers |
| Offline POS | - Create a sale - Refund - Transfers |

## []()Integrations

When creating a product using the Retail Express Unified API or Inventory Planning API, the barcode will be validated to ensure it's unique (including checking existing Multiple Barcodes).
