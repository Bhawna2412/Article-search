# Windows 7 End of Life

Microsoft officially discontinued support for Windows 7 as of **January 14, 2020**. As the product is now "End-of-Life" Microsoft no longer offers technical assistance, software updates and most importantly, security updates. [Read more about the update from Microsoft here](https://www.microsoft.com/en-au/windows/windows-7-end-of-life-support-information?ocid=MoveToWindows10).

In addition, Google has forced an update to Chrome that prevents Windows 7 users to be able to perform specific functions from the browser.
