# Post Code Ranges for Shopify Fulfilment Groups

You can configure post code ranges for Shopify fulfillment groups in **Retail Express** to optimize order routing based on customer location and outlet proximity. This feature helps reduce shipping costs and improve delivery times by automatically assigning orders to the most appropriate fulfillment location based on geographic criteria, supporting more efficient omnichannel operations.

Setting up post code ranges involves defining geographic boundaries for each fulfillment group and establishing outlet priorities within those regions. When properly configured, post code-based fulfillment typically leads to reduced shipping expenses through shorter delivery distances, faster delivery times for customers, and better utilization of multi-store inventory across your retail network.

# Post Code Ranges for Shopify Fulfilment Groups

**Release Date:** 17th May 2021

Post Codes can now be used in Shopify Fulfilment Groups, providing greater control on how you dispatch your products to your customers. With this new feature you can add:

- Individual Post Codes e.g. 4000
- Multiple Post Codes separated by a semi colon e.g. 4000;4001;4002;4003
- A range of post codes separated by a hyphen e.g. 4001-4003

Some notes about using Post Codes:

- Post Codes can only contain numbers, hyphens and semi-colons (other characters will be removed automatically, including spaces)
- A Post Code must be unique across all Fulfilment Groups i.e. the Brisbane Post Code of 4001 cannot be in two separate groups (including in a range 4000-4002).
- Post Codes can start with a zero e.g. 0800 for Darwin in NT
- Post Codes can be defined where the Country is Australia or New Zealand, and the state is set to Any or a state selected (Australia only)

### Why you should use Post Codes

The ability to use Post Code ranges allows you to create more efficient rules for shipping and fulfilment, cutting down your shipping costs and increasing the speed of delivery to your customers.

For example:

- You have several locations in Queensland and Sydney (New South Wales)
- A customer from northern New South Wales places an order
- The Queensland locations are closer to the customer than Sydney

Using Post Code ranges, you can enter the post code range for Northern New South Wales so that any customer from that region places an order online it's automatically fulfilled by your Queensland stores!
