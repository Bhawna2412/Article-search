# Style Performance Report

You can evaluate sales success at the product style level by accessing the Style Performance Report, which aggregates sales data across all variants within a style family. This style-level analysis helps retail operators assess overall style success beyond individual SKU performance, informing decisions about style continuation, reordering, and new style selection for future seasons.

Using the Style Performance Report helps retail operators identify winning style concepts for repeat orders, discontinue underperforming styles, and develop future assortment strategies based on style-level customer preferences and sales trends.

The Style Performance Report provides summary sales information, grouped by Style Code. View sales quantities, value and GP; current stock quantities and order quantities; and sell through calculations.  This report is recommended as a supplement to the Stock Sales Matrix by Colour when more detail is required.

## []()Criteria for Products to appear on the report

For a product to be grouped by Style Code, each product in the group must share following attributes:

- Matching Manufacturer SKU/Style Code (e.g. shoe123, shirtxyz)
- Matching Brand
- Matching Supplier
- Size (if Size is not applicable to your products, create a Size called N/A and assign it to each of your products)
- Colour (if Colour is not applicable to your products, create a Colour called N/A and assign it to each of your products)

> **Note:** Products grouped together by Manufacturer SKU are synchronised together as groups to various web stores, including Shopify (known as "Variable Products").

## []()Using the Style Performance Report

To use the Style Performance Report:

1. Navigate to **Reports > Product Performance > Style Performance Report**
2. Update **Filters **as required (see below for more detailed information)
3. Click **Search**
4. The **results **will be displayed
5. To export the report use the Excel and CSV export buttons in the top right-hand corner

## []()Report Filters

Most of the filtering fields are self explanatory, however see the detailed information on specific fields:

|  |  |
| --- | --- |
| **Field** | **Description** |
| Product PLU / Style Code | Filters to a specific product. Valid PLU codes are:  - Supplier SKU - Manufacturer SKU - Product ID  Enter multiple codes separated by a comma |
| Description | The Product Name. You can enter a partial description, although it will search for that exact match within the names e.g. searching for "9ct Silver" would find "Pendant - *9ct Silver*" but wouldn't find "9ct Yellow Gold Silver Bracelet". |
| Exclude Zero Stock Units and Exclude Below Zero Stock Units | Exclude products if the product currently has zero stock available i.e. even if sales have occurred during the period the product will still be excluded from the report |
