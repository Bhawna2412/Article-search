# Sales Performance Summary Report

You can quickly assess overall sales performance by accessing the Sales Performance Summary Report, which provides high-level sales metrics and key performance indicators without detailed transaction breakdowns. This summary-level reporting helps retail operators rapidly evaluate business performance, share executive-level sales data, and monitor progress against sales targets efficiently.

Using the Sales Performance Summary Report helps retail operators communicate business performance to stakeholders, quickly identify performance trends, and make rapid assessments without analyzing detailed transactional data.

Sales Performance Summary Report is a comprehensive tool designed to offer detailed insights into product performance and style reporting. This report empowers users to make data-driven decisions and enhance business operations.

-   - Provides aggregate figures based on nominated groupings such as Style/Man SKU, Outlet, Brand, Product Type, and more.
  - Allows users to view summed data for variants within a style, presenting one line per style code.

  **Group-by Concept:**
-   - Includes dedicated columns displaying sales quantities for the previous four weeks, calculated up to the current date at the time of report execution.

  **Previous 4 Weeks Sales Data:**

You will need to grant access to your user profiles via Staff > Profile Security > Edit Features > Sales Performance Summary Report

## Report Access and Configuration:

- The new report can be found under **Reports > Product Performance****> Sales Performance Summary Report**

## Filters and Functionality:

**Standard Filters:**

| **Filter** | **Description** |
| --- | --- |
| Date From | Specifies the start date for the data range-defaults to the start of the current month. |
| Date To | Specifies the end date for the data range-defaults to the current day. |
| PLU | Supports comma-separated values for Product Lookup. Use this to search for Style/s or a specific Product ID. |
| Group By | Allows grouping results by various product fields such as Style/Man SKU, Outlet, Brand, Product Type, etc. |

![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/SSR21.05TJ1000005pjA3MAI.png)

## **Advanced Filters:**

- Additional filters include Product Type, Supplier, Country, Store Type, Outlet, and various attributes such as Brand, Season, Size, and Colour.

> **Note:** Exclude Disabled Products, Exclude All Products With Zero Stock Available, Exclude All Products With Zero Sales and Show Previous Weeks Sales are selected by default. The date range impacts only the sales results. The sales and stock quantities for the previous weeks are reported as of the time the report is generated.

### Results Grid:

| Column | **Description** |
| --- | --- |
| Grouping | Groups result by a selected attribute such as Style, Brand, Product Type, etc. |
| Description | Provides details of the grouped attribute (e.g., product name, supplier name). |
| AVG COGS Ex | Shows the average cost of goods sold. |
| AVG POS Price Inc | Displays the average point of sale price, excluding discounts. |
| AVG Sold Price Inc | Includes discounts, reflecting the average sold price. |
| Qty Received | Sums quantities received or made available during the selected date range. |
| Qty Sold | Sums quantities sold during the date range. |
| Qty Avail | Sums available quantities for the selected outlets. |
| SOH Qty | Shows the sum of stock on hand quantities. |
| GP % | Calculates the average gross profit percentage. |
| Total Sold Ex | Displays the total sold value excluding tax. |
| First Received | Shows the date the product was first received. |
| Last Received | Indicates the date the product was last received. |
| Last Sold | Shows the date the product was last sold. |
| Last Fulfilled | Displays the date the product was last fulfilled. |
| Weeks Since Received | Calculates the number of weeks since the product was first received. |
| Sold Current Week | Shows the quantity sold in the current week. |
| Sold 1 Week Ago | Displays the quantity sold in the week before the current week. |
| Sold 2 Weeks Ago | Shows the quantity sold two weeks before the current week. |
| Sold 3 Weeks Ago | Indicates the quantity sold three weeks before the current week. |
| Sold 4 Weeks Ago | Shows the quantity sold four weeks before the current week. |
