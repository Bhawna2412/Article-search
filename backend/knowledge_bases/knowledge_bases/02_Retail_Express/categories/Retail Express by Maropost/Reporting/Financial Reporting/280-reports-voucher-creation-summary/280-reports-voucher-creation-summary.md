# Reports - Voucher Creation Summary

The Voucher Creation Summary report provides information on vouchers that have been created, summarised by type e.g. Credit Note, Gift Voucher etc.

Using the report you can see the following information for each Voucher Type:

- The initial value
- Amount redeemed
- Amount remaining

![mceclip0 - 2023-07-09T230536.071.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppez4EAA.jpg)

> **Note:** The report scrolls to the right to display additional content

> **Note:** For more detailed information refer to the Voucher Creation Report

To use the report:

1. Navigate to **Reports > Finance Reports > Voucher Creation Summary**
2. Use the filters as required
3. Click **Search**
4. The details will be displayed

> **Note:** Click the arrows next to the column headings to sort the data

5. The total values will be displayed at the bottom of the report
