# Item Detail Export

You can export comprehensive product-level sales data by accessing the Item Detail Export, which provides detailed transaction information for every product sale in spreadsheet format. This export capability helps retail operators perform advanced sales analysis, identify product performance trends, and share detailed sales data with suppliers or business partners for collaborative planning initiatives.

Using the Item Detail Export helps retail operators conduct sophisticated product performance analysis, support vendor-managed inventory programs, and integrate detailed sales data with external business intelligence tools for strategic decision making.

The Item Detail Export report provides detailed information on your Inventory, including the standard product information, but most importantly the number of units in each stock status/level.

As indicated by the name, the report is exported from Retail Express. The format of the report is a CSV format, which is typically opened using Microsoft Excel, but can be accessed by a number of other programs.

To use the report:

1. Navigate to **Reports > Stock Reports > Item Detail Export**
2. Use the filters to refine your search as required

> **Note:** The Qty on Hand filter includes the units in each stock status, not just Available stock

3. Click **Export to Excel**
4. The system will begin generating the file to be exported
5. Once complete, the page will refresh and display a **link under the report heading**; click the link to download the Export file
  ![mceclip0 (83).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppQhMEAU.jpg)
6. The file will download in a **compressed****zip **file (this is to speed up the download process due to the size of the file)
7. Open the **zip **file to access and **open the ****CSV file**
8. The report will be displayed

![mceclip0 (84).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppQhWEAU.jpg)
Please Note

- Packages are excluded by default.
