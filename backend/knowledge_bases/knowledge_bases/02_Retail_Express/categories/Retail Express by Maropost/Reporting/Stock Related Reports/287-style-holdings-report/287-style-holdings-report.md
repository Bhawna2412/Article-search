# Style Holdings Report

You can analyze inventory holdings at the style or product family level by accessing the Style Holdings Report, which shows stock levels grouped by style across colors, sizes, and locations. This report helps retail operators assess style-level inventory investment, identify size or color imbalances, and make strategic decisions about style continuation, discontinuation, or reordering to optimize product assortment.

Using the Style Holdings Report helps retail operators maintain balanced size and color representation within styles, identify styles requiring markdown or clearance, and optimize style-level inventory investment across product categories.

The Style Holdings Reports provides stock quantities, sorted by size, for each SKU and outlet selected. Focus your reporting on Available, Allocated or Stock on Hand quantities and view COGS, Master and Promotional Pricing.

**Considerations when using this report:**

This report only displays products that have a Style Code / Manufacturer SKU AND a Size. For a product to be 'grouped' effectively by Style Code, each product must share the following attributes:

- the same Manufacturer SKU/Style Code (e.g., shoe123, shirtxyz)
- the same Brand
- the same Short Description
- each Product must have a Size (if Size is not applicable to your products, create a Size called N/A and assign it to each of your products)
- for some Style Code-related reports it's mandatory that products share the same Supplier and also have a Colour - however, those requirements are not mandatory for this report.

> **Note:** For more information about grouping your products by Style Code, please see: Edit Product Information (Main tab > Manufacturer SKU)

## Using the Style Holdings Report

To access the Style Holdings Report:

1. Navigate to**Reports > Stock Reports > Style Holdings Report**
2. Use the **Filters** to filter the report (or leave these as default)
  ![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/Screenshot20240201at32631PM.05T5g000018qdtTEAQ.jpg)
3. Click the icons in the top-right-hand corner to export the report in an Excel spreadsheet or CSV
4. Click **Refresh **to view the latest details
  ![Screenshot 2023-02-23 at 7.41.47 PM.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppiFaEAI.jpg)

> **Note:** You can use 'Outlet and Style Code Only' option to group style products based only on style code

> **Note:** Click column headings to sort into ascending or descending order. Ensure to scroll to the right to view more column detail. You can view this report in-page, however, due to its extensive nature we recommend you Export it to Excel.

## Filters

Most of the filtering fields are self-explanatory, however, see the detailed information below for the more advanced fields.

|  |  |
| --- | --- |
| **Filter** | **Details** |
| PLU | Valid PLUs are SupplierSKU, ManufacturerSKU and Product ID. Enter multiple PLUs separated by a comma. The Style Code is the Manufacturer SKU,  used to group all variants of a particular style together. |
| Stock Status | Select the stock quantities to report on:  - Available - stock currently available for sale, that is not Allocated via invoice or transfer - Allocated - Allocated stock can be either, sold but not Fulfilled (such as a Layby) OR Allocated to another store on a Transfer - Stock on Hand - Stock on Hand (SOH) is the sum of Available + Allocated stock |
| Consolidate Styles | Tick to display the highest pricing. |

## Report Results

The Style Holdings Report will display a list of items matching your search filters. Scroll to the right to view all of the details.
