# Picking/Dispatch Summary Report

You can monitor order fulfilment efficiency by accessing the Picking/Dispatch Summary Report, which tracks picking and dispatch activity across customer orders. This report helps retail operators measure fulfilment team productivity, identify processing bottlenecks, and ensure timely order dispatch, improving customer satisfaction through faster delivery turnaround and more efficient warehouse operations.

Using the Picking/Dispatch Summary Report helps retail operators optimize fulfilment workflows, track staff productivity in warehouse operations, and identify process improvements that reduce order-to-dispatch cycle times.

The Picking/Dispatch Summary Report is designed to provide a high-level overview of inventory ready to be Picked or Dispatched across your Outlets. It is exclusively for Transfers (transfer-only statuses). The report consists of the details regarding picking or dispatching every item separately. It displays a detailed view of the total outstanding available quantity, cost and cubic meters to Pick and Dispatch.

## Using the Picking/Dispatch Summary Report

To use the Picking/Dispatch Summary Report:

1. Navigate to **Reports > Stock Reports > Picking/Dispatch Summary Report**
2. Select from the **Filters **(see below for more detailed information)
3. Use the Filters to filter the report (or leave these as default)
4. Click the **Search **button located at the bottom left below the filters
5. The Results will be displayed showing all individual items matching the filters
6. Click the icons in the top-right-hand corner to export the report in an Excel spreadsheet or CSV
  ![Screenshot 2023-01-12 at 10.32.12 AM.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppVgeEAE.jpg)
7. Click **Refresh **to view the latest details

## Filters

![Screenshot 2023-01-06 at 1.15.14 PM.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppVwDEAU.jpg)
We've listed the Filters below in detail.

|  |  |
| --- | --- |
| **Filter** | **Details** |
| Product PLU | Valid PLUs are SupplierSKU, ManufacturerSKU and Product ID. Enter multiple PLUs separated by a comma. |
| Home/WH From and Home/WH To | Used to narrow the results to a particular date range of the Customer's Order for Home Delivery or Warehouse **Pickup**. |
| **Transfer **Number | Each transfer/ITO has its unique code, the Transfer Number. So you can see which transfer number particular stock was on when it was transferred between stores. |
| **Outlet** From | Helps select the outlet from where the order is transferred to the determined outlet. |
| **Outlet** To | Used to select the outlet the order is picked up from or delivered to. |
| Supplier | For choosing the supplier name from the scroll-down list. |

## Summary

![7.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppVzQEAU.jpg)
ITO = Internal Transfer Order
PO = Purchase Order

The Picking/Dispatch Summary shows the total quantity, cost and cubic meters of everything Picked, Dispatched, Fulfilled, and POs Received in the last 7, 14, 28, 56 and 84 days. It also shows the quantity, cost and cubic meters of stock due on POs Inbound in the next 7, 14, 28, 56 and 84 days.
