# Stock Control Report

You can monitor inventory control metrics and stock accuracy by accessing the Stock Control Report, which provides comprehensive inventory management data including stock levels, reorder points, and movement patterns. This report helps retail operators maintain optimal inventory control, identify stock management issues, and ensure inventory accuracy across products and locations.

Using the Stock Control Report helps retail operators maintain effective inventory management practices, identify products requiring attention, and support continuous improvement in inventory accuracy and control processes.

The Stock Control Report offers a comprehensive overview of your inventory, showing the total quantity and value of products, helping you maintain accurate stock levels.

This report allows you to assess stock performance, making it easier to manage inventory across various locations and ensuring you have full control over your product data.

The Stock Control Report is used to get visibility of the current status of products over multiple Outlets, including their total quantity and value.

## Using the Stock Control Report

To use the Stock Control Report:

1. Navigate to **Reports > Stock Reports > Stock Control**
  ![mceclip2 (91).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppi6TEAQ.jpg)
2. Select an **Outlet **(hold Ctrl or Cmd to select multiple)

> **Note:** The search results will be displayed for products that match any Outlet (not both)

3. Select the Product Types from the **Products** list
4. Select the **Sort Order **for the results to be displayed in
5. Select Yes or No to **Group by Product Type **(if left ungrouped individual products will be displayed on their own lines
6. Update the rest of the filters as required
  Products will be displayed that match *any *filter
7. Click **Search**
8. The **results **will be displayed
9. Click **Print **to print a copy of the report

## Stock Control Results

![mceclip3 (60).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000pphtIEAQ.jpg)
The results will display a single line per product containing information covered in more detail below.

|  |  |
| --- | --- |
| **Column** | **Details** |
| Product Description | Displayed if Group by Product Type is set to No. Includes:  - ID (Product ID) - SKU (Supplier SKU) - Code (Manufacturer SKU) - Short Description |
| Buy Ex | Global Buy Ex price (as per the Main tab of a product) |
| Sell Inc | Global POS Price (as per the Main tab of a product) |
| Location & Status | A breakdown of the inventory levels for each outlet and the POS price per Outlet.  - Available - Allocated - Picked - Transfer Out - Transfer In - Received - Requested - Ordered - On return - Faulty  The bottom two rows include the Total Quantity and Total Value per inventory status |

If you've chosen to group your products by Product Type, the products will be listed individually but there'll be an additional subtotals row including the number of products in the group and the total quantity and value for the entire group.
