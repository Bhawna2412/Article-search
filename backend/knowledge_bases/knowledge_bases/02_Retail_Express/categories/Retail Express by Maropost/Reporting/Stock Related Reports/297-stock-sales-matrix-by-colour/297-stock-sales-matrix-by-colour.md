# Stock Sales Matrix by Colour

You can analyze sales and inventory patterns by color across product styles by accessing the Stock Sales Matrix by Colour, which presents data in an easy-to-scan grid showing sales velocity and stock levels for each color variant. This visual analysis tool helps retail operators identify best-selling colors, optimize color assortment, and make informed reordering decisions based on color preference patterns.

Using the Stock Sales Matrix by Colour helps retail operators balance color representation within styles, identify trending colors for increased investment, and phase out underperforming color variants to optimize inventory mix.

Find [**related articles**](https://galaxy.maropost.com/kb/articles/297-stock-sales-matrix-by-colour#Faqs)** **at the bottom of this page

The Stock Sales Matrix by Colour reports on a given style, by colour, so you can easily identify the quantities sold for a style, colour, and size at each Store - as well as review the total sales information for each month.

This report is recommended for reporting on an entire *Style Code s*ales, instead of viewing sales for each individual products colour and size.

## []()Required Product Configuration

For a product to be grouped by Style Code (and appear on this report correctly) all products must meet the following rules:

- Attributes **must match** for all products:  - **Manufacturer SKU/Style Code** must match for all products e.g. shoe123, shirtxyz
  - **Brand**
  - **Supplier Code**
- An attribute **must be selected** for each product (they don't need to match)  - **Size** - if Size is not applicable to your products, create a Size called N/A and assign it to each of your products
  - **Colour** - if Colour is not applicable to your products, create a Colour called N/A and assign it to each of your products

> **Note:** For more information about grouping your by products by Style Code, please refer to the Grouping Products article

## []()Running Stock Sales Matrix by Colour

To use the Stock Sales Matrix by Colour:

1. Navigate to **Reports > Product Performance > Stock Sales Matrix by Colour**
2. Update the **Filters** as required

> **Note:** You can enter a partial description to return results e.g. typing "shift" will display all shift dresses. You can also use the filter 'Group Styles Based On' to filter results based on Style Code, Brand, Suppliers, Size and Colour
