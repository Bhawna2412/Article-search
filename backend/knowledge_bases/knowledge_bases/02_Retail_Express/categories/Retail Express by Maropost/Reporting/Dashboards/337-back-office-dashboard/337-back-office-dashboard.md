# Back Office Dashboard

You can use the **Retail Express** Back Office Dashboard to visualize critical business metrics with comparative analysis across date ranges, in-store versus online channels, and historical performance trends. This improves your retail operations efficiency by providing comprehensive visibility into sales revenue, transactions, gross profit, hourly patterns, top products, customer behavior, and staff performance, with properly configured dashboard analytics typically enabling faster strategic decisions and better overall business performance.

Understanding back office dashboards involves recognizing date comparison capabilities, sales channel filtering options, and multi-tab reporting structures to help retail operators focus on the right performance indicators, ensuring your retail strategy consistently drives better data-driven decision-making and business growth outcomes.

The Back Office Dashboard supplies a visual representation of the most vital and frequently requested information, with the ability to compare data based on date ranges or periods, and for instore vs online sales.

To access the dashboard, click **Dashboard **in the top menu bar within Retail Express.

### Feature Overview

![mceclip0 (3).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000pnLAOEA2.jpg)[Click to view full size](https://ress.zendesk.com/hc/article_attachments/4649611830799/mceclip0.png)

1. **Tabs** - click each tab to view a list of new metrics
2. **Sales Filter **- drill into specific stores or online sales channels, or compare instore and online sales.
3. **Date Filter **- drill into preconfigured or custom date ranges, and the ability to compare to previous periods or years
4. **Date Range **- displays the comparison date range
5. **Trend Indicators **- displays a positive or negative trend based on the compared data

### Sales Filters

![mceclip1 (2).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000pnLAYEA2.jpg)

|  |  |
| --- | --- |
| Filter | Details |
| All Sales | All sales for the selected date period regardless of origin |
| In Store Sales | Orders created in POS |
| Online | Orders were created through an external Web Service/API or Retail Express Shopify Connector |
| In Store vs Online | Compares In Store Sales with Online Sales When this filter is selected you will be able to select a date range to filter results, however, you will be unable to use the date comparison feature to compare two different date periods |

### Date Range

![mceclip2 (3).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000pnLBCEA2.jpg)

|  |  |
| --- | --- |
| Filter | Details |
| Date Range | A drop-down with pre-defined periods to easily select from |
| From (Displayed when using a Custom date) | Allows you to select a specific date from/to range. The total date range is restricted to 12 months when using the "Compare to" feature.  > **Note:** Large data queries may take longer to be displayed |
| Compare To | Used to compare two different date ranges against each other:  - Previous Period - The comparison date range will be automatically selected for the same length of time immediately prior to the selected custom date e.g. selecting "This Month" will automatically select Last Month for comparison, or "Last 30 days" will compare with the 30 days before that - Previous Year - the same period of time from the year before e.g. May 2021 vs May 2020 |

## Report Tabs

### []()Trends

![mceclip3 (1).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000pnLBREA2.jpg)

|  |  |
| --- | --- |
| Report | Details |
| Sales Revenue | The amount of sales excluding tax as per the Sales Report To include sales for the entire last day of the selected period you will need to select the day *after* the last day e.g. to report on all of April, you will need to set the "To" date as at 1st of May |
| Transactions | Number of Transactions over the given period as per the Sales Report |
| Gross Profit $ | The Gross Profit Value as per the Sales Report |
| Gross Profit % | Gross Profit Percent as per Sales Report |
| Units per Transaction | Number of units per transactions as per the Sales Report |
| Average Transaction Value | The average value of all transactions for the period as per the Sales Report |
| Discount $ | The value of discounts given as per Sales Report |
| Discount % | The % value of discounts as per the Sales Report |
| Pickup Items vs Delivery Items | A percentage of the items (both fulfilled and unfulfilled) that were either Pickup items (Store Pickup or Warehouse Pickup) or Delivery items. |

### []()Hourly Sales

![mceclip4 (2).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000pnKIXEA2.jpg)

|  |  |
| --- | --- |
| Report | Details |
| Hourly Sales Value | Spanning 24 hours, this graph indicates the sum of Sales Revenue for all sales placed during the date range, grouped by hour e.g. a sale at 9am on Tuesday and a sale at 9am Wednesday will both appear at 9am. |
| Hourly Sales Count | A sum of transactions for all sales placed during the selected date range grouped by hour |

Sales are timestamped using Sydney/AEST time

Top 20 Customers by Revenue Report:

|  |  |
| --- | --- |
| Column | Details |
| Spend | Sum of the sales ex tax in the selected period for the customer |
| GP $ | Gross Profit dollar value |
| GP % | Gross Profit percentage value |
| IPT | Items Per Transaction average - total quantity of items divided by the number of transactions |
| ATV | Average Transaction Value - sum of sell price (including tax) divided by the quantity sold. |
| Trans | The number of sales per customer for the selected period |
