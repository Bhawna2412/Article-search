# Marketing Cloud Release Notes – 19 Nov 2020

## Overview

With some major upcoming releases, this release includes some minor enhancements and bug fixes. With the new changes, the Growth & Attrition reporting is more accurate and helpful. Also, take note of the changes to the  Navigation menu.

To learn more about these updates, read on!

## Enhancements

### Growth Calculation in Growth & Attrition Reports

The growth reporting in the Growth & Attrition report is now more precise and detailed. As compared to earlier reports where the growth was calculated based on subscribed status being updated within the specific time frame, the calculation now is done based on status being updated from unsubscribed to subscribed. This will help weed out the inflated numbers that occur due to temporary soft bounces.

Now, the report also shows new metrics that provide more details to the growth data. You'll be able to see the exact numbers of contacts that are new to the list and contacts that are resubscribers.
