# Marketing Cloud Release Notes – 7 Jul 2020

## Overview

Starting this release, you get the freedom to design your journeys based on order status. Also, check out the bug fixes article to see which bugs were fixed.

To learn more about the update, read on!

## Enhancements

**Ability to Filter Contacts Based on Order Status in Journeys**

In the Product Purchase journey trigger, you can now filter the contacts based on order status. So, now you can create separate journeys that will trigger specifically for particular order statuses, for example, sale, refund, abandon, or cancel. 

On the Product Purchase dialog box, you will notice the new 'Order Status' filter as a text field along with a checkbox. To filter the contacts based on a specific order status, select the checkbox, and then enter the specific order status in the text box.

To learn more about the Product Purchase journey trigger, see [Journey Triggers](https://galaxy.maropost.com/kb/articles/735-journey-triggers).
