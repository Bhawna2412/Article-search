# Marketing Cloud Release Notes – 2 Jul 2020

## Overview

Released as a single feature enhancement, the revamp of custom campaign reports is one of many improvements planned for the reporting section.

To learn about what changed, read on!

## Enhancements

### **Enhanced User Experience for Custom Campaign Reports**

As part of Maropost's user experience transformation, we reevaluated and improved the report-building flow for the custom campaign reports. The UI has also been redesigned as per our latest design standards, leading to improvements in overall look-and-feel and user experience. While the number of steps remains the same, you will now notice slight changes in how you build the report in each step.

To learn about the custom campaign reports, see [Creating a Custom Campaign Report](https://galaxy.maropost.com/kb/articles/835-custom-email-campaign-report).
