# Marketing Cloud Release Notes – 18 Nov 2020

You might have noticed some things have moved! To better your experience, reduce required clicks to complete tasks, and to give better context into how features are used, we have relocated the following features: 

1. Web tracking is no longer under “Acquisition” but can now be found under "**Audience**"
2. Campaign tag has moved from “Audience” to "**Campaigns**"
3. We have created a new menu section within the main nav called "Social" for all social media marketing needs

# Coming Soon

In accordance with larger user interface enhancements, we will soon be moving and adding the following features to the navigational menu: 

1. New section in main nav "Integrations" which will cover our integrations, which are currently found under “Connections” within the account drop-down menu
2. New page in the account drop-down "API documentation"
3. New page in the account drop down "Settings" which will contain setup for relational tables and FTP transfers, API keys, and JSON web token
