# Marketing Cloud Release Notes – 9 Jun 2020

## Overview

This release is a medley of enhancements touching upon many different features. As part of our commitment to Maropost's user experience transformation initiatives, the email template library and custom reports have been modernized. In custom reports, a new report type - deliverability report - has been added. Also, notable is the ability to export the cleansed list, which has been added based on an idea submitted on the Innovation Portal.

For more information, read on!

## Enhancements

### Revamped the Email Template Library

As part of Maropost's user experience transformation, the template library UI has been improved to meet our new design standards. You will notice efficient workflow, clearly-defined labels, and an attractive look-and-feel.

To learn more about the email template library, see [Template Library](https://galaxy.maropost.com/kb/articles/703-template-library).

### Added Export Functionality for Cleansed Lists

In this release, we added the functionality for exporting the cleansed list. The cleansing rules are run as automated processes and a notification is sent on completion of the process. So, now you will get the notification along with an option to download the list of contacts that were cleansed as part of the cleansing rule process. The downloadable file is available in .csv format and can be downloaded from the notification panel and Cleansing Rules page.

To learn more about the [Cleansing Rules](https://galaxy.maropost.com/kb/articles/1002-cleansing-rules).

### Introduced Deliverability Reports

This release, we introduced a new custom reporting tool - Deliverability Reports. The deliverability report is an analytical report that helps you access the success of your campaigns based on the deliverability data. You can create a deliverability report on the Custom Reports page (Navigation > Analytics > Custom Reports).

To learn more about deliverability reports, see [Creating a Deliverability Report](https://galaxy.maropost.com/kb/articles/839-deliverability-report).
