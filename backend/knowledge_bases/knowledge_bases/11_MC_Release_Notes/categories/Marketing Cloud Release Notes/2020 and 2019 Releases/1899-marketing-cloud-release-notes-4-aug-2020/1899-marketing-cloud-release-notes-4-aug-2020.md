# Marketing Cloud Release Notes – 4 Aug 2020

## **Overview**

We are excited to bring you some amazing enhancements in this release. We added a new custom report - SMS campaign report - that you can use to track your SMS campaigns. The improved image library is now available, so be sure to check that out. Keeping track of Maropost's releases is made easy now with the introduction of the Release Notes and Product Updates widget.

To learn more about the changes, read on!

## **Enhancements **

### **SMS Campaign Reporting in Custom Reports**

In this release, a new custom report - SMS campaign report - has been introduced. Using SMS campaign reports, you can get a detailed summary of all your selected SMS campaigns sent within a time period. The report can be customized to include the SMS campaigns and metrics that you want to see. Like any other custom report, you can determine the report file type, schedule it for once or recurring, send the report over email, and so on.

To learn more about how to create custom SMS campaign reports, see [Custom SMS Campaign Report](https://galaxy.maropost.com/s/article/Custom-SMS-Campaign-Report).

### **Improved Image Library**

As part of Maropost's user experience transformation initiative, the Image Library has been revamped. The improved UI makes adding and managing your images simpler and more efficient. Some of the notable enhancements are the ability to toggle between list view and grid view, navigating between images in preview mode, improved experience when organizing images into folders, clearly-defined actions, and many more!

To learn more about the image library, see [Image Library](https://galaxy.maropost.com/s/article/Image-Library).

### **Release Notes and Product Updates Widget**

The Dashboard now contains a new widget called the "Release Notes and Product Updates" widget. You can use this widget to stay informed of the latest releases and product updates. 

To add the widget, click the Add Widget button on the Dashboard and then select Release Notes and Product Updates.

To learn about the dashboard widgets, see [Dashboard Walkthrough](https://galaxy.maropost.com/s/article/Dashboard-Walkthrough).

### **Contact Overview**

More customer engagement cards were added to the Contact Overview page to bring you commerce-related insights at a glance.
