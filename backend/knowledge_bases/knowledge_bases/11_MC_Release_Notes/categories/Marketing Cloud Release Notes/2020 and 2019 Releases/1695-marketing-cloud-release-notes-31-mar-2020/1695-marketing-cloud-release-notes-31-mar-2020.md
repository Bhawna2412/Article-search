# Marketing Cloud Release Notes – 31 Mar 2020

## Overview

In this release, we are introducing some enhancements and bug fixes. Now, you have the convenience of importing contact tags along with your contacts, and it doesn't require a separate process. The Add Tags field introduced in the user interface enables you to specify tags when adding new contacts or editing contacts in a list. Similar to the journey actions, the data journey actions also allow viewing and using an unlimited Facebook custom audience. Head over to the Bug Fixes article to see what has been fixed.

Read on for more details!

## Enhancements and Changes

### **Import Contact Tags Along with Contacts**

Earlier, you were required to perform two import processes if you wanted to import a contact list and its contact tags. With the new enhancement in place, you can now import the contact tags along with the contact list in a single import.

> **Note:** The process for importing contact tags is still available, and you can still use it to import contact tags separately. 

To learn more about importing contacts and using contact tags, see [Import a Contact List](https://galaxy.maropost.com/kb/articles/11-importing-contacts) and [Contact Tags](https://galaxy.maropost.com/kb/articles/917-contact-tags).

### **Specify Contact Tags When Adding a Contact to a List**

In the user interface, you can now specify the tags for a contact when adding the contact to a list. The **Add Tags** field is introduced on the New Contact page (*Contacts index page > View Contacts [of a list] > New Contact*). The field is also available when editing a contact (*Contacts index page > View Contacts [of a list] > Edit Contact*). 

To learn more about contact lists and contact tags, see [Contact Lists](https://galaxy.maropost.com/kb/articles/5-contact-lists-overview), [How to Create a New Contact List](https://galaxy.maropost.com/kb/articles/4-creating-a-contact-list), and [Contact Tags](https://galaxy.maropost.com/kb/articles/917-contact-tags).

### **Facebook Custom Audience - Data Journey**

Similar to the journey actions, the restriction of 25 custom audiences has been lifted from the data journey too. You can now view and use an unlimited Facebook custom audience in your data journey actions.

To learn more about data journey actions, see [Data Journey Actions](https://galaxy.maropost.com/kb/articles/861-data-journey-actions).
