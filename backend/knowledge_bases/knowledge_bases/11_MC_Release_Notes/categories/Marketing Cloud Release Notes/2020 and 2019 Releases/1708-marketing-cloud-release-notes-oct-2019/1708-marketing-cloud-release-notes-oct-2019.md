# Marketing Cloud Release Notes - Oct 2019

## What’s New

Access the [Maropost Status Page](https://status.maropost.com/) for up-to-date notifications of service interruptions and scheduled downtime. You can also proactively subscribe to get notifications in your inbox. 

## Improvements and Bug Fixes

- Users will now see a 120-character count while writing a pre-header for their campaign. If you want to see more, check out this [Help Center article](https://galaxy.maropost.com/kb/articles/998-creating-sms-campaigns).

- 

## Segmentation Improvements

- Previously, the Random filter combined the number of active and inactive users. Now, users will be able to use the Random filter to create a list displaying the final count of active contacts only. If you want to see more, check out this [Help Center article](https://galaxy.maropost.com/kb/articles/918-segment-rules-in-legacy-segment-builder).
