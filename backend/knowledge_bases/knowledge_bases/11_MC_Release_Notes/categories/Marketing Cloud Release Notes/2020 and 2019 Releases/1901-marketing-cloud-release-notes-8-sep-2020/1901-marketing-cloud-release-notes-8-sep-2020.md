# Marketing Cloud Release Notes – 8 Sep 2020

## **Overview**

This release contains improvements to the merge content feed feature and further enhancements to the campaign tag option in the campaign builder. You can now merge up to 12 content feeds in your email content. You can create new campaign tags on the campaign builder flow, too.

To learn more about the changes, read on!

## **Enhancements**

### **Merge Up to 12 Content Feeds**

You can now merge up to 12 content feeds when using content feeds for your email content. Earlier, this number was limited to 8.

To learn about content Feeds, see [Content Feeds](https://galaxy.maropost.com/s/article/Content-Feeds).

### **Create New Tags in Campaign Creation Flow**

You can now create new tags during the campaign builder flow, too. When creating a campaign, if you realize that you need a new campaign tag, then you don't have to go to the Campaign Tags page to create a tag anymore. You can quickly create the campaign tag on the campaign builder page itself.

On Step 1 of the campaign builder flow, in the Campaign Tags field, enter a name for the new tag. If there is no existing tag with the same name, you will see a Create Tag option next to the new tag name in the drop-down list. Click the Create Tag option, and your tag gets created.

To learn about campaign creation and campaign tags, see [Creating Email Campaigns](https://galaxy.maropost.com/s/article/Creating-Email-Campaigns) and [Campaign Tags](https://galaxy.maropost.com/s/article/Campaign-Tags).
