# Marketing Cloud Release Notes – 2 Apr 2020

## **Overview**

In this release, we bring you an assortment of features and enhancements that have been ideated from your feedback and requests. We brought back a few functionalities (missing in the new user interface) with the added benefit of a better user experience. Check out the enhanced contact overview, folder management, and dashboard widgets.

Read on for details!

## **New Feature**

### **Custom Filter Option in Customer Engagement Journey**

You now have a new filtering option called "Custom" in the Customer Engagement Journey region of the Contact Overview page. You can use this option to view selected records based on a chosen date range. Another noteworthy point is that it is not restricted to the last 30 days. So, if you were to look for a record from a few months back, this is the way to go.

 On the calendar, you can select a range of days for which you want to view your engagement journey record. 

> **Note:** You can select a date range up to 31 days only.

As shown in the following image, when you select the date range, the selected days get highlighted. Once you click OK, the selected date range is displayed in the drop-down field.

To learn more about Contact Overview, see [Viewing a Contact Overview](https://galaxy.maropost.com/s/article/Viewing-a-Contact-Overview).

## **Enhancements and Changes**

### **Campaign Performance Metrics in Customer Engagement Journey**

We reintroduced the performance metrics data for campaigns in the Customer Engagement Journey region of the Contact Overview page. The data includes sends, opens, clicks, and bounces, and provides helpful insights on how the contact is interacting with your campaigns.

To learn more about Contact Overview, see [Viewing a Contact Overview](https://galaxy.maropost.com/s/article/Viewing-a-Contact-Overview). 

### **Introduced Drag-and-Drop Functionality in Folder Management**

You will experience a great deal of ease and convenience in managing your folders now that the drag-and-drop feature is available in the new user interface for the Folder panel. 

To learn more about folder management, see [Folder Management](https://galaxy.maropost.com/s/article/Folder-Management). 

### **Optimized Graph Quality in Dashboard Widgets**

The dashboard widgets with visual cues have been optimized for superior graphical representation. What it means for you is that the graphs in dashboard widgets are now easier. We got rid of the unnecessary breakpoints, making the graphs more efficient and readable.

To learn more about the dashboard and its widgets, see [Dashboard Walkthrough](https://galaxy.maropost.com/s/article/Dashboard-Walkthrough).
