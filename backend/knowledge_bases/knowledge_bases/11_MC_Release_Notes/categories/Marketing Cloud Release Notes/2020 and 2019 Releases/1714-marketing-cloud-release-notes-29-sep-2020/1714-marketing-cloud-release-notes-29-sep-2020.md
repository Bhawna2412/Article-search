# Marketing Cloud Release Notes – 29 Sep 2020

## Overview

Over the last few days, we launched several enhancements and bug fixes. In this release notes, you will find the details of all major enhancements since the last release notes. The notable ones are the UI revamp for Contact Lists features, the introduction of reCAPTCHA in Signup Forms, and duplication functionality in Journey elements.

To learn more about the changes, read on!

## Enhancements

**Contact Lists UI Refresh**

As part of our user experience transformation commitment, we revamped all the UI pages related to contact lists. This includes the Contact Lists index page and related pages (such as contact overview and view contacts pages), the creation and update pages, and the import and export pages.

While there are no changes to the existing functionalities, the user interface and feature accessibility have been improved to give you the best experience with our platform.

To learn about contact lists, get started here: [Introduction to Contact Lists](https://galaxy.maropost.com/kb/articles/5-contact-lists-overview).

**ReCAPTCHA in Signup Forms**

To further strengthen the security aspect in Signup Forms, we introduced the reCAPTCHA option. When enabled, this feature will keep your forms safe from bot and spam attacks. You can enable the reCAPTCHA in Step 1 of the Signup Form creation flow.

![Enable_reCAPTCHA.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/05T5g00000BkWkS.05T5g00000BkWkSEAV.png)

The reCAPTACHA option currently uses Google reCAPTCHA and is very easy to implement. To know more about the feature and how to implement it, see [Signup Forms](https://galaxy.maropost.com/kb/articles/31-signup-forms) and [ReCAPTCHA in Signup Forms](https://galaxy.maropost.com/kb/articles/29-recaptcha-in-signup-forms).

**Journeys - Duplicate Elements**

You can now easily duplicate an existing Journey element when creating a Journey. With a single click, you can create an exact copy of a Journey element along with its filled information.

![Journey_Element_-_Duplicate.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/05T5g00000BkWkT.05T5g00000BkWkTEAV.png)
