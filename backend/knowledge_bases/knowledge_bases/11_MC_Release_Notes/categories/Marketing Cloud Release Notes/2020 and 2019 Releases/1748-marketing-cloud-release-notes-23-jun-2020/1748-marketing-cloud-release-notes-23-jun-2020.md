# Marketing Cloud Release Notes – 23 Jun 2020

## Overview

With the team busy making quality improvements, we still made sure to work on feature enhancements that were promised. So, in this release, we bring you enhancements to the Email Usage widget and Acquisition Forms. Stay tuned for more to come in the next few days.

To learn about the updates from this release, read on!

## Enhancements

### **Added 'Year to date' Option in Email Usage widget**

The Email Usage widget now has the option to show both monthly and annual email usage data. Using the radio buttons, you can toggle between the monthly usage and annual usage.

To learn more about the dashboard widgets, see [Dashboard Walkthrough](https://galaxy.maropost.com/kb/articles/44-dashboard-walkthrough).  

### **Ability to Define a Checkbox as a Mandatory Field in the Acquisition Forms**

You can now define a checkbox as a mandatory field in the acquisition forms. Using this functionality, you can ensure that your customer selects the checkbox before submitting the form. Now, to try it out, in an acquisition form, add a checkbox. Select the checkbox element to view the properties on the left panel. You will notice a new section - Element Validations, where you can select the Required field to make the checkbox element on your acquisition form a mandatory field.

To learn more about the acquisition forms, see [Acquisition Form Builder: Creating Your Lightbox Forms](https://galaxy.maropost.com/kb/articles/718-creating-and-using-acquisition-forms).
