# Marketing Cloud Release Notes – 29 Jan 2025

## **OVERVIEW**

Hello, Maropostians!!

This release introduces custom RFM settings for more accurate insights.

## **eRFM REPORT ENHANCEMENT**

We have added a settings feature in the eRFM module that can be used to modify the **Recency, Frequency, **and** Monetary** definitions.

![erfm19](https://us.v-cdn.net/6038474/uploads/attachments/migrated/erfm19.068J1000008JjTlIAK.png)Each definition is based on a scoring methodology that you can calibrate according to your business requirements. Higher scores correspond to higher ranks, thus indicating a high-value customer.

![erfm20](https://us.v-cdn.net/6038474/uploads/attachments/migrated/erfm20.068J1000008JjTvIAK.png)Adjusting these definitions will help you align the scores to your brand’s purchase cycles more closely.

**Note:** To maintain a unified timeframe and a consistent RFM score, the frequency and monetary calculations are automatically adjusted to the recency period.

To learn more about this feature, read our [**eRFM Report**](https://galaxy.maropost.com/s/article/Unlock-Customer-Insights-with-eRFM-Analysis#Calculating%20RFM%20scores) article.
