# Marketing Cloud Release Notes – 14 Feb 2025

## **OVERVIEW**

As part of our ongoing commitment to security, we have updated our authentication procedures.

To learn more, read ahead!! 

## **AUTHENTICATION-BASED ENHANCEMENT**

Previously, as a first-time log-in user, after you’ve added your username and password, you would be taken to a **Multi-Factor Authentication** (**MFA**) page, wherein you would be asked to choose either Text-based messaging or a Security app code to verify your account credentials.

However, with this update, upon verifying your account credential through an OTP (via SMS or security app), you (as a first-time user) will be redirected to a** Multi-Factor Authentication** (**MFA**) page and be asked to scan the QR code (via an Authenticator app) to receive a confirmation code that completes the MFA setup process. 

![authenticate16](https://us.v-cdn.net/6038474/uploads/attachments/migrated/authenticate16.068J10000091j2lIAA.png)
This is a one-time procedure, and once you’ve completed it, you can henceforth log in to your account by first adding your login credentials and then verifying them with a code from your Authenticator app. 

![authenticate30](https://us.v-cdn.net/6038474/uploads/attachments/migrated/authenticate30.068J10000091j3eIAA.png)
**Note:** All users must enable Authenticator app-based MFA since SMS-based MFA has been officially discontinued.

To learn more about multi-step account validation, read our [**Multi-Factor Authentication (MFA)**](https://galaxy.maropost.com/s/article/Multi-Factor-Authentication) article.

## **IDENTITY APP ENHANCEMENT**

We have moved the **Security** section from the Navigation panel of Maropost’s Identity app to the **User Profile** section.

![authenticate14](https://us.v-cdn.net/6038474/uploads/attachments/migrated/authenticate14.068J10000091j3yIAA.png)By clicking on **Update Password** under the Security section, you can add a new password without entering the old credentials from the password reset page.

![authenticate28(1)](https://us.v-cdn.net/6038474/uploads/attachments/migrated/authenticate281.068J10000091fykIAA.png)
To learn more about this application, read our [**Maropost Identity App**](https://galaxy.maropost.com/s/article/Using-Maropost-Identity-App) article.
