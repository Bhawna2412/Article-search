# Marketing Cloud Release Notes – 27 Jan 2022

## **OVERVIEW**

This release consists of a new enhancement for the Contact Search widget, enabling you to search for SMS contacts from the Dashboard.

For more information, read on!

## **ENHANCEMENT**

### **Contact Search widget - Searching SMS Contact**

The Contact Search widget on the application dashboard now enables you to search for SMS contacts too! The radio buttons on the widget let to select whether you want to search for email contacts or SMS contacts. 

Enter the complete email address (for email contacts) or full phone number in international format (for SMS contacts) and press enter. You'll be redirected to the contact overview page that provides a 360-degree view of the contact.

For more information, see the [Contact Search widget](https://galaxy.maropost.com/s/article/Dashboard-Walkthrough#ContactSearch).
