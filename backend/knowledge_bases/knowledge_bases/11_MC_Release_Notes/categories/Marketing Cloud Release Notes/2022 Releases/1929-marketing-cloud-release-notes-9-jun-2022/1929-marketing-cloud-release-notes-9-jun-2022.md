# Marketing Cloud Release Notes – 9 Jun 2022

**OVERVIEW**

This release contains minor but critical changes to the process of creating Keap connections.

For details, read on!

**ENHANCEMENT**

**Changed Process for Connecting With Keap**

The process of connecting your Marketing Cloud with Keap accounts has been changed. The new connections created through the changed process are more efficient in synchronizing data between Keap and Marketing Cloud.

A new step has been introduced which requires you to provide API Key and Secret. You can continue to authenticate your Keap credentials and then allow the connection, same as before.

![Keap Connection Settings](https://us.v-cdn.net/6038474/uploads/attachments/migrated/KeapConnectionSettings.0685g000008Kz8oAAC.png)

So, remember! Before you begin creating a new connection, you must generate and retrieve the API Key and Secret from Keap.

For detailed information on the process, see [**Adding a Keap Integration (NEW UI)**](https://galaxy.maropost.com/s/article/Adding-a-Keap-Integration-New-UI).
