# Marketing Cloud Release Notes – 1 Nov 2022

## **Overview**

This release consists of a complete UI revamp of the shopping cart integrations supported by Marketing Cloud. Also, in custom SMS campaign reports, you can now include test messages.

To learn more, read on!

## Enhancements

### **Shopping Cart Integrations UI Revamp**

The user interface for the shopping cart integrations supported by Marketing Cloud – [**1ShoppingCart**](https://galaxy.maropost.com/s/article/Integration-with-1ShoppingCart), [**UltraCart**](https://galaxy.maropost.com/s/article/Integration-with-UltraCart), [**Nanacast**](https://galaxy.maropost.com/s/article/Integration-with-Nanacast), [**Shift4Shop**](https://galaxy.maropost.com/s/article/Integration-with-Shift4Shop), and [**ClickBank**](https://galaxy.maropost.com/s/article/Integration-with-ClickBank) – have been revamped as per our current design standards. (Click the links to learn more about each of these integrations)

These integrations can now be created and managed on the Integrations page. In Navigation, select INTEGRATIONS.

![Integrations Page](https://us.v-cdn.net/6038474/uploads/attachments/migrated/IntegrationsPage.0685g00000BFBkRAAX.png)

Following a consistent design template, the index pages show all the existing connections and enable you to create new ones and manage existing ones.

Here's a screenshot of the ClickBank Integration index page:

![ClickBank Index Page](https://us.v-cdn.net/6038474/uploads/attachments/migrated/ClickBankIndexPage.0685g00000BFBjTAAX.png)

Besides the user-friendly design, we have simplified the process of adding new connections – requiring minimal steps from your end.

Here's a screenshot of the Add Integration page for the ClickBank integration:

![ClickBank - Add Integration](https://us.v-cdn.net/6038474/uploads/attachments/migrated/ClickBankAddIntegration.0685g00000BFBjxAAH.png)

### **Custom SMS Report – Include Test Messages**

You can now specify whether you want to include the test messages in the custom SMS campaign report. A new checkbox – **Include Test Messages** – has been introduced on step 2 of the Custom SMS Campaign Report page.

![Custom SMS Report Step 2](https://us.v-cdn.net/6038474/uploads/attachments/migrated/CustomSMSReportStep2.0685g00000BFBm3AAH.png)

When the **Include Test Messages** checkbox is selected, the test messages for the selected campaigns are included in the report.

To learn more about the custom SMS campaign reports, see [**Custom SMS Campaign Reports**](https://galaxy.maropost.com/s/article/Custom-SMS-Campaign-Report).
