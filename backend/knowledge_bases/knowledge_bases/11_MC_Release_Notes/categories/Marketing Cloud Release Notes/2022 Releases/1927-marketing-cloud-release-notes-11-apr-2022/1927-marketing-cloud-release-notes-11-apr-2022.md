# Marketing Cloud Release Notes – 11 Apr 2022

## **OVERVIEW**

This release includes several new improvements to the Drag-and-Drop content editor – all designed to improve your text editing experience.

For details, please read on!

## **ENHANCEMENT**

### **Improvements in Drag-and-Drop Content Editor**

The Drag-and-Drop content editor has been enhanced to provide you with better control in styling your content. To learn about how to create content using the Drag-and-Drop content editor, see [Drag and Drop Editor: Content Tab](https://galaxy.maropost.com/s/article/Drag-and-Drop-Editor-Content-Tab).

The notable changes include:

New content blocks – PARAGRAPH and LIST – have been introduced to replace the TEXT content block. The new content blocks provide better control of how text appears including:
