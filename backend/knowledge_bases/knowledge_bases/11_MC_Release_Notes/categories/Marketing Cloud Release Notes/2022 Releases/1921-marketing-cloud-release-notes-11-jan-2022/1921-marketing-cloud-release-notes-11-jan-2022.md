# Marketing Cloud Release Notes – 11 Jan 2022

## **OVERVIEW**

This release consists of a new enhancement for the Shopify integration feature.

For more details, read on!

## **ENHANCEMENT**

### **Shopify Integration Update – Real-Time Synching of Order Financial Status **

All updates to the financial status of Shopify orders are now synced to Marketing Cloud in real-time. The order financial status is also synched every time an order is created or updated.

This is especially helpful for setting up journeys and segments based on order financial status.

**Important!!! **

While the functionality is automatically available for the new integrations, you'll have to activate it for your existing Shopify integrations. To activate the functionality in your existing integration, disable the integration and then enable it.

To learn about the Shopify integration, see [Integration with Shopify](https://galaxy.maropost.com/s/article/Integration-with-Shopify) and [Adding a Shopify Integration](https://galaxy.maropost.com/s/article/Adding-a-Shopify-Integration).
