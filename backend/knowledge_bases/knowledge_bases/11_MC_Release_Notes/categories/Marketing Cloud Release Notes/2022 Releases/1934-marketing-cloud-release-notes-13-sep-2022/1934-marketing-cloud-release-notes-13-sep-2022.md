# Marketing Cloud Release Notes – 13 Sep 2022

## **Overview**

We are excited to have made your archiving simpler. In this release, we bring you better Archive management. Also, Dynamic Content now exhibits a revamped index page.

To learn more, read on!

## **Enhancement**

### **Improved Archive Management**

Marketing Cloud now enables you to see and manage all your archived items in one place. For more information, see [**Archive Management**](https://galaxy.maropost.com/s/article/Archive-Management).

In Navigation, go to ARCHIVE to access the Archives page. The page displays all the archived items that can also be filtered based on the item type; for example, content or contact list. You can unarchive and delete an archived item using the Actions drop-down menu on this page.

![Archives Page](https://us.v-cdn.net/6038474/uploads/attachments/migrated/ArchivesPage.0685g000009tmh8AAA.png)

The archive functionality is available for [**Campaign Tag**](https://galaxy.maropost.com/s/article/Campaign-Tags), [**Contact List**](https://galaxy.maropost.com/s/article/Introduction-to-Contact-Lists), [**Content**](https://galaxy.maropost.com/s/article/Managing-Email-Content), [**Segment**](https://galaxy.maropost.com/s/article/Introduction-to-Segments), and [**Dynamic Content**](https://galaxy.maropost.com/s/article/Dynamic-Content). On the index page of the items supporting archiving, you'll notice a new button – VIEW ARCHIVES – that takes you to the Archives page as well.

![Archive - Content](https://us.v-cdn.net/6038474/uploads/attachments/migrated/ArchiveContent.0685g000009tmiDAAQ.png)
> **Note**: Use the Archive option from the Actions drop-down menu to archive an item.

### **Revamped Dynamic Content Index Page**

The Dynamic Content index page is now revamped as per our design standards. To learn about Dynamic Content, see [**Dynamic Content**](https://galaxy.maropost.com/s/article/Dynamic-Content).
