# Marketing Cloud Release Notes – 5 Jan 2022

## **OVERVIEW**

This Maropost Marketing Cloud release consists of a single new feature furthering our self-service strategy. Using a few simple steps, you can now set up a custom return path for your domain.

Scroll down to read the details!

## **NEW FEATURE**

### **Setting Up Custom Return Path**

Maropost Marketing Cloud now gives you the option of setting up a custom Return-Path for your domain. A custom Return-Path enables you to achieve a full DMARC alignment in addition to improving your deliverability and sending reputation.

You can set up the custom Return-Path in the verification step of your sending domain setup.

On the Domain Verification page of your sending domain, scroll down to the Custom Return Path section. Here, you can set up and verify the custom return path. As part of the setup process, you’ll also be required to add a CNAME record in your DNS settings using the Host and value from the Domain Verification page.

For more details and step-by-step procedure, see [Set Up and Manage Sending Domains](https://galaxy.maropost.com/s/article/Set-Up-and-Manage-Sending-Domains).
