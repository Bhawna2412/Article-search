# Marketing Cloud Release Notes – 20 Oct 2023

### **OVERVIEW:**

In this launch, we are introducing a revamped layout of our Mobile App profile section.

To learn more about this feature, kindly read ahead!

### **ENHANCEMENT:**

#### **A Dedicated Feature**

You can now view your Mobile App SDKs from a single place known as the **Mobile Profile,** present in the **Settings** section of your Maropost Marketing Cloud account. 

![mobapp6](https://us.v-cdn.net/6038474/uploads/attachments/migrated/mobapp6.0685g00000HgysjAAB.png)
#### **Cleaner UI and a Simplistic flow**

The user interface has been updated, making the existing flow more understandable. Furthermore, the Mobile SDK page has a fresh look to it, making it easier for developers and users to understand the current versions and platforms on which the mobile apps are operating. 

![mobapp7](https://us.v-cdn.net/6038474/uploads/attachments/migrated/mobapp7.0685g00000HgysyAAB.png)﻿To know more about Mobile SDKs and how to manage them, kindly read our [**Mobile App Profiles**](https://galaxy.maropost.com/s/article/Mobile-App-Profiles) page.
