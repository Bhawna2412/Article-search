# Marketing Cloud Release Notes – 10 Jan 2023

## **Overview**

In this release, we launched some minor UI enhancements in the A/B Campaign creation flow.

For details, please read on!

**  **

## **Enhancement**

### **A/B Campaign UI Enhancements**

In step 2 of the A/B Campaign creation flow, we introduced some minor UI changes to help manage the split groups. The new UI is visually more intuitive and simplifies duplicating and deleting the split groups.

The Delete and Duplicate icons are now permanently visible on the page. When no split group is selected, the icons remain disabled. When you select a split group (by clicking the check box), the icons get enabled. You can then click the Delete or Duplicate icon to perform the desired action.

To learn about creating A/B Campaigns, see [**Creating A/B Campaigns**](https://galaxy.maropost.com/s/article/Creating-AB-Campaigns).

> **Reminder!!** A/B Campaign Report consists of both vertical and horizontal scrolls. When scrolling horizontally, the metric labels remain static, making it easier to read and compare data from all the split groups.

To learn more about A/B Campaign Reports, see [**A/B Campaign Reports**](https://galaxy.maropost.com/s/article/AB-Campaign-Reports).
