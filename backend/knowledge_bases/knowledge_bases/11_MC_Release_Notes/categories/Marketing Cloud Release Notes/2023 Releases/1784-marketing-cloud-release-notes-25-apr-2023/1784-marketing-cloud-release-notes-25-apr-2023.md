# Marketing Cloud Release Notes – 25 Apr 2023

**OVERVIEW**

In this release, we added a new metric to the product feed feature and introduced a consolidated view of a contact's order history in the form of the Order Details page.

For more details, please scroll down!

**ENHANCEMENTS**

**New Product Feed Metric – Top Trending**

You can now create a product feed consisting of your top trending products, meaning products showing a surge in popularity. When creating the product feed, select the new metric – Top Trending and select the time frame for data to be considered.

**Introducing the Order Details Page**

You can now see all the orders placed by a contact in one place, that is, on the Order Details page. The page is accessible from the Contact Overview by clicking the order count on the engagement card.

To learn more about contact overview, see [**Viewing a Contact Overview**](https://galaxy.maropost.com/s/article/Viewing-a-Contact-Overview).

![Contact Overview - Orders](https://us.v-cdn.net/6038474/uploads/attachments/migrated/ContactOverviewOrders.0685g00000DatIcAAJ.png)

The Order Details page displays details that include the order number, order status, total order value, order date, and the source from which the order details were captured.
