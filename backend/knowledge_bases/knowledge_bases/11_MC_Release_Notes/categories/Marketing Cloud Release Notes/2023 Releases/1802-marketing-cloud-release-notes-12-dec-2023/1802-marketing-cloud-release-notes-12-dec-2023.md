# Marketing Cloud Release Notes - 12 Dec 2023

### **OVERVIEW:**

In this release, we have made some layout and UI modifications to our Email Campaign Custom Report.

To learn more about this feature, read ahead!! 

### **ENHANCEMENT:**

Initially, while generating a one-time schedule report, in the Report Details section of step 2, the overall UI interface would appear cluttered and not so easy to follow, thus causing the report-creation process to be quite taxing and time-consuming.

To streamline the process, we have updated the UI fields and clubbed the campaign types into one single field.

![custrep18](https://us.v-cdn.net/6038474/uploads/attachments/migrated/custrep18.0685g00000Io3luAAB.png)Hence, from now on, users can simply choose the campaign type from the list and select the brands, names, and campaign tags, respectively.

![custrep20](https://us.v-cdn.net/6038474/uploads/attachments/migrated/custrep20.0685g00000Io3mEAAR.png)Also, we have a preview mode wherein you can have a sneak peek at all the selection criteria you have chosen to generate the custom email report. 

![custrep21](https://us.v-cdn.net/6038474/uploads/attachments/migrated/custrep21.0685g00000Io3mTAAR.png)To learn more about generating a custom email report, visit and read our [**Email Report**](https://galaxy.maropost.com/s/article/Custom-Email-Campaign-Report) topic.
