# Marketing Cloud Release Notes – 27 Apr 2023

*This article was created from a *[*community discussion*](https://maropost.vanillacommunities.com/discussion/127/marketing-cloud-release-notes-27th-april-2023).

**OVERVIEW**

In this release, we launched some very exciting enhancements to Shopify Integration feature. Also, check out the Content Feeds feature for minor UI updates. 

For more details, please scroll down!

**ENHANCEMENTS**

**Shopify Integration Upgrades**

We made several improvements in the Shopify integration feature enabling you to efficiently import more data from Shopify into your Marketing Cloud account.

**Note**: Enabling these new features in your existing Shopify connections needs a one-time permission and therefore, requires you to do a quick reconnection. See the announcement [**Reconnect Your Existing Shopify Integrations**](https://maropost.vanillacommunities.com/home/leaving?allowTrusted=1&target=https%3A%2F%2Fgalaxy.maropost.com%2Fs%2Fquestion%2F0D55g00002S66wMCAR%2Freconnect-your-existing-shopify-integrations) for more details.

- 
- 
  **Imports SMS Contact Details**: The Shopify integration now enables you to import SMS contact information from Shopify into Marketing Cloud. If set up when adding an integration, the integration imports the mobile number and consent status for the contact.

  ![Shopify](https://us.v-cdn.net/6038474/uploads/attachments/migrated/ShopifyIntegrationRN.0685g00000DazDSAAZ.png)

To learn about Shopify integration, see [**Integration with Shopify**](https://maropost.vanillacommunities.com/home/leaving?allowTrusted=1&target=https%3A%2F%2Fgalaxy.maropost.com%2Fs%2Farticle%2FIntegration-with-Shopify) and [**Adding a Shopify Integration**](https://maropost.vanillacommunities.com/home/leaving?allowTrusted=1&target=https%3A%2F%2Fgalaxy.maropost.com%2Fs%2Farticle%2FAdding-a-Shopify-Integration).

**Content Feed – UI Changes**

The labels on the **New Feed** dialog box have been changed to bring clarity when setting up the content feed update schedule.

The drop-down fields are now labeled ‘**Day of the Week**’ and ‘**Hour of the Day**’, indicating the day and time on which the content feed is scheduled to be updated. Also, note that the fields are now mandatory and require input to successfully create a content feed.
