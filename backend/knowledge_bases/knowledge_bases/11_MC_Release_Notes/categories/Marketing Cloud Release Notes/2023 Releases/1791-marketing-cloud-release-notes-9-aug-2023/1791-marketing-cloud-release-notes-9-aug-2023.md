# Marketing Cloud Release Notes – 9 Aug 2023

**OVERVIEW**

In this release, we launched the enhanced UI for segment details.

**ENHANCEMENT**

**Redesigned Segment Details UI **

The segment details UI has been redesigned to enhance both usability and look and feel. To learn about segments including the new enhancement details, see [**Introduction to Segments**](https://galaxy.maropost.com/s/article/Introduction-to-Segments).

On the **Email Segments** page, (In Navigation > AUDIENCE > Segments), click the segment name to go to its details view. You’ll notice a significant upgrade to how information is displayed on the UI. The segment rules are now visually presented in a way that’s more readable and comprehensible.

![Segment Overview](https://us.v-cdn.net/6038474/uploads/attachments/migrated/SegmentOverview.0685g00000G89CMAAZ.jpg)

A Timeline feature has been introduced enabling you to see all the Email Campaigns, Journeys, and Data Journeys in which the segment is used.
