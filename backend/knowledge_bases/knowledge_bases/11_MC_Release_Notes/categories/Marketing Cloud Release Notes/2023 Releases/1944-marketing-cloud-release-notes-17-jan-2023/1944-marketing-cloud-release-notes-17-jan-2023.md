# Marketing Cloud Release Notes – 17 Jan 2023

## **OVERVIEW**

In this release, we are very happy to announce the launch of Product Feed enhancements.

To learn the details, please read on!

## **ENHANCEMENT – PRODUCT FEEDS**

We made several enhancements to the Product Feed feature to give you more control in building product recommendations. You now have more filtering options that help you promote specific product categories and exclude inactive products.

To learn about Product Feeds, see [**Product Feeds**](https://galaxy.maropost.com/s/article/Product-Feeds).

### **Ability to Filter Product Categories**

When creating a product feed, you can now filter the categories based on a source and then further restrict it based on Brand. The categories are displayed in a hierarchical structure, enabling you to select specific categories.

**What changed in the UI?**

The **New Product Feed** dialog box now contains two new drop-down fields – **Source** and **Brand**. Also, the **Categories** drop-down fields have been improved to allow hierarchical selection.

**How does it work?**

In the Source field, select the e-commerce platform from which you want to derive the product categories. Upon selecting the source, the Brand and Categories fields are populated with the results based on the source selection.

The Categories drop-down field features a hierarchy-based selection giving you more flexibility to include and exclude child categories within a parent category.

Optionally, if you want to further limit the product categories to specific Brands, you can select the Brands from the Brand drop-down field.﻿
