# Marketing Cloud Release Notes – 21 Feb 2023

## **Overview**

This release is a step closer towards reinforcing your marketing automation. Check out Maropost Marketing Cloud's first Journey Template.

Scroll down to learn the details!

## **Enhancement**

### **Journey Template – Welcome Journey Template**

Marketing Cloud has launched its first Journey template – **Welcome Journey Template**. This pre-designed template is the first of many to be launched and provides you with the simplest way to set up Journeys for sending introductory emails to new subscribers.

While Journeys are instrumental in automating and simplifying your customer journey mapping process, it does require some learning to set up an effective Journey. But now, using our predefined templates, you can quickly set up high-performing Journeys without having to learn the underlying schematics.

The templates are available within the Journey creation flow. In Navigation, go to AUTOMATION, and select Journeys. Then on the Journeys page, click NEW JOURNEY. Here, on the Journey Selection page, you can create Journey from the scratch or select a template to use.

> **Note**: Currently, the feature availability is limited to the Welcome Journey Template. Stay tuned as we launch more templates!

To learn more about the Journey Template feature, see [**Journey Templates: Overview**](https://galaxy.maropost.com/s/article/Journey-Templates-Overview).
