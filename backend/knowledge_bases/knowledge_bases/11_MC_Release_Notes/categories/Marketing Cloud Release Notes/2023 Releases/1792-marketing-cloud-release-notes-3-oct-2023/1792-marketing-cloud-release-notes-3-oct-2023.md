# Marketing Cloud Release Notes – 3 Oct 2023

### **OVERVIEW**

In this release, we are delighted to bring out some of the major enhancements to the SMS and Integration features.

For more information, please read on!

### **NEW FEATURE**

#### **Introduced SMS Link Tracking Domains**

You can now use your own branded domains for shortened links in SMS. Similar to the custom domains for email tracking, we are introducing custom domains for SMS link tracking. To learn about the SMS Link Tracking Domains, see** **[**Setup and Manage the Link Tracking Domains**](https://galaxy.maropost.com/s/article/Set-Up-and-Manage-the-SMS-Link-Tracking-Domains).

These custom domains can be set up and managed on the **SMS Link Tracking Domains** page. 

![smslinktrackingdomain](https://us.v-cdn.net/6038474/uploads/attachments/migrated/smslinktrackingdomain.0685g00000I4cWEAAZ.png)To successfully set up a domain for SMS link tracking, you must complete the process of adding and verifying the domain.

![smslinktrackingdomain2](https://us.v-cdn.net/6038474/uploads/attachments/migrated/smslinktrackingdomain2.0685g00000I4cWYAAZ.png)

### **ENHANCEMENTS**

#### **Revamped ClickBank Integration**

The ClickBank integration has undergone some important changes, making the integration easy to set up, along with other additional functionalities. To learn about ClickBank Integration in detail, see [**Integration with ClickBank**](https://galaxy.maropost.com/s/article/Integration-with-ClickBank) and [**Adding a ClickBank Integration**](https://galaxy.maropost.com/s/article/Adding-a-ClickBank-Integration). 

#### **UI Enhancements**

The ClickBank Integration index page is redesigned with subtle UI changes to improve your user experience. On the page, you’ll find more details, including all the lists associated with the integration.

![ClickBankIndex](https://us.v-cdn.net/6038474/uploads/attachments/migrated/ClickBankIndex.0685g00000I4ccQAAR.png)The Add Integrations slider has been revamped with a new UI that also features additional fields.

![CB1](https://us.v-cdn.net/6038474/uploads/attachments/migrated/CB1.0685g00000I4cz0AAB.png)![CB2](https://us.v-cdn.net/6038474/uploads/attachments/migrated/CB2.0685g00000I4cqaAAB.png)

#### **New Functionalities**

Some of the notable enhancements are as follows:

- The integration works in real time, ensuring faster data synchronization.
- 
- You can associate multiple lists with a single integration and thereby sync the contacts from ClickBank into all the associated lists automatically.
- The **Sync historical data** functionality enables you to sync the existing data from ClickBank when creating the integration.
- 

#### **Enabled Click Tracking for Mobile Keywords**

The Click Tracking feature is now also supported for mobile Keywords. Click tracking converts your URLs into short, unique links, which in turn take up less space in your message and are also trackable.

To learn about the mobile keywords, see [**Mobile Keywords**](https://galaxy.maropost.com/s/article/Mobile-Keywords).

![clicktrackingkeywords](https://us.v-cdn.net/6038474/uploads/attachments/migrated/clicktrackingkeywords.0685g00000I4cWxAAJ.png)You can view the number of clicks for each keyword on the **Mobile Keywords** index page and find a detailed report by clicking on the click count.

![mobilekeywordsindex](https://us.v-cdn.net/6038474/uploads/attachments/migrated/mobilekeywordsindex.0685g00000I4caNAAR.png)

### **DEPRECATION**

#### **Removed the Recent Maropost Tweets Widget**

The **Recent Maropost Tweets **widget has been removed from the Marketing Cloud dashboard and is no longer available for use.
