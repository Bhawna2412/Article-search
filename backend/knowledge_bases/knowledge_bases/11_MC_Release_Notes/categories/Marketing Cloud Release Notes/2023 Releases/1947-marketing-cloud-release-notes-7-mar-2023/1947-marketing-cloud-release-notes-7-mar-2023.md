# Marketing Cloud Release Notes – 7 Mar 2023

## **Overview**

In this release, we launched an integration between Marketing Cloud and Retail Express, enabling data flow between the systems so you can take advantage of both their features.

For more information, please read on!

** **

## **Enhancement**

### **Integration with Retail Express**

Maropost Marketing Cloud now offers easy-to-set-up integration with Retail Express, enabling you to sync data between the systems. The integration enables Retail Express users to leverage Marketing Cloud features for sending campaigns and implementing successful marketing strategies.

To learn more details about the integration, see [**Integration with Retail Express**](https://galaxy.maropost.com/s/article/Integration-with-Retail-Express).

You can create and manage your Retail Express integrations on the **Retail Express Integration** page.

![Retail Express Integration Index - RN](https://us.v-cdn.net/6038474/uploads/attachments/migrated/RetailExpressIntegrationIndexRN.0685g00000DXk5OAAT.png)

Adding a new integration simply requires you to provide the shop URL and API Key (primary or secondary) from Retail Express and select the Marketing Cloud contact list into which you want to sync the contact information. You can then select whether you want to sync the historical data and click CONNECT.
