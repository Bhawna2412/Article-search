# Marketing Cloud Release Notes – 17 Oct 2023

### **OVERVIEW:**

Great news, Maropostians!!! We are delighted to announce the highly awaited revamp of our Landing page feature. Now you can create vivid and amazing standalone pages for your users.

To know more, continue reading ahead!

### **ENHANCEMENT – Revamped Landing Pages UIs:**

#### **An Informative Index Page**

Landing Pages now have a much more detailed layout, wherein you can view key details of a page from a single place. Plus, you can now sort and view web pages as per **WYSIWYG** or **Drag & Drop** format. 

![land45](https://us.v-cdn.net/6038474/uploads/attachments/migrated/land45.0685g00000I4dD6AAJ.png)To learn more, see [**Introduction to Landing Pages**](https://galaxy.maropost.com/s/article/Introduction-to-Landing-Pages).

#### **Simplistic Designing Process**

New to landing pages? Not to worry, we provide a straightforward design process. Simply click on new page >> Start Designing >> Choose your format >> Fill out basic details>> And begin with your web page design.

![land46](https://us.v-cdn.net/6038474/uploads/attachments/migrated/land46.0685g00000I4cipAAB.png)See [**Creating Landing Pages**](https://galaxy.maropost.com/s/article/Creating-Landing-Pages) to learn the steps in detail.

#### **An array of Customization Tools**

Whether it be **WYSIWYG** or **Drag & Drop** format, you’ll have a number of design tools at your disposal.

Below is what a WYSIWYG editor page looks like. Another advantage of this format is that you can copy and paste your own HTML code into this editor. This takes personalization to a whole new level.

![land47](https://us.v-cdn.net/6038474/uploads/attachments/migrated/land47.0685g00000I4dBnAAJ.png)Drag & Drop format also has a rich collection of options to help you create highly engaging pages for your users.

![land48](https://us.v-cdn.net/6038474/uploads/attachments/migrated/land48.0685g00000I4dCBAAZ.png)

#### **A Dedicated Preview Mode**

Using the preview mode, you can see how your Landing Page will look to your users and decide whether any changes are required or not.

![land50](https://us.v-cdn.net/6038474/uploads/attachments/migrated/land50.0685g00000I4caqAAB.png)To learn more, see [**Previewing a Landing Page**](https://galaxy.maropost.com/s/article/Preview-a-Landing-Page).

#### **Library Template Collection**

Not designer savvy? Use our vast library of ready-to-use templates consisting of themes ranging from newsletters and promotional flyers to occasion-based dynamic content. You can search for templates using the Categories filtering options.

![land51](https://us.v-cdn.net/6038474/uploads/attachments/migrated/land51.0685g00000I4dCkAAJ.png)
