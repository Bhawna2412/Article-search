# Marketing Cloud Release Notes – 16 May 2023

**OVERVIEW**

This release includes the launch of a new Journey template.

**NEW FEATURE**

**New Journey Template – Abandoned Cart Journey Template**

Working towards our aim of simplifying the automation setups, we launched our second Journey template – **Abandoned Cart Journey Template**. This pre-designed template enables you to set up your abandoned cart Journey in two quick steps.

To learn about setting up Abandoned Cart emails, see [**Sending Abandoned Cart Emails to Your Neto Contacts**](https://galaxy.maropost.com/s/article/Sending-Abandoned-Cart-Emails-to-Your-Neto-Contacts). And to learn about Abandoned Cart Journey Template, see [**Abandoned Cart Journey Template**](https://galaxy.maropost.com/s/article/Abandoned-Cart-Journey-Template). 

In step 1, you will configure the Journey by entering or selecting the name, end date, activation status at the time of creation, and retrigger preference.

![Abandoned Cart Journey Step 1](https://us.v-cdn.net/6038474/uploads/attachments/migrated/AbandonedCartJourneyStep1.0685g00000EhcSWAAZ.jpg)

In step 2, you will select the web store connection that you want to use as the trigger source and the from name and content for emails that will be sent.
