# Marketing Cloud Release Notes – 23 Feb 2023

## **Overview**

In this release, we launched a very special feature that will help you measure the effectiveness of your campaigns.

To learn more, please read on!

## **New Feature**

### **Conversion Attribution**

Marketers are metrics-driven, and so we know you’ll love this new feature – Conversion Attribution.

Conversion Attribution is a strategy used to identify which email campaign led your contact to make a purchase and then associate the campaign with the order. Marketing Cloud implements this strategy using the Last-Touch attribution model that assigns the conversion credit to the last touchpoint before a lead is converted.

Marketing Cloud enables you to set up the Conversion Attribution strategy simply by setting up a set of parameters, without any coding effort.

On the **Settings** page, select Conversion Attribution. Then, on the **Conversion Attribution** page, enable the **Enable Last-Touch attribution** toggle switch and then set up the rest of the parameters as you see fit. To learn how to set up the conversion attribution parameters, see [**Conversion Attribution (Last-Touch Attribution)**.](https://galaxy.maropost.com/s/article/Conversion-Attribution)
