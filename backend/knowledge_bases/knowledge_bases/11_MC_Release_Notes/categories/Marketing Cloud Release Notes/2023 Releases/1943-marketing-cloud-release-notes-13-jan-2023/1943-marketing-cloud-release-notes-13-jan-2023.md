# Marketing Cloud Release Notes – 13 Jan 2023

## **Overview**

In this release, we are very excited to launch a unified URL for all our Marketing Cloud users. Also, check out the new location for setting your profile preferences.

For details, please read on!

** **

## **Enhancements**

### **Unified URL for All Maropost Marketing Cloud Users**

Starting today, all our Marketing Cloud users can access their accounts at the web address:

[**app.maropost.com**](https://app.maropost.com)

No more separate URLs for different users! All users now have one simple URL for logging in. If you have multiple accounts, it's now even easier to switch accounts – the URL for all your accounts remains the same.

Please remember to change your bookmarks to the new URL. If you still happen to use the old URL, you’ll be automatically redirected to the new URL.

### **Setting Your User Profile Preferences**

The PROFILE tab has been moved from the **Account & Billing** page (in the **Maropost Marketing Cloud** application) to the **Maropost Identity** application[https://id.maropost.com/](https://id.maropost.com/) and is renamed to **PREFERENCES**.

The actions available on the tab remain the same. This means that you will be able to select your default account and set subscription and notification preferences on the PREFERENCES tab in our Identity application.

To learn more about setting your profile preferences, see [**Using Maropost Identity App**](https://galaxy.maropost.com/s/article/Using-Maropost-Identity-App).
