# Marketing Cloud Release Notes – 9 Oct 2024

### **OVERVIEW:**

In this release, we are introducing the ability to set up the timezone for a user account. This will greatly help users who are based both locally and globally to create account-based time zones and enable better management of several processes taking place in the marketing cloud.

To learn more, read ahead!! 

### **NEW FEATURE:**

You can now select the timezone for your account depending on your location or preference. This eliminates the confusion of using the Eastern Standard Timezone (EST) if you are located in a different timezone and increases efficiency in using Marketing Cloud features such as campaigns, reports, and so on.

A new field, namely “**Timezone,**” has been introduced on the **Preferences** page of the **Maropost Identity** app. To update the timezone, simply select from the drop-down list and save the changes.

![Preferences-Timezone](https://us.v-cdn.net/6038474/uploads/attachments/migrated/PreferencesTimezone.068J1000005YHuhIAG.png)You can see your account’s current timezone displayed at all times in the top panel of the Marketing Cloud application. To change the timezone, you can click on it, and it will take you to the **Preferences** page of the **Maropost Identity** app, where you can select your preferred timezone.

![TimezoneDisplay](https://us.v-cdn.net/6038474/uploads/attachments/migrated/TimezoneDisplay.068J1000005YHurIAG.png)Some Marketing Cloud features/functionalities don't support local timezone settings yet and still function based on Eastern Standard Time (EST). To learn more about the timezone setup and the exception list, see [**Set Up Your Account Timezone**](https://galaxy.maropost.com/s/article/Set-Up-Your-Account-Timezone). Also, you can learn more about the Identity app by seeing** **[**Using Maropost Identity App**](https://galaxy.maropost.com/s/article/Using-Maropost-Identity-App).
