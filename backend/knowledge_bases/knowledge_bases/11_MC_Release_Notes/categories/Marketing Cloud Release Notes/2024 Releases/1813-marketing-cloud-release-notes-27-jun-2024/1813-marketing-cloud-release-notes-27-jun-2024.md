# Marketing Cloud Release Notes – 27 Jun 2024

### **OVERVIEW:**

Greetings Maropostians!!

We are pleased to announce that we have introduced a significant update to our next-generation segment builder.

Kindly read ahead to learn more!! 

### **ENHANCEMENT:**

Previously, marketers would have to manually input the rules and criteria while building a segment. Although the builder is intuitive in nature, it tends to be time-consuming while building large and complex segments.

To navigate around this issue, we have implemented an **AI Builder** toolkit to assist you in constructing segment rules to ease your task.

With the help of **Natural Language Processing** (**NLP**) logic, you only need to add a keyword or a sentence that depicts how the segment rule should be built and what type of contact it needs to fetch.

![segment32](https://us.v-cdn.net/6038474/uploads/attachments/migrated/segment32.068J1000001YKj6IAG.png)You can preview the rule layout and the contact records that fulfill the criteria. Once satisfied, you can generate the rules and criteria in no time. 

![segment33](https://us.v-cdn.net/6038474/uploads/attachments/migrated/segment33.068J1000001YIluIAG.png)With the AI builder, marketers can generate multiple rules and criteria in a segment fairly quickly.
