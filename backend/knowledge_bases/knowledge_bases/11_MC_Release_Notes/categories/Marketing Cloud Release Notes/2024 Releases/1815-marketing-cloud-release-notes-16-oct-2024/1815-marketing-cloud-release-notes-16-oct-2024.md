# Marketing Cloud Release Notes – 16 Oct 2024

### **OVERVIEW:**

In this release, we are offering better privacy and more control over your API Keys.

To learn more, read ahead!!

### **ENHANCEMENT:**

#### **API Keys - Privacy and Security Update**

The API Keys are now encrypted on the **API Keys** page, with only the last five characters of the original API Key being exposed for identification. Further, a date of creation section is added so you can easily sort through multiple API keys from a single place.

![api10](https://us.v-cdn.net/6038474/uploads/attachments/migrated/api10.068J1000005ZJH6IAO.png)You can now maintain a list of whitelisted IP addresses, which can be shared with the Maropost servers in such a way that it allows only those IP addresses included in the list. Such whitelisting adds another layer of safety and mitigates the risk of misusing a user’s API keys.

![api14](https://us.v-cdn.net/6038474/uploads/attachments/migrated/api14.068J1000005ZJ1jIAG.png)For URL-based requests, you’ll need to insert the API key in the Authorization Header as depicted in the image below: 

![api11](https://us.v-cdn.net/6038474/uploads/attachments/migrated/api11.068J1000005ZJJTIA4.png)
To learn more about creating and managing API Keys, see [**API Keys**](https://galaxy.maropost.com/s/article/API-Keys). Also, see our [**API Reference Guide**](https://api.maropost.com/api/) for an exhaustive list of the APIs offered.
