# Marketing Cloud Release Notes – 14 Mar 2024

### **OVERVIEW:**

In this release, we have introduced some improvements to our contact overview, landing page, and user profile sections.

To learn more details, please read ahead!!

### **CONTACT OVERVIEW ENHANCEMENT:**

A new tab, known as **Abandoned Cart List**, has been introduced. Here, you can see the abandoned carts and their associated campaigns on the contact overview page. Note that for the abandoned cart feature to work, you must have an **active integration** with an e-commerce store. Currently, Maropost supports **Neto, Shopify,** and **ClickBank** integrations for this purpose.

![abco0](https://us.v-cdn.net/6038474/uploads/attachments/migrated/abco0.0685g00000LpYMKAA3.png)To learn more about this feature, have a glance at our [**Contact Overview**](https://galaxy.maropost.com/kb/articles/10-contact-360-overview?utm_source=community-search&utm_medium=organic-search&utm_term=contact+360) article.

### **LANDING PAGE PREVIEW AND NOTIFICATION ENHANCEMENT:**

We have modified the preview layout's user interface. Initially, the page's preview was displayed within a device framework, making the viewing region quite constrained. From now on, you can preview your landing page in full-screen mode, so you can evaluate the different components you have added to your landing page. 

![land52](https://us.v-cdn.net/6038474/uploads/attachments/migrated/land52.0685g00000LoXp2AAF.png)We have also developed a notification service, wherein you’ll be notified to verify your landing page’s domain status two or three days prior to its publication release. Make sure to verify your page’s URL status, since failing to do so could cause the pages to not function properly upon release. 

![land53](https://us.v-cdn.net/6038474/uploads/attachments/migrated/land53.0685g00000LoXpHAAV.png)To learn more about Landing Pages, read our [**Introduction to Landing Pages**](https://galaxy.maropost.com/s/article/Introduction-to-Landing-Pages) article. 

### **MAROPOST'S PROFILE SECTION ENHANCEMENT:**

We have made some minor UI changes in the Identity App module. As part of the update, the tabs are replaced with the left navigation to resemble the uniform UI design across Maropost platforms. Thanks to its change in appearance, you can now easily manage numerous accounts across several of Maropost’s products. 

![maropostiden2](https://us.v-cdn.net/6038474/uploads/attachments/migrated/maropostiden2.0685g00000LoXqSAAV.png)To learn more about Maropost’s user profile platform, read our [**Using Maropost Identity App**](https://galaxy.maropost.com/s/article/Using-Maropost-Identity-App) article.
