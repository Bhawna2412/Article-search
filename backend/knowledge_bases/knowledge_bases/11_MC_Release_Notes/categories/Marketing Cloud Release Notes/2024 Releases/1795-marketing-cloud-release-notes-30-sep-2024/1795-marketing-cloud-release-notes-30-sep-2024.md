# Marketing Cloud Release Notes – 30 Sep 2024

### **OVERVIEW:**

In this release, we have launched a new segment category in our Next Generation Segment Builder that enables you to fetch contacts from relational tables.

Kindly read ahead to know more!!

### **NEW FEATURE:**

#### **New Rule Category in the Next Generation Segment Builder**

A new category - **Relational Data** - has been introduced in our Next Generation Segment Builder**. **This enables you to segment and retrieve contacts from relational tables and SQL queries.

New to Relational Tables? Relational tables offer quick and powerful ways to extend personalization data beyond the standard 1:1 relationship of a contact field. For example, managing various pets and their associated demographics in a household. To start learning about relational tables and SQL queries in Marketing Cloud, see [**Introduction to Relational Tables**](https://galaxy.maropost.com/s/article/Introduction-to-Relational-Tables).
