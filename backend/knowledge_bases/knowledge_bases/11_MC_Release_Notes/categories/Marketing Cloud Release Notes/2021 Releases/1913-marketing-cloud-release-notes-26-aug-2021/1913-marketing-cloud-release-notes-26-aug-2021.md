### **Revamped Coming Soon Page**

With the newly revamped Coming Soon page, you'll have a better understanding of our product roadmap. The page is up-to-date with information on planned upcoming features.

Click your username in the top-right corner of the application, and then from the drop-down menu, select Coming Soon.

### **Updated Growth & Attrition Reporting**

The Growth & Attrition reporting has been updated with a minor UI change and a new functionality addition. To learn about Growth & Attrition reports, see [Growth & Attrition Report: Overview](https://galaxy.maropost.com/s/article/Growth-Attrition-Report-Overview).

The Report Name field has been moved from step 2 to step 1.

![Updated_G&A_Reports_1](https://us.v-cdn.net/6038474/uploads/attachments/migrated/UpdatedGAReports1.0685g000006wWSXAA2.png)

A new toggle button - "Include all new Lists" - has been added in step 2. If the toggle is turned on, all the new lists created between the report creation date and the report delivery date will be included in the report.

![Updated_G&A_Reports_2](https://us.v-cdn.net/6038474/uploads/attachments/migrated/UpdatedGAReports2.0685g000006wWScAAM.png)

### **Added New Status to Identify Journey SMS Campaigns**

On the SMS Campaigns index page, you can now easily identify and filter Journey SMS campaigns. The statuses of Journey SMS campaigns are shown as "Jrny" (short for Journey). Also, the filter drop-down list now has Journey as a filtering option, enabling you to view only the Journey SMS campaigns.

To learn about SMS Campaigns, see [SMS Campaigns](https://galaxy.maropost.com/s/article/SMS-Campaigns).

![SMS_Campaign_Status_Field](https://us.v-cdn.net/6038474/uploads/attachments/migrated/SMSCampaignStatusField.0685g000006wWShAAM.png)

### **Updated Mobile SDK for Android**

Maropost Marketing's Mobile SDK for Android has been updated to support the latest Kotlin version. To learn more about Mobile SDK for Android, see [Mobile SDK for Android – Installation and Configuration](https://galaxy.maropost.com/s/article/Mobile-SDK-for-Android-Installation-and-Configuration).
