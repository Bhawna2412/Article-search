## **Enhancements**

### Added Functionalities in Neto Integration

You can now import historical data from Neto to Maropost when creating an integration. To learn about the Neto integration, see [Integration with Neto](https://galaxy.maropost.com/s/article/Integration-with-Maropost-Commerce-Cloud).

The 'Sync historical data' check box is available on the Integration Settings dialog box; enabled only when adding a new connection. If the check box is selected, the historical customer and order data from Neto is synced to Maropost when the integration is created.

![Neto_Integration_Settings](https://us.v-cdn.net/6038474/uploads/attachments/migrated/NetoIntegrationSettings.0685g000006w88AAAQ.png)

The Actions drop-down menu now contains the delete option, enabling you to delete the Neto integrations that are not useful to you anymore.

![Neto_Integration_Actions](https://us.v-cdn.net/6038474/uploads/attachments/migrated/NetoIntegrationActions.0685g000006w88FAAQ.png)

### **Journey Information in Contact Tags**

You’ll now be able to see all the Journeys associated with a Contact tag, enabling you to easily identify which Contact tag is used to trigger which Journey. To learn about Contact tags, see [Contact Tags](https://galaxy.maropost.com/s/article/Contact-Tags).

**Navigation:** On the Contact Tags index page, click the Contact tag name to go to the Tag Details page. This page shows the tag details and lists out the Journeys associated with the tag.
