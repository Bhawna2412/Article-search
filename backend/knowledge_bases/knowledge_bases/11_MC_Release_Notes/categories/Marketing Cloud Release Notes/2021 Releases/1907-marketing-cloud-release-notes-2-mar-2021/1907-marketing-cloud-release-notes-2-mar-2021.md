# Marketing Cloud Release Notes – 2 Mar 2021

## **Overview**

In this release, you'll find enhancements to the Data Journey feature, including the UI redesign. Also noteworthy is the change to the Account Name field on the Account Details page, which was changed to a read-only field.

To learn more about the updates, read on!

## **Enhancements**

### **End Date Field in Data Journey**

A new field, End Date, has been introduced in the Data Journey creation flow. This field enables you to specify a date on which the Data Journey will be disabled automatically, saving you the task of remembering to manually disable the Data Journey.

To learn more about Data Journeys, see [Data Journeys](https://galaxy.maropost.com/s/article/Data-Journeys).

![New_Data_Journey](https://us.v-cdn.net/6038474/uploads/attachments/migrated/NewDataJourney.0685g000006w861AAA.png)

### **UI Refresh of Data Journey Pages**

The Data Journey pages were redesigned to meet our latest design standards. The UI is now clearer and actions are easier to access; the functionality however remains the same.

To learn more about Data Journeys, see [Data Journeys](https://galaxy.maropost.com/s/article/Data-Journeys).

![Data_Journey_Index](https://us.v-cdn.net/6038474/uploads/attachments/migrated/DataJourneyIndex.0685g000006w86BAAQ.png)

### **Account Name Changed to Read-only Field**

We changed the Account Name field in the Account Details page (Account Details page > ACCOUNT tab) to a read-only field to avoid downstream backend impacts. You cannot update your account name on the application UI; however, you can reach out to our support team to get it updated.

To learn about managing your accounts, see [Managing Your Accounts](https://galaxy.maropost.com/s/article/Managing-Your-Accounts).
