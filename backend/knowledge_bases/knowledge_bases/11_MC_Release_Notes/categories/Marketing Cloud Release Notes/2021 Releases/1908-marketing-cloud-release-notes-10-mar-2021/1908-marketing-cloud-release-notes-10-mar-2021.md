# Marketing Cloud Release Notes – 10 Mar 2021

## **Overview**

In this special release, you'll find a simpler and more efficient process for Shopify integration.

## **Enhancement**

### **Shopify Integration Revamp**

The redesigned Shopify Integration setup is aimed at making the setup process as simple and quick as possible. Now, you'll be required to enter only the shop URL and Maropost list name for the integration. And then with just a few clicks, you'll have the integration all set up and ready to use!

There will be no impact on your existing integrations. However, you'll not be able to delete those integrations. Remember, you can always disable the integration if not being used.

To learn about Shopify integration, see [Integration with Shopify](https://galaxy.maropost.com/s/article/Integration-with-Shopify), and to get step-by-step instructions on how to add a Shopify integration, see [Adding a Shopify Integration](https://galaxy.maropost.com/s/article/Adding-a-Shopify-Integration).
