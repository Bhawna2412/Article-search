# Marketing Cloud Release Notes – 12 Oct 2021

## **Overview**

This release includes some notable UI refreshes and minor nifty features intended to improve your experience.

To learn what's new, read on!

## **Enhancements**

### SMS Contacts UI Refresh

Several UI screens for the SMS Contact Lists have been revamped, including the [SMS Contacts] page, where you can view, add, and manage the contacts for an SMS list.

To learn more about SMS Contact Lists, see [SMS Contact Lists](https://galaxy.maropost.com/s/article/SMS-Contact-Lists).

The dialog box for adding a new contact to the list has been redesigned. Besides the new look, a toggle has been introduced that you can use to associate the contact with your account's Unique Identifier.

To learn more about creating new SMS lists and adding contacts to lists, see [Creating SMS Contact Lists](https://galaxy.maropost.com/s/article/Creating-SMS-Contact-Lists).

### **SMS Contacts Overview**

You can now get a consolidated view of all the information and insights for an SMS Contact. On the SMS Contact Overview page, you can do the following: 

- 
- View and manage a contact's subscription to lists
- 
- View and understand the contact's engagement with your SMS

On the SMS Contact Lists index page, from the Actions drop-down menu of a list, select View Contacts. Then, on the [SMS Contacts] page, click the contact's phone number to go to the contact overview.

For more information on managing SMS Contact Lists, see [SMS Contact Lists](https://galaxy.maropost.com/s/article/SMS-Contact-Lists).

### **Shortcut for Creating New Lists During Integration**

A shortcut for creating a new contact list is now available in the integration setup UI. When adding an integration, you are required to select a contact list for the integration. If you want to use a new list instead of an existing one, the Create List option will help you avoid the additional steps of going back to create a new list before proceeding with the integration.

> **Note:** This feature is currently available in Shopify integration and Maropost Commerce Cloud integration. Very soon, it will be extended to other integrations as well.
