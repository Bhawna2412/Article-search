# Marketing Cloud Release Notes – 11 Feb 2021

## **Overview**

This release rolls out one of the prime features we have been working on - Mobile Keywords. This feature is aimed at helping you build contact lists by encouraging easy interaction with contacts over SMS Campaigns.

## **New Feature**

### Mobile Keywords

Mobile Keywords are texts or phrases that you can define to be associated with certain actions. You can use these Keywords in your SMS Campaigns to facilitate two-way interaction with your contacts.

You can create and manage the Mobile Keywords on the Mobile Keywords index page. On this page, you can also set up an Invalid Response Message that will be sent in response to texts that are received with the wrong or missing keyword.

 To learn about Mobile Keywords and how to create and use them, see [Mobile Keywords](https://galaxy.maropost.com/s/article/Mobile-Keywords).
