# Marketing Cloud Release Notes – 21 Dec 2021

## **Overview**

This release marks the launch of the Maropost Identity, an application where you can manage your Maropost account profile. Also, notable is the improved security with the enhancements in the Two-factor Authentication feature.

To learn more, scroll down!

## **New Feature**

### **Introducing Maropost Identity Application**

Maropost Identity is an application where users can manage their account profile and security settings, such as Multi-Factor Authentication (also referred to as Two-factor Authentication).

The Maropost Identity application is designed with a vision that all our Maropost users will have a platform to manage a single profile for all their accounts across different Maropost products. As of this release, the feature is only accessible to Marketing Cloud users.

Click your username in the top-right corner of the Marketing Cloud application, and then from the drop-down menu, select **Profile Settings**. Or, you can directly access the Maropost Identity application at: [https://id.maropost.com](https://id.maropost.com/).

## **Enhancement**

### **Strengthening of the Multi-Factor Authentication**

Multi-Factor Authentication provides an additional layer of security by employing two verification steps. In this release, we introduced a few enhancements to our existing multi-factor authentication feature, making your accounts more secure.

Instead of getting the verification code in your registered email address, you'll now get the code on the phone number that you provided during setup. This will ensure your account safety in case the email account itself is compromised.

To learn more about Multi-Factor Authentication in Marketing Cloud, see [Multi-Factor Authentication](https://galaxy.maropost.com/s/article/Multi-Factor-Authentication). 

You'll automatically set up the Multi-Factor Authentication when logging into your account. During setup, you will be asked to provide a phone number on which you'll want to receive the verification code.

![Multi-Factor Auth](https://us.v-cdn.net/6038474/uploads/attachments/migrated/MultiFactorAuth.0685g000006wiZGAAY.png)

On the next UI screen, you must enter the verification code that you receive on the phone and finish logging in.
