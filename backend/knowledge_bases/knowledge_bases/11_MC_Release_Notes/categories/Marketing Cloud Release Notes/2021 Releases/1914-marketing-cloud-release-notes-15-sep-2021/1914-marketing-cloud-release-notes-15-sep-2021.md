# Marketing Cloud Release Notes – 15 Sep 2021

## **Overview**

As part of this release, we bring you a new feature - DNS Setup. Now, you'll be able to set up your sending and link tracking domains through a self-driven process.

To learn about the feature, read on!

## **New Feature – DNS Setup**

You can now manage your DNS setup on the application UI. The feature enables you to add and manage Sending Domains and Link Tracking Domains for your account.

> **Note**: If you are a new user, the DNS setup is a mandatory step that you must complete before you can start sending email campaigns.

Click your username in the top-right corner of the application and from the drop-down menu, select Settings. On the Settings page, you'll find the newly added section – DNS Setup.

![DNS_Setup](https://us.v-cdn.net/6038474/uploads/attachments/migrated/DNSSetup.0685g000006wWT1AAM.png)

### **Sending Domains**

Sending Domain refers to the domain that is used for sending email campaigns. The Sending Domains index page (Settings page > Sending Domains) is where you can view all the Sending Domains added to your account. Here, you can add a new Sending Domain and manage the existing ones.

To learn more, see [Set Up and Manage Sending Domains](https://galaxy.maropost.com/s/article/Set-Up-and-Manage-Sending-Domains).

![Sending_Domains](https://us.v-cdn.net/6038474/uploads/attachments/migrated/SendingDomains.0685g000006wWSrAAM.png)

### **Link Tracking Domains**

Link Tracking Domain refers to the domain that is used for tracking the sent email campaigns. The Link Tracking Domains index page (Settings page > Link Tracking Domains) is where you can view all the Link Tracking Domains added to your account. Here, you can add a new Link Tracking Domain and manage the existing ones. 

To learn more, see [Set Up and Manage Link Tracking Domains](https://galaxy.maropost.com/s/article/Set-Up-and-Manage-Link-Tracking-Domains).
