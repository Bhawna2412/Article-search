# Marketing Cloud Release Notes – 2 Dec 2021

## **Overview**

In this release, we bring you an alternative process for integrating with Keap that is simpler and faster. This new feature includes a completely new UI and shortened steps.

To learn more, scroll down!

## **New Feature**

Maropost Marketing Cloud now offers a more intuitive UI and process for creating and managing your Keap integrations.

> **Note**: The old UI for Keap/InfusionSoft integration will still be available to all customers until all our customers have moved to the new UI/process. To learn more about the existing Keap integration feature, see [Integration with InfusionSoft/Keap (Old UI)](https://galaxy.maropost.com/s/article/Integration-with-InfusionSoft-Keap).

In Navigation, go to Integrations, and then on the Integrations page, select Keap. You'll be redirected to the Keap Integration index page, where you can create new integrations with Keap as well as view and manage your previously created integrations.

The following screenshot shows the Keap Integration index page:

![Keap_Integration_Index](https://us.v-cdn.net/6038474/uploads/attachments/migrated/KeapIntegrationIndex.0685g000006wcPUAAY.png)

If you are adding a new integration for the first time, you'll be required to create a connection first. After the connection is established, it is displayed on the Keap Integration index page. You can then select Add from the Actions drop-down menu to add an integration. Adding an integration is a quick step where you just select the integration type and the Maropost list into which you want to import the data.

To learn more about the Keap integration feature and how to set it up, see [Integration with Keap (New UI)](https://galaxy.maropost.com/s/article/Integration-with-Keap-New-UI) and [Adding a Keap Integration (New UI)](https://galaxy.maropost.com/s/article/Adding-a-Keap-Integration-New-UI).
