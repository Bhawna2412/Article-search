### **SMS Campaigns – Support for Special Characters**

You can now use special characters (UCS-2), such as icons, in the SMS message text. This, however, reduces the message text limit from 160 characters to 70 characters.

For more information about SMS Campaigns and how to create them, see [SMS Campaigns](https://galaxy.maropost.com/s/article/SMS-Campaigns) and [Creating SMS Campaigns](https://galaxy.maropost.com/s/article/Creating-SMS-Campaigns).
