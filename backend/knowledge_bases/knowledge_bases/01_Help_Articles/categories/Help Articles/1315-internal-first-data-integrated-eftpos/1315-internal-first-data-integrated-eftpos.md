# Internal - First Data Integrated EFTPOS

Retail Express is an exclusive provider of payment solutions via First Data, using the PC-EFTPOS integrated payment functionality.

In this article:

- [Who are First Data](#who-firstdata)
- [Sales Process](#sales)
- [Support Process](#support)

## []()Who are First Data

First Data are the largest global merchant service card provider in the world. First Data integrates the Retail Express platform with the customer's bank using the PC-EFTPOS functionality to process payments. They are the "middle man", similar to Tyro.

First Data offer customers incredibly competitive pricing, accepts all major cards from 3,700 financial institutions.
