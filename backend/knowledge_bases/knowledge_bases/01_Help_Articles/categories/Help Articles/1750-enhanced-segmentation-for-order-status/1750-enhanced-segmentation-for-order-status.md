# Enhanced Segmentation for Order Status

**Release Date:** 29th June 2020

## Overview

As a continuation of the earlier release, this release is also dedicated to the enhancements that were promised. This release, we added the ability to save changes in step 4 of the campaign builder. We also introduced a new option "Contains" for the Product condition in Segments.

To learn more about how these updates impact you, read on!

## Enhancements

**Added a Save Button in Step 4 of Campaign Builder**

  

Similar to other steps in the campaign builder, a Save button has been added in step 4 as well. The ability to save changes in step 4 enables you to use the step jumper feature when you want to send another test email or update the schedule of the campaign.

  

To learn more about the campaign builder, see [Managing Email Campaigns](http://galaxy.maropost.com/home/leaving?allowTrusted=1&target=https%3A%2F%2Fsupport.maropost.com%2Fhc%2Fen-us%2Farticles%2F360042044113) and [Creating Email Campaigns](http://galaxy.maropost.com/home/leaving?allowTrusted=1&target=https%3A%2F%2Fsupport.maropost.com%2Fhc%2Fen-us%2Farticles%2F360041572374).

  
  
  
  
  

**Simplified the Product Condition Queries in Segments**

  

In order to facilitate fetching results from a wider range of products that share a common aspect, we added the Contains option in the Product condition of the segment.

  

The Contains option searches for the entered keyword within the item id and product description of the products. For example, consider your product inventory includes five models of cars that you wish to include in the segment. Then, instead of creating five rules to include all five car models, you simply use the Contains option and enter the keyword 'car' into the value text box.

  

To learn more about the segments, see [Introduction to Segments](http://galaxy.maropost.com/home/leaving?allowTrusted=1&target=https%3A%2F%2Fsupport.maropost.com%2Fhc%2Fen-us%2Farticles%2F360048252733).
