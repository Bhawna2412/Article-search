# Offline POS fails to Install - Error: 0x84be0bc2

**If Offline POS occurs an error whilst installing with error message - 0x84be0bc2**

1. Perform a **RESTART**! IMPORTANT !
2. Check for Admin privileges
3. **Webroot** security (if installed) will block install

   ```
   Ask for permission to UNINSTALL
   ```
4. Always perform a **RESTART**if error occurs
5. Proceed after reboot with installation
