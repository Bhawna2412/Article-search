# Xero / MYOB Trial Accounts for Testing

## **Xero**

Xero will allow a full functioning account as a free 30 day trial which can be used with the automatic Xero Integration.

Go to [https://www.xero.com/au/](http://galaxy.maropost.com/home/leaving?allowTrusted=1&target=https%3A%2F%2Fwww.xero.com%2Fau%2F)

* Click Try Xero for free on the top right
* Create an Account by entering your details
* Setup a password via the confirmation email

![rtaImage - 2023-07-10T061745.985.jpeg](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000pplFsEAI.jpg)

* Enter a **Business Name**
* Select an **Industry**
* Under **Do you have employees?** select **No**
* Click **Start Trial**

## **MYOB**

MYOB will only allow you to create Local Files as a free 30 day trial, which cannot be used with the automatic MYOB Integration.  
  
It can, however, be used for testing via the Accounting Data Export.

Go to [https://www.myob.com/au/accounting-software/accountright](http://galaxy.maropost.com/home/leaving?allowTrusted=1&target=https%3A%2F%2Fwww.myob.com%2Fau%2Faccounting-software%2Faccountright)

* Scroll down to the Pricing area
* Under AccountRight Plus click Try Free for 30 Days
* Create an Account by entering your details
* Click Download AccountRight if the software is not already installed
* Setup a password via the confirmation email
