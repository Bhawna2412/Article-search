# Deploy - Enable Shippit Integration

This article explains how to deploy the Shippit Integration Feature once a signed solutions agreement/completed request form has been submitted:

Enabling the Shippit Integration

![rtaImage (35).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000powZhEAI.jpg)

1. Login to clients Retail Express

2. Navigate to **Settings > Feature Flag Settings**

3. Select checkbox **Enable Fulfilment Service?**

4. Select checkbox **Enable Shippit Integration?**

4. Click **Save Changes**

Zendesk - Sending the email 

![rtaImage (36).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000powZiEAI.jpg)

1. Add a new ticket

2. Add the email address for the contact on the solutions agreement/zendesk ticket

3. Select the macro **Implementation > Shippit > Getting Started**

4. Submit the ticket as **Solved**

Zendesk - Updating the Organisation

![rtaImage (37).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000powZrEAI.jpg)

1. Search for the clients *Organisation*in Zendesk

2. Select Checkbox **ShippIT**
