# How can I find which port my USB adapter is connected to ?

You can see the port name for you port adapter using the computer's Device Manager.

1. Click on your Start button at the bottom-left of screen.  
2. Using your Right mouse button click on Computer (or My Computer)  
3. In the popup menu click on Manage  
4. In the Computer Management window that opens, click on Device Manager.  
5. In the Device Manager window you will see in the list PORTS (Com & LPT)  
6. Double click on it and you will see the list of ports in use.

You should see the name of your adapter in the list. If you can't tell which one it may be, unplug the adapter from the computer and it should disappear from the list. Plug it back in to verify the port name next to it. If your adapter is not in the list, it is not installed properly. Please refer to the setup instructions that came with the device or refer to the manufacturer's website.
