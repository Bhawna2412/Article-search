# Deploy - All APIs

This article explains how to deploy all the APIs once a signed contract has been received:

Enabling the Web Services

![rtaImage (32).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000powYtEAI.jpg)

1. Login to clients Retail Express
2. Navigate to **Quick Links > Web Services Configuration**
3. Set **WMS** to enabled
4. Set **Inventory Planning** to enabled
5. Set **WebStoreV2** to enabled
6. Click **Save Changes**

Getting the Client ID

![rtaImage (33).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000powXIEAY.jpg)

1. Login to clients Retail Express
2. Navigate to **Help & Suppor**t**> What's my Client ID**
3. Press CTRL+C to copy the Client ID (you'll need this for the next step)

Zendesk - Sending the email 

![rtaImage (34).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000powYyEAI.jpg)

1. Add a new ticket
2. Add the email address for the contact on the solutions agreement
3. Select the macro **Finance > API >** **ALL API Deployment**
4. Replace the [ClientId] section with the Client ID from Retail Express
5. Replace the xxx section with the Retail Express URL for the client
6. Submit the ticket as **Solved**
