# View Previously Created Customer Stores

### Steps

From the partner portal homepage, navigate to **Stores** > **View Stores**.

The **Stores** page displays any previously created customer webstores in a tabular layout.

| Column Name | Definition |
| --- | --- |
| Store Name | Name of the Webstore. |
| Webstore | Webstore URL address. |
| Plan | Customer webstore subscription state. 			If a customer changes their plan, this will be reflected here. |
| Status | Predefined statuses for webstores at different stages in their lifecycle.  - **Trial** 				Maropost Commerce webstore trial - either 14 / 30 / 60 / 90 days. - **Staging** 				A webstore with an active subscription but not yet trading and processing orders in Maropost Commerce. - **Live** 				A webstore currently trading and taking orders. - **Cancelled** 				A webstore previously in staging or a live status and has now cancelled - **Expired Trial** 				A webstore that did not convert to a live site and has since expired their trial period. - **Suspended** 				A webstore that has access to the control panel suspended due to non-payment. 				In these circumstances, orders can still be processed online and via Point of Sale (POS) however customers cannot access the control panel to process orders. |
| Created | Webstore creation date. |
| Expires | Webstore trial expiry date. |

#### Filter stores by status

1. 
  Select the relevant status from the **Status** dropdown.
2. 
  Click the **Apply filters** button.

#### View a specific webstore

1. 
  Click the relevant **Webstore** link or **Store Name**.
2. 
  Click the **Apply filters** button.

### Result

You have successfully viewed previously created customer webstores within the **Maropost Commerce Partner Tools Home** page.
