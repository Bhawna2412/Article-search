# Retail Express Certificate Utility

The Retail Express Certificate Utility is a small application designed to streamline the process of installing the two different certificates that are required in order for Retail Express POS Hardware such as receipt printers and integrated eftpos, to function.  
  
**Notes Before You Begin:**

* The Retail Express Hardware Utility is still required for printing, however for updating or installing of certificates, this tool is all that is required, [view more details.](https://galaxy.maropost.com/s/article/Installing-Receipt-Printers-and-Integrated-EFTPOS)
* You may require admin priveledges on your Windows PC in order to run this utility. Please consult with your Store Manager/Owner/IT Staff if you do not have the access to open and run the file.

[**Click Here To Download the Zip File**](http://galaxy.maropost.com/home/leaving?allowTrusted=1&target=https%3A%2F%2Fwww.retailexpressmedia.com%2Fdownloads%2FCertificateUtility.zip)  
  
**Certificate Installation Instructions**

* After downloading the zip file, you must first extract it to a folder or location on your computer.
* Extract the Zip File:
  + Right Click On the Downloaded File and select extract
  + Or Open the zip file and select extract.
* Once extracted, vistit the folder and open the "Certificate Utility.exe" file.

![image.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g0000116ioYEAQ.jpg)

* Once opened succesfully you will see the below window as below:

![image.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g0000116iUIEAY.jpg)

  
  
**First Certificate Installation:**

* Click the first "Install" button - this will install the first of two certificates.
* **Note:** If the "Install" button is not able to be clicked/selected, you already have the certificate installed
  + Please Click **"Uninstall"** To remove the Old Certificate to ensure the new one is installed correctly.
  + Then Click "Install" to re-install the new copy of the certificate.

**Second Certificate Installation:**

* Complete the Client ID field by entering your client ID
  + This is found in Back Office > Settings  > What's my Client ID
* Select your network adaptor your computer is using to connect to the internet
  + Note: It is usually pre-selected with the correct value and you can skip this step.
  + If you receive an error, please ensure you have extracted the zip file as per the earlier steps, or that you have the correct client ID value entered.
* Click Install to complete the 2nd certificate install.
* **Note:** If the "Install" button is not able to be clicked/selected, you already have the certificate installed
  + Please Click **"Uninstall"** To remove the Old Certificate to ensure the new one is installed correctly.
  + Then Click "Install" to re-install the new copy of the certificate.

With Both now installed, you can close the application.  
  
**To ensure the new certificates are recognised by Retail Express POS:**

* Return to Retail Express POS
* Log Out of POS (Note: It is vital you select the Log Out button, not just closing the browser tab).
* Close the Web Browser (Chrome, Edge etc)
* Re-Open the Browser and log back into POS.
* Test & Confirm Printing/EFTPOS functionality is available.
