# MYOB Cloud Data Integration Overview

# MYOB Cloud Data Integration Overview

The information below displays data synchronised automatically - daily from Retail Express with the corresponding transactions in MYOB AccountRight 2019+.

Please note: This article refers to the cloud integration method - [click here for information on the Manual Data Export option](https://galaxy.maropost.com/s/article/MYOB-AccountRight-Accounting-Data-Export).

For more detailed information regarding how transactions are integrated, please refer to [Accounting Integrations - Integrated data sent from Retail Express](https://galaxy.maropost.com/s/article/Integrated-Data)

|  |  |
| --- | --- |
| **Retail Express** | **MYOB** |
| Cash and Carry sales  Layby movements  Debtors movements  Cost of Goods Sold (COGS)  Freight on sales  Voucher sales  Expired Vouchers  Surcharges | Daily Sales Invoice (per store) |
| Payments (per Payment Type including Direct Deposits) | Summarised or Individual Payments (applied to the Daily Sales Invoice) |
| Purchase Orders (received) | Bills (per Purchase Order) |
| Direct Costs | Bills (per Purchase Order) |
| Suppliers | Contacts |
| Duty | General Journal |
| Stock Transfers | General Journal |
| Stock Adjustments | General Journal |
| Cash Up Variances | General Journal |
| Purchase Variances | General Journal |
| Petty Cash (Money In/Out) | General Journal |
