# Customer Search

# Customer Search

The Customer Manager tool allows you to search and edit your customers, while simultaneously reporting on useful information at a glance. You can also [create a customer](https://galaxy.maropost.com/s/article/Create-a-Customer-in-Back-Office) directly from the Customer Manager tool.

## Using Customer Manager

1. Navigate to **Customers > Customers** (also available via Reports > Customer Reports > Customers or the "Customers" option from the Quick Links)
2. Use the **Search Filters**(see below for detailed information on each filter)  

   **Tip**: The Name fields support partial searches e.g. searching for the First Name "im" would display all results containing "im" - Imogen, Jim, Tim etc.

     
     

   **Note**: You can search for up to 25 commas or space-delimited customer IDs and Customer Numbers on the filter page
3. Click **Search**
4. The **Search Results** will be displayed  
   ![mceclip1 (23).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000powNbEAI.jpg)  
   [Click to view full size](http://galaxy.maropost.com/home/leaving?allowTrusted=1&target=https%3A%2F%2Fress.zendesk.com%2Fhc%2Farticle_attachments%2F4402426014223%2Fmceclip1.png)
5. To **sort the results** in a specific order click on each column heading; repeat to reverse the order between Ascending and Descending
6. To **Edit**the customer click the **Customer No.** field or the **Pencil** icon
7. To create a new **CRM Contact**(i.e. comments on an interaction with the customer) click the **Person** icon  

   **Related:**For information on entering comments see [POS - Public/Private Customer Comments](https://galaxy.maropost.com/s/article/Add-Comments-and-Notes-to-Sales))
8. Click **Print Report**to print the results to a printer
9. Click **Export to Excel**to download the results into a compressed zip file (the file is compressed due to the large amount of data - open the zip file to locate the spreadsheet)  

   **Tip:**The export file can be used for reporting purposes, or even as a template for Mail Merges/EDMs as it contains names, addresses and contact information. For even more functionality refer to the [Customer Activity Report](https://galaxy.maropost.com/s/article/Customer-Activity-Report) - bulk create Gift Vouchers, modify Price Groups and more.

## Search Filters

|  |  |
| --- | --- |
| **Field** | **Details** |
| Customer Number | Reference number internally assigned by Retail Express - this could be in the format of 300053 or for newer records AA-123456. Used for internal identification purposes only |
| Customer ID | Unique Customer Identification number - this will be in the format of 300053 |
| Company | The Company name |
| First Name | The contact's first name |
| Last Name | The contact's last name/surname |
| Email | The contact's email address |
| Phone/Mobile | The contact's phone number |
| Points Balance | If [Loyalty](https://galaxy.maropost.com/s/article/Loyalty-Configuration) has been enabled and the customers are accruing points, this field will display the total outstanding points yet to be redeemed |
| Points Earned | If [Loyalty](https://galaxy.maropost.com/s/article/Loyalty-Configuration) has been enabled and the customers are accruing points, this field will display the total points earned by the customer |
| Account Customer | Tick this option to filter by customers configured as an [Account Customer](https://galaxy.maropost.com/s/article/Creating-Account-Customers) |
| Created Date From/To | Used to select a date range of when the customer record was added to the Retail Express database |
| Last Updated From/To | Used to select a date range of when the customer record was last updated e.g. the address details were updated. This includes any modifications made by editing the customer in POS |
