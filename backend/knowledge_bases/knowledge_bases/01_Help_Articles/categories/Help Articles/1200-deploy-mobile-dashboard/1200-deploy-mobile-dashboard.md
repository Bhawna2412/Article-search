# Deploy - Mobile Dashboard

This article explains how to deploy the Mobile Dashboard Feature once a signed solutions agreement/completed request form has been submitted:

Enabling the Mobile Dashboard

![rtaImage (48).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000powcMEAQ.jpg)

1. Login to clients Retail Express

2. Navigate to **Settings > Global Settings**

3. Select checkbox **Mobile Dashboard**

4. Click **Save Changes**

Zendesk - Sending the email 

![rtaImage (49).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000powcREAQ.jpg)

1. Add a new ticket

2. Add the email address for the contact on the solutions agreement/zendesk ticket

3. Select the macro **Finance > Mobile Dashboard Activated**

4. Submit the ticket as **Solved**
