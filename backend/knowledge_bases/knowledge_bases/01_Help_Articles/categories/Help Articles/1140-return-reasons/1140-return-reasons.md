# Return Reasons

This feature allows you to create and configure custom reasons for product returns. Staff will be able to select the reason when process a return or exchange. View more details in the [Return Reason](https://galaxy.maropost.com/s/article/1Return-Reason-Report) report.

To configure your Return Reasons:

1. Navigate to **Settings > Return Reasons**
2. Enter the **Return Reason** name  
   ![mceclip0 - 2023-07-09T231804.646.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000pphaEEAQ.jpg)
3. Tick **Enabled**
4. Click **Insert**
5. Use the **pencil** icon to edit the name for an existing Reason
6. Click the **trash can**icon to delete Return Reasons (you will be unable to delete the Return Reason if it's already been used for a sale)
