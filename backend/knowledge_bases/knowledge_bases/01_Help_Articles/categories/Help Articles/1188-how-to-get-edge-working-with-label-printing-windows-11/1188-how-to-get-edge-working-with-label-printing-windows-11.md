# How to get Edge working with label printing (Windows 11)

This guide can remove the option of using IE and use Edge instead (works for both Windows 10/11)

Still requires [EPSON Virtual Port Assignment Tool](http://galaxy.maropost.com/home/leaving?allowTrusted=1&target=https%3A%2F%2Fdownload.epson-biz.com%2Fmodules%2Fpos%2Findex.php%3Fpage%3Dsingle_soft%26cid%3D6481%26scat%3D36%26pcat%3D3)

and [First Install](http://galaxy.maropost.com/home/leaving?allowTrusted=1&target=https%3A%2F%2Fwww.retailexpressmedia.com%2Fsupport%2Fdrivers%2Ffirstinstall.exe)

---

1. Open **Settings** in Windows
2. Type in **Find a setting**
3. ```
   Optional
   ```
4. Press **Add optional feature**  
   ![rtaImage - 2023-07-08T112111.971.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppNpLEAU.jpg)
5. Ensure **Internet Explorer Mode** is enabled  
   ![rtaImage - 2023-07-08T112116.095.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppORCEA2.jpg)
6. Open **Microsoft Edge > Settings > Default Browser**  
   ![rtaImage - 2023-07-08T112124.231.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppORWEA2.jpg)  
     
   ![rtaImage - 2023-07-08T112132.694.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppO0lEAE.jpg)
7. Set **Internet Explorer compatibility**to **Never**
8. White-List the REX URL with **Internet Explorer mode pages**  
   ![rtaImage - 2023-07-08T112137.440.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppORlEAM.jpg)
9. If the **Restart**tab is not available / change the **Allow sites to be reloaded in Internet Explorer mode**to **Default**and back to **Allow**  
     
   ![rtaImage - 2023-07-08T112142.628.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppO4sEAE.jpg)
10. Open **Internet Options**(Type in Windows search)  
    ![rtaImage - 2023-07-08T112147.300.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppOTIEA2.jpg)
11. Adjust **Active X**Settings (same as Windows 10 using IE)  
    [Knowledge base how to adjust Active X settings](http://galaxy.maropost.com/home/leaving?allowTrusted=1&target=https%3A%2F%2Fress.zendesk.com%2Fhc%2Fen-us%2Farticles%2F203100254)

    ```
    https://ress.zendesk.com/hc/en-us/articles/203100254
    ```
12. Now **Explorer Mode**websites will have a Internet Explorer logo in the URL  
    ![rtaImage - 2023-07-08T112155.709.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000ppORmEAM.jpg)
