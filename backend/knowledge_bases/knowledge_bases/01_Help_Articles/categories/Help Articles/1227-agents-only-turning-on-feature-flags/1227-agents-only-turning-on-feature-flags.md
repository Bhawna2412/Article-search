# Agents only - Turning on Feature flags

Feature flags are the ability to turn on/off features within a customer database. Previously this was typically managed within MGMT, however this is now being moved directly within the database itself (this aids with future development purposes).

**Note:**Currently the features are not specifically designed to be switched *off* i.e. "test this feature out, then we'll switch it off again". Features should only be switched on when appropriate for licensing or internal testing purposes.

## Turning on a feature flag

To turn on a feature flag you will need to do the following:

**Note:**You will need the Retail Express "god mode" login - credentials can be obtained from your manager

1. Log into Retail Express
2. Navigate to **Settings > Global Settings**
3. Scroll down to **7. Feature Settings**
4. Tick the required feature
5. Click **Save Changes**
