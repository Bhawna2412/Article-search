# Gross Profit Report - Fulfilled

Use the Gross Profit Report - Fulfilled to view details of Fulfilled products within the selected period, including Fulfilled COGS, Fulfilled GP % and $.

When using this report, please bear the following in mind:

* A maximum date range of one month can be applied
* The report displays both the date the invoice was created and fulfilled, however, the filters provided are for the Fulfilment date only.

To use the Gross Profit Report - Invoiced:

1. Navigate to **Reports > Sales Report > Gross Profit Report - Fulfilled**
2. Select from the Filters as required
3. ![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/Screenshot20240131at41348PM.05T5g000018qHEiEAM.jpg)
4. Click **Search**
5. The results will be displayed
6. Click the **Save** disk to export the report in a variety of formats
