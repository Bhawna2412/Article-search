# Admin Forms - Mobile Dashboard De-activation

This article explains how to disable the Mobile Dashboard feature in an existing clients Retail Express database.

Disabling the Feature

1. Login to clients Retail Express

2. Navigate to **Settings > Global Settings**

3. Untick the *Mobile Dashboard Enabled* checkbox

4. Click **Save Changes**

Zoho Subscription - Updating Monthly Hosting

1. Login to *Zoho Subscriptions*

2. Search for clients monthly subscription

3. Edit subscription to remove the line item for Mobile Dashboard

4. Click **Update**

5. Add comment stating "Mobile Dashboard Disabled #[*Zendesk ticket number*]"

6. If an Ezypay customer login to Ezypay to update their ongoing monthly fees from the next monthly invoice cycle and note the Ezypay variation id as an internal note in the Zendesk ticket and a comment against Zoho Subscriptions

Zendesk - Sending the email

![User-added image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/100.05T5g00000nHH13EAG.png)

1. Update the email address to be the one on the completed online form in original ticket

3. Select the macro **Finance > Mobile Dashboard De-activated**

5. Submit the ticket as **Solved**
