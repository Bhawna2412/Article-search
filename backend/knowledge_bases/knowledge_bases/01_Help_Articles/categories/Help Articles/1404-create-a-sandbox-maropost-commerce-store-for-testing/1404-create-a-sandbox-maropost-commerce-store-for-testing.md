# Create A Sandbox Maropost Commerce Store For Testing

### Steps

1. 
  Log in to the Maropost Commerce [partner portal](https://galaxy.maropost.com/s/partner-portal).
2. 
  From the **Create New Store** page, enter the customer's **Business Name**.
3. 
  Enter the customer's email address in the **Merchant's email** field.
4. 
  Select the customer's location **Country** from the dropdown.
5. 
  Select the customer's business **Industry vertical** from the dropdown.
6. 
  Select 'Sandbox' from the **Trial type** dropdown.
7. 
  Select 'Just investigating' from the **Described as** dropdown.
8. 
  Click the radio button next to the relevant trial period ('Choose the length of the trial').
9. 
  Click the **Create store **button.

  Maropost Commerce will advise you that a store is being created. Once completed, the store details will display.

### Result

You have successfully created a Sandbox Maropost Commerce store for testing.

You can either log in to the newly-created store, view the webstore or view the list of previously-created stores.

> **Note:** If you require a permanent webstore that can be used to showcase Maropost Commerce to clients:
> 
> Complete the steps above.
> 
> Create a case for our Support team, listing the webstore URL and the request to make the store a sandbox account.
