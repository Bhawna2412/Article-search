# Change Product Description

Once a product has been added to the sale you can change the description for that product for this sale only.

1. Using the mouse **double click on the product****row**to open the Edit Product window (press and hold on tablets or touch screens)
2. Edit the **description**
3. Click **Done**to save the changes and return to the sale window
