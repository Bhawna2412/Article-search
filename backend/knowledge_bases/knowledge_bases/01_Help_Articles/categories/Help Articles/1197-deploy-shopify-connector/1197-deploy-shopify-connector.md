# Deploy - Shopify Connector

This article explains how to deploy a Shopify Connector once a signed contract has been received and paid for:

Enable Shopify Connector

![rtaImage (58).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000powboEAA.jpg)

1. Login to MGMT

2. Navigate to **BackOffice > BO Site List**

3. Filter by company name in **Admin Link**

4. Click on **Edit** and select **Basic** from the dropdown options under **Shopify License**

5. Select **Update**

Enable Multiple Shopify Connectors

![rtaImage (59).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000powVgEAI.jpg)

1. If client has purchased more than 1 Connector then go to **Shopify > Shopify Site List**

2. Filter by company name in **Admin Link**

3. Clink on **Edit** and select the relevant number of connectors under **Licensed Stores**

4. Select **Update**

Zendesk - Sending the email 

![rtaImage (60).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000powfzEAA.jpg)

1. Add a new ticket

2. Add the email address for the contact on the solutions agreement

3. Select the macro **Implementation > Shopify BYO: Getting Started**

5. Submit the ticket as Solved
