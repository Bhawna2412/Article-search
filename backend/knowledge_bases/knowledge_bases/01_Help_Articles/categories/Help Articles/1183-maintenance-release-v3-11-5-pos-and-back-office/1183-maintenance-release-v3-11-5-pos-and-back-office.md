# Maintenance Release v3.11.5 - POS and Back Office

**Release date:** 10th March 2020

In this release we've introduced a new Payment Type for The Card Network, as well as made 5 improvements and resolutions.

In this release:

- [New Payment Type - The Card Network](#card-network)
- [Improvements and Resolutions](#improvements)

## []()**New Payment Type - The Card Network**

We've added a new Payment Type to use in POS for [The Card Network](http://galaxy.maropost.com/home/leaving?allowTrusted=1&target=https%3A%2F%2Fthecardnetwork.com.au%2F), a popular Gift Card provider. Using this Payment Type, payments can be allocated to The Card Network for reconciliation and reporting.

> **Note:** The Card Network Payment Type is used for reporting purposes only and is not an integrated payment system

To use the new Payment Type:

1. Navigate to **Settings > payment Types**
2. Locate the line for **The Card Network**
3. Tick the options for Enabled, POS and Loyalty as required (you can also adjust the Loyalty Multiplier as necessary)
4. Click **Save Changes**
