# Empty Hardware Services remover Automation Script

# Empty Hardware Services remover Automation Script

**This Automation script is to remove all the Empty, non-used Hardware services, that are not assigned to a register, within the Hardware Services.**

The reason for the script is to clean all empty entries, making the hardware service menu easier to located computers/hardware-services/IP-addresses, during troubleshooting.

![mceclip2.png](https://ress.zendesk.com/hc/article_attachments/5069476983567/mceclip2.png)

---

## Installation

Before beginning, you will be required to install an open-source Browser Extension from [UI Vision](http://galaxy.maropost.com/home/leaving?allowTrusted=1&target=https%3A%2F%2Fui.vision%2F) (unless already installed you may skip this step)

From there once the installation is complete, you may import the Scripts into the extension and continue to the Database with the error entries to remove.

---

## Running the Script

1. Download [Ui Vision](http://galaxy.maropost.com/home/leaving?allowTrusted=1&target=https%3A%2F%2Fui.vision%2F) - [https://ui.vision/](http://galaxy.maropost.com/home/leaving?allowTrusted=1&target=https%3A%2F%2Fui.vision%2F) (**unless already installed you may skip this step**)
2. Download the Zip file containing a JSON file named:
   1. Empty Hardware Service remover
3. Import the file using the import-file/zip  
     
   ![mceclip1.png](https://ress.zendesk.com/hc/article_attachments/5056807326223/mceclip1.png)
4. Open a NEW Browser window
5. Log into the Database
6. Open UI Vision in your Extension list  
     
   ![mceclip2.png](https://ress.zendesk.com/hc/article_attachments/5056810467727/mceclip2.png)
7. Go to **Settings > POS Settings > Hardware Services** in the Database
8. Select the script - Empty Hardware Service Remover
9. Play the Script in a loop with UI Vision  
     
   ![mceclip4.png](https://ress.zendesk.com/hc/article_attachments/5056870013583/mceclip4.png)  
     
   ![mceclip6.png](https://ress.zendesk.com/hc/article_attachments/5056877027471/mceclip6.png)
10. Once complete you may stop the Loop  
      
    ![mceclip0.png](https://ress.zendesk.com/hc/article_attachments/5069419253775/mceclip0.png)
