# Error when using Tyro payment option in Retail Express Hardware Utility

If receive an error when using Tyro payment option.

There are 2 Possible errors with the Tyro payment option.

1. **Tyro adapter is installed and a REX utility error occurred - REF Picture 1**
   1. When opening the Tyro Adapter and upon closing an error occurs in Retail Express Hardware Utility (crashing the service).
2. **Unsupported 16-bit Application Error - REF Picture 2**

Ref pictures below:

b. ![rtaImage (87).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000pox25EAA.jpg)

## Tyro adapter is installed and a REX utility error occurred - REF Picture 1

Ensure you are logged in as Admin

1. Download the Registry edit file below, open and run the file in the PC with the error.
2. Rerun the Retail Express Hardware Utility and the Tyro adapter should no longer crash the service.

---

## Unsupported 16-bit Application Error - REF Picture 2

1. Press **OK** on the error message
2. Press **Back** in the REX utility
3. Download the [Tyro Adapter](http://galaxy.maropost.com/home/leaving?allowTrusted=1&target=https%3A%2F%2Fhelp.tyro.com%2Fs%2Farticle%2FHow-do-I-install-the-Tyro-Terminal-Adapter) from [Tyro's Knowledge Base](http://galaxy.maropost.com/home/leaving?allowTrusted=1&target=https%3A%2F%2Fhelp.tyro.com%2Fs%2F)
   1. You may Google **Tyro Adapter**
   2. Or use this link [https://help.tyro.com/s/article/How-do-I-install-the-Tyro-Terminal-Adapter](http://galaxy.maropost.com/home/leaving?allowTrusted=1&target=https%3A%2F%2Fhelp.tyro.com%2Fs%2Farticle%2FHow-do-I-install-the-Tyro-Terminal-Adapter)
4. Download the adapter and install  
   ![rtaImage (88).png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05T5g00000pox2AEAQ.jpg)
5. Press Next in REX utility and the configuration button should be enabled.
