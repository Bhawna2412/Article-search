# Customer Contact Log Overview

The Contact Log is used to record interactions and follow up requests with your customers.

### View the Contact Logs for a Customer

Each customer card contains a contact log specific to that customer. To view the contacts for a customer:

1. 
  In your Maropost Commerce control panel select **Customers** > **View Customers & Prospects**.
2. 
  Select the the customer would like to view the **Contact Log** for.
3. 
  Scroll to the bottom of the **Contact Card** and you will see the **Contact Log**

From here, you can:

- Create a new entry.
- Mark an existing entry as complete.
- Edit an existing entry.
- Delete an existing entry.

### Create new Customer Follow Up Types

Maropost Commerce allows you to define your own **Follow Up Types** for **Contact Logs**. You can define anything you would like for this section. Defining a **Follow Up Type** allows you to filter all Contact Logs by a specific type for quick follow up.

Some suggested types are:

- Competition Winner
- Customer Disputes
- VIP Customers

To add customer follow up types:

1. 
  In your Maropost Commerce control panel select **Settings & tools** > **All settings & tools**.
2. 
  On the **Settings & tools** page, locate and select **Customer Follow Up Types**.
3. 
  On the **Customer Follow Up Types** page, select **Add New**
4. 
  Enter a Name and Description for your **Follow Up Type** and click **Save**.

### View All Contact Logs

Maropost Commerce allows your to view all Contact Logs in a list, making it easier to process customer follow ups.

> **Note:** Install the To-do List widget on your dashboard to see a quick list of all customers that need to be contacted.

1. 
  In your Maropost Commerce control panel select **Customers** > **Contact Logs**.
2. 
  The **Customer Contact Log** page will display all Entries that require follow up by default. You can apply filters to view different types of entries. You can also read the notes on an entry from this page.
