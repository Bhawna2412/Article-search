# Deploy: Unified API

This article explains how to deploy the Retail Express unified API once a customer has purchased access. This includes enabling the licence through MGMT, as well as the resulting macro to be sent to the customer to confirm they can now utilise the API.

**Note:**Customers will need to generate their own API keys from within Back Office - [click here](https://galaxy.maropost.com/s/article/Generating-Unified-API-Keys) to view the process.

## Enabling the API Licence

To allow customers to access the Generate API Licence Key feature you will need to add a "Core API" Licence for the customer in MGMT.

1. Log into MGMT  

   **Tip:**If you can't access MGMT you will need to request approval from your Manager to be added
2. Navigate to **Licenses > Add Client License**
3. Select the client domain from the **Value**column
4. Select **Core API**
5. Ensure **Enabled**is ticked
6. Click **Save**

To check existing Client Licences navigate to **Licenses > Client License List**.
