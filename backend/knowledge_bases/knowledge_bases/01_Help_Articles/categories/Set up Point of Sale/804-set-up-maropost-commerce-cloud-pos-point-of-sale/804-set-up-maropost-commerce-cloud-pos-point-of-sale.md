### POS Settings

Maropost POS comes with a number of built in settings to change functionality of the system. To access these settings in your Maropost Commerce control panel, select **Point of Sale**, and choose **POS Settings**.

#### Barcode Scanning

Select which UPC/EAN fields are searched when scanning a barcode in POS.

#### Rounding

Cash payments rounding mode - Rounding mode to be followed when processing cash transactions

Cash round to nearest - Nearest rounding value

#### Discounts

Maximum cashier discount percentage - This is the maximum discount a staff member with cashier permissions can apply to a order

Maximum store manager discount percentage - This is the maximum discount a staff member with store manager permissions can apply to a order

### Create a POS Desktop Shortcut

Using Maropost POS on a Windows PC? So that you do not need to manually go to [https://pos.neto.com.au](http://galaxy.maropost.com/home/leaving?allowTrusted=1&target=https%3A%2F%2Fpos.neto.com.au) each time you want to access the register through Chrome, you can create a desktop shortcut instead:

1. Open Google Chrome
2. Enter [https://pos.neto.com.au](http://galaxy.maropost.com/home/leaving?allowTrusted=1&target=https%3A%2F%2Fpos.neto.com.au) into the address bar and press **Enter**.

   ![image1](https://us.v-cdn.net/6038474/uploads/attachments/migrated/SetupMaropostCommerceCloudPOSPointofSale1.05T5g00000szV55EAE.png)
3. Navigate to the Chrome menu at the top right of screen.
4. Rollover **More Tools** and click **Add To Desktop**.

   ![image2](https://us.v-cdn.net/6038474/uploads/attachments/migrated/SetupMaropostCommerceCloudPOSPointofSale2.05T5g00000szV6IEAU.jpg)
5. A desktop icon for Maropost POS will now be created.
6. Return to your desktop and click on this icon to launch Maropost POS.
