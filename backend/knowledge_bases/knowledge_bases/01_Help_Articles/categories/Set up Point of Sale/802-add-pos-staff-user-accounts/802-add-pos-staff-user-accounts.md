> **Important:** The Maropost Point of Sale add-on must be installed to set up POS permissions.

### Add User Accounts for Staff

So that your staff users can login to your register(s) to process sales, you must setup staff user accounts. We recommend assigning each staff user their own account so that you can report on sales by user.

1. 
  In your Maropost Commerce control panel navigate to **Settings & tools** > **Staff Users** and select **Add New**.

  ![image1](https://us.v-cdn.net/6038474/uploads/attachments/migrated/AddPOSStaffUserAccounts1.05T5g00000szcXVEAY.png)

  **Please Note:** You might have to re-sync your Maropost POS registers for new staff members to be available to log in.
2. 
  Enter a username, first and last name, email address, password and permission group for the user. You should also set up a **Pin** for added security. When configured, **Save & Close**.

### User Permissions

You can control the permissions a staff user has by assigning them to a user permission group and can define what permissions each group has.

1. In your Maropost Commerce control panel navigate to **Settings & Tools** > **All Settings & Tools**.
2. 
  In the left side menu select **Staff User Accounts & Permissions** and choose **Staff Permission Groups**.

  ![image2](https://us.v-cdn.net/6038474/uploads/attachments/migrated/AddPOSStaffUserAccounts2.05T5g00000szcYEEAY.png)
3. 
  By default, only staff with **Admin** privileges will have access to process orders where the customer pays later. If you need to enable this option for other staff, click the **POS** tab and toggle the setting for pay later to **Allow**.

  ![image3](https://us.v-cdn.net/6038474/uploads/attachments/migrated/AddPOSStaffUserAccounts3.05T5g00000szcYxEAI.png)

### Password vs PIN

Each staff user has a password and a PIN. For security reasons, the password is used to authenticate a user when logging into your control panel and is used when setting up a register for the first time. The PIN is used for quickly logging into a register during a shift. Using a PIN negates the need for a user to remember and type in a lengthy password throughout the day.

![image4](https://us.v-cdn.net/6038474/uploads/attachments/migrated/AddPOSStaffUserAccounts4.05T5g00000szbNrEAI.png)
PIN codes are optional and are useful for when you want to add more security to your register. When switching to a user with a PIN code, your staff members will need to quickly enter in their four digit PIN code.

When logging in as a staff member who does not have a PIN code set, you will not be prompted for a PIN code. This can be handy if you are in a fast paced environment and don't need this functionality.

### Update Staff Passwords or PINs

In your Maropost Commerce control panel navigate to **Settings & tools** > **Staff Users**. From here you can select the staff member you wish to update, and update their Password or PIN Code.

![image5](https://us.v-cdn.net/6038474/uploads/attachments/migrated/AddPOSStaffUserAccounts5.05T5g00000szcaFEAQ.png)
**Please Note:** You may have to resync your Neto POS registers for new PIN codes to work.

### Add a Staff Profile Image

When you have a number of staff regularly logging in to a register, it can sometimes be tricky for them to find their user on the login screens. To make it easier to identify each staff member quickly, you can add a picture of your staff that will appear on the log in screens.

1. 
  From the Neto dashboard, navigate to **Settings & tools** > **Staff users**.
2. 
  Select the staff member to add the image to.
3. 
  Select the profile image to upload, and click **Save**.

![image6](https://us.v-cdn.net/6038474/uploads/attachments/migrated/AddPOSStaffUserAccounts6.05T5g00000szcbDEAQ.png)
**Please Note:** You might have to resync your Neto POS registers for the profile images to appear.
