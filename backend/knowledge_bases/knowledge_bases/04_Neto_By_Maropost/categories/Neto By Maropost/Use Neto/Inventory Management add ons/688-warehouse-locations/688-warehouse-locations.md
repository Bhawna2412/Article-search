## Set up Warehouse Locations

To set up the warehouse locations add-on:

1. In your Neto control panel click on the **Addons** menu.
2. Find the Warehouse Locations add-on and click the **Install** button.
3. Click the **Install** button in the pop-up that appears.

Once activated, the **Default Fulfilment Warehouse / Pick Locations** will be added to the **Warehouse & Picking** section of the product details in the control panel (except for variation parent products).

![Warehouse Locations2.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05TJ100000FtwOpMAJ.jpg)
You can use the priority field to set up a warehouse order, for fulfilment of products. Sales orders will be fulfilled from the warehouse with the priority set to the lowest value first (ascending order). To use this you must ensure that the advanced configuration setting `USE_WAREHOUSE_WITH_STOCK` is enabled.
