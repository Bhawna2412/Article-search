# Inventory Planner

### This add-on was built by Inventory Planner.

[Visit Add-On Website](https://inventory-planner.com/)

For support with this add-on [contact Inventory Planner](http://help.inventory-planner.com/en/).

## Set up the Inventory Planner add-on:

1. In your Neto control panel click **Addons** in the main menu.
2. In the Inventory Management section locate the Inventory Planner add-on and click the **Install** button.
3. Create an account on [inventory-planner.com](https://inventory-planner.com) if you haven't already.
4. Select your platform as **Neto **or Neto.
5. Follow instructions on the set-up page on the Inventory Management site.
6. It will request an API key which is associated with a staff user in Neto. To get this, navigate to **Settings & Tools** > **Staff Users**.
7. Click on the staff user you wish to associate the account with. Scroll down to **API Key** and click the **Regenerate** button.
