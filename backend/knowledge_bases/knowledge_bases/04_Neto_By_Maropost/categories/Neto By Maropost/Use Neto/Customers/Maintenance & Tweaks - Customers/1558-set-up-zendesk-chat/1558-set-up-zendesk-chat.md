# Set up Zendesk Chat

### This add-on was built by Zopim.

Visit Add-On Website

For support with this add-on [contact ]().

## To Install and Set up the Zendesk Chat Add-on

1. In your Neto control panel navigate to the **Addons** store and look for **Zendesk Chat** and click **Install**.
2. Insert the **Header Script** to insert into the pop up window that appears.
3. To find this header script, go to [https://www.zendesk.com/service/messaging/live-chat/](https://www.zendesk.com/service/messaging/live-chat/) and either sign-in or register your account.
4. From the left hand side menu, click **Widgets** and copy the Zopim provided live chat widget code and paste it into the header script field in Neto.
