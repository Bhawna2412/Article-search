# Set up Automatic Payment Reminders

1. In your Neto control panel navigate to **Settings & Tools** > **All settings & tools**.
2. Click on the **Payments & Tax** tab and click on **Payment Reminders**.
3. Set the number of days for the 3 automatic reminders and click **Save**.

If you do not want to use the payment reminder system then leave the payment reminder fields on the above setup page blank. The system will only send payment reminders if these fields are populated.

You can edit the wording of your payment reminder emails for each of the 3 payment reminders from within the email templates.

1. To access this, simply navigate to **Settings & Tools** > **All settings & tools**
2. Click on the **System Templates** tab and click **Email Templates**.
3. Click on **ID 11 - Payment Reminder Emails**.
4. Select which reminder email you wish to edit.
5. Make your changes and click **Save**. If you are not familiar with HTML or Neto’s B@SE language, you can [contact a design partner](https://forms.neto.com.au/services/designtweak.html) to create a new template for you.

Payment reminder emails also include a list all outstanding invoices on a customers account (much like an account statement).

A link for customers to pay outstanding invoices online is also included.
