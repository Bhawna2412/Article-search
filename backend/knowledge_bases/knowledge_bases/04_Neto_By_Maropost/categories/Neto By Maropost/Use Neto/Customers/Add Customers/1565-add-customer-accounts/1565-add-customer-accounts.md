# Add Customer Accounts

## Add a Customer Manually

To add a customer manually:

1. In your Neto control panel navigate to **Customers** > **Add New Customer**.
2. Fill out the details for the customer and press the **Active** option in the top right hand corner. If you need help with understanding any of the fields in the customer card, see our [overview guide here](https://galaxy.maropost.com/kb/articles/1566-customers-overview)
3. Once completed, press **Save.**

## Add Customers in Bulk

Use the Data Import Wizard to import [.CSV files (spreadsheets)](https://en.wikipedia.org/wiki/Comma-separated_values) into your website with all of your customers.

Here is a sample copy of our popular customer fields to use as a template: [Download here](https://docs.neto.com.au/assets/files/Templates/import_customers_popular.csv).
Here is a sample copy of all customer fields to use as a template: [Download here](https://docs.neto.com.au/assets/files/Templates/import_customers_all.csv).

1. To use the import wizard, navigate to **Settings & tools** > **Import Data** then click Perform complex import and select Start Data Import Wizard.
2. Upload the template file and click **Continue to Next Step**. Map the fields in your CSV file to the fields in Neto. Once complete, click **Continue to Next Step**.
3. Set your update options and click **Continue to Next Step**.
4. Click **Process File Now**. This will now generate a batch process which will take time depending on other tasks in the queue.
