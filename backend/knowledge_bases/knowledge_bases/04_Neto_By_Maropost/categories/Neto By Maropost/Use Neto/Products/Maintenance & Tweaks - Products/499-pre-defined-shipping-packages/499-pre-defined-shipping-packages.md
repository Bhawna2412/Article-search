## Install the Add-on

To install the add-on:

1. In your Neto control panel navigate to the **Addons** menu.
2. In the product management section, find the Pre-defined Shipping Packages add-on and click the **Install** button.
3. Confirm by clicking the **Install** button in the pop-up that appears.

Once activated, a **Package** column will be added when creating a consignment for an order:

![Pre-defined Shipping Packages1.jpg](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05TJ100000PMahtMAD.jpg)
The package dimensions will be entered for you based on the package settings.

## Add Pre-defined Packages

To set up a pre-defined package.

1. In your Neto control panel navigate to **Settings & tools** > **All Settings & Tools **
2. Select **Shipping** from the left side menu, and choose **Pre-defined Packages**.
  ![Pre-defined Shipping Packages2.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05TJ100000PMak9MAD.jpg)
3. Select **Add New**.
  ![Pre-defined Shipping Packages3.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05TJ100000PMamZMAT.jpg)
4. ![Pre-defined Shipping Packages4.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05TJ100000PMameMAD.jpg)
  Configure the fields for the package and **Save**.

Your pre-defined package will now be selected during order processing, should it meet the size requirements of the order.

## Create New Package Types

By default you will have one package type, carton. However, you can also add package types for identification purposes, such as envelope, cylinder, pallet etc.

1. In your Neto control panel navigate to **Settings & Tools** > **All Settings & Tools**.
2. Select **Shipping** from the left side menu and choose **Pre-defined Packages Types**.
  ![Pre-defined Shipping Packages5.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05TJ100000PMamtMAD.jpg)
3. Click the **Add New** button.
  ![Pre-defined Shipping Packages6.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05TJ100000PMY31MAH.jpg)
4. Enter a **Package Type** and **Package Description** and **Save**.
  ![Pre-defined Shipping Packages7.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05TJ100000PMaoGMAT.jpg)

The package type can now be chosen when creating a pre-defined package.

## Automate Sales Order Packaging

Neto can automatically package items on orders into your predefined packages, based on the dimensions and cubic weight of your items. In such cases, the system will automatically create the correct number of shipping labels based on the number of pre-defined packages needed to pack the order.

You can configure rules around the max fill percentage per package and more:

1. In your Neto control panel navigate to **Settings & Tools** > **All Settings & Tools**.
2. Select **Sales Orders & Invoices** from the left side menu, and choose **Order Processing Rules**.
  ![Pre-defined Shipping Packages8.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05TJ100000PMaoVMAT.jpg)
3. ![Pre-defined Shipping Packages9.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05TJ100000PMap4MAD.jpg)  - **Minimum Packaging Split Percentage**: If a pre-defined package is not filled to this percentage, split the order into smaller pre-defined packages.
  - **Dynamic Packaging Percentage**: If a pre-defined package is not filled to this percentage, allow items without a cubic or weight to be added to the pre-defined package.

  In the **Warehouse Auto Packaging Rules** section, configure the settings and **Save Changes**.

  If you do not setup predefined packages for automatic packaging of orders, consignments will be populated as per the dimensions, weight, and cubic of products in your database.
