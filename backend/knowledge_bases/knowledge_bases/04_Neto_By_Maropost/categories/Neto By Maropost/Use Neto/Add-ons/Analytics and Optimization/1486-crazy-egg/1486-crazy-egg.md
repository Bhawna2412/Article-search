# Crazy Egg

### This add-on was built by Crazy Egg.

[Visit Add-On Website](http://www.crazyegg.com/)

For support with this add-on [contact Crazy Egg](https://support.crazyegg.com/hc/en-us/requests/new).

## Set up the Crazy Egg add-on

1. Register your account for [Crazy Egg](http://www.crazyegg.com/).
2. Locate your code by following the instructions provided [here](https://app.crazyegg.com/instructions). Copy this code so that it can be entered into Neto.
3. In your Neto control panel select **Addons** from the menu.
4. Find the Crazy Egg add-on and click the **Install** button.
5. Paste the code you copied previously, and click the **Save**button.
