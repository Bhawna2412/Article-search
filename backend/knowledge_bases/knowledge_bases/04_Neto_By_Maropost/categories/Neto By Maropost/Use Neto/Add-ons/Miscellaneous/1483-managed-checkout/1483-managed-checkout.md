# Managed Checkout

Managed Checkout is designed with responsiveness in mind, helping you capture more sales, allowing you to focus on running your business and less time chasing abandoned carts. By using Neto's managed checkout, any updates will be applied automatically keeping you using the latest technology and security features.

**Please Note**: The design of your Managed checkout is separate to the rest of your Neto website. Customising managed checkout is not available outside of the available settings.

Managed Checkout is a standard feature for customers who created an account after September 2016. If you're an existing Neto customer using Neto version 6.4 and onward, you can choose to make **Managed Checkout** the default checkout on your web store.

## Turn on Managed Checkout

Before you turn on managed checkout it's important to check if you have any existing customisations on your checkout page. Customisations will be lost if you enable managed checkout without consultation from our design specialists.

To verify if managed checkout is already enabled, or if you can turn it on, please follow the steps below.

### Visual check

One way to confirm if managed checkout is enabled is to compare the legacy checkout page to our [managed checkout](https://inspirationtheme.neto.com.au/_mycart?ts=1497844939174507&fn=payment&sku=173) page.

**Legacy Checkout**
- Under the title **Secure Checkout** it will have the **Returning Customer** alert section.
- In Step 3 (**Payment Details**) it will have the payment methods as buttons and also a dropdown selection.

**Managed Checkout**
- Under the **Secure Checkout** title, it will have three columns; **Returning Customers**,** Checkout With Payment**, or **Proceed To Checkout**, with available payment methods.
- In step 3 (**Payment Details**), it will have the payment methods as radio buttons and display the payment method's description when you select a method.

### Control Panel Settings

If you're still not sure if managed checkout is enabled, you can also check by logging into your Neto control panel to check the settings.

In your Neto control panel navigate to **Settings & Tools** > **Payment Methods**.

If you don't have the button that says **Enable New Methods** then you're already using managed checkout, and your account is ready for the newer payment methods to be set up.

If you do see the button that says **Enable New Methods** then you do not have managed checkout enabled.
