# Reviews.io

Visit Add-on Website!

For support with this add-on [contact Reviews.io](https://www.reviews.io/contact/contact-us).

## Overview

Reviews.io is a comprehensive reviews platform that has deep integrations with social media platforms including:

- Facebook
- Instagram
- Google
- Pinterest

It can collect and aggregate reviews across those platforms and publish them on official channels such as Google. They also have tools to generate social proof, banners and ad campaigns using the reviews.

## Benefits & Features

With the **Reviews.io** add-on you can:

- Export Neto sales orders from a given start date which are used to trigger follow up emails to collect reviews.
- Export all active and approved products to populate ad units, campaigns and review emails.
- Export customers to link to your Reviews.io customer database, which spans multiple social media sites and sources.

## Set up the Reviews.io Add-on

To install and set up the Reviews.io add-on:

1. [Add an API user account](https://galaxy.maropost.com/kb/articles/1614-add-an-api-user-account) so that Review.io can receive information from your Neto control panel. We recommend duplicating the API User permission group, and allowing access to:
2. In your Neto control panel navigate to the **Addons** menu.
3. In the Marketing section, find the Reviews.io add-on and click the **Install** button.
4. In the pop-up window that appears, click the **Install Button**.
5. The add-on will be installed. Click the **Configure Reviews.io** button when it appears.
6. Log in to your existing Reviews.io account, or create a new account if you don’t have one.
7. Follow the steps on the [Reviews.io](https://intercom.help/reviews-io/en/articles/3339568-installation-neto) website to complete configuration.
