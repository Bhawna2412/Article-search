# Download Image from URL

## This add-on was built by Maropost

Add-ons and Integrations built in-house by Maropost are documented and supported by our customer service team. If you're having issues with this integration, [let us know](https://www.netohq.com/support/s/ask-us)!

To install and set up the download image from URL add-on:

1. In your Neto by Maropost control panel click on the **Addons** menu.
2. In the Product Management section, find the Download Image from URL add-on and click the **Install** button.
3. Click the **Install** button in the pop-up that appears.
  A **Download Image from URL** button will appear in the **Images** section of each product in the control panel:

  [Image: ]
4. Click on the **Download Image from URL** button, the field will appear on the right.
5. Enter the full web address of the image. Click on the arrow to the right of the field and select the **Force Check** checkbox. Then click the **Save** button.

A process runs once an hour (however this can be changed) to download the image to a folder on your site. A second process runs every 15 minutes (the schedule can be changed) which creates the three sizes of the image that is displayed on your website.

**Force Check** will download and process the image from the entered web address next time the processes are run.

You can also set to check download the image from the web address on a regular basis by entering the number of days in **Image Download Check Interval**. Set to zero if you don't need periodic checking.

## Tips for Downloading Images

- Make sure the image web address is valid.
- Make sure that the image can be accessed. Some web servers block other sites accessing the images.
- The image can be in JPEG, PNG or GIF formats. It is recommended that you use JPEG as this format has smaller file sizes.
- Make sure there is no question mark in the image web address.
