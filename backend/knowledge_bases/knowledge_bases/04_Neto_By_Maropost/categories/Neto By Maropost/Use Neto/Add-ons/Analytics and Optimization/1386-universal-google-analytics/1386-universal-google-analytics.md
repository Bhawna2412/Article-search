# Universal Google Analytics

> **Warning:** This add-on has been deprecated, please use the [Google Analytics 4](https://galaxy.maropost.com/kb/articles/1254-google-analytics-4) steps.

### This add-on was built by Google.

Visit Add-On Website

Google Analytics can track visitors from all referrers, including search engines and social networks, direct visits and referring sites. It also tracks display advertising, pay-per-click networks, email marketing and digital collateral such as links within PDF documents.

> **Note:** Neto partners can set this up for you. [Visit the job request portal](https://www.netohq.com/job-request-portal) for a quote.

## Install the Google Analytics Add-on

**Please Note**: Ensure your Google Analytics account is set up using the Universal Analytics property, not the GA4 property.

1. In your Neto control panel select **Addons**.
2. Locate **Google Analytics** and click install. In order to locate the Google Analytics addon, please ensure you have the **All Add-ons** filter selected to see all available addon's.
3. Once the Google Analytics add-on has been installed, click the **Configure** button.
4. Enter your **Google Analytics tracking ID, which starts with UA, eg. UA-12345678-1**. To find your tracking ID, see [Google's help article here](https://support.google.com/analytics/answer/1008080?hl=en#trackingID).
5. **Save **changes.
  **Please Note**: Changes to a custom script require you to authenticate your access. Click the **Send Token Now** button and an email will be sent to you with a security token. Copy the token and paste it into the verification field. Click the **Verify Token** button and your changes can be saved. The token will last two hours in case you need to change multiple scripts.

  If you need to verify your site ownership with Google, you can opt to use the "meta code" verification method. [Click here](https://galaxy.maropost.com/kb/articles/426-google-search-console-setup) for an article on how to verify site ownership.

The Google analytics add-on is now installed. To set up ecommerce tracking, Tag Manager, or search console, see the related articles below.
