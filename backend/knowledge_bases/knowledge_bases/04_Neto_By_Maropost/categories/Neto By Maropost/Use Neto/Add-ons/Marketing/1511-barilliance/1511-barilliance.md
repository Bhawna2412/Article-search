For support with this add-on [contact Barilliance](https://www.barilliance.com/best-ecommerce-personalization-solution/).

## Set up Barilliance

1. Create an account on [Barilliance](http://www.barilliance.com/) and copy the code provided to you.
2. In your Neto by Maropost control panel click on the **Addons** menu.
3. Find the Barilliance add-on and click the **Install** button.
4. Paste the code you copied from Barilliance into the pop-up window, and click the **Install** button.
