# Accounting Integration Set up Guides and Feature Comparison

Sync and track the status of your Neto by Maropost orders and RMAs in MYOB. Novel Integrations allows mapping of locations in MYOB with warehouses in Neto.

#### This add-on was built by Novel Integrations

[Visit add-on website!](https://novelintegrations.com.au/)

For support with this add-on [contact Novel Integrations](https://novelintegrations.com.au/contact/).

## Overview

Neto has a number of accounting integrations available to suit your needs. If you already use an accounting program that Neto integrates with, use the links below to go straight to the setup steps.

- Xero (direct)
- MYOB Live (direct)
- [Crossbeam (Xero)](https://galaxy.maropost.com/kb/articles/1604-crossbeam)
- MYOB by Channelup
- MYOB AccountRight by Novel Integrations

Use the table below to determine which accounting integration has the capabilities you require.

- Export to Accounting System
- Import from Accounting System

## Export to Accounting System

|  |  |  |  |  |  |  |
| --- | --- | --- | --- | --- | --- | --- |
|  |  |  |  |  |  |  |
| **Accounting System** | **Sales orders** | **Customers** | **Products** | **Payments** | **Stock levels** | **RMA** |
| Xero (direct)** | Yes | Yes | Yes | Yes | Yes | Yes |
| MYOB (direct) | Yes | Yes | No | Yes | No | Yes |
| MYOB by Channelup | Yes | Yes | Yes | Yes | No | No |
| MYOB AccountRight by Novel Integrations | Yes | Yes | Yes | Yes | No | Yes |

** Xero direct integration is compatible with Neto Commerce Inventory purchase orders.
