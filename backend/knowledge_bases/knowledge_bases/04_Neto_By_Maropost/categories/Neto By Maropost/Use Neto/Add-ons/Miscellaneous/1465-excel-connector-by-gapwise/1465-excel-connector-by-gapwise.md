# Excel Connector by Gapwise

**This add-on was built by Gapwise.**

[Visit add-on website!](https://gapwise.consulting/maropost-commerce-cloud-connector-excel/)

For support with this add-on [contact Gapwise](https://gapwise.consulting/contact-us/).

## Overview

Fetch, edit and save your data directly from Microsoft Excel in seconds with this powerful add-in. This easy-to-use interface is designed to save you time and simplify the task of managing your store data.

## Benefits & Features

- Fetch and edit most product fields.
- Bulk edit 1000's of products simultaneously.
- Support for very large stores (100,000+ products).
- Wildcard search filtering on Name and SKU.
- Changes are validated before saving to Maropost.
- Update multiple stores from one place.
- Fetch content and orders.
- Fetch and edit product category assignments.
- Fetch and edit warehouse stock levels.
- Fetch and edit product price groups.
- Fetch and edit product specifics (formerly item specifics).
- Fetch and edit sales channels.

**Data that is transferred between systems:**

- Orders
- Customers
- Products
- Content/Categories
