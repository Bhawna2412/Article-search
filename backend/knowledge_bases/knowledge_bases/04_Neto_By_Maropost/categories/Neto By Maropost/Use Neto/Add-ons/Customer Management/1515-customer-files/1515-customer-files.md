# Customer Files

### This add-on was built by Maropost

Add-ons and Integrations built in-house by Maropost are documented and supported by our customer service team. If you're having issues with this integration, [let us know](https://www.netohq.com/support/s/ask-us)!

## Set up Customer Files

To install and set up the customer files add-on:

1. In your Neto by Maropost control panel click the **Addons** menu.
2. In the customer management section find the Customer Files add-on, and click the **Install** button.

Once activated, the **Files** section will appear on each customer's card in the control panel:

To add a file, drag-n-drop the file on the grey box or click on the **Browse** / **Choose File** button and select the file. You can remove the file by clicking on the red cross of the right-hand side of the file details.
