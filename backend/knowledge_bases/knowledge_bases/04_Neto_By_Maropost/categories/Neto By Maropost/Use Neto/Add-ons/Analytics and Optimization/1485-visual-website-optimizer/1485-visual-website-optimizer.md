## Set up the Visual Website Optimizer Add-on

1. Create a campaign in [Visual Website Optimizer](https://vwo.com/) and copy the smart code generated.
2. In your Neto by Maropost control panel select **Addons** in the menu.
3. Locate the Visual Website Optimizer add-on and click the **Install** button.
4. Paste the code into the add-on and click the **Save** button.
