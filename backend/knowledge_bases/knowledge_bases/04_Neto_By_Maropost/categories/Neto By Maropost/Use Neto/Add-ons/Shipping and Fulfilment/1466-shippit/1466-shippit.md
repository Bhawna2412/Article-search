# Shippit

### This add-on was built by Shippit

[Visit Add-on Website](https://www.shippit.com/)

For support with this add-on [contact Shippit](https://www.shippit.com/contact-us/).

## Overview

With the Shippit add-on you can:

- Automatically sync your orders from Neto to Shippit.
- Map your Shipping options to a Shippit service type.
- Automatically fulfil your orders once they are shipped.
- Manually send orders to Shippit to be fulfilled.

## Generate an API Key

So that Shippit and Neto can communicate you will first need to generate an API key.

1. In your Neto control panel navigate to **Settings & tools** and select **All settings & tools**.
2. In the Developer Tools section, click on **API Settings**.
3. Check the **Enable Neto API Webhook** box, and copy your **Neto API Key**.
4. Click **Save Changes** button. Don’t close this page as you’ll need to enter the webhook into Shippit soon.

## Install and Set up the Shippit Add-on

1. In your Neto  control panel click the **Addons** menu.
2. In the Shipping section find **Shippit** and click on **Get App**, then **Get App** again.
3. You’ll be directed to Shippit to enter your **Domain** (website address) and the **API Key** you copied earlier. Once entered click **Continue**.
4. You’ll now be given the API webhook URL to enter into your Neto API settings page. Then click the **Save Changes** button.

Your Neto control panel and Shippit are now linked to transfer orders and dispatch information.
