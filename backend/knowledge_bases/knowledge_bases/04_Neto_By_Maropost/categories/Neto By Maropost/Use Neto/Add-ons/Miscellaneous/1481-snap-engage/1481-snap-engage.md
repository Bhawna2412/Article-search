# Snap Engage

## This add-on was built by Snap Engage.

Visit Add-On Website

For support with this add-on [contact Snap Engage](https://help.snapengage.com/).

Deliver exceptional customer experiences with Snap Engage. Their flexible, powerful chat solution handles business complexity with ease. Capture more leads, reduce costs, and skyrocket customer satisfaction. To set up the Snap Engage add-on:

1. Create an account with [Snap Engage](https://snapengage.com/) and copy the code for your website.
2. In your Neto by Maropost control panel click on the **Addons** menu.
3. Locate the Snap Engage add-on and click the **Install** button.
4. Paste the code from Snap Engage and click save changes.
