## Overview

Create consignments (labels) for orders in batches and mark dispatched at the click of a button. For large order volumes, this can speed up your order processing by creating multiple consignment numbers in one easy step.

## Set up the Batch Order Processing Add-on

To install the batch order processing add-on:

1. In your Neto control panel navigate to the **Addons** menu.
2. In the order management section, find **Batch Order Processing** and click the install button.

> **Note:** To remove the add-on, click the **Uninstall** button.

Once installed, the **Bulk Pick** and **Bulk Create Consignments & Dispatch** buttons will appear at the bottom of the order listing in the control panel. For more information on processing orders in bulk see this article.

**Please Note**: Only **Bulk Pick** will appear in the **New** orders list.
