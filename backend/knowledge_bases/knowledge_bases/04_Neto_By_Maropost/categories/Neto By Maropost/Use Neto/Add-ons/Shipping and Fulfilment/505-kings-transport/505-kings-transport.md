# Kings Transport

## This add-on was built by Maropost

Add-ons and Integrations built in-house by Maropost are documented and supported by our customer service team. If you're having issues with this integration, [let us know](https://www.netohq.com/support/s/ask-us)!

Shipping label and manifest features require the **Neto Commerce Ship** add-on.

To install this add-on:

1. In your Neto control panel navigate to the **Addons** menu.
2. In the Shipping section locate Kings Transport and click the **Install** button.
3. Fill in the [Shipping Carrier Label Setup Form](https://forms.neto.com.au/services/shippingcarrierlabelsetup.html) and Maropost staff will set up the shipping labels for you for free.
