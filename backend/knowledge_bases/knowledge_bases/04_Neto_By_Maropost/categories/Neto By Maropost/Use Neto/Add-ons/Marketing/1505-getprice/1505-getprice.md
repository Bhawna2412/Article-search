# GetPrice

### This add-on was built by Maropost

Add-ons and Integrations built in-house by Maropost are documented and supported by our customer service team. If you're having issues with this integration, [let us know!](https://www.netohq.com/support/s/ask-us)

## Overview

GetPrice is a shopping comparison website which allows customers find and compare millions of products, all in one location.

Neto uses data feeds to send information to GetPrice to update your products through the shopping comparison website.

**Tip**: Neto partners can set this up for you. [Request a data service job](https://www.netohq.com/job-request-portal) for a quote.

## Set up the GetPrice Add-on

To install and set up GetPrice:

1. Sign up for a [GetPrice](http://www.getprice.com.au/MerchantSignUp.aspx) account.
2. In your Neto control panel navigate to **Other** > **Shopping Comparison Feeds**.
3. Locate GetPrice and click the **Install** button.
4. Enter your **Getprice Account ID** and click **Install**.
5. You’ll be provided with a Feed URL. Provide this URL to GetPrice. This is a link to a file that contains a list of your products in the required format.
6. You can modify the Feed URL, filtering products that you don’t wish to push to GetPrice. To do this, click the **Edit** button.
7. Scroll down to the **File Builder** section and click the **Filter** tab to make products, categories, pricing and other changes. Scroll down and click **Save Changes**.
8. To make changes to the feed, simply scroll back up the top and click the **Generate Data File** button and click **Download Data File** button to review the results of your changes.
