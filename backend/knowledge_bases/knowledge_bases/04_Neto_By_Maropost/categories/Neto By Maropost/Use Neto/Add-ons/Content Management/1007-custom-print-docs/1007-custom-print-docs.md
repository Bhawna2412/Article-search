## Set up the Custom Print Docs Add-on

To create new template sets you need to install the Custom Print Docs add-on:

1. In your Neto control panel click the **Addons** menu.
2. In the content management section, find the Custom Print Docs add-on and click the **Install** button.
3. Click the **Install** button in the pop-up window that appears.

Once activated, the **Custom Documents** menu option will appear in the **Admin** menu.

## Create a Custom Print Doc

To create a custom document template:

1. Get your web designer or a [Neto partner](https://forms.neto.com.au/services/designtweak.html?) to create the template for you.
2. Upload it to your site using SFTP to the /private/www/netosuite/SysDoc/printdocs/custom/order/  folder.
3. In your Neto control panel navigate to **Setup & Tools** > **All Setting & Tools**.
4. Select **System Templates** from the left side menu, and then choose **Custom Order Templates** and click on **Add New**.
5. Enter the **Template Name** - this will appear in the **Print** menu.
6. Enter the Template Reference. Thie is normally the same at the Template File.
7. Select the **Template File** from the dropdown menu.
8. Your web designer or our design team will tell you with the options in the **Settings** section. Just contact whoever designed the template.
9. Click on **Save**.
10. Go to **Sales Orders** > **View Orders** and scroll down to the **Bulk Actions** section at the bottom of the page. You will see the new print template at the bottom of the **Print** menu.
