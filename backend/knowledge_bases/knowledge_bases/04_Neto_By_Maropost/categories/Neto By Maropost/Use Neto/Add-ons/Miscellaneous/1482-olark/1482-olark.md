# Olark

## This add-on was built by Olark.

Visit Add-On Website

For support with this add-on [contact Olark](https://www.olark.com/contact).

Olark’s live chat software and customer data tools help you learn from every online interaction. To set up the Olark add-on:

1. Create an account with [Olark](https://www.olark.com/) and copy the code for your website.
2. In your Neto by Maropost control panel click on the **Addons** menu.
3. Locate the Olak add-on and click the **Install** button.
4. Paste the code from Olark and click save changes.
