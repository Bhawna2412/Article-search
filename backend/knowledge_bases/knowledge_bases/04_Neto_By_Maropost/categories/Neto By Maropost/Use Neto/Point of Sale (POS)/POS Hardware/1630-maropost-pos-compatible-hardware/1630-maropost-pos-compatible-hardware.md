# Maropost POS Compatible Hardware

**Please Note**: Neto POS is not available outside of Australia and New Zealand.

To buy hardware for your Point-of-sale system please [visit this page](https://www.pos99.com.au/pos-software-bundles/neto-pos-receipt-printers-and-scanners). Maropost POS can be run on any device that connects to the internet, running the latest version of [Google Chrome](https://www.google.com/chrome/browser/desktop/index.html) or the [Apple iOS app](https://itunes.apple.com/app/apple-store/id1081280067?pt=2020760&ct=Neto Website&mt=8). However, for optimal use, we recommend using touch screen devices with Maropost POS.

**Tip**: All Windows based printers and scanners will work with Maropost POS.

The below table outlines peripherals that have been thoroughly tested by Maropost development team. If your device is not listed below, it may work but will not be supported by Maropost.
