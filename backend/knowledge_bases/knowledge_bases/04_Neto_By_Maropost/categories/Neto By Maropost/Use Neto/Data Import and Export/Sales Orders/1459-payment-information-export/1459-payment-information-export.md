# Payment Information Export

Customers can be exported into a CSV (comma separated values) file so that you can manually report or import into a third party system. You can also export customer lists to make changes, and then import back into Neto. There's 2 ways to export your customer details:

> **Note:** Neto partners can set this up for you. [Visit the job request portal](https://www.netohq.com/job-request-portal) for a quote.

## Complex Export

The complex export wizard is the only way to export payment information, as there is no simple export method.

**Please Note:** Neto does not store credit card information. Only the first and last digits of the card number will be exported.

To use the complex export wizard:

1. In your Neto control panel navigate to **Settings & tools** > **Export data**.
2. Select the **Order Payments** radio button.
3. Click the **Perform complex export** button and choose the **Start Data Export Wizard** option.
4. Enter a name for your file and click the **Continue to Next Step** button.
5. You now need to choose which payment fields you want to export. Click the **Add New Field To File** button, and choose the fields from the drop down menus.
  You need to select a field type first (payment details) and then choose the field name.
