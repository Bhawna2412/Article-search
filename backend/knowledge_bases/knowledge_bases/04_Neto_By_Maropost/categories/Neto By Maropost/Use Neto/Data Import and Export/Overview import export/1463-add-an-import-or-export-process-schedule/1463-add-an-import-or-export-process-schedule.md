# Add an Import or Export Process Schedule

Your Neto control panel has a number of pre-made schedules that you can use straight away. If they do not suit your needs, you can add a custom schedule to run on specifics days and times. To add a custom schedule:

1. 
  In your Neto control panel navigate to **Settings & tools** > **All settings & tools**.
2. 
  In the Processes & Scheduled Jobs section click on **Task / Feed Schedules**.
3. 
  The default schedules will display. Click on these to edit your schedules, or click the **Add New** button to create a custom schedule.
4. 
  Enter a **Name** for your schedule. It’s best to make it as descriptive as possible so that you know how often the schedule will trigger a task/export.
5. 
  Using the checkboxes, select the hours, minutes, days of month, months, and days of the week the schedule will trigger the process or task to run. Most schedules will run every day, so select all the boxes in the days of month, months, and days of week sections.

  For example, to run a schedule every half hour, select all the boxes in the hours. Then select two boxes, 30 minutes apart, in the minutes section.
6. 
  Click the **Save & Close** button.

Your schedule is now added, and can be assigned to a task, import or export process.
