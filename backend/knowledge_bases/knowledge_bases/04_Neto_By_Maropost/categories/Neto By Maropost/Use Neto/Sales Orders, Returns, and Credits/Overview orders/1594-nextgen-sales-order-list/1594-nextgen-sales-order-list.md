# NextGen Sales Order List

## Overview

The NextGen sales order list is designed as a streamlined page to make it quicker and easier to find the information you need. It combines a new interface with improved technology behind the scenes to deliver a simplified user experience.

Currently, the NextGen orders page is in a functional state while we work on bringing it up to parity with the existing sales order list page. In particular we're focusing on adding:

- Bulk order actions
- Saved custom filters
- More search filters

NextGen sales orders is managed per user, so staff at your business can use use the NextGen page or the original legacy page, whichever they prefer.

## Switch to the Legacy Page

You can switch your view between the NextGen and legacy pages whenever you need to.

1. Click the more options button.
2. Click on **Revert to Legacy Orders**.

You'll be taken back to the legacy order list page. Please provide feedback on why you're switching back, so that we can improve the NextGen sales order list page to better meet your needs.

## Search Filters

NextGen sales orders features a new way search your order list. Gone is the time consuming hunt searching for the right advanced filter. Instead all search filters are found within the search bar itself.

Click into the search bar and you'll see a list of the available filters. You can also start typing the name of the field you want to search with to find the filter.

If you're searching by order number you can leave out the filter entirely.

**Please Note**: The current list of available filters are the ones that are used most often. More filters will be added soon.

Once you've selected your filter you can enter a value to search by. Some filters, like the order status, will give you a list of available options to choose from.

Add multiple filters to the search to see orders that match all the filters selected.

## Import or Export Sales Orders

Sales order import and export is now accessed from the **more actions (...)** button.

You'll then be directed to the import or export wizard to complete the process through the simple or complex wizards. Steps for these process can be found below:

- [Sales Order Import Wizard](https://galaxy.maropost.com/kb/articles/1580-sales-order-import-wizard)
- [Sales Order Export Wizard](https://galaxy.maropost.com/kb/articles/1457-sales-orders-export-wizard)

## Sales Order Actions

Each sales order has **more actions (...)** button that allows you to perform a variety of tasks:

- Clone the order
- Create an RMA (return/credit)
- Add a sticky note
- Add a new order for the customer
- View the customer
- View the customer's other orders
