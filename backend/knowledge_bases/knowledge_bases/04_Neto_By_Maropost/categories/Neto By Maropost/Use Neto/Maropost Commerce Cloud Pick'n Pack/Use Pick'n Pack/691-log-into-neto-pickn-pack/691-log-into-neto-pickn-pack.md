# Log Into Neto Pick'n Pack

You can login to Neto Pick n Pack from your Apple iOS device or via web interface if you use a non-apple device.

## Apple iOS Device

1. Download and install the [Neto Pick n Pack app](https://itunes.apple.com/au/app/neto-pick-n-pack/id626685026?mt=8).
  **Compatibility**: Requires iOS 8.0 or later. Compatible with iPhone, iPad and iPod touch.
2. Open the app and enter the [user credentials setup for Pick'n Pack](https://galaxy.maropost.com/kb/articles/1579-neto-pickn-pack-staff-accounts).
  There can only be one active session per user account. If two users login with the same account, the most recently logged in user will become the active user and the older session will be logged out.

## Desktop & Non-Apple Devices

1. In your Neto control panel dashboard, click on the username in the top right hand side toolbar and select **Mobile pick & pack interface** from the dropdown menu.
2. Enter the [user credentials setup for Pick'n Pack](https://galaxy.maropost.com/kb/articles/1579-neto-pickn-pack-staff-accounts) and click **Login**.
  There can only be one active session per user account. If two users login with the same account, the most recently logged in user will become the active user and the older session will be logged out.
