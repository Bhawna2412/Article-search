# Reverse or Cancel a Pick In Maropost Commerce Cloud Pick'n Pack

If you make a mistake whilst picking, such as incorrectly picking or back-ordering an order line, you can adjust the order whilst it is still in Pick status. If you have already moved it to pack, it will need to be sent back to pick first.

1. Touch or Click on** Show Processed Lines**, this will display a list of order lines that have already been processed.
  ![Image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/ReverseorCancelaPickInMaropostCommerceCloudPicknPack1.05T5g00000SSvosEAD.png)
2. Touch or click the black **Qty Ordered** **button **you want to edit.
  ![Image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/ReverseorCancelaPickInMaropostCommerceCloudPicknPack2.05T5g00000SSvpREAT.png)
3. This will reset your previous action and ask you to re-enter the Qty. Use the **+** and **-** symbols to increase and decrease the Qty, then press **Apply**.
  ![Image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/ReverseorCancelaPickInMaropostCommerceCloudPicknPack3.05T5g00000SSvpbEAD.png)
4. Here's how the process looks from start to finish.
  ![Image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/ReverseorCancelaPickInMaropostCommerceCloudPicknPack4.05T5g00000SSvplEAD.gif)
