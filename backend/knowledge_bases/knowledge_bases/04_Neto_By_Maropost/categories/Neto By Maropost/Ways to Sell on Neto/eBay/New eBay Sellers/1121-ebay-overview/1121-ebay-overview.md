# eBay Overview

Whether you're new to eBay or an experienced seller, the Neto eBay integration has the tools and features to control your listings, stock management, sales orders and dispatch processes.

## Getting Started

To get started with the Neto** **eBay integration, ensure that you have the following:

- [Set up, activate and verify your eBay account.](https://sellercentre.ebay.com.au/content/set-your-selling-account)
- [Create Products](https://galaxy.maropost.com/kb/articles/26-add-and-manage-products) (can be created from existing eBay listings).
- [Set up Shipping Options](https://www.netohq.com/support/s/knowledge#category=Support__c:Configure_Shipping__c) (**Recommended:** Australia Post).
- [Configure Payment Options](https://www.netohq.com/support/s/knowledge#category=Support__c:Configure_Payments__c) (**Recommended:** PayPal).
- [Set up eBay Store Categories](https://www.ebay.ie/pages/help/sell/store-categories.html) (Highly recommended).
- [Set up Product Categories](https://galaxy.maropost.com/kb/articles/685-add-a-product-category)
- Ensure your product descriptions are clean/simple (If installing an eBay Custom Design).

**Optional Requirements:**

- [eBay Custom Listing Design](https://galaxy.maropost.com/kb/articles/785-ebay-listing-design-images)

## Duplicate Listings

If you have duplicate products listed on eBay that you intend to list within Neto, you can either:

- Set a unique custom label for each duplicate listing in eBay, or
- Create a unique custom label using the eBay import wizard.

Each listing should be linked to its own product (unless listed on different eBay accounts), so that it isn't removed by eBay.

To set a unique custom label on eBay, complete the steps in the custom labels section below.

**Please Note**: Should you find multiple listings linked to the same product, creating kitted products will allow you to have separate listings, with stock managed from a single source.

## Custom Labels

Setting up custom labels in eBay is important as this will become your product SKU (stock keeping unit) once imported into Neto. SKU's need to be unique in Neto, so setting them correctly before connecting will save you a lot of time in the long run.

If you already have products setup in Neto and you wish to link these to your products on eBay, you must set your custom label to the SKU listed in Neto for it to be recognised.

To access your custom labels on eBay login to your eBay account and in **Selling Manager Pro,** click **Active**.

To bulk update these, you are best to use [eBay's file exchange](http://sellercentre.ebay.com.au/file-exchange).

Within file exchange, navigate to the **Download Files** section and submit a download request for **Active** listings. Once complete, obtain the file from **Completed Downloads**.
