# Listing Rules Template Details

[Listing rules templates](https://galaxy.maropost.com/kb/articles/784-listing-rules-templates-overview) contain all the settings to list products to eBay and make it quick and easy to apply the same settings to many products.

The first tab of a rules template is the **Template Details**. Here you give the template a name, ans select the account it will list to. To set up your rules template details:

1. In your Neto by Maropost control panel navigate to **eBay** > **Listing rule templates**
2. Click on the template you're working on or click the **Add new** button to create a new template.
  [Image: image1]
3. On the **Template Details** tab, enter a **Template name**. We recommend describing the shipping calculation method/costs and if the template is for variations or standard listings.
4. Select an **eBay Store / Account**.
  **Please Note**: You won't be able to save the template until you've set default categories on the [Categorisation & Listing Rules](https://galaxy.maropost.com/kb/articles/786-ebay-categorisation-listing-rules) tab.

**Tip**: Set up the next tab of your listing rules template, [eBay Categorisation & Listing Rules](https://galaxy.maropost.com/kb/articles/786-ebay-categorisation-listing-rules#category=Support__c:Launch_eBay_Store__c).
