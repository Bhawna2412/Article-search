# eBay Returns Settings

Your returns settings let customers know the terms for returning purchases they've made. If you're using Business Policies, set your returns on the **Payment, Shipping & Returns**. Otherwise, returns are set on the **Payment & Returns** tab.

## Set up Returns Settings in Neto

> **Important:** eBay doesn't allow return settings to be changed through the integration if the listing has bids or sales. If you need to change the returns setting for a listing log in to your eBay account and change it manually.

To set the returns settings for your listings in Neto:

1. In your Neto control panel, navigate to **eBay** > **Listing Rules Templates** and open the template you're working on.
2. Click on the **Payment & Returns** tab.
3. Select a **Listing Returns Policy**. We recommend returns accepted.
4. Select a duration from the **Item must be returned within** drop-down menu. We recommend 39 days.
5. Leave the **Refund will be given as** blank.
6. Select an option from the **Return shipping will be paid by** drop-down menu. We recommend setting it to the buyer.
7. Select a **Sales Tax Jurisdiction** if it applies to you (e.g. eBay US accounts).
  ![image1](https://us.v-cdn.net/6038474/uploads/attachments/migrated/eBayReturnsSettings1.05T5g00000t0SfbEAE.png)
8. Click the **Save template** button.

## Set up Returns via eBay Business Policies

To set your returns settings from your eBay business policies:

> **Note:** Create your business policies in your [eBay account](https://www.bizpolicy.ebay.com.au/businesspolicy/manage) before starting this step.

1. In your Neto control panel, navigate to **eBay** > **Listing Rules Templates** and open the template you're working on.
2. Click on the **Payment, Shipping & Returns** tab.
  ![image2](https://us.v-cdn.net/6038474/uploads/attachments/migrated/eBayReturnsSettings2.05T5g00000t0SgoEAE.jpg)
3. Click the **Return Policies** link to see the returns policies from your eBay account.
4. Select a returns policy using the radio buttons.
5. Click the **Save template** button.

> **Note:** Your listing rules template is now configured. You're now ready to [list products on eBay](https://galaxy.maropost.com/kb/articles/781-list-products-on-ebay#category=Support__c:I_m_new_to_eBay__c).
