# Troubleshoot Amazon Australia Error Messages

Neto synchronises your product information with Amazon Australia approximately every 60 minutes. To check the status of your listings, and when they were last synchronised, in your Neto control panel navigate to **Amazon Australia** > **Listings**.

The Amazon Australia listings page displays key information relating to:

- The number of products successfully listed.
- Products not yet listed (pending).
- Any errors that need to be resolved.
- Products on Amazon that are not managed in Neto.

Errors with Amazon listings can occur for a number of reasons. The best way to prevent listing errors is to ensure the following points are addressed:

- Required fields are mapped in the[product field mapping](https://galaxy.maropost.com/kb/articles/700-set-up-amazon-mapping-for-product-registration).
- Fields that require multiple fields mapped in order to display correctly.
- Fields that only have a certain range of available options (check the "i" icon) are mapped to send data Amazon wants.
- Variation theme is sending the correct data. Theme needs to match to the specific type in Neto (e.g. Colour, Size) and not the value (e.g. Blue, XL).
