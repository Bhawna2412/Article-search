# eWAY Credit Card Payment Method

## Overview

| Name | Is this easy to install? | Type of Gateway | Triggered Payments | Benefits |
| --- | --- | --- | --- | --- |
| eWAY Secure Panel | Yes | Onsite | No | Stored cards |

## Set up eWAY Secure Panel

1. In your Neto control panel navigate to **Settings & Tools** > **Payment Methods**.
2. In the **Looking for more payment options?** section click the **Add payment method** button.
![eWay1.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05TJ100000OQWeLMAX.jpg)
3. Click the Mastercard Visa + more tile, and choose **eWay Rapid** from the drop down menu that appears.
![eWay2.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05TJ100000OQeloMAD.jpg)
4. Select **Visa**, **Mastercard,** and any other card types you have access to with your eWay agreement. You can also choose to allow **Saved Cards**.
![eWay3.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05TJ100000OQeqFMAT.jpg)
5. Enter your **eWAY Rapid Secure Panel Public API Key **(called the **Pay Now Button Public API Key** in eWAY), **eWAY Rapid API Key** and **eWAY Rapid API User Password** which is the information provided by eWAY.  
![eWay4.png](https://us.v-cdn.net/6038474/uploads/attachments/migrated/image.05TJ100000OQeU5MAL.jpg)
  **Please Note**: eWAY will send you instructions on creating a user. This user's password is required in the above field.
6. Refer to our [Payment Method Settings](https://galaxy.maropost.com/kb/articles/1036-payment-method-settings) guide for reference on how to set up functions like minimum spend and surcharges.
7. Click the **Complete** button at the bottom to finish the setup of eWAY Rapid.
