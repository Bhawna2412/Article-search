# Add an API User Account

To add an API user for a 3rd party developer:

1. In your Neto by Maropost control panel, navigate to **Settings & tools** > **Staff users**.
2. Click the **Add new staff user** button.
3.   - Username
  - First Name
  - Last Name
  - Email address (cannot be the same as another staff user)
  - Password

  Complete the required fields:
4. Click the **Regenerate** button to create an API key for this user.
5. In the **User Permission Group** dropdown menu select **API User**.
  Tip

  : You can also create a custom permission group by clicking on the

  Add or Edit User Permission Groups

  link. We recommend that you clone the API user group and then add or remove permissions.
6. Click the **Save** button.
7.   - Username
  - API Key
  - API URL ([https://www.yourdomain.com.au/do/WS/NetoAPI](https://www.yourdomain.com.au/do/WS/NetoAPI))

  Provide your developer or third party the below information:

Your developer or 3rd party integration will now be able to connect to your control panel using these credentials.

**Tip**: To revoke access make the user account Inactive.
