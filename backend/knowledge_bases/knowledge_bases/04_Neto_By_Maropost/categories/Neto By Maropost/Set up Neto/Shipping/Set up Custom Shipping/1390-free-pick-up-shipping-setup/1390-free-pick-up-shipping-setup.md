# Free Pick up Shipping Setup

> **Note:** Neto partners can set this up for you. [Visit the job request portal](https://www.netohq.com/job-request-portal) for a quote.

## Set up a Free Pickup Shipping Option

To set up a free pickup shipping option.

1. In your Neto control panel navigate to **Shipping** and select **Shipping Options**.
2. Click the **Add new Shipping Option** button.
3. Select **Set My Own Pricing**.
4. Click **Free Pickup**.
5. Enter a **Name** and **Description**, and set who should see this shipping option in checkout, then click **Next Step**.
6. Review the settings, and click **Complete**.

Your **Local Pickup** shipping option will now be created for all shipping categories.

## Display Free Pickup Last In Checkout

We also suggest setting the **Sort Order** for your free pickup option, so that it isn't selected by default during checkout.

1. In your Neto control panel navigate to **Shipping** > **Shipping Options**.
2. Click the name of the free pickup shipping option you created to edit it.
3. Enter a number in the **Sort Order** field. The shipping option with the lowest number will appear first, so you should enter to a high number (e.g. 100). This should make it appear last, but you may also need to set the sort order for your other shipping options as well. Click **Save**.
4. Navigate to **Webstore** > **Checkout Settings**.
5. n the **Shipping Rules** section, change the **Default shipping option** to **Sort Order** and **Save**.

If some products do are not able to be picked up locally, you'll need to configure your shipping categories for the items that cannot be picked up. See our [article on shipping categories](https://galaxy.maropost.com/kb/articles/1628-add-a-shipping-category) to help you further.
