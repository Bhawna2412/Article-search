# Couriers Please Shipping Setup

You can configure Couriers Please shipping integration to provide automated carrier-based pricing and label generation for Australian deliveries, which helps streamline your shipping workflow and reduces manual rate calculation. Couriers Please integration enables real-time shipping quotes, service selection, and automated label printing, which typically improves operational efficiency by eliminating manual data entry and reducing shipping errors.

Setting up Couriers Please involves using the shipping wizard to configure your carrier account, selecting available services, and establishing label printing preferences. This helps merchants automate shipping operations and provide customers with accurate delivery costs at checkout, which generally enhances fulfillment speed and supports business scalability as order volume increases through multi-channel operations.

## Couriers Please Setup Wizard

To set up your Couriers Please shipping option using the wizard:

1. In your Neto control panel navigate to **Shipping** > **Shipping Options**.
2. Select **Add New Shipping Option**.
  ![image2](https://us.v-cdn.net/6038474/uploads/attachments/migrated/CouriersPleaseShippingSetup2.05T5g00000xsVN0EAM.png)
3. Select **Set Up Shipping Carrier Based Pricing**, and in the pop-up box click **I Have This Info, Continue Setup**.
  ![image3](https://us.v-cdn.net/6038474/uploads/attachments/migrated/CouriersPleaseShippingSetup3.05T5g00000xsVTSEA2.png)
4. Select **Couriers Please** and click **Next Step.**
  ![image4](https://us.v-cdn.net/6038474/uploads/attachments/migrated/CouriersPleaseShippingSetup4.05T5g00000xsVVEEA2.png)
5. Enter your **Carrier Account Name**, **Account ID** and set your **Pickup** details. Click **Next Step** once complete.
  ![image5](https://us.v-cdn.net/6038474/uploads/attachments/migrated/CouriersPleaseShippingSetup5.05T5g00000xsVSGEA2.png)
6. Choose which services you want to offer. Click **Show All Services** to see a full list. Click **Next Step** once complete.
  ![image6](https://us.v-cdn.net/6038474/uploads/attachments/migrated/CouriersPleaseShippingSetup6.05T5g00000xsUPzEAM.png)
7. Name your shipping services and give them a description. This information will be visible to customers on your website. Once complete, move on to the **Next Step**.
  ![image7](https://us.v-cdn.net/6038474/uploads/attachments/migrated/CouriersPleaseShippingSetup7.05T5g00000xsVfdEAE.png)
8. Set if the shipping options should **Show To Customers And Staff** or **Show To Staff Only** (for creating orders manually). You can also configure if delivery to PO Boxes is available. Once complete, move on to the **Next Step**. To set different options for each service, click the **Apply Visibility And PO Box Delivery Settings For Selected Services** box.
  ![image8](https://us.v-cdn.net/6038474/uploads/attachments/migrated/CouriersPleaseShippingSetup8.05T5g00000xsVvCEAU.png)
9. You can now add a **Discount Incentive** when a customer orders a certain amount. These are optional, set them if you wish and then move on to the **Next Step**.
  ![image9](https://us.v-cdn.net/6038474/uploads/attachments/migrated/CouriersPleaseShippingSetup9.05T5g00000xsVvqEAE.png)
10. Choose which type of printer you will use to print shipping labels, and click **Next Step**.
  ![image10](https://us.v-cdn.net/6038474/uploads/attachments/migrated/CouriersPleaseShippingSetup10.05T5g00000xsVxDEAU.png)
11. Review your settings. You can still make any changes by going back to the relevant section. If you're happy with the configuration, select **Complete**.
  ![image11](https://us.v-cdn.net/6038474/uploads/attachments/migrated/CouriersPleaseShippingSetup11.05T5g00000xsWBZEA2.png)

> **Note:** For all carrier shipping options we also recommend setting backup rates in case the carrier's API suffers an outage.

## Couriers Please Shipping Labels

To set up the Couriers Please labels:

1. In your Neto control panel navigate to the **Addons** menu.
2. In the Shipping section locate Couriers Please and click the **Install** button.
3. Fill in the [Shipping Carrier Label Setup Form](https://forms.neto.com.au/services/shippingcarrierlabelsetup.html) and Neto staff will set up the shipping labels for you for free.
