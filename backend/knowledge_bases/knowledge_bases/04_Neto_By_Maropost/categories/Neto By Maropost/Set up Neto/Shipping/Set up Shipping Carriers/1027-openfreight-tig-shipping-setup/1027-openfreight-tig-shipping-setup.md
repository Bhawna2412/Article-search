## OpenFreight Shipping Wizard

TIG consolidate the national freight management of any business through one central point with a tailored freight solution. To set up OpenFreight shipping:

1. In your Neto control panel navigate to **Shipping** > **Shipping options**.
2. Click the **Add new shipping option** button.
3. Click the **Set up shipping carrier based pricing** button. Confirm you have the required information in the pop-up window.
4. Select **OpenFreight** and click the **Next step** button.
  ![image2](https://us.v-cdn.net/6038474/uploads/attachments/migrated/OpenFreightTIGShippingSetup2.05T5g00000xsCIBEA2.png)
5. Click **Add New** and enter your **Openfreight API Username**, **Openfreight API Password** and set your **Pickup** details.
  ![image3](https://us.v-cdn.net/6038474/uploads/attachments/migrated/OpenFreightTIGShippingSetup3.05T5g00000xsCJYEA2.png)

  Set the **Danger Items** drop down and when ready, move on to the **Next Step**.
6. Select which services you wish to offer through OpenFreight TIG by selecting from the available options. Once complete, move on to the Next Step.
7. Name your shipping services and give them a description. This information will be visible to customers on your website. Once complete, move on to the **Next Step**.
8. Set if the shipping methods should **Show To Customers** or **Show To Staff Only** (for creating orders manually). You can also configure if delivery to PO Boxes is available. Once complete, move on to the **Next Step**.
9. You can now enable additional **Pricing Rules** to offer shipping discounts when a customer orders a certain amount. These are optional, set them if you wish and then move on to the **Next Step**.
10. Review your settings, you can still make any changes by scrolling up to the relevant section. If you're happy with the configuration, select **Complete Setup**.

> **Important:** For API shipping rates we also recommend setting backup rates in case the carriers API has downtime.

## Set up Shipping Labels

To set up the OpenFreight shipping labels:

1. In your Neto control panel navigate to the **Addons** menu.
2. In the Shipping section locate OpenFreight - TIG and click the **Install** button.
3. Fill in the [Shipping Carrier Label Setup Form](https://forms.neto.com.au/services/shippingcarrierlabelsetup.html) and Neto staff will set up the shipping labels for you for free.
