# Set up Sendle for Multiple Warehouses

You can setup multiple warehouse locations for Sendle by adding a new **Carrier and Label** configuration in the shipping matrix. The configuration will tell Sendle couriers where to pickup the products from and who to contact.

This is important if you have multiple warehouses and/or dropship suppliers. You will need to create a configuration for each warehouse you have.

By default, a carrier and label configuration has already been setup to attend one single location for all warehouses. To start with, we want to assign this default configuration to one particular warehouse location. Once this is done, we’ll add new configurations for other warehouses.

1. In your Neto control panel navigate to **Settings & Tools** > **Shipping**.
2. Click on the **Carriers & Labels** configuration.
3. Find your default **Sendle for Neto** label configuration and edit the configuration.
4. Under the **Carrier Details**, change the name to represent your warehouse, eg. Sendle (Warehouse 1) and then set a warehouse location from the dropdown menu.
  [Image: image1]
5. Click **Save** and go back to the Carriers & Labels configuration menu.
6. Click the **Add new** button on the top right hand side and click **Sendle**.
  [Image: image2]
7. In the Carrier Details section, give your label configuration a name, such as eg. Sendle (Warehouse 2) and from the warehouse drop-down menu, select a specific warehouse you want the configuration to be assigned to.
8. Fill out the **Sender details** section, this is the address Sendle will collect from and the contact details that will appear on the label.
  [Image: image3]
9. Scroll down to the **Shipping Services** section. Tick **Sendle for Neto** and click **Save & Close**.
10. Repeat this process for any other warehouses you have. When processing orders from a particular warehouse, Sendle will now print consignments and pickup from the associated warehouse.
