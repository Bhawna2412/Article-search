# MyFastway Shipping Option Setup

### This add-on was built by Maropost

Add-ons and Integrations built in-house by Maropost are documented and supported by our customer service team. If you're having issues with this integration, [let us know](https://www.netohq.com/support/s/ask-us)!

> **Note:** Neto partners can set this up for you. [Visit the job request portal](https://www.netohq.com/job-request-portal) for a quote.
> This support article requires the **Neto Commerce Ship** add-on.

## MyFastway Shipping Wizard

To set up MyFastway as a shipping option using your account rates:

1. In your Neto control panel navigate to **Shipping** > **Shipping Options**.
2. Click the **Add New Shipping Option** button.
3. Click the **Set Up Shipping Carrier Based Pricing** button, and in the pop-up box click **I Have This Info, Continue Setup**.
4. Select **Fastway** and click the **Next Step** button.
5. Enter your **Carrier Account Name**, **My Fastway Client ID** and **My Fastway Secret**. This information can be found in your [MyFastway account](https://help.aramex.com.au/hc/en-us/articles/360001154895-Does-MyFastway-offer-any-API-integration-). Click **Next Step** once complete.
6. Choose which services you want to offer. Click **Show All Services** to see a full list. Click the **Next Step** button once complete.
7. Name your shipping services and give them a description. This information will be visible to customers on your website. Once complete, move on to the **Next Step**.
8. Set if the shipping options should **Show To Customers And Staff** or **Show To Staff Only** (for creating orders manually). You can also configure if delivery to PO Boxes is available. Once complete, move on to the **Next Step**. To set different options for each service, click the **Apply Visibility And PO Box Delivery Settings For Selected Services** box.
9. You can now add a **Discount Incentive** when a customer orders a certain amount. These are optional, set them if you wish and then move on to the **Next Step**.
10. Choose which type of printer you will use to print shipping labels, and click **Next Step**.
11. Review your settings. You can still make any changes by going back to the relevant section. If you're happy with the settings, click **Complete**.

Your shipping options for MyFastway are now set up. We recommend putting all MyFastway services in the same [routing group](https://galaxy.maropost.com/kb/articles/1617-display-the-cheapest-shipping-option-to-customers) so that only the cheapest applicable service shows to your customers.

> **Note:** For all carrier shipping options we also recommend setting backup rates in case the carrier's API suffers an outage.

**Please Note**: Maximum lengths should be added to the [rate tables](https://galaxy.maropost.com/kb/articles/1223-add-or-edit-shipping-rates) for Fastway Satchel services, based on the services limits.

## MyFastway Shipping Labels

**Please Note**: Your pickup address is set in your MyFastway account, rather than in the shipping labels.

To set up MyFastway labels install the Neto ship add-on and fill in the [Shipping Carrier Label Setup Form](https://forms.neto.com.au/services/shippingcarrierlabelsetup.html) and Neto staff will set up the shipping labels for you for free.
