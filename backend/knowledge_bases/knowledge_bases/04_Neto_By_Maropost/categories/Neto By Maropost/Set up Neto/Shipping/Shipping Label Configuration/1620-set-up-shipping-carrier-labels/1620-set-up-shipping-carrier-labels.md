# Set up Shipping Carrier Labels

> **Note:** If you are using an integrated carrier we will assist you to configure your integrated labels and manifests as part of the setup process. Complete the [Shipping Carrier Label Form](https://forms.neto.com.au/services/shippingcarrierlabelsetup.html) for each of the carriers you wish to use, and our customer support team will contact you when complete.

To set up your shipping carrier labels:

1. In your Neto control panel navigate to **Settings & Tools** > **Shipping**.
2. Click on **Carriers & Labels**.
3. On the top right hand side, click the green **Add New** button and select the carrier you wish to create a label for. For this purpose we will be creating a generic label, so select **Universal**.
4. Add your **Carrier Details**.
5. Fill out the **Return Address** which will appear on the labels.
6. Select the **Shipping Services** you wish to use the label with.

## Want a personalised label?

> **Note:** Most integrated couriers do not allow modifications to their labels. However if you wish to make design changes to a generic label, [contact a design partner](http://forms.neto.com.au/services/designtweak.html) for a quote.
