# Shipping Label Setup for Laser and Inkjet Printers for Neto

## Internet Explorer Set up

1. In Internet Explorer, click the **Settings (cog icon)** > **Print** > **Page Setup**
2. Change all page header and footer options to **Empty** so that time, date, page number etc do not print in the header or footer.
3. Change all margins (left, right, top and bottom) to **0**.
4. Enable **Print Background Colours and Images**.
5. Disable **Shrink To Fit**.
6. **Save** changes and close.

## Firefox Set up

1. From within Firefox, go to the top right-hand **Settings **button.
2. Click **Print** > **Page Setup**.
3.   1. Enable **Print Background (Colours & Images)**.
  2. Disable **Shrink to Fit Page Width**.

  On the **Format & Options** tab:
4.   1. Change all **Headers & Footers** options to **-blank-**. By doing this the time, date, page number etc will not print in the header or footer.
  2. Change all margins (left, right, top and bottom) to **0**.

  Select the **Margins & Header/Footer** tab:
5. Click **Ok** to save these changes, and print your labels.

## Google Chrome Set up

1. From within Google Chrome, go to the top right hand **Settings button** > Click **Print**.
2. Click **More Settings**.
3. Set **Margins** to **None**.
4. Enable **Background graphics**.

You're now ready to print your labels from Google Chrome.

## Order eParcel Label Stationary

Australia Post also offers the following single-page envelope size labels for printing Express and Regular eParcel consignments. These can be ordered directly from Australia Post. We recommend this stationery for all eParcel users as it reduces paper wastage and allows for regular and express label printing using the same printer settings.

- Australia Post eParcel - Code: 8839405
- Australia Post Express Post eParcel - Code: 8839407

## Tips

- If you are having trouble fitting the labels on your stationery, adjust your printer margins slightly in your browser's page settings.
- To test that everything is working, open the [test label](http://www.neto.com.au/assets/label/label2.html) in your browser and print the page using the new settings above. You may need to make adjustments where necessary.
- eParcel stationary is not required when printing labels from Neto. You can use generic labels. If you are using generic labels, please ensure that you have headings enabled.
