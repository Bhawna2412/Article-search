## TNT Shipping Wizard

To set up your TNT shipping options using the set up wizard:

1. In your Neto control panel navigate to **Shipping** > **Shipping Options**.
2. Select **Add New Shipping Option**.
  ![image2](https://us.v-cdn.net/6038474/uploads/attachments/migrated/TNTExpressShippingSetup2.05T5g00000xsRO0EAM.png)
3. Select **Set Up Shipping Carrier Based Pricing**, and in the pop-up box click **I Have This Info, Continue Setup**.
  ![image3](https://us.v-cdn.net/6038474/uploads/attachments/migrated/TNTExpressShippingSetup3.05T5g00000xsUR6EAM.png)
4. Select **TNT Express** and click **Next Step.**
  ![image4](https://us.v-cdn.net/6038474/uploads/attachments/migrated/TNTExpressShippingSetup4.05T5g00000xsUUKEA2.png)
5. Enter your **Carrier Account Name**, **Account Number**, **Username**, **Password** and set your **Pickup** details. Click **Next Step** once complete.
  ![image5](https://us.v-cdn.net/6038474/uploads/attachments/migrated/TNTExpressShippingSetup5.05T5g00000xsUWaEAM.png)
6. Choose which services you want to offer. Click **Show All Services** to see a full list. Click **Next Step** once complete.
  ![image6](https://us.v-cdn.net/6038474/uploads/attachments/migrated/TNTExpressShippingSetup6.05T5g00000xsUaXEAU.png)
7. Name your shipping services and give them a description. This information will be visible to customers on your website. Once complete, move on to the **Next Step**.
  ![image7](https://us.v-cdn.net/6038474/uploads/attachments/migrated/TNTExpressShippingSetup7.05T5g00000xsSvKEAU.png)
8. Set if the shipping options should **Show To Customers And Staff** or **Show To Staff Only** (for creating orders manually). You can also configure if delivery to PO Boxes is available. Once complete, move on to the **Next Step**. To set different options for each service, click the **Apply Visibility And PO Box Delivery Settings For Selected Services** box.
  ![image8](https://us.v-cdn.net/6038474/uploads/attachments/migrated/TNTExpressShippingSetup8.05T5g00000xsUQiEAM.png)
9. You can now add a **Discount Incentive** when a customer orders a certain amount. These are optional, set them if you wish and then move on to the **Next Step**.
  ![image9](https://us.v-cdn.net/6038474/uploads/attachments/migrated/TNTExpressShippingSetup9.05T5g00000xsUluEAE.png)
10. Choose which type of printer you will use to print shipping labels, and click **Next Step**.
  ![image10](https://us.v-cdn.net/6038474/uploads/attachments/migrated/TNTExpressShippingSetup10.05T5g00000xsUj1EAE.png)
11. Review your settings. You can still make any changes by going back to the relevant section. If you're happy with the configuration, select **Complete**.
  ![image11](https://us.v-cdn.net/6038474/uploads/attachments/migrated/TNTExpressShippingSetup11.05T5g00000xsUtKEAU.png)

> **Note:** For all carrier shipping options we also recommend setting backup rates in case the carrier's API suffers an outage.

## TNT Shipping Labels

To set up the TNT labels:

1. In your Neto control panel navigate to the **Addons** menu.
2. In the Shipping section locate TNT and click the **Install** button.
3. Fill in the [Shipping Carrier Label Setup Form](https://forms.neto.com.au/services/shippingcarrierlabelsetup.html) and Neto staff will set up the shipping labels for you for free.
