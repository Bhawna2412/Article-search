# Set up FedEx Carrier Labels

> **Important:** FedEx labels are only available to merchants in the USA.

## Before you Begin

Before you set up your FedEx labels, you need zones, rates, and a shipping option for FedEx. Follow the steps below to set these up.

- [Add zones for FedEx](https://galaxy.maropost.com/kb/articles/601-custom-shipping-overview-and-setup#shipping-zones) or use an existing carrier's zones and skip this step.
- [Import shipping rates](https://galaxy.maropost.com/kb/articles/1223-add-or-edit-shipping-rates#2) for FedEx.
- [Add a shipping option](https://galaxy.maropost.com/kb/articles/601-custom-shipping-overview-and-setup#shipping-methods) for FedEx.

> **Note:** Neto partners can set this up for you. [Visit the job request portal](https://www.netohq.com/job-request-portal) for a quote.

## Set up FedEx Carrier Labels

To set up your FedEx carrier labels:

1. In your Neto control panel navigate to **Settings & tools** > **Shipping**.
2. Click on **Carriers & Labels**.
3. Click the **Add New** button.
4. Search for and click on **FedEx**.
5. Enter your FedEx account details in the **Connect FedEx Account** section.
6. Enter your warehouse address details in the **Pickup Address** section.
7. Select additional settings in the **Shipment Configuration** section.
8. In the **Label Configuration** section map fields from your order to the shipping label.
9. The **Shipping service** section displays all of the rate tables you've set up in Neto. Toggle the rates for FedEx so that they are active. In the **Service code** drop down menu select the FedEx service that the rates relate to.
  If you have multiple FedEx rates and services, click the **Add shipping service** button and repeat the previous step.
10. Click the **Save** button.
