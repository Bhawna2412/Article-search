# Add a Shipping Category

**Please Note**: This is not necessary for all businesses and most Neto users generally use the default and dangerous goods categories that has been set up.

Shipping categories let you control which shipping options are available to individual products. They provide the link between your products and the shipping options you add in Neto.

## Add a Shipping Category

1. To create a new shipping category In your Neto control panel navigate to **Settings & Tools** > **Shipping**.
2. Click on **Shipping Categories**.
3. Click the **Add New** button on the top right hand side.
4. Enter a **Name** for the category. This will not display to customers. You can also enter a **Internal Description** for your reference. This will not display to customers and is optional.
5. Click on **Save**.

## Add Shipping Options to a Shipping Category

Following on from setting up your categories, you will need to assign these shipping categories on the product level. Before that happens, you will need to assign the categories to the shipping options you want to use.

Doing this will prevent customers from using options which are not available with certain couriers.

1. In your Neto control panel navigate to **Settings & tools** > **Shipping**.
2. Click on **Shipping Options**.
3. Edit the shipping option(s) you wish to add your category too.
4. Scroll down to the **Shipping Category** section and click the green **Add Shipping Category** button.
5. In the newly created drop down box, select your new shipping category and the rates you wish to use for that category (if unsure, use the same rates as the other categories) and click **Save**.
6. Repeat the process for any other shipping options.

## Add Shipping Categories to Products

Now this is complete, you just need to assign products to this newly created shipping category so that your shipping calculator can recognise that it's different.

For the purpose of this article, we will show you how it's done individually, however this can be completed in bulk using [Neto's import wizard](https://galaxy.maropost.com/kb/articles/668-product-import-wizard) as well.

1. In your Neto control panel navigate to **Products** > **View Products**.
2. Edit the product you wish to change.
3. Scroll down to the **Shipping** tab and select the **Shipping Category** you wish to use and click **Save**.
