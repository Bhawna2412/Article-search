# Change Return Address Details on Shipping Consignment Labels

1. To change the return address details printed on your shipping labels from your Neto by Maropost control panel, navigate to **Settings & Tools** > **Shipping**.
2. Click on **Carriers & Labels**.
3. Click on the carrier you wish to edit the return address for.
4. Under **Carrier and Account Details**, scroll down to **Return Address Shown on Labels** and make the necessary changes and press **Save**.
