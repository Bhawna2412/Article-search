# NZ Post eShip Setup

You can setup NZ Post eShip integration to work with Neto by following the guide on the eShip website.

We recommend that **Pack** is entered in the order status field in step 4 of their guide, as orders should not be exported for consignments until this order status is met.

**Please Note**: If orders coming into Neto go straight to the **Pick** status, you will need to enter **Pick** or **Pack** in the order status to make sure all orders are imported.
