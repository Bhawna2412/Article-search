# Set up a Weight-based Shipping Option

## Set up the Shipping Option

To charge for shipping in weight bands:

1. In your Neto control panel navigate to **Shipping** > **Shipping Options**.
2. Click the **Create shipping option** button.
3. Click the **Set my own pricing** button.
4. In the Advanced Options section, click the **Weight or size** button.
5. Select a **Where to ship** option. **Use preset carrier locations** for domestic, or **Select countries** for Australia wide and international).
6. Select a **Carrier**. This will populate the preset zones for that carrier, or the [zones you've uploaded](https://galaxy.maropost.com/kb/articles/601-custom-shipping-overview-and-setup#shipping-zones) separately.
7. Select the carrier zones you'd like to ship to and click the **Next step** button.
8. Enter a **Pricing table name** in the available field.
9. Enter a **Weight range** in the available fields (e.g. 0 - 1(kg)). You can also leave this empty to charge per kg.
10. Enter a cost for the **1st Parcel**.
11. Enter a cost for the **Subsequent parcels**.
12. If you have additional costs the more a package weighs, enter a cost for the **Per KG** rate. This will be added on to the 1st parcel cost when calculating the total.
13. Enter a cost for the **Min charge**. You can also select if the minimum charge is per article, or per consignment. Check with your carrier if you're unsure how they charge.
14. Enter a **Delivery time** in days.
15. Click the **Add weight range** button to add an additional pricing band, and repeat these steps.

## Set Prices Per Location (Zone)

2. Select the carrier zones you'd like to ship to.
3. Enter your shipping costs as per the table below:
4. Select if specific locations should have their own pricing. If you do this, repeat the steps for each zone.

## Select Shipping Categories

1. Click the **Dangerous Goods** checkbox to apply the rates to any products in the dangerous goods category.
2. Click the **Default** checkbox to apply the rates to products in the default category.
3. If you have any other shipping categories you want these rates applied to, select them here.
4. Enter a **Cubic weight modifier** (usually '250') provided by your carrier.
5. Click the **Next step** button.

## Add Naming and Visibility

1. Select a **Visibility** option.
2. To allow delivery to PO Boxes, toggle the **Available to PO box addresses** setting.
3. Enter a **Name** and a **Description** for the shipping option that will be visible to customers.
4. Click the **Next step** button.

## Select Label Options

1. Select a labelling dimension (either 'A4' or 'Thermal printer').
2. Click the **Next step** button.

## Add Discounts (optional)

1. Click the **Add discount incentive** button.
  This displays additional fields where a discount can be added to orders over an specified amount.
2. Click the **Next step** button.

## Review and Save

1. Review the details of your shipping option.
  To make changes, click the **Previous step** button.

  To complete the shipping option set up click the **Complete** button.

Your shipping option is now set up.
