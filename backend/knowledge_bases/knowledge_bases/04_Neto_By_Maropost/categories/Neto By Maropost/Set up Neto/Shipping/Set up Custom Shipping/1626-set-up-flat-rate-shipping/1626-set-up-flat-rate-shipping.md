# Set up Flat Rate Shipping

> **Note:** Neto partners can set this up for you. [Visit the job request portal](https://www.netohq.com/job-request-portal) for a quote.

To set up flat rate shipping using the shipping wizard:

1. In your Neto control panel navigate to **Shipping** > **Shipping Options**.
2. Click the **Create shipping option** button.
3. Click the **Set my own pricing** button. The **Create Shipping option** page updates with a range of new options.
4. Click the **Flat-rate** button.
5. Click either the **Use preset carrier** locations or **Select countries** link to select the location you want the flat rate shipping to be applied to.

## Preset carrier locations

1. Click the **Use preset carrier locations** button.
2. Click the radio button next to the carrier whose locations you want to use.
3. Click the checkbox next to each carrier location you wish to ship to.
4. Click the **Next step ** button.

## Select countries

1. Click the **Select countries ** button.
2. Click the checkbox next to each country you wish to ship to.
3. Click the **Next step ** button.

## **Enter fixed postage prices**

1. Enter a **Flat-rate price** for each selected location.
2. Click the **Next step ** button.

## Enter name and visibility options

1. Select the relevant **Visibility** radio button (either **Show to customers and staff** or **Show to staff only**).
2. Adjust the **Available to PO box addresses** toggle to allow for shipping to PO Boxes.
3. Enter a **Name** for the shipping option.
4. Enter a **Description** of the shipping option.
5. Click the **Next step** button.

## Review your shipping options

1. Validate all previous entries.
  To change any entries, click the **Previous step ** button.
2. Click the **Complete ** button.

You have successfully added a flat-rate shipping option. The shipping option will display in the **Shipping Options** page. To add another, click the **Add new shipping option** button.

**Please Note**: If some products need to be excluded from the flat-rate option you will need to configure your shipping categories for these items. See [Create a shipping category ](https://galaxy.maropost.com/kb/articles/1628-add-a-shipping-category) for further information.
