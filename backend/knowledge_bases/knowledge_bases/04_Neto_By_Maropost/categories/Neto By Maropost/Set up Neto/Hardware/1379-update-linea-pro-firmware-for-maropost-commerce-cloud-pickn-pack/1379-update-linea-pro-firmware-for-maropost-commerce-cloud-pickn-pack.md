# Update Linea Pro Firmware For Maropost Commerce Cloud Pick'n Pack

For your Linea Pro hardware to work on the newest version of iOS you will be required to run the latest version of the Linea Pro firmware for your device.

Important Note:

If you are running outdated versions of iOS, you must run the Linea Pro firmware update before upgrading your iOS devices to the latest version.

1. On your iDevice, navigate to the App Store and install the [**IPCSupport**](https://apps.apple.com/us/app/ipcsupport/id1015160712) App.
2. Once the app has downloaded, open it.
3. Select the **Firmware** tab.
4. Click the **Auto Update** button.
5. Select the newest firmware update.

This will begin the update to the latest firmware. A warning message will appear to advise against interruption to the installation.

Once the firmware update has completed, you can update your iOS devices to the latest version.
