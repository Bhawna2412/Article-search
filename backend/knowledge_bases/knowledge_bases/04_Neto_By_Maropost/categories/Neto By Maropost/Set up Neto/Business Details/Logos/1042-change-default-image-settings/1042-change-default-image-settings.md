# Change Default Image Settings

## Steps

1. In your Neto control panel navigate to the **Settings & Tools** > **All Settings & Tools** page.
2. Search for and select **Image Settings**.
3. On the **Image Settings** page, you can change dimensions of images, quality and number of images to display on your webstore at any given time.
  ![image1](https://us.v-cdn.net/6038474/uploads/attachments/migrated/ChangeDefaultImageSettings.05T5g00000xpZ6QEAU.png)

### Change the number of images that display per product

Modify the number in the **Number of images per product** field.

### Change the default image size

Change the number in the corresponding **width/height** field.
