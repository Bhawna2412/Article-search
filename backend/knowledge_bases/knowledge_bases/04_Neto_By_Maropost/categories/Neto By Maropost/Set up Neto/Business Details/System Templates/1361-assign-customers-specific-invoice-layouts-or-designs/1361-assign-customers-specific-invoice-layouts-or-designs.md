# Assign Customers Specific Invoice Layouts or Designs

To assign a specific layout to a customer you first need to [create or edit your own document layouts](https://galaxy.maropost.com/kb/articles/1107-customise-email-and-print-templates). If you're not comfortable with HTML, CSS and Neto's B@SE language, [Neto partners](https://www.netohq.com/job-request-portal) can create one for you.

**Example usage:** If a specific customer requires a special invoice layout then you can create a custom invoice template that can then be assigned to that customer's card. All invoices/orders received by the customer will be in this format.

The documents that can be customised are:

- Invoices / Orders
- Adjustment Notes
- Quotes

> **Note:** Neto will automatically convert the HTML versions of the above documents into a PDF document.

- [Assign a Custom Template to a Customer](https://galaxy.maropost.com/kb/articles/1361-assign-customers-specific-invoice-layouts-or-designs#2)
- [Assign a Custom Document Template to an Order/Invoice](https://galaxy.maropost.com/kb/articles/1361-assign-customers-specific-invoice-layouts-or-designs#2)

## Assign a Custom Template to a Customer

You can assign the custom document template that will be used for a customer on the customer's card:

1. In your Neto control panel navigate to **Customers** > **View customers & prospects**.
2. Click on the customer you wish to change.
3. Scroll down to the **Payment details** section. Select the template set from the drop down list.
  ![Image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/AssignCustomersSpecificInvoiceLayoutsorDesigns2.05T5g00000hwVoeEAE.png)
4. Click the **Save** button.

The assigned document template will then be used for all future orders, adjustment notes and quotes for that customer.

## Assign a Custom Document Template to an Order/Invoice

You can assign the default document format that will be used for a specific order:

1. In your Neto control panel navigate to **Sales Orders** > **View Orders**.
2. Click on the order you wish to change.
3. Scroll down to the billing address section and select the document format.
  ![Image](https://us.v-cdn.net/6038474/uploads/attachments/migrated/AssignCustomersSpecificInvoiceLayoutsorDesigns.05T5g00000hwUz0EAE.png)
4. Click the **Save** button.

You can now print or send the order/invoice using the new template.
