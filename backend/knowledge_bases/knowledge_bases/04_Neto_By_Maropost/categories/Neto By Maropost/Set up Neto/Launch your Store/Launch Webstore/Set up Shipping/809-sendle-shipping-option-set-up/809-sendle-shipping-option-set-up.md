# Sendle Shipping Option Set up

Sendle is a fully integrated shipping carrier services that can be set up in minutes.

> **Note:** Neto partners can set this up for you. [Visit the job request portal](https://www.netohq.com/job-request-portal) for a quote.

## Set up Sendle

To set up Sendle in Neto:

1. In your Neto control panel navigate to **Shipping** > **Shipping Options**.
2. Click the **Add New Shipping Option** button.
3. Click the **Setup Sendle** button.
  ![image1](https://us.v-cdn.net/6038474/uploads/attachments/migrated/SendleShippingOptionSetup1.05T5g00000syyg4EAA.png)
4. Select from **Sendle Premium Plan** or **Sendle Pro**.
  **Please Note**: There are minimum requirements for Sendle Pro which is approved on a case-by-case basis. If you're sending a significant number of parcels every month, it's worth [enquiring about this plan.](http://forms.neto.com.au/products/sendle-pro.html)
5. Follow the steps and add your email address and credit card information for weekly billing.
  ![image2](https://us.v-cdn.net/6038474/uploads/attachments/migrated/SendleShippingOptionSetup2.05T5g00000sz7hoEAA.png)
6. **Agree** to the Dangerous goods declaration and terms & conditions.
7. Confirm the address and contact details of each warehouse where products will be picked up from.
8. Press **Complete setup** button.

## Set the Maximum Weight

Sendle has a maximum weight allowance of 25kg for an order, which needs to be configured after setup.

1. Click on the **Sendle** shipping option.
2. Enter 25 into the **Maximum Weight** field and then click **Save**.
  ![image3](https://us.v-cdn.net/6038474/uploads/attachments/migrated/SendleShippingOptionSetup3.05T5g00000sz6vHEAQ.png)

## Change the Default Rate Location

You can change the location Sendle uses to calculate your shipping from which is useful when you move or simply use another warehouse as your primary location. There are two places where the pickup location should be updated.

### Sendle Settings

1. In your Neto control panel navigate to **Shipping** > **Sendle**.
2. Click **Edit services/rates** button.
  ![image4](https://us.v-cdn.net/6038474/uploads/attachments/migrated/SendleShippingOptionSetup4.05T5g00000sz7lYEAQ.png)
3. Scroll down to **3rd Party Calculators** and next to **Calculator**, click the configure icon.
  ![image5](https://us.v-cdn.net/6038474/uploads/attachments/migrated/SendleShippingOptionSetup5.05T5g00000sz7nDEAQ.png)
4. Scroll down to **Sendle** and click the edit button.
  ![image6](https://us.v-cdn.net/6038474/uploads/attachments/migrated/SendleShippingOptionSetup6.05T5g00000sz7nwEAA.png)
5. Add the new suburb and postcode location and click **Okay**.

### Carrier Config Settings

1. In your Neto control panel navigate to **Settings & Tools** > **Shipping**.
2. Select **Carriers & Labels**.
3. Select **Sendle for Neto**.
4. Update the address in the **Sender Details** section.
5. When complete, click the **Save & Close** button.

Your Sendle integration will now utilise the new postcode for calculating shipping.
